## 功能

本 skill 给出"日主五行在十二地支上的位次判定"——即根据出生日的天干，判定该五行在四柱任一地支上处于十二长生（长生、沐浴、冠带、临官、帝旺、衰、病、死、墓、绝、胎、养）哪一阶段，并贴上对应的命理标签（吉位 / 中性 / 凶位 / 条件吉凶两端）。

AI 执行者在以下场景调用本 skill：

- 用户给出八字四柱，想知道每一柱地支对日主而言处于"什么状态"
- 排定大运流年地支与日主的"位次关系"（如"行至长生""走到墓库"等）
- 论格局时需先看日主是否得月令长生、是否"坐临官"等
- 判定四柱地支是否为日主提供"通根"（即该地支是否在日主十二长生序列中）

**目标读者**：AI 执行本 skill 后的输出是给**普通用户**看的，非专业研究者。每个位次标签都附通俗解释（如"长生（初生之旺，主人温和、长寿）"），结论须配合语境说明。

**重要立场声明**：十二长生体系内部存在多处流派分歧——尤其是"阴干长生是否有力""墓库是否需要刑冲开启""五行阴干死位是否通用"三处议题。本 skill 不强行独尊一说，而是按"以用神为上"原则，看位次在命局中是喜是忌方下断；分歧处并列陈述子平真诠与三命通会两方立场。

## 输入

| 参数名 | 描述 | 类型 | 格式/约束 | 必填 |
|--------|------|------|-----------|------|
| 四柱天干 | 年/月/日/时四柱的天干 | string[4] | 每字一字天干（甲乙丙丁戊己庚辛壬癸），共 4 字 | 是 |
| 四柱地支 | 年/月/日/时四柱的地支 | string[4] | 每字一字地支（子丑寅卯辰巳午未申酉戌亥），共 4 字 | 是 |
| 日主 | 出生日的天干（用于"日主五行"） | string | 单字天干 | 是 |

**输入不合法时**（如日主不在十天干内、四柱地支数 ≠ 4），输出报告须明确指出「输入非法、无法完成判定」，不得静默失败。

> **实现提示**：若用户只给出日主而未给出完整四柱，则只能给出"日主五行的十二长生位次表"（共 12 条地支 → 位次的对应表），无法对"年/月/时"地支做贴标签。AI 须据实告知。

## 处理逻辑

### 前置加载（按顺序）

1. `cat skills/命/八字/长生十二位/rules/子平真诠.md` —— 子平体系对阳顺阴逆、墓库冲与不冲、通根透出之理的辨正（Step 1-3 全程参考）
2. `cat skills/命/八字/长生十二位/rules/滴天髓阐微.md` —— 任铁樵对"阴生阳死、阳顺阴逆"的破执 + "得时失时非死法"的根本立场（Step 1 必读，决定是否独尊阳顺阴逆）
3. `cat skills/命/八字/长生十二位/rules/三命通会.md` —— 阳干顺行/阴干逆行的具体起例表 + 十干逐位禄位吉凶（Step 2、Step 4 必查）
4. `cat skills/命/八字/长生十二位/rules/渊海子平.md` —— 十二长生诗诀的位次吉凶判定（Step 3 按位次对应吉凶标签）
5. `cat skills/命/八字/长生十二位/rules/千里命稿.md` —— 干支起例基础（推年、推月、推日、推时 + 大运顺逆，确保四柱正确）

### 判定流程

1. **Step 1：定日主五行与阴阳，决定行运方向**
   - 按日主天干查其五行与阴阳归属：
     - 阳干（甲丙戊庚壬）→ 阳顺，五行沿十二支**顺行**排位次
     - 阴干（乙丁己辛癸）→ 阴逆，五行沿十二支**逆行**排位次
   - **破执检查**：加载 `rules/滴天髓阐微.md` 的"任氏破阴生阳死之执"节——任铁樵认为**不可执定**阳顺阴逆推断吉凶，乙木死亥（"乙木死亥，亥中有壬水乃其嫡母，何为死哉"）之疑需在报告中提示
   - 若日主为阳干 → 加载 `rules/三命通会.md` 的"五阳长生起例"表
   - 否则（阴干）→ 加载 `rules/三命通会.md` 的"五阴长生起例"表 + `rules/子平真诠.md` 的"阴干长生不甚有力"立场

2. **Step 2：定十二位次的地支序列**
   - 加载 `rules/三命通会.md` 的"十干十二位起例表"——阳干查阳干起例表、阴干查阴干起例表，得到"日主五行 × 12 地支"的位次对照表
   - **例（甲木阳干顺行）**：长生亥、沐浴子、冠带丑、临官寅、帝旺卯、衰辰、病巳、死午、墓未、绝申、胎酉、养戌
   - **例（乙木阴干逆行）**：长生午、沐浴巳、冠带辰、临官卯、帝旺寅、衰丑、病子、死亥、墓戌、绝酉、胎申、养未
   - **土长生位特例**：戊土按"附火长生"说长生在寅、帝旺在午（与火同宫）；按"附水长生"说长生在申、帝旺在子。报告应**并列陈述两种说法**，按命局实际情况判定

3. **Step 3：对四柱地支逐位贴标签 + 通根判定**
   - 对四柱每一地支（年支、月支、日支、时支），查 Step 2 的对照表得到位次
   - **位次 → 吉凶标签映射**（加载 `rules/渊海子平.md` 对应位次节）：
     - 长生（吉位）→ 主人温和、长寿、得祖业
     - 沐浴（条件吉凶两端）→ 须看喜忌；与桃花（咸池）重叠则主感情复杂
     - 冠带（吉位）→ 兴家立业之象
     - 临官（吉位）→ 科甲、仕途之象
     - 帝旺（吉位）→ 旺之极、承祖业、显祖宗
     - 衰（凶位，但看吉神救助）→ 盛极初变
     - 病（凶位，但看吉神救助）→ 衰之甚
     - 死（凶位，但看吉神救助）→ 气数之尽
     - 墓（条件吉凶，看刑冲开启）→ 既藏亦埋
     - 绝（位次终始交接，"绝中逢生"可化）→ 前气已绝后气将续
     - 胎（半凶半吉，"胎神难为绝"）→ 新气初萌
     - 养（半凶半吉，须细审详）→ 气尚微、未成形
   - **同步判定通根**（加载 `rules/子平真诠.md` 的"通根透出之理"节 + `rules/滴天髓阐微.md` 的"四柱有根"节）：该地支是否在日主十二长生序列中？是 → 视为通根，分重根（长生禄旺）与轻根（墓库余气）两等
   - **墓库位特别提示**：加载 `rules/子平真诠.md` 的"墓库冲与不冲"节 + `rules/渊海子平.md` 的"墓库"节——并列陈述"逢库必冲"俗说（渊海诗诀）与"逢库不须冲"立场（子平真诠），按"以用神为上"原则，看墓库所藏之干与日主的生克关系方下断

4. **Step 4：看位次与吉神/凶煞的会遇**
   - 该地支上是否有吉神（天乙贵人、禄神、月德、天德）？位次 + 吉神 → 加力吉断
   - 该地支上是否有凶煞（七杀、阳刃、空亡）？位次 + 凶煞 → 加力凶断
   - **禄位会遇的细分吉凶**：加载 `rules/三命通会.md` 的"十干逐位禄位吉凶"节——甲见丙寅为福星禄（吉），甲见庚寅为破禄（半吉半凶），甲见壬寅为截路空亡（凶）等
   - **沐浴位特别警示**：若命局"子午卯酉"全见，触发桃花重叠判定（**旧时判语带时代局限**——参考 `rules/渊海子平.md` 的"沐浴"节末段"时代局限标注"）

5. **Step 5：流派分歧提示（仅在出现时插入）**
   - 若四柱地支落在**阴干长生位**（如乙木见午、丁火见酉、己土见酉、辛金见子、癸水见卯）→ 加载 `rules/子平真诠.md` 的"阴干长生不甚有力"立场，与 `rules/三命通会.md` 的"明列阴干长生"立场并列陈述，**不强行独尊一说**
   - 若四柱地支落在**墓库位**（辰戌丑未）→ 加载 `rules/子平真诠.md` 的"墓库冲与不冲"立场（认为不必冲开亦可用），与 `rules/渊海子平.md` 的"刑冲开启"说并列
   - 若日主为**戊己土**且月支非寅巳申亥 → 加载 `rules/三命通会.md` 的"土长生附火附水二说"节，并列陈述"戊土随丙火长生在寅"与"戊土随壬水长生在申"两种说法

```yaml
  年柱: {地支} → {位次}（{标签}）— {通俗解释 + 关键警示}

  日柱: {地支} → {位次}（{标签}）— {通俗解释}
  时柱: {地支} → {位次}（{标签}）— {通俗解释}

```

### 字段说明

| 字段 | 类型 | 取值范围 | 说明 |
|------|------|----------|------|
| 日主五行 | enum | 甲木/乙木/丙火/丁火/戊土/己土/庚金/辛金/壬水/癸水 | 按日干查 |
| 行运方向 | enum | 顺行/逆行 | 阳干顺、阴干逆 |
| 位次 | enum | 长生/沐浴/冠带/临官/帝旺/衰/病/死/墓/绝/胎/养 | 12 位 |
| 标签 | enum | 吉位/中性/凶位/条件吉凶两端/半凶半吉 | 见 Step 3 分支表 |
| 通根判定 | enum | 重根/轻根/无根 | 长生禄旺=重根；墓库余气=轻根；非位次=无根 |
| 通俗解释 | string | 1-2 句白话 | 必含 |

**通俗说明要求**：每个专业术语必须在报告中附一句通俗解释。例：

- 长生（初生之旺，主温和、长寿、得祖业）
- 沐浴（如婴儿浴于水中，体弱易摇动；位次与桃花重叠，主人多情或情感复杂）
- 临官（五行初入旺地，主科甲、仕途；与现代"当官"含义有别，指命局得力的状态）
- 墓库（既是收藏之所也是掩埋之所；须看"刑冲开钥"或"相生顺势"才可取用）
- 通根（日主在地支中找到了"立足点"——重根=长生禄旺，轻根=墓库余气）

**自包含性**：所有通俗解释必须在 skill 报告内部产出，不依赖外部引用。

- 四柱地支: 子 寅 午 卯

### 示例输出

日主五行: 丙火（阳干）
行运方向: 顺行

十二位次表（丙火阳干顺行）：
- 寅 → 长生（吉位；初生之火，主人温和、有文艺气质）
- 卯 → 沐浴（条件吉凶两端；五行初衰之位，主人体弱易动，须看喜忌）
- 辰 → 冠带（吉位；火气初盛，主人少年可立）
- 巳 → 临官（吉位；火气当权，主人身旺可用）
- 午 → 帝旺（吉位；火气之极，主人壮盛、贵气足）
- 未 → 衰（凶位；盛极而衰，主初转退）
- 申 → 病（凶位；衰之甚，须看吉神救助）
- 酉 → 死（凶位；气数之尽，旧说"火无西向"即此理）
- 戌 → 墓（条件吉凶；火之库，须刑冲或相生才可取用）
- 亥 → 绝（位次终始交接；"绝中逢生"可化）
- 子 → 胎（半凶半吉；新气初萌）
- 丑 → 养（半凶半吉；气尚微未成形）

四柱位次判定：
- 年柱：子 → 胎（半凶半吉）— 子水为丙火之胎，新气初萌；旧说"火明则灭"提示此时若见吉神则吉、见凶煞则殃
- 月柱：寅 → 长生（吉位）— 寅为丙火长生之方（"火生寅"），主温和、有文艺气质；月柱主父母宫与少年运，长生加临则少年得父母庇荫
- 日柱：午 → 帝旺（吉位）— 午为丙火帝旺之地，主人身旺可用；日支为婚姻宫，帝旺之火主配偶个性强、关系热烈
- 时柱：卯 → 沐浴（条件吉凶两端）— 卯为丙火沐浴位，旧说"沐浴桃花重叠"主多情或情感复杂；时柱为子女宫与晚年，沐浴在此须看全局喜忌方能下断

通根判定：
- 年柱子水 → 无根（不在丙火十二长生序列中）
- 月柱寅木 → **重根**（长生位，丙火在此有根且根深）
- 日柱午火 → **重根**（帝旺位，丙火在此根最重）
- 时柱卯木 → **轻根**（沐浴位，丙火气足而力未充，按子平真诠立场"沐浴在十二长生序列中，亦视为有根，但比长生禄旺次之"）

吉神/凶煞会遇：
- 月柱寅：寅中藏甲丙戊——甲木为丙火之印绶，印加长生则主人少年得学；戊土为丙火之食神，食神加长生则主人有口福
- 日柱午：午中藏丁己——丁火为丙火之劫财，劫财加帝旺则主人独立但有争财之象；己土为丙火之伤官，伤官加帝旺须看是否"伤官配印"或"伤官生财"
- 时柱卯：卯中藏乙——乙木为丙火之正印，正印加沐浴则主人晚年有学；但沐浴位须看喜忌

流派分歧提示：本命不见阴干长生位 / 不见墓库位 / 日主为丙火非戊己土，无相关分歧需提示。
