## 阳顺阴逆的根源辨正

> 【徐注】阳主聚，以进为进，故主顺；阴主散，以退为退，故主逆。长生沐浴等项，所以有阳顺阴逆之殊也。
> 【徐注】阳之所生，即阴之所死，彼此互换。

**判定规则**：

1. **阳顺阴逆的根源**——阳气主收敛凝聚（以进为进），阴气主发散流布（以退为退）
2. 一进一退落到十二支的同一段位上而方向相反——甲木长生在亥顺行至卯而帝旺；乙木长生在午逆行至子而帝旺
3. **"阳之所生，即阴之所死；彼此互换"**——这是子平真诠的根本立场
4. 报告应：阳顺阴逆作为基础判定口使用，但同时提示任氏对此的破执（见 `rules/滴天髓阐微.md` 的"任氏破阴生阳死之执"节）

## 五行非十干之辨

> 【徐注】生旺墓绝者，五行之生旺墓绝，非十干之旺墓绝也。十干之名称，为代表五行之阴阳；五行虽分阴阳，实为一物。甲乙，一木也，非有二也。
> 【徐注】从来术数书中，仅言五阳长生，而不言五阴长生，仅言阳刃而不言阴刃，后世未察其理，而欲自圆其说，支离曲解，莫知所从。

**判定规则**：

1. **十二长生是"五行"的循环，不是"十干"的循环**——甲乙同属一木，阴阳之分只在进退之顺逆，不在长生之有无
2. 此辨是子平真诠对历代术数的关键义理辨正——批判"误将十干视作两物"
3. 后世注家或言五阴无刃、或以进退一位为刃，皆因未明此理而强行自圆
4. **报告应据实陈述**：本 skill 的"位次表"既适用于阳干也适用于阴干——其根源在于五行而非十干

## 阴干长生不力

> 【徐注】若阴长生，则无关时令之气，地支藏用，不因之而有所增损也。
> 【徐注】阳长生有力，而阴长生不甚有力，然亦不弱。若是逢库，则阳为有根，而阴为无用。

**判定规则**：

1. **子平真诠立场**——阴干长生（如乙长生在午）只是阴阳进退之数序的对应点，**并非时令真实给阴干以生气**
2. **阴干长生不甚有力、然亦不弱**——既不能全然否定阴干长生有微力，也不能视同阳干长生之有根
3. **阴干逢库为无用**——阳大阴小，阳得兼阴、阴不能兼阳；阴干逢库则全无根气（与阳干逢库有别）
4. 论命取用时，**阴干之长生位不作为"根气"看待**
5. **此立场与三命通会形成流派分歧**——三命通会明列阴干长生（乙午、丁酉、己酉、辛子、癸卯），子平真诠立场是"阴干长生不甚有力"（详见 `rules/三命通会.md`）
6. **报告应**：
   - 当四柱地支落在阴干长生位时，**并列陈述双方依据**
   - 子平真诠立场："阴干长生不甚有力、然亦不弱、逢库为无用"
   - 三命通会立场："阴干长生是数序对应点，明列其位"
   - **不强行独尊一说**——按"以用神为上"原则，看该阴干长生在命局中是喜是忌方下断

## 十二位之名的譬喻

> 【徐注】长生者，犹人之初生也。沐浴者，犹人既生之后，而沐浴以去垢；如果核既为苗，则前之青壳，洗而去之矣。冠带者，形气渐长，犹人之年长而冠带也。临官者，由长而壮，犹人之可以出仕也。帝旺者，壮盛之极，犹人之可以辅帝而大有为也。衰者，盛极而衰，物之初变也。病者，衰之甚也。死者，气之尽而无余也。墓者，造化收藏，犹人之埋于土者也。绝者，前之气已绝，后之气将续也。胎者，后之气续而结聚成胎也。养者，如人养母腹也。自是而后，长生循环无端矣。

**判定规则**：

1. 十二位之名取譬于人身一生之历程，是借日常经验形容气之微变
2. 十二位的气数变化：长生（初生）→ 沐浴（去垢）→ 冠带（年长）→ 临官（出仕）→ 帝旺（壮盛）→ 衰（初变）→ 病（衰甚）→ 死（气尽）→ 墓（收藏）→ 绝（前气已绝后气将续）→ 胎（结聚）→ 养（母腹）
3. 报告应：每个位次标签的通俗解释**优先用此譬喻系统**（与渊海诗诀口径一致）

## 墓库冲与不冲

> 【徐注】人之日主，不必生逢禄旺，即月令休囚，而年日时中，得长禄旺，便不为弱，就使逢库，亦为有根。时产谓投库而必冲者，俗书之谬也。
> 【徐批】逢库必冲之说，谬误可嗤。

**判定规则**：

1. **子平真诠批判"逢库必冲"俗说**——库中藏有可用之干，**无需冲开**亦可为有根
2. 日主强弱不只看月令——年日时中得长生、禄、旺或余气、墓库**皆为有根**
3. **阴阳库之别**：
   - 阳干逢库**仍有根气**（阳大阴小，阳得兼阴）
   - 阴干逢库**则无用**（阴不能兼阳）
4. 辰为东方木地，若在清明后十二日内乙木司令、余气犹旺——**根本谈不上"投库"**
5. 土为辰之本气无所谓库；金火库中本无，冲亦无益；唯壬癸水遇辰为库，透出则同一可用
6. **此立场与渊海诗诀"刑冲开启"说形成流派分歧**——渊海诗诀墓库诗诀明确"刑冲是开之钥"；子平真诠立场是"逢库不须冲"（详见 `rules/渊海子平.md` 的"墓"节）
7. **报告应**：
   - 当四柱地支落在墓库位（辰戌丑未）时，**并列陈述双方依据**
   - 渊海立场："刑冲是开之钥、闭而不通则凶"
   - 子平立场："逢库不须冲、库中有根即可用"
   - **不强行独尊一说**——按"以用神为上"原则，看墓库所藏之干与日主的生克关系方下断

## 通根与透出之理

> 【徐注】地支所藏之干，本静以待用，透出干头，则显其用矣。故干以通根为美，支以透出为贵。
> 【徐注】天干通根，不仅禄旺为美，长生、余气、墓库皆其根也。如甲乙木见寅卯，固为身旺，而见亥辰未，亦为有根也。

**判定规则**：

1. **地支藏干是"体"（静以待用），天干透出是"用"（显于外）**——体用须相济
2. 天干在地支中有根为美（气有所本）
3. 地支所藏能透出天干为贵（用得以显）
4. **通根不只禄旺**——长生位（如亥）、余气位（如辰中所藏乙）、墓库位（如未中所藏乙）皆为天干之根
5. 通根之理不独适用于日主——**年月时之干同此**
6. 根之轻重两等：
   - **重根**——长生、禄、旺位（如甲木见亥寅卯）
   - **轻根**——墓库余气位（如甲木见辰未）
7. 报告应：判定四柱地支是否为日主通根时使用此规则，标注"重根/轻根/无根"

## 月令人元司令分野表

> 【徐注·月令人元司令表】

**判定规则**：

1. 一年三百六十日，按五行各分七十二日——月令用事的根本框架
2. 逐月分配五行用事日数：

   | 月 | 司令（按节气后日数）|
   |----|---------------------|
   | 寅月 | 戊土七日 → 丙火七日 → 甲木十六日 |
   | 卯月 | 甲木十日 → 乙木二十日 |
   | 辰月 | 乙木九日 → 癸水三日 → 戊土十八日 |
   | 巳月 | 戊土五日 → 庚金九日 → 丙火十六日 |
   | 午月 | 丙火十日 → 己土九日 → 丁火十一日 |
   | 未月 | 丁火九日 → 乙木三日 → 己土十八日 |
   | 申月 | 戊己土十日 → 壬水三日 → 庚金十七日 |
   | 酉月 | 庚金十日 → 辛金二十日 |
   | 戌月 | 辛金九日 → 丁火三日 → 戊土十八日 |
   | 亥月 | 戊土七日 → 甲木五日 → 壬水十八日 |
   | 子月 | 壬水十日 → 癸水二十日 |
   | 丑月 | 癸水九日 → 辛金三日 → 己土十八日 |

3. **日数不可拘泥**——地支藏干之理极为精密，与《周易》卦爻阴阳消息之理相通
4. 十二支由此分为三类：
   - **四生支（寅申巳亥）**——藏五行长生临官之干，故亦为四禄
   - **四专旺支（子午卯酉）**——藏五行帝旺之干
   - **四墓支（辰戌丑未）**——藏五行余气、墓地及土本气之干，是为杂气
5. 寅为甲木临官、丙戊长生之月，故藏甲丙戊三干；卯专藏乙（春木专旺之地）；辰藏戊乙癸（杂气）
6. 报告应：判定月令地支的"司令五行"时参考此表，**日数仅作参考不可拘泥**

## 得时失时非死法（破"得时即旺、失时即衰"之执）

> 【原文·沈氏】书云，得时俱为旺论，失时便作衰看，虽是至理，亦死法也。然亦可活看。夫五行之气，流行四时，虽日干各有专令，而其实专令之中，亦有并存者在。
> 【徐注】四时之中，五行之气，无时无刻不俱备，特有旺相休囚之别耳。

**判定规则**：

1. **破"得时即旺、失时即衰"死法**——五行之气流转于四时，不当令不等于无力
2. 判日主旺衰不可只看月令专令，须兼看四柱整体气的并存
3. 不得时中又有三等细分（旺相休囚）：火为方生之气（相），金土虽绝而其气将来，水为刚退而当休息
4. 退避不等于消失——春土何尝不生万物，冬日何尝不照万国
5. **根的等级体系**——长生禄旺为重根，墓库余气为轻根
6. **干多不如根重**——天干得比肩不如地支得墓库余气（"比劫如朋友、通根如家室"）
7. 报告应：判定日主强弱时，**不得单以月令论**，须综合四柱整体气的分布

## 根的特例与例外

> 【徐注】此节所论至精。墓库者，本身之库也，如未为木库，戌为火库，辰为水库，丑为金库。不能通用，与长生禄旺同，余气亦然。辰为木之余气，未为火之余气，戌为金之余气，丑为水之余气。

**判定规则**：

1. **墓库须各有所属**——未为木库、戌为火库、辰为水库、丑为金库，**不能通用**
2. **余气亦各有所属**——辰为木之余、未为火之余、戌为金之余、丑为水之余
3. **乙逢戌、丁逢丑不作通根论**——戌中无藏木、丑中无藏火，本库必须真藏本干之气方为通根
4. 沈氏原文"阴长生亦为有根可比一余气"——徐注明确驳议："木至午、火至酉皆为死地，岂得为根？盖亦拘于俗说而曲为之词也"
5. 报告应：判定通根时严守"本库中是否真藏本干之气"的标准

## 阳刃之五阳独享与"刃即劫之甚"

> 【原文】阳刃者，劫我正财之神，乃正财之七煞也。禄前一位，惟五阳有之，故为阳刃。不曰劫而曰刃，劫之甚也。
> 【徐注】刃者，就气候而言之也，甲木生卯月为刃，若非卯月而干透乙，或年日时支为卯，则应名之为劫而不名为刃。

**判定规则**：

1. **阳刃=禄前一位**——甲丙戊庚壬五阳干各有一位刃位（甲卯、丙午、戊午、庚酉、壬子），五阴干无此专位
2. **刃与劫之分**——就月令气候而定：得月令旺气为"刃"（旺过其极），非月令之支为"劫"（力较轻）
3. 刃宜伏制，官煞皆宜，财印相随尤为贵显
4. 阳刃与十二长生**直接相关**——帝旺位的别名即"刃"（如甲木卯为帝旺、亦为阳刃）
5. 报告应：判定某地支为日主的"帝旺位"时，同时提示其为"阳刃"——若此位再遇七杀、则成"杀刃格"（见 `rules/三命通会.md` 的"禄之诸格"节）

## 用神随月令藏透与会合而变化

> 【徐注】十二支中，子午卯酉为专气，所藏仅一神也；寅申巳亥为生地，所藏为长生禄旺之气也。十干即五行，五行仅四生地。阴长生者，乃阳极而阴生，非真生地，故子午卯酉中，无长生之神也。

**判定规则**：

1. 月令所藏非止一干——除子午卯酉专气外，其余八位皆有所藏
2. **寅中藏甲丙戊**（甲为本主、丙戊为长生）；**巳中藏丙戊庚**；**申中藏庚壬戊**；**亥中藏壬戊甲**
3. **土寄宫的特殊处理**——土寄旺于巳、寄生于申：巳中有丙火相生故土旺可用，亥中有金水相泄故土弱不可用
4. "假使寅月为提，不透甲而透丙"——本气虽当令，透干与否方是用神取定之关键
5. 报告应：判定月令地支的"司令五行"时，必须看透干之神而非只看本气

## 杂气取用——辰戌丑未的湿燥之分

> 【徐注】辰丑为湿土，戌未为燥土，其用各别，亦不能概论也。

**判定规则**：

1. **辰戌丑未四墓库=杂气**——所藏三干各不相谋、各司其用（与寅申巳亥所藏气势相通者不同）
2. **湿土与燥土之分**：
   - **辰丑为湿土**——能生金蓄水（辰为水库、丑含金库）
   - **戌未为燥土**——能制水焙火（戌为火库、未藏木库）
3. 取用总则——**透干会支，取其所清**：支中所藏透出天干则用清，支辰会合成局则力大
4. 辰月透癸用正印、丑月透辛用正官、戌月透辛用正官、未月透己用正印——同属四墓、用神可能完全不同
5. 报告应：判定四墓库位的"用神"时，必须先分湿燥、再看透干

## 杂气之有情与无情（透干会支的吉凶）

> 【沈氏】何谓有情？顺而相成者是也。何谓无情？逆而相背者是也。

**判定规则**：

1. **有情=顺而相成**——干支所透所会之气相互资生、彼此成就
2. **无情=逆而相背**——所透所会之气相互克战、各伤其用
3. **无情未必尽败，须视有无救应**：
   - 壬生未月官被伤克 → 柱有重印制伤护官可救
   - 甲生辰月财印两透交战 → 中隔以官/劫/伤可救
4. **有情而卒成无情**——表面似成，究其实则不成（位置配置失当所致）
5. 报告应：判定月令地支的"杂气取用"时，按"有情/无情/有救应/无救应"四象陈述
