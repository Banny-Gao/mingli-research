## 长生

> 【原文】长生管取命长荣，时日重临主性灵，更得吉神相会遇，少年及第入王庭。
> 【原文】长生若也得相逢，生下须招祖业隆，父母妻儿无克陷，安然享福保初终。
> 【眉批】五行长逢之地，木生於亥，火生於寅，金生於巳，水生於申，如能遇之，则福。

**判定规则**：

1. 阳干五行长生位：甲木→亥 / 丙火→寅 / 庚金→巳 / 壬水→申 / 戊土→寅（附火长生）
2. 阴干五行长生位：乙木→午 / 丁火→酉 / 己土→酉 / 辛金→子 / 癸水→卯
3. 长生位主"寿命悠长、命格荣显、得祖业、少年得志"
4. 长生位见吉神（天乙贵人、禄神、月德、天德）→ 吉力倍增，少年科场得意
5. **土长生位留疑**——按后世通行说戊土长生亦在申（与水同长生），亦有以土长生在亥者；本诗诀原眉批未列入土，本 skill 沿用「戊土随丙火长生在寅」的附火说

## 沐浴

> 【原文】沐浴凶神切忌之，多成多败少人知，男人值此应孤独，女命逢之定别离。
> 【原文】沐浴那堪吉位逢，更兼引从在其中，读书必定登科甲，莫比诸神例作凶。
> 【原文】桃花沐浴不堪闻，叔伯姑姨合共婚，日月时胎如犯此，定知无义乱人伦。
> 【原文】咸池无禄号桃花，酒色多因败坏家，更被凶神来克破，疾赢病死莫咨嗟。
> 【原文】女命若还逢沐浴，破败两三家不足，父母离乡寿不长，头男长女须防哭。
> 【眉批】沐浴，五行强以见弱之地，如人初生浴于水中无力，掌持极恰也。在命前为引，在命后为行。沐浴又为桃花杀同看，子午卯酉全之局也主淫乱。咸池星，即桃花杀，主酒色亡家，女人更忌生於此。

**判定规则**：

1. 沐浴是十二长生中"吉凶两端"之位——五行由长生之强转入"初衰"，气足而力未充
2. **沐浴 ≠ 铁定凶神**：第二诀"吉位可化"明确指出，若沐浴恰在喜用神位置且得引从（禄、贵等吉神）配合，可催文运、利科考
3. **沐浴与桃花（咸池）重叠**：子午卯酉四桃花支全见亦主淫乱；若命局四柱见子午卯酉之一以上 + 沐浴位，须警示
4. 沐浴位见凶神（七杀、羊刃）→ "疾赢病死"之凶象
5. **时代局限标注**（必读）：本诗诀中"主酒色亡家""女人更忌生于此""乱人伦""破败两三家""父母离乡寿不长"等判语，**带有宋明时期社会伦理色彩**，反映旧时男权中心对女性的道德苛求。今时观之应：
   - 取其"沐浴与桃花须谨慎辨喜忌"的方法论内核
   - **不可机械套用**"淫乱""败家""乱伦"等判语
   - **不可作为评价他人品行或现代两性关系的依据**

## 冠带

> 【原文】命逢冠带少人知，初主贫寒中主宜，更得贵人加本位，功成名遂又何疑。
> 【原文】人命若还逢冠带，兄弟妻孥无陷害，因何接祖绍箕裘，只为胎中有冠盖。
> 【眉批】冠带如人年长至十五，可以冠中，更带以在。五行始生，招之故为吉庆，人命当此，可以兴家立业。

**判定规则**：

1. 冠带是十二长生中"由蒙入壮"过渡关口之吉位
2. 冠带主"时序吉断"——初运未必遂意（"初主贫寒"），中运渐宜；若得贵人（天乙、月德）加临本位则功名可期
3. 冠带主"六亲吉断"——所在不冲不害兄弟妻孥之宫，主家门和睦、克绍祖业
4. 冠带为"兴家立业"之象——本质在"物已长成、可以施用"
5. **本诗诀早期子平诗诀重视"运限流转"的笔法**：单以冠带一字断吉不成立，须兼得"贵人加本位"或"胎中有冠盖"方可言遂

## 临官

> 【原文】临官帝旺最为奇，禄贵同宫仔细推，若不状元登上第，直须黄甲脱麻衣。
> 【眉批】临官帝旺者，如人年至四十五十，居天下之广居，立天下之正位，行天下之大道，得志与民由之不得志，独行其道，不离此身也。

**判定规则**：

1. 临官是十二长生第五阶（如木在寅、火在巳、金在申、水在亥）
2. **临官主"禄贵同宫"**——日主禄位（地支）与天乙贵人位（地支）落在同一支，是贵格的标准象
3. 临官位主"科甲高位"——若不状元，亦必中进士（"黄甲脱麻衣"指考中进士脱去布衣换官服）
4. 临官位之人**人到中年立身行道**——"得志"则与民由之、"不得志"则独行其道，不失本心
5. 戊土临官在巳（与火同宫）、己土临官在午；其余按五行各有所属

## 帝旺

> 【原文】临官帝旺两相逢，业绍箕裘显祖宗，失位纵然居世上，也须各姓达天聪。

**判定规则**：

1. 帝旺是十二长生"旺之极"（如木在卯、火在午、金在酉、水在子）
2. 帝旺主"承祖业、显祖宗"——"业绍箕裘"出自《礼记·学记》"良冶之子必学为裘、良弓之子必学为箕"
3. 帝旺主"失位亦达天聪"——"失位"（失去禄位或官位）但因命带帝旺之旺气，仍能"各姓达天聪"（通达智慧、闻名于众）
4. 帝旺是命局"自旺之极"的态势——"显祖宗"与"达天聪"分别对应"承祖"与"自显"两层吉象
5. 戊土帝旺在午（与火同宫）、己土帝旺在未；其余按五行各有所属

## 衰

> 【原文】纳音衰病死重逢，成败之中见吉凶，若得吉神来救助，变灾为福始亨通。
> 【原文】衰病两逢兼值死，世人至老无妻子，不惟衣食不丰隆，灾病绵绵终损己。
> 【眉批】五行以此比人之至六七十，由老衰弱而又死命中遇此，主人孤苦之断。

**判定规则**：

1. 衰 / 病 / 死 三位是十二长生"由盛转衰"到"完全气尽"的三环——衰是气势初退、病是衰到失力、死是完全失生气
2. **衰病死本身不一定主凶**——"成败之中见吉凶"，关键是看是否得"吉神来救助"（天乙贵人、禄神之类）
3. 衰病死三凶齐聚 + 无吉神救助 → 主"无妻子 / 衣食不丰隆 / 灾病绵绵终损己"
4. 此三凶对应人生六七十岁（"由老衰弱而又死"）——若命局原带此三位叠加（非自然老去），主"孤苦之断"
5. 报告应取"凶位重逢"作客观陈述，**不**简单以"命不好"理解

## 病

（见"衰"节。病与衰、死三位合并论述，无独立诗诀。）

**判定规则**：

1. 病位是衰之甚——按三命通会"母生子、子成而母老"之喻，病为气数将尽
2. 病位须看全局吉神救助；与衰、死合论时尤其警惕三凶齐聚

## 死

> 【原文·三命通会佐证】经云：木不南奔 / 水泛木浮 / 火无西向 / 火明则灭 / 土虚则崩 / 火燥土裂 / 金沉水底 / 土重金埋 / 死水横流 / 水不西流

**判定规则**：

1. 死位是气数之尽——五行于此完全失生气
2. 五阳干死位：甲木→午 / 丙火→酉 / 庚金→子 / 壬水→卯 / 戊土→酉（附火死位）
3. 五阴干死位：乙木→亥 / 丁火→寅 / 己土→寅 / 辛金→巳 / 癸水→申（与阳干长生位对冲）
4. 死位之论须有"经云"佐证（如"木不南奔"释甲木逢午火之焚、"金沉水底"释庚金逢子水之溺）
5. 死位与临官、帝旺、吉神同柱或相邻 → 可减凶力；纯死位无救助则按"衰病死"节凶断

## 墓

> 【原文】墓库元来是葬神，一为正印细推论，相生相顺无相克，富贵之中次第分。
> 【原文】人命若还逢墓库，积谷堆财难计数，悭贪不使一文钱，至老人呼守钱奴。
> 【眉批】墓库死者，埋葬于地，五行至此亦如是，如财官印於早有闭而不须用刑冲取用其财者。

**判定规则**：

1. **墓库是"葬神"——既是「入土」（墓）也是「储藏」（库）**，辰戌丑未四支的双重定位：
   - 「墓」重在"终"（五行之气终结，如木墓在未、火墓在戌、金墓在丑、水墓在辰）
   - 「库」重在"蓄"（收藏以待用，如辰为水库、丑为金库、未为木库、戌为火库）
2. **墓库的吉凶双面**：
   - 吉路：墓库所藏之干与日主相生顺、无相克 → 富贵可期，**「次第分」**（按透与不透、旺与不旺分高下）
   - 凶路：墓库所藏与日主不相生顺 → "积谷堆财难计数，悭贪不使一文钱"（守财奴之象）
3. **「刑冲」是「开」之钥**——墓库闭合时，须刑冲或相生开启才可取用
4. **"丑为正印"的特例**：辰戌丑未四墓库中，丑为金的墓库、又是金的印绶之库（己土生辛金），故丑中"一为正印"尤须细察
5. 墓库有"闭藏"之性——所藏之干不流通、不施出，故虽有财而财不出
6. **报告应明确**：墓库之"刑冲取用"说与子平真诠批判的"逢库必冲"俗说形成流派分歧，须并列陈述（见 `rules/子平真诠.md` 的"逢库必冲俗说批判"节）

## 绝

> 【原文·胞胎诗诀】绝中逢生少人知，却去当生命里推，反本还元直细辨，忽然迍否莫猜疑。
> 【原文·胞胎诗诀】胎神一位难为绝，克陷妻奴家道劣，不惟朝暮走忙忙，羊食狼贪无以别。
> 【眉批】胞胎者，其行如人受气於母亲腹中以至胎成而生。

**判定规则**：

1. **绝中逢生的玄机**——"绝"是五行气数已尽，但"绝中逢生"是少有人知的命理玄机：绝位之后会再次逢生（新一轮气数开始）
2. 绝位"不在别的位置推，要在『当生』（命主出生那一刻的八字）里推"
3. **胎神一位"难为绝"**——胎神这一位，论命时不能轻易下"绝"断；胎是受气成胎的新生状态，不是死局
4. **绝中逢生 + 忽然困顿**——即便"绝中逢生"的好命，遇到忽然的困顿也不要胡乱猜疑（"莫猜疑"）
5. **胎位若被克陷**（"克陷妻奴家道劣"）——主伤及妻妾奴仆、家道衰劣、朝暮奔忙
6. "羊食狼贪"——胎位受克者贪欲无度的统称（与羊刃的"羊食"性质相呼应）
7. 报告应：先说"绝中逢生"的好命底色，再点出"胎位被克"的凶象警示

## 胎

> 【原文】绝中逢生少人知，却去当生命里推，反本还元直细辨，忽然迍否莫猜疑。
> 【原文】胎神一位难为绝，克陷妻奴家道劣，不惟朝暮走忙忙，羊食狼贪无以别。
> 【眉批】胞胎者，其行如人受气於母亲腹中以至胎成而生。

**判定规则**：

1. 胎是十二长生中"新一周期受气成胎"之位——新生命的最早阶段
2. 胎位 ≠ 死局——"胎神一位难为绝"明确点出胎位不能轻易断为"绝"
3. 胎位主"反本还元"——返回命局最原始状态去重新审视（"直细辨"）
4. 胎位若被克陷（"克陷妻奴家道劣"）——主伤及妻妾奴仆、家道衰劣、朝暮奔忙
5. 胎位 + 凶煞（羊刃、贪狼）→ 主"贪欲无度"（"羊食狼贪无以别"）
6. 报告应：先说"胎位新生"的底色，再警示"胎位被克"的具体人事推断

## 养

> 【原文】胎养须宜细审详，半凶半吉两相当，贵神相会应为福，恶杀重逢见祸殃。
> 【眉批】养者，人已出乎母胎，如五行之长，吉星由大为吉。

**判定规则**：

1. 养是十二长生"新一周期养于母腹"之位——气尚微、未成形
2. **养位主"半凶半吉"**——"半凶半吉两相当"明确点出基调：不是一边倒的吉或一边倒的凶
3. **养位判定须"细审详"**——不轻下断语，须看旁边的"贵神"还是"恶杀"
4. 养位 + 贵神（天乙贵人）→ "应福"（吉祥会被放大，"吉星由大为吉"）
5. 养位 + 恶杀（羊刃、七杀）→ "见祸殃"（潜在的不安被激化）
6. 养位是"潜在而非现实"的状态——单独论断意义不大，必须配合喜忌定位
