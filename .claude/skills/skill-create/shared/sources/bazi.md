## 适用范畴

本技能适用于**正统子平八字**相关议题，包括：

- 八字命理理论探讨
- 经典原文解读与义理辨析
- 流派分歧的客观陈述
- 观点正误的经典依据判定

---

## 超出适用范畴的声明

以下议题**不在本技能适用范畴**，需明确告知用户：

| 超出范畴 | 说明 |
|----------|------|
| 紫微斗数 | 星命术，与子平体系不同 |
| 六爻/梅花易数 | 卜筮类，非命理体系 |
| 奇门遁甲 | 式盘类，非命理体系 |
| 大六壬 | 壬遁类，非命理体系 |
| 风水堪舆 | 地理术数，与命理并列 |
| 七政四余 | 星学类，非子平体系 |

当遇到超出范畴的议题时，应：
1. 明确告知该议题不在子平八字范畴
2. 可建议用户查阅对应术数的专项技能或资源
3. 保持专业礼貌，不贬低其他术数体系
