# 步骤 S4：信息生成

## 规范

| 字段 | 规则 |
|------|------|
| type | feat/fix/refactor/docs/style/test/chore/perf |
| scope | 模块名（小驼峰）或 root |
| description | 中文，≤50字 |

## type 匹配规则

- 新增功能 → feat
- 修复 bug → fix
- 重构 → refactor
- 文档 → docs
- 样式 → style
- 测试 → test
- 构建/依赖 → chore
- 性能 → perf
