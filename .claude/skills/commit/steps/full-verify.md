# 步骤 S6：全量验证

## 验证项

```bash
pnpm lint || npx eslint . --ext .vue,.ts,.tsx
```

```bash
tsc --noEmit --project tsconfig.json
```

### Build

```bash
pnpm build || npm run build
```
