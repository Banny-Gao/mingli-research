## L3 关键差异（vs L1/L2）

| 维度 | L1 | L2 | L3 |
|------|----|----|----|
| 触发条件 | WebFetch 拿清晰正文 | HTML 复杂 + 用户给 selector | WebFetch 拿不到正文（JS 渲染）|
| 工具 | WebFetch | WebFetch + selector | WebFetch → Playwright MCP |
| 适配站点 | 任意静态 HTML | 复杂静态 HTML | JS 渲染 SPA |

## Playwright 集成路径

```
1. mcp__playwright__browser_navigate {url}
2. mcp__playwright__browser_wait_for {time: 2}  # 等待 2s
3. mcp__playwright__browser_snapshot 拿 DOM
4. LLM 在渲染后 DOM 上做 L1 同等提取
5. 不持久化浏览器会话
```

## 限制

- v1 不处理登录态 / cookies
- v1 不处理验证码
- v1 不持久化浏览器 session
- v1 不抗反爬
- 每次新建浏览器，用完即关（无状态）
