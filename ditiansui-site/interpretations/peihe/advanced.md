### B-2. 配合篇 ← 理气篇（第五章）

> **理气篇核心：** 五行旺相休囚死，进气退气

**任铁樵在配合篇的话：** "审日主之衰旺，用神之喜忌"

这个"衰旺"从哪里来？→ **理气篇的"旺相休囚"**

```
理气篇：进气退气是判断衰旺的核心
  三月甲木 → 春木当令，其气始进
  九月甲木 → 秋金当令，其气始退
  
配合篇：审日主衰旺
  ↓
判断日主是进气还是退气
决定用神选取方向
```

**第一造为何失败？（联系理气）**

```
庚金生于辰月：
  辰月 → 土令 → 金相（次旺）
  但支会水局 → 水势反旺 → 金被水泄
  壬水进气（辰月水进气）
  庚金退气（相对水旺而言）
  
→ 这就是"衰旺失度"，不是简单的旺或衰
→ 用神难取
```

**第二造为何成功？（联系理气）**

```
乙木生于亥月：
  亥月 → 水令 → 木相（次旺）
  但有己土透出 → 止水卫火
  一阳后 → 木火进气（十月冬至一阳生）
  伤官秀气 → 进气之力

→ 虽然水势旺，但木火进气
→ 衰旺配合恰到好处
→ 用神得力
```

---

### B-3. 配合篇 ← 衰旺篇（第十四章）

> **衰旺篇核心：** 得令者旺，失令者衰；旺者冲衰则发，衰者冲旺则败

配合篇引用了衰旺判断：

```
"当抑则抑，当扶则扶"
  ↓
旺 → 抑（冲克抑之）
衰 → 扶（生助扶之）
  ↓
但不能机械判断，要看配合
```

**任注的精妙之处：** 衰旺不是独立判断，要看配合

```
第一造：
  庚金在辰月 → 本为金相（不弱）
  但水局冲之 → 旺变衰
  欲扶（水已旺）→ 矛盾
  欲抑（水更旺）→ 无力

第二造：
  乙木在亥月 → 本为木相（水令）
  但土卫火 → 衰中有救
  抑水（用己土）→ 扶木火
  配合得宜 → 发贵
```

---

### B-4. 配合篇 ← 中和篇（第十五章）

> **中和篇核心：** 中和为贵，偏枯则夭

配合篇的"配合有情"实质是**中和的实现手段**：

```
中和：五行流通，不偏不枯
  ↓
如何实现中和？
  → 干支配合有情
  → 有病得药
  → 相互护卫
  ↓
第一造为何不中和？
  水势过旺 → 偏枯
  无药可救 → 偏枯加剧
  结局：刑妻克子，孤苦不堪

第二造为何中和？
  水势虽旺，有土止
  火势虽弱，有土卫
  木火进气 → 平衡
  结局：青云直上
```

---

### B-5. 配合篇 ← 月令篇（第十二章）

> **月令篇核心：** 月令为提纲，司令用神由此定

配合篇的"审日主衰旺"依赖**月令判断**：

```
第一造：辰月
  月令本气：戊土
  司气：癸水（余气）
  → 水气司权 → 水势旺
  → 庚金虽在辰月相，但水冲土埋
  
第二造：亥月
  月令本气：壬水
  司气：甲木（余气）
  → 甲木司气 → 伤官秀气
  → 乙木有根，用神得力
```

### B-6. 配合篇 ← 知命篇（第四章）

> **知命篇核心：** 用神不可伤，旺极宜泄，衰极宜扶

配合篇的方法论是对**知命篇的展开**：

```
知命篇："能知衰旺之真机"
  ↓
配合篇："审日主之衰旺，用神之喜忌"
  ↓
知命篇："顺用神者吉，逆用神者凶"
  ↓
配合篇："当抑则抑，当扶则扶"
```

**两个命造的"用神不可伤"验证：**

```
第一造：官星午火为用神
  → 支会水局 → 用神受伤
  → 任注："官星必伤，不足为用"
  → 违背知命篇"用神不可伤"

第二造：伤官秀气为用神
  → 己土止水卫火 → 用神得护
  → 任注："用神生旺"
  → 符合知命篇"用神不可伤"
```

### C-3. 关键版本差异详解

#### 差异一：开篇有无"详"字

```
故宫本/坊刻本：
  "配合干支仔细，定人福祸与灾祥"
  
任铁樵增注本：
  "配合干支仔细详，定人福祸与灾祥"

考证：
  《四库全书》收录的《滴天髓》原文为：
  "配合干支仔细，定人福祸与灾祥"
  
  任注本将"仔细"拆分为"仔细详"
  一字之差，语义微变：
    "仔细" → 强调谨慎
    "仔细详" → 强调详尽推究
    
  任注本更强调"详推"之意
```

#### 差异二：任注有无"辟谬"定性

```
这是最重要的版本差异：

刘伯温原注本/坊刻本：
```
