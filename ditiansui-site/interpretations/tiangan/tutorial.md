### 三、序论精解：丙最癸至

> **原注：** 甲、丙、戊、庚、壬为阳，独丙火秉阳之精，而为阳中之阳；乙、丁、己、辛、癸为阴，独癸水秉阴之精，而为阴中之阴。

#### 3.1 阴阳十干分类

| 类别 | 天干 | 说明 |
|------|------|------|
| 五阳干 | 甲、丙、戊、庚、壬 | 阳刚健动 |
| 五阴干 | 乙、丁、己、辛、癸 | 阴柔顺静 |

#### 3.2 丙癸为最

| 天干 | 任注定性 | 核心特性 |
|------|---------|---------|
| **丙火** | 阳中之阳 | 纯阳之精，万物莫不由此而发 |
| **癸水** | 阴中之阴 | 纯阴之精，万物莫不由此而生 |

#### 3.3 阴阳相济

```
丙辛化水 → 阳极阴生（丙火+辛金 → 水）
戊癸化火 → 阴极阳生（戊土+癸水 → 火）
↓         ↓
阴阳相济，万物生生之妙
```

### 四、总论一：十干性情批判

#### 4.1 任注核心观点

> 甲乙一木也，丙丁一火也，戊己一土也，庚辛一金也，壬癸一水也，即分别所用，不过阳刚阴柔，阳健阴顺而已。

**【任注批判的世俗谬说】**

| 谬说 | 任注批语 |
|------|---------|
| 甲木为梁栋，乙木为花果 | 比拟失伦 |
| 丙作太阳，丁作灯烛 | 失伦 |
| 戊作城墙，己作田园 | 失伦 |
| 庚为顽铁，辛作珠玉 | 失伦 |
| 壬为江河，癸为雨露 | 失伦 |

#### 4.2 两大具体谬误批驳

**谬误一：死木活木之分**

```
世俗：甲木为无根死木，乙木为有根活木

任注批：
  同是木而分生死
  岂阳木独禀死气，阴木独禀生气乎？
  ↓
  阳木≠死木，阴木≠活木
  生死应从配合判断，非从天干阴阳属性
```

**谬误二：活木畏水，死木不畏水**

```
世俗：活木畏水泛，死木不畏水泛

任注批：
  岂活木遇水且漂，而枯槎遇水反定乎？
  ↓
  活木遇水未必漂（得地得用则不畏）
  枯木遇水未必定（朽木逢水反溃）
  ↓
  关键看配合，不在死活之名
```

**任注结论：**
> 论断诸干，如此之类，不一而足，当尽避之，以绝将来之谬。

### 五、总论二：五阳从气、五阴从势

#### 5.1 原注原文

> 五阳得阳之气，即能成乎阳刚之势，不畏财杀之势；五阴得阴之气，即能成乎阴顺之义，故木盛则从木，火盛则从火，土盛则从土，金盛则从金，水盛则从水。于情义之所在者，见其势衰，则忌之矣，盖妇人之情也。

#### 5.2 原文关键解读

```
五阳从气：
  气 → 阳刚之气
  不畏财杀之势
  解释：五阳干得阳刚之气，有独立之性，不易随从

五阴从势：
  势 → 阴柔之势
  木盛从木，火盛从火，水盛从水
  见势衰则忌 → 如妇人见势利
  ↓
  五阴干从势无情义
```

#### 5.3 任注详论：阳刚阴顺之辨

| 维度 | 五阳干 | 五阴干 |
|------|--------|--------|
| **本性** | 刚健 | 柔顺 |
| **处世** | 不苟且 | 骄谄 |
| **内心** | 有恻隐之心 | 有鄙吝之心 |
| **行为** | 不畏财杀 | 见势忘义 |
| **结果** | 刚不能制克柔 | 柔能克制刚 |

#### 5.4 任注：阳中之阴、阴中之阳

| 类型 | 外表 | 内心 | 命理含义 |
|------|------|------|---------|
| **阳中之阴** | 外仁义 | 内奸诈 | 伪君子 |
| **阴中之阳** | 外凶险 | 内仁慈 | 真豪杰 |
| **阳外阴内** | 外阳 | 内阴 | 包藏祸心 |
| **阴外阳内** | 外阴 | 内阳 | 秉持直道 |

#### 5.5 任注结论

> 要在气势顺正，四柱五行停匀，庶不偏倚，自无损人利己之心。

#### 6.1 甲木

> **原文：** 甲木参天，脱胎要火。春不容金，秋不容土。火炽乘龙，水宕骑虎。地润天和，植立千古。

**【原注要义】**

| 句 | 原注 | 任注展开 |
|----|------|---------|
| 甲木参天 | 纯阳之木，参天雄壮 | 体本坚固，极雄壮 |
| 脱胎要火 | 木嫩气寒，得火而发荣 | 强木得火，方化其顽 |
| 春不容金 | 旺木得火而愈敷荣，欺金不能容金 | 衰金克旺木，木坚金缺 |
| 秋不容土 | 枝叶凋落，根气收敛下达，受克者土 | 秋土虚薄，不能培木，反遭倾陷 |
| 火炽乘龙 | 寅午戌多见而坐辰 | 辰为水库，湿土生木泄火 |
| 水宕骑虎 | 申子辰多见而坐寅 | 寅为火土生地，能纳水气 |
| 地润天和 | 土不干，水不消 | 非植立千古而得长生 |

**【任注核心解读】**

```
甲木生于春：
  春初 → 木嫩气寒 → 得火发荣（要火）
  仲春 → 旺极之势 → 宜泄菁英（强木得火，方化其顽）

甲木与金：
  春金休囚 → 以衰金克旺木 → 木坚金缺
  势所必然 → 故"春不容金"

甲木与土：
  秋土虚薄 → 生金泄气
  虚气之土遇下攻之木 → 倾陷
  故"秋不容土"

甲木与火：
  丙丁多见（炽）→ 木被焚
  宜坐辰（水库）→ 乘龙
  湿土能生木泄火

甲木与水：
  壬癸多见（宕）→ 木浮
  宜坐寅（木禄旺）→ 骑虎
  能纳水气，不致浮泛
```

**【配合要点速记】**

```
甲木四时配：
  春 → 要火（发荣），不容金（克旺）
  夏 → 强木得火方化顽
  秋 → 不容土（虚薄倾陷），根气收敛
  冬 → 水润则生（但非乘龙骑虎之时）
```

#### 6.2 乙木

> **原文：** 乙木虽柔，刲羊解牛。怀丁抱丙，跨凤乘猴。虚湿之地，骑马亦忧。藤萝系甲，可春可秋。

**【原注要义】**

| 句 | 原注 | 任注展开 |
|----|------|---------|
| 刲羊解牛 | 生于丑未月，坐丑未土，可制柔土 | 丑乃湿土可以受气，未乃木库得以蟠根 |
| 怀丁抱丙 | 生申酉月，有丙丁透出天干 | 有水不相争克，制化得宜 |
| 跨凤乘猴 | 申酉月，得丙丁，不畏金强 | 凤=酉，猴=申 |
| 虚湿之地，骑马亦忧 | 生亥子月，无丙丁，又无燥土 | 午难发生 |
| 藤萝系甲 | 甲寅多见，藤萝附乔木 | 藤萝系松柏，春秋皆可 |

**【任注核心解读】**

```
乙木为甲之质，承甲之生气

乙木四季配：
  春如桃李 → 金克则凋 → 宜火
  夏如禾稼 → 水滋得生 → 宜水润燥
  秋如桐桂 → 金旺火制 → 宜火克金
  冬如奇葩 → 火湿土培 → 宜火土

刲羊解牛：
  丑未月 → 土柔 → 乙木可制
  坐丑 → 湿土受气
  坐未 → 木库蟠根

虚湿之地：
  亥子月 → 水旺
  无丙丁 → 无火发荣
  无燥土 → 无力培根
  坐午 → 亦难发生

藤萝系甲：
  天干透甲，地支藏寅
  鸢萝系松柏 → 附甲而生
  春得助，秋得扶 → 可春可秋
```

**【甲乙之辨（任注精华）】**

```
甲木 → 参天雄壮之木（阳）
乙木 → 甲之质，承甲之生气（阴）

本质区别：
  甲木是独立之木，参天之象
  乙木是依附之木，藤萝之性

但：
  乙木非弱者
  得丙丁 → 不畏金
  有甲寅 → 可春可秋
  丑未坐 → 可刲羊解牛
  ↓
  乙木之柔，非懦弱之柔
  乃能柔克刚、以弱胜强之柔
```

```

| 可秋可冬 | 生于秋冬，得一甲木则倚之不灭 | 干透甲乙，秋生不畏金 |

**【任注核心解读】**

```
丁火：柔中，内性昭融

  丙 → 纯阳猛烈

  ↓

丁火特性：
  旺而不烈 → 赫炎亦有度
  衰而不穷 → 熄灭有节

## 📋 附：完整原文

### 原文一：序论

```
五阳皆阳丙为最，五阴皆阴癸为至。

【原注】甲、丙、戊、庚、壬为阳，独丙火秉阳之精，而为阳中之阳；
乙、丁、己、辛、癸为阴，独癸水秉阴之精，而为阴中之阴。

【任氏曰】丙乃纯阳之火，万物莫不由此而发，得此而敛；
癸乃纯阴之水，万物莫不由此而生，得此而茂。
阳极则阴生，故丙辛化水；阴极则阳生，故戊癸化火。
阴阳相济，万物有生生之妙。
```

### 原文二：十干批判

```
夫十干之气，以先天言之，固一原同出，以后天言之，亦一气相包。
甲乙一木也，丙丁一火也，戊己一土也，庚辛一金也，壬癸一水也，
即分别所用，不过阳刚阴柔，阳健阴顺而已。

窃怪命家作歌为赋，比拟失伦，竟以甲木为梁栋，乙木为花果；
丙作太阳，丁作灯烛；戊作城墙，己作田园；
庚为顽铁，辛作珠玉；壬为江河，癸为雨露。
相沿已久，牢不可破，用之论命，诚大谬也。
```

### 原文三：五阳从气，五阴从势

```
五阳从气不从势，五阴从势无情义。

【原注】五阳得阳之气，即能成乎阳刚之势，不畏财杀之势；
五阴得阴之气，即能成乎阴顺之义，故木盛则从木，火盛则从火，
土盛则从土，金盛则从金，水盛则从水。
于情义之所在者，见其势衰，则忌之矣，盖妇人之情也。

【任氏曰】五阳气避，光亨之象易观；五阴气翕，包含之蕴难测。
五阳之性刚健，故不畏财煞，而有测隐之心，其处世不苟且；
五阴之性柔顺，故见势忘义，而有鄙吝之心，其处世多骄谄。
是以柔能克制刚，刚不能制克柔也。
```
