# 《滴天髓》·知命篇

> **原著：** 《滴天髓》上篇第四章·知命
> **正文：** 要与人间开聋聩，顺逆之机须理会

---

## 一、核心定理

### 定理一：知命之要
```
知命 = 理会顺逆之机
  ↓
顺者吉，逆者凶
```

### 定理二：四句核心
```
用之为财不可劫
用之为官不可伤
用之印绶不可坏
用之食神不可夺
  ↓
关键在一"用"字
```

### 定理三：旺极之命
```
旺之极者，宜泄不宜克
  ↓
顺其气势则吉
悖其性则凶
```

### 定理四：水流之命
```
昆仑之水，可顺而不可逆
  ↓
顺其流则吉
逆其流则凶
```
