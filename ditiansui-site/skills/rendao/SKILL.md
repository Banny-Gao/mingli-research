# 《滴天髓》·人道篇

> **原著：** 《滴天髓》上篇第三章·人道
> **正文：** 戴天履地人为贵，顺则吉兮凶则悖

---

## 一、核心定理

### 定理一：人贵于万物
```
人得五行之全（土居中央，木火金水中气）→ 为贵
万物各得五行之一偏 → 贱
```

### 定理二：顺悖定吉凶
```
顺 = 接续相生 = 有情 = 吉
悖 = 反克为害 = 无情 = 凶
```

### 定理三：中和为贵
```
命贵中和，偏枯终有损
理求平正，奇异不足为凭
```

### 定理四：救应四法（日主被克时）
```
① 地支生助日主（生）
② 天干化忌（化）
③ 地支通根（根）
④ 天干制忌（制）
有一种则吉，全无则凶
```

## 二、开篇正文

> 戴天履地人为贵，顺则吉兮凶则悖

| 关键词 | 命理含义 |
|--------|---------|
| 戴天履地 | 人立于天地之间 |
| 人为贵 | 人禀五行之全 |
| 顺则吉 | 五行相生有情则吉 |
| 凶则悖 | 五行反克无情则凶 |

---

## 三、【原注】核心

> 万物莫不得五行而戴天履地，惟人得五行之全，故为贵。其有吉凶之不一者，以其得于五行之顺与悖也。

- 人贵于万物：人得五行之全
- 吉凶各异：顺悖之别

---

## 四、【任氏曰】核心

### 4.1 顺悖四种情况

**顺（吉）：**
- 天干气弱，地支生之
- 地支神衰，天干辅之

**悖（凶）：**
- 天干衰弱，地支抑之
- 地支气弱，天干克之

### 4.2 木被金克救应
```
木畏金克 → 四种救应：
① 地支亥子水生木
② 天干壬癸水化金
③ 地支寅卯木通根
④ 天干丙丁火制金
有一种则吉，全无则凶
```

### 4.3 偏枯批判
```
四戊午、四癸亥等皆偏枯论
谬书妄称为圣帝之造，皆后人讹传
```
