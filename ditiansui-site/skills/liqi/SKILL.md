# 《滴天髓》·理气篇

> **原著：** 《滴天髓》上篇第五章·理气
> **正文：** 理承气行岂有常，进兮退兮宜抑扬

---

## 一、核心定理

### 定理一：进退有常
```
进之极 → 退之机
退之极 → 进之机
  ↓
进退循环，无有常
```

### 定理二：旺相休囚
```
相 = 将来者进（未来之星）
旺 = 进而当令（正当时令）
休 = 功成者退（功成退位）
囚 = 退而无气（退而无用）
```

### 定理三：日主喜忌
```
日主/喜神 → 宜旺相，不宜休囚
凶煞/忌神 → 宜休囚，不宜旺相
```

### 定理四：相与旺
```
旺 = 极盛之物 → 退反速
相 = 方长之气 → 进无涯
```
