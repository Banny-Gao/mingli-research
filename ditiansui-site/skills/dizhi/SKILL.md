# 《滴天髓》·地支篇

> **原著：** 《滴天髓》上篇第八章·地支
> **正文：** 阳支动且强，速达显灾祥；阴支静且专，否泰每经年

---

## 一、核心定理

### 定理一：阳动阴静
```
阳支（子丑寅卯辰巳）→ 动且强 → 气化流行，变化快；吉凶速达
阴支（午未申酉戌亥）→ 静且专 → 质凝形聚，变化慢；否泰经年乃见
```

### 定理二：藏干主次规则
```
本气为重（最重要）
  ↓
中气得时令可用（次要）
  ↓
余气非当令不足为用（最次）
```

### 定理三：冲合总纲（任注精华）
```
冲者，必是相克
支神以冲为重，刑与穿总非紧要
四库之冲，必须冲开，非闭藏之谓
冲有吉有凶，不可执一
```

### 定理四：生方旺方
```
生方怕动（亥子寅卯辰）→ 气化速变
旺方忌冲（申酉戌）→ 本气当令，冲开反吉
```

## 五、冲吉凶判断树

```
所冲之地支是否为用神所在？
  ↓是
  用神旺相？→ 冲起作用 → 大吉
  用神休囚？→ 冲坏用神 → 大凶

所冲之地支是否为忌神所在？
  ↓是
  忌神旺相？→ 冲坏忌神 → 大吉（去忌得清）
  忌神休囚？→ 冲起忌神 → 凶（忌神借势）

冲的是四库（辰戌丑未）吗？
  ↓是
  冲开 → 吉
  冲坏 → 凶
```
