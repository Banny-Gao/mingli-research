## 功能

本 skill 用于根据用户给定的八字四柱（年柱、月柱、日柱、时柱），按《滴天髓阐微》八格篇的「立格四步法」与「正变两类十六格」体系，判定该命局的格局归属，并给出用神定位与病药分析。

适用场景：命理研究者、学习者拿到一张八字四柱后，需要快速判别其格局正变、用神喜忌，并据此推断命主一生大致的富贵贫贱趋势。

目标读者：执行本 skill 后输出的判定报告须面向**普通用户**——读者不一定精通子平命理，故术语与结论需附带通俗解释。AI 执行者需在每个判定步骤同时给出「专业判语」与「通俗说明」。

## 输入

| 参数名 | 描述 | 类型 | 格式/约束 | 必填 |
|--------|------|------|-----------|------|
| year_pillar | 年柱 | string | 2 字符（天干地支） | 是 |
| month_pillar | 月柱 | string | 2 字符（天干地支） | 是 |
| day_pillar | 日柱 | string | 2 字符（天干地支），日干为命主 | 是 |
| hour_pillar | 时柱 | string | 2 字符（天干地支） | 是 |
| gender | 命主性别 | string | 取值 `男` / `女` | 否（默认 `男`） |

**输入不合法时**（如某柱不是合法天干地支组合），输出报告须明确指出「该柱格式非法、无法完成判定」，不得静默失败。

## 处理逻辑

### Step 1: 取月支、查司令、定透干

> 【原注】格之真者，月支之神，透于天干也。
>
> 【任氏曰】若月逢禄刃，无格可取，须审日主之喜忌另寻别支透出天干者，借以为用。

判定：

- 取月柱地支（月支）作为「立格之根」
- 查月支藏干（本气/中气/余气），按司令时节取当令者为主
- 查四柱天干是否有月支藏干透出
- 如果月支本气透出天干 → 进入 Step 2 立格判定
- 否则如果月支中气/余气透出 → 注明「格气不纯」，进入 Step 2 但需标注
- 否则（透干完全与月支无关）：
  - **如果其他柱（年/时）有全局最强神透出且与日主形成明显生克关系** → 注明「借以为用」，进入 Step 2 立格，但需在「格局真伪」字段标 `假格借透`，并在「判定依据」引用任氏「另寻别支透出天干者借以为用」
  - 否则（透干散乱无全局最强神）→ 进入 Step 4「无格可取」分支

### Step 2: 按正格七组 / 变格九组分类

> 【任氏曰】正者必兼五行之常礼也，曰官印，曰煞印，曰财煞，曰食神制杀，曰食神生财，曰伤官佩印，曰伤官生财；变者，必从五行之气势也，曰从财，曰从官杀，曰从食伤，曰从强，曰从弱，曰从势，曰一行得气，曰两气成形。

判定：

- 如果月令是四正支（子/午/卯/酉）、日主在月令得禄或得刃 → 考虑**变格九组**（从格类）
- 如果月令是辰/戌/丑/未 → 按月支所藏本气透干立正格，不可一律判为「杂气财官」
- 否则（月令为寅/申/巳/亥）→ 按透干所透之神归入**正格七组**之一
  - 官印格：正官透干 + 正印透干
  - 煞印格：七杀透干 + 正印透干
  - 财煞格：正/偏财透干 + 七杀透干
  - 食神制杀格：食神透干 + 七杀透干
  - 食神生财格：食神透干 + 正/偏财透干
  - 伤官佩印格：伤官透干 + 正印透干
  - 伤官生财格：伤官透干 + 正/偏财透干
- 如果上述判定都不能覆盖（如四柱散乱、月支不立格） → 进入 Step 4「无格可取」分支

### Step 3: 取用神、定病药

> 【任氏曰】格正用真，行运不悖，名利自如；格破用损，谓这有病，忧多乐少。

判定：

- 按日主旺衰取用神——日主旺宜抑（日主五行所克者为财、克日主者为官杀，均为用神候选）；日主衰宜扶（生我者为印、助我者为比劫，均为用神候选）
- 按格局定「忌神」与「病」——如印绶格忌财星坏印（财为病）；官杀格忌伤官克官（伤官为病）
- 找出「药」——能克去病神、护住用神者即为药；如印绶格以比劫护印、克制财星
- 判定「格正用真」或「格破用损」：
  - 配合得宜、病有药、格清纯 → 格正用真（属贵格）
  - 配合失衡、病无药、格混杂 → 格破用损（属贫格，需运程补救）
- 否则（既非「格正用真」也非「格破用损」） → 输出「格局中等、需视运程」的兜底判定

### Step 4: 兜底分支——无格可取

> 【任氏曰】大凡格局真实而纯粹者，百无一二，破坏而杂气者，十有八九，无格可取者甚多。

判定：

- 如果四柱天干散乱、月支之神未透出 → 输出「无格可取」
- 此时不再强立格，转而**只取用神**：按日主五行喜忌定用神
- 用神又无所取 → 看大势：日主旺衰、五行流通、气势方向
- 兜底结论：「碌碌营生、可免饥寒」或「不贫亦贱」取决于运程能否补其所喜
- **不**采用「飞天禄马」「杂气财官」「神煞」一类俗格立论（见 Step 5 批驳逻辑）

### Step 5: 排除俗格干扰

> 【任氏曰】其余外格我端，余务考群书，俱不从五行正理，尽属谬谈；至于《兰台妙选》，所定一切奇格异局，纳音诸法，尤属不经。

判定：

- 如果四柱出现「壬骑龙背」「六阴朝阳」「鼠贵格」一类俗格标识 → **不**采纳为正格
- 改回 Step 2 / Step 3 按月令透干原则重新立格
- 同样排除「飞天合禄」「墓库逢冲必发」「四种禄」一类附会之说
- 兜底：所有判定必须回溯到「月令本气透干」「日主旺衰」「生克制化」三原则

### 字段说明

| 字段 | 类型 | 取值范围 | 说明 |
|------|------|----------|------|
| 四柱 | string | 4 段，每段 2 字符（天干地支） | 命主完整四柱（年/月/日/时），原样回显 |
| 命主 | string | 十天干 | 日柱天干即命主 |
| 月令 | string | 12 地支 | 月柱地支 |
| 地支藏干 | string | 本气\|中气\|余气 | 月支所藏天干（如未藏己土/丁火/乙木），按司令时节标主用（如 `己土★ \| 丁火 \| 乙木`）|
| 格局归属 | enum | 16 分格名 / 无格 | 正格七组或变格九组，无格时为「无格可取」 |
| 格局真伪 | enum | 真格 / 假格 / 杂气 | 真格指月令本气透干且清纯；假格借透指月令本气未透、借别支透出立格（任氏曰"借以为用"）|
| 用神 / 病 / 药 | string | 天干或地支 | 三者各一 |
| 判定 | enum | 格正用真 / 格破用损 / 格局中等 / 无格 | 综合判定 |
| 通俗说明 | string | 1-2 句白话 | 例：「此命有真格、配合得宜，主贵」 |
| 判定依据 | string[] | 引文列表 | 必须包含至少 1 条 `> 【任氏曰】` 原文 |
| 进一步了解 | string[] | 篇名列表 | 建议阅读的 related articles |

**通俗说明要求**：每个专业术语必须在报告中附一句通俗解释。例：「正官（管束命主的力量）」「正印（生助命主的力量）」。

**自包含性**：所有通俗解释必须在 skill 报告内部产出，不依赖外部引用。

### 验证要点

- Step 1 应触发「月支本气未透、但其他干透出全局最强神」分支（年干庚金独发而清 + 时干癸水偏印），按任氏「借以为用」立格，进入 Step 2
- Step 2 应归入「官印格」组（正官庚金 + 正印癸水 透干），不是「正财格」也非「印绶格」
- Step 3 应判定「格正用真」——配合得宜、无病无药、格清纯
- 通俗说明应使用「管束命主的力量」「生助命主的力量」等白话解释，不可直接用「正官」「正印」无解释
- 判定依据必须包含至少 1 条 `> 【任氏曰】` 原文
- 报告首行 `四柱` 字段须原样回显用户输入的 4 段天干地支，不得省略或重排
- `地支藏干` 字段必须显示月支所藏全部天干（本气/中气/余气），并用 `★` 标注司令时节的主用藏干；不能仅显示透干
- `各柱天干十神` 与 `各柱地支藏干十神` 必须按相对日主的生克关系逐一标注（克我者官/杀、我克者财、同我者比劫、生我者印、我生者食伤），不能省略
- `日主旺衰` 须按月令司令严格判定（休/囚/旺/相/死），不得仅看根气强弱
- `格局真伪` 必须标 `假格借透` 而非 `真格`，因月支己土未透、借年干庚金立格
