## 功能

在收到八字四柱后，根据月令的当令五行，判断日主在月令所处的旺相休囚死状态，从而确定日主是"得令顺势"还是"失令逆势"，并据此给出用神选取方向和运势趋势建议。此技能应在用神分析之前调用，用于建立命局的时令大环境判断。

## 输入

- **四柱八字**：年柱、月柱、日柱、时柱对应的天干地支（字符串，格式如"癸卯 乙卯 甲寅 乙亥"）
- **性别**：乾造（男）或坤造（女）（可选，影响大运方向对顺逆的影响）

## 处理逻辑

1. **月令当令五行判定**
   a. 根据月支确定当令五行：
      - 寅卯月 → 木令
      - 巳午月 → 火令
      - 申酉月 → 金令
      - 亥子月 → 水令
      - 辰戌丑未月 → 土令
   b. **依据**：顺逆篇【原注】"得时令者为顺，失时令者为逆"

2. **日主旺相休囚死状态判断**
   a. 将日主五行与月令进行比较：
      - 日主五行 == 月令五行 → "旺"（得令，最顺）
      - 日主生月令 → "休"（失令，退气）
      - 月令生日主 → "相"（得令，次顺）
      - 月令克日主 → "囚"（失令，被困）
      - 日主克月令 → "死"（失令，最逆）
   b. **依据**：顺逆篇【任氏曰】"日主得令，顺时令之势，此顺势也；日主失令，逆时令之势，此逆势也"

3. **顺逆状态判定**
   a. 日主旺或相 → "顺势"（得令）
   b. 日主休、囚或死 → "逆势"（失令）

4. **用神方向建议**
   a. 顺势格局 → 用神宜顺月令之势（生助当令之气）
   b. 逆势格局 → 用神宜逆月令之势（调补助弱者）
   c. **依据**：顺逆篇【原注】"顺势者吉，逆势者凶"

5. **综合评估**
   a. 结合用神在月令中的顺逆状态给出最终评判
   b. 用神得令 → 格局得力
   c. 用神失令但有制化 → 险中求胜
