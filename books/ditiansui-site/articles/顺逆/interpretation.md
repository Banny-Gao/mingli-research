# 《滴天髓阐微·顺逆》解读

> 篇目编号：28 | 难度分级：基础级
> 原文来源：`articles/顺逆/source.md`

---

## 一、入门导读

### 1.1 这一篇在学什么

顺逆篇讲的是八字与时令的关系——日主应当"顺"时令之势还是"逆"时令之势。顺应时令者为吉，违逆时令者为凶。但"顺"不是无条件的顺从，"逆"也不是绝对的凶。你要学习判断在何种情况下应当顺势而为，在何种情况下又需要逆势而动。

### 1.2 本篇在整个子平体系中的地位

顺逆篇是刚柔篇（27）的延续——**刚柔论其性，顺逆论其势**。刚柔篇教你判断一个人天性的阴阳刚柔，顺逆篇则教你判断这个人对他所处的时令（月令大环境）是顺从还是违逆。两者结合，才能完整判断一个命局的"性情+时势"。顺逆篇与紧随其后的**寒暖篇（29）** 一样，都着眼于"命局与环境的关系"——顺逆讲的是"是否顺应时令大势"，寒暖讲的是"气候是否适宜"。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **判断日主的顺逆状态**——识别日主是得令（顺）还是失令（逆）
2. **理解顺不等于吉，逆不等于凶**——在特定条件下，逆势也可能吉利
3. **将顺逆法则应用于用神选取**——顺势取生扶，逆势取调济
4. **结合时令判断命局的整体态势**——不仅是四季旺衰，更是"顺势而为"还是"逆势抗争"

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
顺逆者，时令之从违也。顺势者吉，逆势者凶，此顺逆之辨也。
```

【原注】
顺逆者，时令之从违也。得时令者为顺，失时令者为逆。顺势者吉，逆势者凶。

【任氏曰】
顺逆之论，乃时令之辨也。日主得令，顺时令之势，此顺势也；日主失令，逆时令之势，此逆势也。

### 2.2 原文逐句精解

【原句】"顺逆者，时令之从违也。"

→ 解读：顺和逆的概念，来自于"对时令的服从还是违抗"。"从"就是顺着时令走，"违"就是逆着时令来。

【原句】"顺势者吉，逆势者凶，此顺逆之辨也。"

→ 解读：这个判断看起来简单直白——顺着时令的势头就吉，逆着时令的势头就凶。但这只是基本法则，实际运用中需要结合具体八字的格局来判定。后面将会看到，在特定条件下，逆势也能化凶为吉。

【原注】"得时令者为顺，失时令者为逆。顺势者吉，逆势者凶。"

→ 解读：原注把"时令之从违"具体化为"得时令"和"失时令"。得时令就是日主或者用神在月令上处于旺相状态；失时令就是处于休囚状态。顺势则吉，逆势则凶——这是基本原理。

【任氏曰】"顺逆之论，乃时令之辨也。日主得令，顺时令之势，此顺势也；日主失令，逆时令之势，此逆势也。"

→ 解读：任铁樵把顺逆的判断重点指向"日主"——日主得令（在月令旺相）就是顺势，日主失令（在月令休囚）就是逆势。这是一个简单实用的判断标准。但需要注意的是：日主得令固然好，但如果过旺而缺少制化，也会走向反面。日主失令虽然偏弱，但如果格局配置得当，也能成器。

## 三、核心义理深度解析

### 3.1 得令与失令的旺相休囚判定

**原文依据**

【原注】"得时令者为顺，失时令者为逆。"

**义理解读**

"得时令"和"失时令"的核心判断依据是五行在四季中的旺相休囚状态：

- **当令者旺**：木旺于春（寅卯月），火旺于夏（巳午月），金旺于秋（申酉月），水旺于冬（亥子月），土旺于四季末（辰戌丑未月）
- **生我者相**：如木旺时火相，火旺时土相——被当令所生的五行处于"相"（次旺）的状态
- **我克者休**：如木旺时土休——当令所克的五行处于"休"（退休）的状态
- **克我者囚**：如木旺时金囚——克制当令的五行处于"囚"（被困）的状态
- **我生者死**：如木旺时水死——生当令的五行处于"死"（力量耗尽）的状态

**日主得令**意味着日主在月令上处于旺或相的状态——力量强大，顺势而为容易成功。
**日主失令**意味着日主在月令上处于休、囚或死的状态——力量不足，需要逆势奋争或用神扶助。

**命造验证**

【真神篇 山东刘中堂造】甲子 丙寅 己丑 甲子

己土日主生于寅月（春季）。春季木旺、火相、土休。己土在寅月处于"休"的状态——得令否？答案是否定的，日主己土在月令处于休囚状态（失令）。但为什么此造仍然大贵？因为用神甲木（正官）得令（寅月当令为旺）、相神丙火（正印）得时（得寅月相气），官印相生，所以日主虽失令但格局清贵。这说明顺逆法则不仅要看日主本身，还要看用神与月令的关系。

### 3.2 顺势与逆势的深层意义——不仅是吉凶，更是格局的品格

**原文依据**

【任氏曰】"日主得令，顺时令之势，此顺势也；日主失令，逆时令之势，此逆势也。"

**义理解读**

顺逆不仅是吉凶判断，它更是命格的"品格"——决定了一个人是顺势而为的"调节型人格"还是逆势抗争的"战斗型人格"。

- **顺势之人**：日主得令，在整个命局中占主导地位。这类人通常适应环境能力强、在顺境中如鱼得水，但也可能缺乏危机意识。
- **逆势之人**：日主失令，需要在逆境中奋斗来证明自己。这类人通常坚韧、有闯劲，但容易走极端、人生起伏较大。

任铁樵说的"顺势者吉，逆势者凶"是从自然法则的角度说的——好比竹子随风弯曲（顺势）就不易折断，树木迎风挺立（逆势）就容易被吹断。但八字中的"顺逆"并不是一个简单的二元判断，它需要结合用神来综合分析：**日主顺势 + 用神得力 → 大吉；日主顺势 + 用神无力 → 平庸；日主逆势 + 用神得力 → 险中求胜；日主逆势 + 用神无力 → 困顿。**

**命造验证**

【天道篇 木专旺格】癸卯 乙卯 甲寅 乙亥

甲木日主生于卯月（春季木旺），日主得令——此为顺势。全局水木一气，无金相破，顺势而为的力量极强。此类格局中顺势取用才是正途——用神取水木，顺木令之势。如果强行用金（逆势）来克制旺木，反而会破坏格局的纯粹性。

## 四、命造实例拆解

### 命造一：日主得令顺势而为——木专旺格

【命造一】乾造：癸卯 乙卯 甲寅 乙亥

大运：癸丑 → 壬子 → 辛亥 → 庚戌 → 己酉 → 戊申 → 丁未
（出自任铁樵《滴天髓阐微·天道》）

| 柱 | 干支 | 月令状态 | 顺逆分析 |
|---|------|---------|---------|
| 年 | 癸卯 | 卯为木旺 | 印星生木，顺应木势 |
| 月 | 乙卯 | 卯为木令当旺 | 日主得令，顺势 |
| 日 | 甲寅 | 寅为木旺 | 日主坐寅木强根 |
| 时 | 乙亥 | 亥为木长生 | 印星生木，顺势 |

**格局**：木专旺格。甲木日主生于卯月（仲春木旺之时），日主得令为"顺势"。

**用神**：取水木（印星、比劫），顺木令之势。

**结论**：此造是最典型的"顺势"命局——日主甲木得令，全局水木一气贯通，无一金克制。顺势而为的结果就是格局纯粹、气势专一。这类命主往往一生顺风顺水，但一旦遇到逆势的大运（如庚戌运金来破格），就会有大的波动。

### 命造二：日主失令逆势求存——杀旺身弱格

【命造二】乾造：庚申 甲申 甲申 庚午

大运：乙酉 → 丙戌 → 丁亥 → 戊子 → 己丑 → 庚寅 → 辛卯
（出自任铁樵《滴天髓阐微·天道》）

| 柱 | 干支 | 月令状态 | 顺逆分析 |
|---|------|---------|---------|
| 年 | 庚申 | 金旺之地 | 七杀得令，逆势 |
| 月 | 甲申 | 申为金令 | 甲木日主失令（囚） |
| 日 | 甲申 | 金旺 | 日主坐绝地 |
| 时 | 庚午 | 午火制金 | 唯一的顺势救应 |

**格局**：七杀格（杀旺身弱）。

**用神**：取火（食神制杀），以逆制逆。

**结论**：此造甲木日主生于申月（秋季金旺），日主失令（木处囚地）。全局金气极旺，日主处于"逆势"状态——被时令所克。但神奇的是"逆势者凶"在这里并不完全成立：因为时支午火制住申金，以火之"逆"（火在金月失令）制金之"顺"（金在秋月当令），反而形成了"以逆制逆"的救应格局。这说明：逆势并不一定全凶，关键在于是否有制化之力。

## 六、常见误解与辨析

### 误解一：顺势一定吉，逆势一定凶

**错误说法**：有人认为"顺势者吉，逆势者凶"是绝对的、不可变通的铁律。

**经典原文反驳**：

【原注】"得时令者为顺，失时令者为逆。顺势者吉，逆势者凶。"

【任氏曰】"日主得令，顺时令之势，此顺势也；日主失令，逆时令之势，此逆势也。"

**正确理解**："顺势者吉，逆势者凶"是基本法则，但不是绝对法则。命造实例二（庚申甲申甲申庚午）清楚表明：日主失令（逆势）但有制化之力，同样可以富贵。"逆势"不等于"必凶"，关键在于命局中是否有能够制化逆势的力量。好比一个人逆流而上，虽然费力，但如果体魄强健、方向正确，也能抵达目的地。

### 误解二：顺逆只看日主和月令的关系就够了

**错误说法**：有人只看日主是否得令就断言顺逆吉凶。

**经典原文反驳**：

【任氏曰】"顺逆之论，乃时令之辨也。日主得令，顺时令之势，此顺势也；日主失令，逆时令之势，此逆势也。"

**正确理解**：虽然任铁樵以日主为切入点和重点来阐述顺逆，但实际运用中，顺逆的判断应该包括日主、用神、格局三个层面的综合评估。日主得令但是用神失令——日主虽然是顺势，但用神处于逆势，全局仍然不顺。顺逆是一个"全局"的概念，不是"日主"一个人的事情。

### 误解三：弱命就要逆势，旺命就要顺势

**错误说法**：有人认为命弱就要反抗命运（逆势），命旺就要顺应天命（顺势）。

**正确理解**：这其实是一种人生观的投射，并非命理法则。命局的"顺逆"不是一种选择，而是客观状态。日主得令就是顺，失令就是逆——这是由出生时间决定的客观事实。顺势命局如果行运不当也可能逆境丛生，逆势命局如果行运配合也能顺风顺水。不要用人生哲学来替代命理分析。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"顺逆者，时令之从违也"**——顺逆的概念：对时令的服从或违抗
2. **"得时令者为顺，失时令者为逆"**——顺逆的判断标准：是否得令
3. **"顺势者吉，逆势者凶"**——基本法则：顺势为吉，逆势为凶
4. **"日主得令，顺时令之势，此顺势也"**——日主得令是顺势的核心判断

### 7.2 与其他篇章的关联脉络

- **前序关系**：顺逆篇紧接**刚柔篇（27）**——刚柔论心性，顺逆论时势
- **直接关联篇目**：
  - **寒暖（29）**——顺逆讲时令的"势"，寒暖讲时令的"候"，势与候一体两面
  - **月令（15）**——旺相休囚死是顺逆判断的具体工具
  - **理气（04）**——顺逆是理气的一种具体表现

### 7.3 进阶学习路径建议

1. **寒暖篇（29）**——从时令的"顺逆"进入时令的"寒暖调候"（基础级）
2. **月令（15）**——深入掌握旺相休囚死的判断方法（基础级）
3. **实战练习**：拿到一个八字后，先判断日主是否得令、处在何种状态（旺相休囚死），再用顺逆法则评估整体的"势"
