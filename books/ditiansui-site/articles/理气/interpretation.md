# 理气 解读

> 《滴天髓阐微》通神论第五篇
> 难度分级：基础级

---

## 一、入门导读

### 1.1 这一篇在学什么

理气篇探讨五行之气在四时流转中的进退规律——气不是固定不变的，而是在春夏秋冬中不断进退、消长。掌握了气的进退之机，才能准确判断命局的衰旺，这是所有命理分析的基础。

### 1.2 本篇在整个子平体系中的地位

理气篇位于知命（第四篇）之后，配合（第六篇）之前，是通神论方法论部分的第二环节。知命篇建立了"顺逆之机"的辩证方法论，而理气篇将这些方法论落实到气的层面——具体说清楚五行之气在四时中如何进退、旺相休囚如何判定。没有理气篇，知命的顺逆判断就缺少了具体的依据。本篇属于**基础级**难度，但与后续的衰旺、天干、地支等篇有直接的技术关联。

### 1.3 学习目标

学完本篇后，你应能：

1. 说清楚五行之气"旺相休囚"四态的含义，以及"进"与"退"的区别
2. 掌握"将来者进，是谓相；进而当令，是谓旺；成功者退，是谓休；退而无气，是谓囚"四句口诀
3. 区分"相"与"旺"、"休"与"囚"之间的微妙差异
4. 将进退之机应用于日主衰旺的实际判断

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
理承气行岂有常，进兮退兮宜抑扬。

【原注】阖关往来皆是气，而理行乎其间。行之始而进，进之极则为退之机，如三月之甲木是也；行之盛而退，退之极则为进之机，如九月之甲木是也。学者宜抑扬其浅深，斯可以言命也。

【任氏曰】进退之机，不可不审也。非长生为旺，死绝为衰，乃按五行之气势，进退之机，明其衰旺之真机也。凡五行旺相休囚，按四时而定之。将来者进，是谓相；进而当令，是谓旺；成功者退，是谓休；退而无气，是谓囚。须察其旺相休囚，以知其进退之机。为日主，为喜神，宜旺相，不宜休囚；为凶神，为忌神，宜休囚，不宜旺相。然相妙于旺，旺则极盛之物，其退反速，相则方长之气，其进无疆也。休甚于囚，囚则既极之势，必将渐生；休则方退之气，未能骤复也。此理气进退之正论也，举两造为例。

丁丑 壬寅 甲辰 癸酉
己酉 庚戌 辛亥 壬子 癸丑 甲寅
甲木休困已极，酉金禄旺克之，一点丁火难以相对，加以两财生杀，杀似重身轻。不知九月甲木进令，癸水贴身，正要丙火明透，辰土止水生金，木之根已拔，比肩亦不能为力。年月两干，又透土金，只得从其弱势，顺财之性，以比肩为病。故运至水旺之地，制去比肩，事业巍峨；丙寅帮身，刑丧破貌。所谓弱极者扶之，徒劳无功，反有害也。此等格局颇多，以俗论之前造必以金水为用，此造必以木为用，以致吉凶颠倒，反归咎于命理之无凭，故特书两造为后证云。

戊寅 丙辰 甲戌 乙亥
丁巳 戊午 己未 庚申 辛酉 壬戌
甲木生于春初，乙木并透，通根月令，春木得火而发荣，丙丁火木之性情，而木性渗入膀胱而得令，正所谓藤萝系甲，可春可秋也。
```

### 2.2 原文逐句精解

**【原文首句】** "理承气行岂有常，进兮退兮宜抑扬。"

→ 解读：开篇点题。理（命理规律）是依托于气（五行之气）运转的，岂有固定不变的模式？关键在于把握气的"进"与"退"，并在判断时懂得"抑扬"（把握分寸、权衡轻重）。"岂有常"三字就否定了死板教条的命理方法。

**【原注】** "阖关往来皆是气，而理行乎其间。行之始而进，进之极则为退之机，如三月之甲木是也；行之盛而退，退之极则为进之机，如九月之甲木是也。学者宜抑扬其浅深，斯可以言命也。"

→ 解读：原注用形象的比喻说明气的进退规律。气的运行有始有终，开始的时候是"进"，进到极点就转化为"退"——比如三月的甲木（春季木旺至极，开始向夏季火转化）；反之，退到极点又转化为"进"——比如九月的甲木（秋季木衰至极，开始向冬季水转化、酝酿新一轮的生长）。学习者要根据气运的"浅深"（程度）来"抑扬"（把握分寸），这样才能谈论命理。三月甲木和九月甲木是理解进退之机的两个经典例证。

**【任氏曰第一段】** "进退之机，不可不审也。非长生为旺，死绝为衰，乃按五行之气势，进退之机，明其衰旺之真机也。凡五行旺相休囚，按四时而定之。将来者进，是谓相；进而当令，是谓旺；成功者退，是谓休；退而无气，是谓囚。须察其旺相休囚，以知其进退之机。为日主，为喜神，宜旺相，不宜休囚；为凶神，为忌神，宜休囚，不宜旺相。然相妙于旺，旺则极盛之物，其退反速，相则方长之气，其进无疆也。休甚于囚，囚则既极之势，必将渐生；休则方退之气，未能骤复也。此理气进退之正论也。"

→ 解读：这是任铁樵对理气进退理论的系统阐述，需要逐层理解：

第一层——"非长生为旺，死绝为衰"：纠正了一个常见的误区，即不能简单地用十二长生的生旺死绝状态来判断衰旺。

第二层——"将来者进，是谓相；进而当令，是谓旺；成功者退，是谓休；退而无气，是谓囚"：这一段是经典口诀。以春季为例——冬季之气退去、春季之气即将来临，这叫"相"（比如冬末的寅木）；春季之气正式当令，这叫"旺"（比如仲春的卯木）；春季之气完成使命开始退去，这叫"休"（比如春末的辰土）；完全退去没有气了，这叫"囚"（比如夏季的火当令时，冬季的水就是囚）。

第三层——"相妙于旺，旺则极盛之物，其退反速，相则方长之气，其进无疆也"：一个精妙的观点——"相"比"旺"更好，因为旺到了极点就会快速衰退，而"相"是正在生长的气，成长空间无限。

第四层——"休甚于囚，囚则既极之势，必将渐生；休则方退之气，未能骤复也"：另一个精妙观点——"休"比"囚"更差，因为"囚"是退到了极点，接下来反而会逐渐新生；而"休"是刚刚开始退，短期内难以恢复。

**【任氏曰命造一】** "丁丑 壬寅 甲辰 癸酉 ……所谓弱极者扶之，徒劳无功，反有害也"

→ 解读：任氏举第一个例证来说明"从弱"格局。甲木日主生于寅月看似得令，但分析全局：酉金正官当令克身，癸水正印贴身生木但力量微弱，丁火伤官虚浮，辰土财星耗木。全局金土力量占优，甲木看似有根实则"休困已极"。

关键判断：此命甲木表面有寅木禄位和癸水印星，但金旺土重，"木之根已拔"，因此不能扶助日主，而应该"从其弱势"。大运到水旺之地（壬子、癸丑）时，水制去比肩（乙木），反而事业有成；丙寅运帮身（扶助甲木），却"刑丧破貌"。

结论："弱极者扶之，徒劳无功，反有害也"——这就是理气进退之机的实际运用：不能机械地认为日主弱就要扶，要看气运的进退之势。

**【任氏曰命造二】** "戊寅 丙辰 甲戌 乙亥 ……藤萝系甲，可春可秋也"

→ 解读：第二个例证是甲木生于辰月（春末），乙木透出通根月令。丙丁火透出发荣木性。这个命造与前一造的对比意义在于：同样是甲木，同样是春季，但一个"从弱"，一个"发荣"——关键在于气的进退程度不同，全局配置不同。

## 三、核心义理深度解析

### 3.1 进退之机——五行之气在四时中的动态规律

**论点名称**：进退之机

**原文依据**：
【原注】"行之始而进，进之极则为退之机"
【任氏曰】"进退之机，不可不审也……将来者进，是谓相；进而当令，是谓旺；成功者退，是谓休；退而无气，是谓囚"

**义理解读**：
进退之机的核心是：五行之气不是在静止状态中存在的，而是在四时流转中不断变化。每个季节都有一种气当令（旺），一种气正在生长（相），一种气刚刚退去（休），一种气完全退去（囚）。

以春季为例：
- 木旺（当令）
- 火相（木生火，火气正在生长）
- 水休（水生木，水气刚完成使命开始退）
- 金囚（金被旺木所克，完全退去）

任氏有一个极其重要的观点："相妙于旺，旺则极盛之物，其退反速"。这意味着在判断命局时，正在生长的气（相）比已经到顶的气（旺）更有发展潜力。同理，"休甚于囚"——刚刚退去的气（休）比已经退到底的气（囚）更难恢复。

**命造验证**：
第一个命造（丁丑 壬寅 甲辰 癸酉）就是进退之机的绝佳例证。表面看甲木在寅月当令，但全局金旺土重、木根已拔，气的实际走向是"从弱"，因此不能扶木。

### 3.2 旺相休囚的真正含义——超越十二长生

**论点名称**：旺相休囚与四时的对应关系

**原文依据**：
【任氏曰】"非长生为旺，死绝为衰，乃按五行之气势，进退之机，明其衰旺之真机也。凡五行旺相休囚，按四时而定之。"

**义理解读**：
任铁樵在这里纠正了一个重要的误解：不能用十二长生的生旺死绝状态来判断五行衰旺。十二长生（长生、沐浴、冠带、临官、帝旺、衰、病、死、墓、绝、胎、养）是另一套体系，而旺相休囚是专门按四时定的。

旺相休囚的判定标准是：
- **旺**：正当令，气最盛（如春木）
- **相**：即将当令，气在生长（如春火）
- **休**：刚刚退令，气在消退（如春水）
- **囚**：完全被克，气最弱（如春金）

**命造验证**：
两个命造都是甲木日主，一个在寅月（春初），一个在辰月（春末）。同样在春季，但气的进退程度已经不同：寅月甲木初旺，辰月甲木已退。这就解释了为什么一个需要从弱，一个可以发荣。

## 四、命造实例拆解

### 【命造一】丁丑 壬寅 甲辰 癸酉——从弱之局

**乾造**：丁丑年 壬寅月 甲辰日 癸酉时

**大运**：己酉 庚戌 辛亥 壬子 癸丑 甲寅

**格局**：甲木日主生于寅月，虽得月令但全局金旺土重，木根已拔。此为从弱格。

**逐项分析**：
- **年柱丁丑**：丁火伤官虚浮，丑土财星耗木
- **月柱壬寅**：壬水偏印看似生木，实则自身不强；寅木为日主禄位，但被金克土耗
- **日柱甲辰**：辰土偏财耗木
- **时柱癸酉**：癸水正印力量微弱，酉金正官当令克身
- 全局金水势力占优，"水旺"制去比肩时事业有成
- 丙寅运帮身（扶助甲木），反而"刑丧破貌"

**结论**：此造甲木"休困已极"，看似得令却不任扶，需从弱。顺其弱势则吉，强扶日主则凶。这正是进退之机的具体运用——不能因为甲木生在寅月就断旺，要看全局气的真实走向。

### 【命造二】戊寅 丙辰 甲戌 乙亥——藤萝系甲

**乾造**：戊寅年 丙辰月 甲戌日 乙亥时

**大运**：丁巳 戊午 己未 庚申 辛酉 壬戌

**格局**：甲木日主生于辰月（春末），乙木劫财透出通根月令，丙丁火透出发荣木性。此为身旺食伤泄秀格。

**逐项分析**：
- **年柱戊寅**：戊土偏财，寅木为甲木禄位
- **月柱丙辰**：丙火食神透出泄木之秀气，辰土蓄水养木
- **日柱甲戌**：甲木日主，戌土偏财
- **时柱乙亥**：乙木劫财帮身，亥水偏印生木
- 全局木得火而发荣，"藤萝系甲，可春可秋"——如藤蔓缠绕大树，四季皆可

**结论**：此造与前造同为甲木，但配置完全不同。木得火泄秀，得水滋润，得土培根，气运流通顺畅。进退之机在这里体现为：甲木在春末有火引化，不似前造那样"木根已拔"，所以可以"发荣"。

## 六、常见误解与辨析

### 误解一：长生就是旺，死绝就是衰

**错误说法**：用十二长生的生旺死绝状态来代替旺相休囚判断五行衰旺。

**经典原文反驳**：
【任氏曰】"非长生为旺，死绝为衰，乃按五行之气势，进退之机，明其衰旺之真机也"

**正确理解**：十二长生（长生、沐浴、冠带、临官、帝旺等）和旺相休囚是两套不同的体系，不能混用。旺相休囚按四时季节定：春木旺、夏火旺、秋金旺、冬水旺、季土旺。而十二长生描述的是五行从生到死的完整周期。任铁樵明确指出，判断衰旺要用旺相休囚体系。

### 误解二："相"不如"旺"，"休"好于"囚"

**错误说法**：认为"旺"一定比"相"好，"囚"一定比"休"差。

**经典原文反驳**：
【任氏曰】"然相妙于旺，旺则极盛之物，其退反速，相则方长之气，其进无疆也。休甚于囚，囚则既极之势，必将渐生；休则方退之气，未能骤复也。"

**正确理解**：任氏这个观点非常精妙——"相"比"旺"更有潜力，因为旺到了极点就会衰退，而"相"正在生长，未来可期。"休"比"囚"更差，因为"囚"已经到了不能再差的极点，反而孕育着新生；而"休"刚开始退，短期内难以恢复。这完全符合事物发展的辩证法——否极泰来、盛极而衰。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"将来者进，是谓相；进而当令，是谓旺；成功者退，是谓休；退而无气，是谓囚"**——旺相休囚四态标准定义，必须熟练默诵
2. **"相妙于旺"**——正在生长的气比已经到顶的气更有潜力
3. **"休甚于囚"**——刚刚退去的气比已经退到底的气更难恢复
4. **"非长生为旺，死绝为衰"**——不要混淆十二长生与旺相休囚两套体系
5. **进退之机核心**：五行之气在四时中不断进退，没有固定不变的状态

### 7.2 与其他篇章的关联脉络

- **知命（第四篇）**：知命篇的"顺逆之机"是方法论，理气篇的"进退之机"是该方法在气的维度上的具体应用
- **配合（第六篇）**：配合篇讲干支的具体配合，需要以理气篇的气运进退判断为前提
- **衰旺（第十七篇）**：理气篇为衰旺篇提供了理论基础——旺相休囚的判定是衰旺判断的核心依据
- **天道（第一篇）**：天道篇讲五行生克的宏观规律，理气篇将其落实到气的进退层面

### 7.3 进阶学习路径建议

- **继续**：配合（第六篇）——在掌握进退之机后，学习干支配合的具体方法
- **辅助阅读**：配合理气篇理解，可参考《穷通宝鉴》中五行在四时的宜忌论述
- **难度对比**：理气篇为基础级，概念看似简单但"相妙于旺""休甚于囚"的精妙之处需要反复体会
