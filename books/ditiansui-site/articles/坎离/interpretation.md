# 《滴天髓阐微·坎离》解读

> 篇目编号：34 | 难度分级：专题级
> 原文来源：`articles/坎离/source.md`

## 一、入门导读

### 1.1 这一篇在学什么

坎离篇讲八字中水（坎）与火（离）两种力量相济相持的问题。坎为水、离为火，二者在五行上相克（水克火），在性质上一寒一热。本篇提出了水火相济的"五法"体系（升、降、和、解、制），告诉你当水火在命局中不相容时，如何通过正确的策略使它们从"交战"走向"既济"。

### 1.2 本篇在整个子平体系中的地位

坎离篇是通神论第34篇，与第33篇"震兑"构成完整的"四象调候"闭环——震兑讲金木（东西相战），坎离讲水火（南北相冲）。两篇共同构成了《滴天髓》最核心的"两两相对、相成相济"哲学。此外，坎离篇与"寒暖"（29）"燥湿"（30）也有深度关联。本篇属于**专题级**难度，需要你对水火五行的特性有深入理解。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **理解"坎离相持"的含义**——水火不能独成，必须相持（互相制衡）才能成事
2. **掌握坎离五法**——升（木升水气）、降（金降水气）、和（木运调和）、解（金运化解）、制（岁运制强助弱）
3. **区分既济与未济**——天干水地支火 ≠ 天干火地支水，两种局势的处理完全不同
4. **看懂五种水火战局的命造**——从任铁樵的五个命造中理解五法的实际运用

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
坎离宰天地之中气，成不独成，而有相持者在。

【原注】
天干透壬癸，地支属离者，乃为既济，要天气下降；
天干透丙丁，地支属坎者，乃为未济，要地气上升。
天干皆水，地支皆火，为交媾，交媾身强则富贵；
天干皆火，地支皆水，为交战，交战身弱，岂能富贵？
坎外离内，谓之未济，主之所喜在离，要水竭；
主之所喜在坎，则不详。
离外坎内，谓之既济，主之所喜在坎，要离降；
主之所喜在离，要木和。
水火相见于天干，以火为主，而水盛者存；
坎离相见于地支，喜坎而坎旺者昌。
夫子、午、卯、酉专气也，其相制相持之势，宜悉辨之；
若四生四库之神，皆所以党助子午卯酉者，其理亦可推详。

【任氏曰】
坎阳也，先天位右七之数，故为阳也；
离阴也，先天位左三之数，故为阴也。
坎为中男，天道下济，故一阳生于北；
离为中女，地道上行，故二阴生于南。
离为日体，坎为月体，一润一暄，水火相济，男女媾精，万物化生矣。
夫坎离为日月之正体，无消无灭，而宰天地之中气，
是以不可独成，必要相持为妙也。
相持之理有五，升、降、和、解、制也。
升者，天干离衰，地支坎旺，必得地支有木，则地气上升；
降者，天干坎衰，地支离旺，必得天干有金，则天气下降；
和者，天干皆火，地支皆水，必须有木运以和之；
解者，天干皆水，地支皆火，必须有金运以解之；
制者，水火交战于干支，必须岁运视其强者而制之。
此五者，坎离之作用如此，则无独成之势，而有相持礼智之性矣。
```

### 2.2 原文逐句精解

【原句】"坎离宰天地之中气，成不独成，而有相持者在"

→ 解读：开篇总纲。坎（水）和离（火）主宰着天地之间的"中气"（核心能量）。它们不能独自成功（成不独成），必须互相支持、互相平衡（相持）。"相持"不是对抗僵持，而是互相制衡、互相成就。

【原注第1句】"天干透壬癸，地支属离者，乃为既济，要天气下降；天干透丙丁，地支属坎者，乃为未济，要地气上升"

→ 解读：原注定义了"既济"和"未济"两个重要概念（借用《易经》卦名）：

- **既济**（已完成的交汇）：天干是水（壬癸），地支是火（巳午）。火性炎上、水性润下，方向相反，所以需要"天气下降"——让天干的水气下降来与地下的火交汇
- **未济**（未完成的交汇）：天干是火（丙丁），地支是水（亥子）。同样需要"地气上升"——让地支的水气上升来与天上的火交汇

【原注第2句】"天干皆水，地支皆火，为交媾，交媾身强则富贵；天干皆火，地支皆水，为交战，交战身弱，岂能富贵？"

→ 解读：两种极端场景：

- **交媾**（阴阳和谐）：天干全是水、地支全是火——水火各居其位，各行其道，如果日主强旺，则为富贵之命
- **交战**（水火相冲）：天干全是火、地支全是水——这是水火在同层面直接对抗，日主削弱，不可能富贵

【原注第3句】"坎外离内，谓之未济……离外坎内，谓之既济"

→ 解读：继续深化既济/未济的定义。坎外离内（天干水、地支火）= 未济；离外坎内（天干火、地支水）= 既济。注意这和前文有点不同，原注在不同的语境下给出了两种表述，需要综合理解。

【任氏曰第1段】"坎阳也……离阴也……坎为中男……离为中女……离为日体，坎为月体，一润一暄，水火相济，男女媾精，万物化生矣"

→ 解读：任铁樵用八卦原理解释坎离。坎虽然是水（阴的五行），但它是阳卦（中男）；离虽然是火（阳的五行），但它是阴卦（中女）。坎为月体（阴性天体），离为日体（阳性天体）。一润（水之润）一暄（火之暖），水火相济，就像男女结合一样，万物由此化生。

【任氏曰第2段】"夫坎离为日月之正体，无消无灭，而宰天地之中气，是以不可独成，必要相持为妙也"

→ 解读：任铁樵强调坎离的恒久性——不像其他五行有旺相休囚的循环（春木旺、秋金旺等），坎离（日月）是永恒的，"无消无灭"，所以它们必须有"相持"（互相制衡）的关系。

【任氏曰第3段】"相持之理有五，升、降、和、解、制也"

→ 解读：**坎离五法**——和震兑五法（攻成润从暖）并列的另一个五法体系。

【任氏曰第4段】"升者，天干离衰，地支坎旺，必得地支有木，则地气上升"

→ 解读：**升法**。天干离衰（天干火弱）、地支坎旺（地支水旺）。这时候需要地支有木（寅卯）来"升"——木在地支，能把水气向上引导（木从水中吸水向上），实现"地气上升"与天干之火交汇。

【任氏曰第5段】"降者，天干坎衰，地支离旺，必得天干有金，则天气下降"

→ 解读：**降法**。天干坎衰（天干水弱）、地支离旺（地支火旺）。这时候需要天干有金（庚辛）来"降"——金在天干，能把火气向下引导（金能生水，水气向下），实现"天气下降"与地支之水交汇。

【任氏曰第6段】"和者，天干皆火，地支皆水，必须有木运以和之"

→ 解读：**和法**。天干全是火、地支全是水——水火各行其道但没有交汇。需要"木运"来调和（和之），因为木能通水火——木从水中吸收水气，向上蒸腾供给火。

【任氏曰第7段】"解者，天干皆水，地支皆火，必须有金运以解之"

→ 解读：**解法**。天干全是水、地支全是火——水火各行其道但方向相反。需要"金运"来化解（解之），因为金能为水火提供中间媒介——金生水、金也生火（金被火克但能传递能量）。

【任氏曰第8段】"制者，水火交战于干支，必须岁运视其强者而制之"

→ 解读：**制法**。水火直接交战于四柱中（同层次互相冲击），需要看岁运中哪一方更强，然后制强扶弱——让水火恢复平衡。

## 三、核心义理深度解析

### 3.1 既济与未济——水火交汇的两种态势

**原文依据**

【原注】"天干皆水，地支皆火，为交媾，交媾身强则富贵；天干皆火，地支皆水，为交战，交战身弱，岂能富贵？"

**义理解读**

"交媾"和"交战"一字之差，吉凶天壤之别：

- **天干水、地支火**（水上火下）= **交媾**。水火各在其位，水往下流、火往上窜，各行其道、互不干扰，阴阳和谐。日主强则为富贵。
- **天干火、地支水**（火上水下）= **交战**。火性本炎上（往上走），水性本润下（往下走），二者方向一致向下/向上（不是相向而行），会产生直接冲突，是"交战"状态。

关键在于：水火不是越"平衡"越好，而是位置对、方向对才能和谐。水在上、火在下是自然之道（天降雨露滋润大地之火），反之则是逆天之道。

**命造验证**

【既济例】丙子 己亥 丙寅 戊子

此造是"天干离衰、地支坎旺"的既济格局。丙火生于亥月（孟冬，水旺火弱），地支双子水——"天干离衰，地支坎旺"。用寅木"升之"——寅木在地支引水气上升。壬寅、癸卯东方木运发挥作用，采芹折桂、仕至观察。

【未济例】壬午 壬寅 壬戌 庚戌

此造壬水生于寅月（孟春），地支午戌半合火局——"天干坎衰，地支离旺"。用庚金"降之"——庚金在天干引火气下降。可惜运走东南木火（卯辰巳午未），没有发挥庚金的作用，一生漂泊无成。

### 3.2 坎离五法——水火相持的五种策略

**原文依据**

【任氏曰】"相持之理有五，升、降、和、解、制也。"

**义理解读**

|  策略  |      适用场景       |         操作         |          比喻          |
| :----: | :-----------------: | :------------------: | :--------------------: |
| **升** | 天干火弱 + 地支水旺 | 用地支的木引水气上升 |  像水泵把地下水抽上来  |
| **降** | 天干水弱 + 地支火旺 | 用天干的金引火气下降 | 像用金属棒引导电流入地 |
| **和** | 天干皆火 + 地支皆水 |  用木运调和两种势力  |  像中间人调解水火之争  |
| **解** | 天干皆水 + 地支皆火 |    用金运化解冲突    |   像催化剂让二者相容   |
| **制** |    水火直接交战     | 视强者而制之（岁运） |  像裁判压制更凶的一方  |

这五种策略的本质是：改变水火的气流方向，让它们从"对抗"变为"交汇"。"升"和"降"是改变气的运行方向（垂直方向），"和"和"解"是借助中间介质（木和金），"制"是直接干预（靠岁运）。

## 四、命造实例拆解

### 命造一：天干离衰、地支坎旺——升法

【命造一】乾造：丙子 己亥 丙寅 戊子

大运：庚子 → 辛丑 → 壬寅 → 癸卯 → 甲辰 → 乙巳
（出自任铁樵《滴天髓阐微·坎离》）

| 柱  | 干支 | 分析                               |
| --- | ---- | ---------------------------------- |
| 年  | 丙子 | 丙火微弱，子水旺                   |
| 月  | 己亥 | 己土伤官泄火，亥水当令             |
| 日  | 丙寅 | 丙火日主，寅木为关键（引水气上升） |
| 时  | 戊子 | 戊土食神，子水加重水势             |

**格局**：正官格（身弱官旺）。**用神**：寅木（升法之枢纽）。**喜**：木火运。**忌**：金水运。

**结论**：此造是"升法"的经典案例。丙火日主生于亥月，天干丙火弱（离衰），地支双亥子水旺（坎旺）。全局的关键是日支寅木——这个寅木是"升"的枢纽，它扎根在地支水中，把水气向上引导（地气上升），供给天干的丙火。

前两运庚子、辛丑，金生水旺，不利。到壬寅、癸卯运，东方木地，"升法"发挥作用——采芹折桂。甲辰、乙巳东南木火运，仕至观察。

### 命造二：天干皆水、地支皆火——解法和法

【命造二】乾造：癸巳 壬戌 壬午 壬寅

大运：辛酉 → 庚申 → 己未 → 戊午 → 丁巳 → 丙辰
（出自任铁樵《滴天髓阐微·坎离》）

**格局**：从财格。**用神**：金（解法——金运以解之）。**喜**：金水运。**忌**：火土运。

**结论**：此造"天干皆水（壬、癸），地支皆火（巳午戌成火局）"。天干壬癸水三透，地支巳午戌三合火局——水在上、火在下，各行其道但无交汇，需要"解之"。用什么解？金（庚辛申酉）。为什么是金？因为金能生水（给水续命），同时火虽然克金但金能"传导"能量（金在中间缓冲水火）。

初运辛酉、庚申，正是金运——"解"法得用，丰衣足食。一交己未（土运），土克水、土晦火，刑耗异常。戊午运财杀并旺（火土之势冲天），出外遇盗丧身。

### 命造三：水火交战于干支——制法

【命造三】乾造：壬子 丙午 壬子 丙午

大运：丁未 → 戊申 → 己酉 → 庚戌 → 辛亥 → 壬子
（出自任铁樵《滴天髓阐微·坎离》）

**格局**：水火交战。**用神**：视强者而制之（午火当令则制午火）。**喜**：金运。**忌**：木火运。

**结论**：此造水火交战——壬子水（纯水）对丙午火（纯火），天干壬丙冲、地支子午冲。火当令（午月生）、水休囚。任铁樵说"喜其无土，日主不克"——没有土来混战，还有一线生机。初运丁未，逢戊午年天克地冲，父母双亡、流为乞丐。申运（戊申己酉），金运制火（制强者），好转发财娶妻。

## 六、常见误解与辨析

### 误解一：坎离就是简单的水火平衡

**错误说法**：有人认为坎离篇就是讲水火要平衡——水多了加火、火多了加水。

**经典原文反驳**：

【原注】"天干皆水，地支皆火，为交媾，交媾身强则富贵；天干皆火，地支皆水，为交战，交战身弱，岂能富贵？"

**正确理解**：坎离篇讲的不只是"平衡"。水在天干、火在地支是**交媾**（吉），但同样是水与火，反过来变成天干火、地支水就是**交战**（凶）。"平衡"不是最重要的，"位置"和"方向"才是。水在上火在下是自然之道，火在上水在下是逆天之道——这是坎离篇的核心洞见，不是简单的"缺啥补啥"。

### 误解二：五法可以用来法所有水火命局

**错误说法**：遇到水火的命局，任选五种方法之一就能解决。

**经典原文反驳**：

【任氏曰】"此五者，坎离之作用如此，则无独成之势，而有相持礼智之性矣。"

**正确理解**：五法各有适用范围，不可混淆。升法只适用于"天干离衰、地支坎旺"的场景；降法只适用于"天干坎衰、地支离旺"的场景。用错了方法，效果适得其反。比如地水火旺需要降法（用金），你却用了升法（用木），木生火只会让火更旺。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"坎离宰天地之中气，成不独成，而有相持者在"**——水火必须相持（互相制衡）而不能独成。
2. **"天干皆水，地支皆火，为交媾，交媾身强则富贵；天干皆火，地支皆水，为交战"**——位置决定吉凶的关键论断。
3. **坎离五法口诀**："升者地有木则地气上升，降者天有金则天气下降，和者木运以和之，解者金运以解之，制者视强者而制之。"
4. **"不可独成，必要相持"**——水火不能独自存在，必须互相成就。

### 7.2 与其他篇章的关联脉络

- **前序关联**：**震兑（33）**——震兑五法（攻成润从暖）与坎离五法（升降和解制）并列对照，震兑谈金木、坎离谈水火
- **直接关联**：**寒暖（29）**——寒暖的温度判断是坎离水火分析的基础
- **关联篇目**：**燥湿（30）**——燥湿的水分判断与坎离水火调候互为补充

### 7.3 进阶学习路径建议

坎离篇之后推荐顺序：

1. **从象（46）**——坎离"从"的思路延伸，深入"从格"理论（专题级）
2. **合局（53）**——水火既济中的"合"（如寅午戌合火、申子辰合水）的深层理论（专题级）
3. **战局（52）**——水火交战之后的格局分析与处理（专题级）

建议将震兑（33）+ 坎离（34）连读，建立完整的四象调候知识体系——金木有攻成润从暖五法，水火有升降和解制五法，对照学习效果最佳。之后可进入下篇六亲论各篇，按需学习。
