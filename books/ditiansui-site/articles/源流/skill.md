## 功能

在完成衰旺判定和月令分析后，对命局的五行之气进行更深层的"源流"分析。检查日主（或命局中心五行）的长生之位是否完整、根基是否稳固，判断命局的源深源浅，以及空亡对五行根基的影响。此技能揭示的不是"日主强不强"（衰旺层面），而是"根基深不深"（源流层面），适用于八字分析的进阶阶段。

## 输入

- **四柱八字**：年柱、月柱、日柱、时柱对应的天干地支（字符串，格式如"甲子 丙寅 戊午 庚申"）
- **性别**：乾造（男）或坤造（女）（可选）
- **前期分析结果**（可选）：日主衰旺判定、月令分析等前期分析结果

## 处理逻辑

1. **长生之位定位**
   a. 确认日主五行对应长生之位：
      - 木日主 → 长生在亥
      - 火日主 → 长生在寅
      - 金日主 → 长生在巳
      - 水/土日主 → 长生在申
   b. **依据**：源流篇【任氏曰】"生方即长生之方，如木生亥，火生寅，金生巳，水土生申是也"

2. **长生之位完整性判定**
   a. 检查四柱中是否出现了日主的长生地支
   b. 如果出现，检查该地支是否被冲（如亥见巳冲、寅见申冲）
   c. 如果被冲 → "居影堂空"，根基受损
   d. 如果未出现被冲且地支完好 → 根基完整
   e. **依据**：源流篇【任氏曰】"居于影堂空者，如木之根在亥，而亥被冲破；火之根在寅，而寅被申冲是也"

3. **源深源浅判定**
   a. 长生之位完整且无冲克 + 日主有根 + 生扶得力 → 源深
   b. 长生之位缺失或被冲克 + 日主无根或少根 → 源浅
   c. **依据**：源流篇【总结句】"源深则流长，源浅则流短"

4. **根气状态判定**
   a. 检查日主的根气（禄位、旺位、余气位）状态
   b. 根被冲克或被合化 → 根枯气泄
   c. 根完好且得生扶 → 根盛气足
   d. **依据**：源流篇【原注】"气泄根枯者，元神已败也"；【总结句】"根枯则气泄，根盛则气足"

5. **空亡判定**
   a. 根据日柱天干所在旬，确定空亡地支
   b. 检查日主对应的根基地支是否逢空亡
   c. 如果逢空 → "空则不实，亡则无根"
   d. **依据**：源流篇【任氏曰】"十干不到之方，谓之空亡。空则不实，亡则无根。如甲以子为根，子空则甲无所托；乙以丑为根，丑空则乙无所凭"

6. **综合评级**
   a. 综合长生、根气、空亡三项，给出源流评级（源深流长 / 源浅流短 / 根枯气泄 / 根盛气足）
   b. **依据**：源流篇【总结句】"当先明五行之源流，辨其衰旺生死，而后方可言用神也"

## 输出

```typescript
interface YuanLiuReport {
  /** 日主基础 */
  dayMaster: {
    stem: string;
    element: string;
    changSheng: string;         // 长生之位的地支
  };
  /** 长生之位状态 */
  changShengStatus: {
    present: boolean;           // 长生之位是否在四柱中出现
    location: string;           // 出现在哪一柱（年/月/日/时）
    isChonged: boolean;         // 是否被冲
    is完整: boolean;            // 是否完整无损
    description: string;
  };
  /** 源深源浅判定 */
  depth: {
    level: "源深" | "源浅" | "中平";
    assessment: "流长" | "流短" | "中平";
    description: string;
  };
  /** 根气状态 */
  rootQi: {
    status: "根盛气足" | "根枯气泄" | "中平";
    rootBranches: string[];     // 提供根气的地支列表
    damagedRoots: string[];     // 受损的根气
    description: string;
  };
  /** 空亡判定 */
  emptiness: {
    emptyBranches: string[];    // 空亡地支
    affectedStems: string[];    // 受空亡影响的天干
    isCritical: boolean;        // 是否涉及根基性空亡
    description: string;
  };
  /** 综合评级 */
  overall: {
    rating: "源深流长" | "源浅流短" | "根枯气泄" | "根盛气足" | "中平";
    keyInsight: string;         // 基于源流篇的关键发现
    usabilityNote: string;      // 对用神选取的影响说明
  };
}
```
