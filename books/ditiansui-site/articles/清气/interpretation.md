# 《滴天髓阐微·清气》解读

> 篇目编号：通论 | 难度分级：进阶级
> 原文来源：`articles/清气/source.md`

## 一、入门导读

### 1.1 这一篇在学什么

"清"是八字命局的一种理想状态——所有五行各安其位、生克有序、目标一致。清气的命局如同一幅布局精密的画，每一笔都有它的位置和作用。本篇教你如何判断一个命局是"清"还是"浊"，以及如何从看似混杂的命局中"澄浊求清"——通过岁运或争神来净化格局。

### 1.2 本篇在整个子平体系中的地位

清浊理论是贯穿《滴天髓》全书的评价标准，与浊气篇为姊妹篇。前面各章（八格、官杀、伤官等）教的是"怎么分析具体格局"，清浊两篇教的是"怎么评价格局的高低好坏"。清浊的判断标准也用于验证官杀格、伤官格、通关等分析是否到位——格局对了但不清，仍然不是好命。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **理解"清"的本质**——清不是五行单一，而是各神各安其位、各有作用
2. **判断命局之"清"**——能够识别"清而有精神"和"清而无气"的区别
3. **掌握"澄浊求清"的方法**——知道如何在杂乱的格局中清除浊气
4. **理解"精神"的含义**——清而不枯、有精神才是真的清
5. **将清浊分析应用到具体格局**——在官杀、伤官等格局中评估格局的清浊

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
一清到底有精神，管取生平富贵真。澄浊求清清得去，时来寒谷也回春。

【原注】
清者不徒一气成局之谓也。如正官格，身旺有财，身弱有印，并无伤官七杀杂之，纵有比肩食神财煞印绶杂之，皆循序得所，有安顿，或作闲神，不来破局，乃为清奇。又要有精神，不为枯弱者佳。

浊非五行并出之谓。如正官格，身弱混之以煞，混之以财，以食神杂之，不能伤我之官，反与官星不和；以印绶杂之，不能扶我之身，反与财星相戕，俱为浊。或得一神有力，或行运得所，以扫其浊气，冲其滞气，皆为澄浊以求清，皆富贵命矣。

【任氏曰】
命之最难辨者，清浊两字也。此章所重得，"澄浊求清"四字也。

清而有气，则精神贯足；清而无气，则精神枯槁。精神枯即邪气入，邪气入则清气散，清气散则不贫即贱矣。

夫清浊者，八字皆有也，非正官一端而论也。

如正官格，身弱有印，忌财，财星不现，清可知矣。即使有财，不可便作浊论，须要看其情势。如财与官贴，官与印贴，印与日主贴，则财生官，官生印，印生身，印之源头更长矣，至行运再助其印绶，自然富贵矣。即使无财，不可便作清论，亦要看其情势，或印星无气，与官星不通，或印星太旺，日主枯弱，不受印星之生；或官星贴日，印星远隔，日主先受官克，印星不能生化，至行运再逢财官，不贫亦夭矣。

如正官格，身旺喜财，所忌者印绶，伤官其次也。亦看情势，如伤官与财贴，财与官贴，官与比肩贴，不特官星无碍，抑且伤官化劫生财，财生官旺，官之源头更长，至行运再遇财官之地，名利两全矣。如伤官与财星远隔，反与官星紧贴，财不能为力，至行运再遇伤官之地，不贫亦贱矣。如伤官在天干，财星在地支，必须天干财运以解之；伤官在地支，财星在天干，必须地支财运以通之。或财官相贴，而财神被合神绊住，或被闲神劫占，亦须岁运冲其合神，制其闲神，皆为澄浊求清。虽举正官而论，八格皆同此论。

总之喜神宜得地逢生，与日主紧贴者佳；忌神宜失势临绝，与日主远隔者美。日主喜印，印星贴身，或坐下印绶，此即日主之精神也；官星贴印，或坐下官星，此即印绶之精神。余可类推。
```

### 2.2 原文逐句精解

【原句】"一清到底有精神，管取生平富贵真。澄浊求清清得去，时来寒谷也回春。"

→ 解读：全篇主旨的四句诗。意思是：如果一个命局从头到尾清纯贯通、又有精神气，那必然是富贵命。即使命局有浊气，只要能澄浊求清（清除掉浊气），时机到来时连寒冷的山谷也能回春转暖。

【原注第1段】"清者不徒一气成局之谓也。如正官格，身旺有财，身弱有印，并无伤官七杀杂之，纵有比肩食神财煞印绶杂之，皆循序得所，有安顿，或作闲神，不来破局，乃为清奇。"

→ 解读：原注对"清"的定义非常关键——清不等于五行单一。清的根本是"各神各有安顿"，每个十神各在其位、各行其是。即使比肩食神财煞印绶都有，只要"循序得所"——即生克顺序合理——也是清。那些不参与核心生克的"闲神"，只要不破局，也算清。

【原注第2段】"浊非五行并出之谓。如正官格，身弱混之以煞，混之以财……俱为浊。"

→ 解读：浊也不是指五行种类多少。真正的浊是指无用之神破坏了有用之神的运作——比如身弱需要印，但财却来坏印；身弱怕杀，但杀混在其中破坏官星。关键是"不能伤我之官，反与官星不和"，招来了无用甚至有害的五行。

【原注第3段】"或得一神有力，或行运得所，以扫其浊气，冲其滞气，皆为澄浊以求清。"

→ 解读：原注给出两种"澄浊求清"的方案：一是原局中有一神"有力"（可能是冲、合、克等作用）能够扫除浊气；二是依靠岁运清除浊气。只要能做到"澄浊求清"，命局仍可富贵。

【任氏曰第1段】"命之最难辨者，清浊两字也。此章所重得，'澄浊求清'四字也。"

→ 解读：任铁樵开门见山：清浊是八字中最难判断的概念。本章的核心是"澄浊求清"——不是静态地判断清还是浊，而是动态地看能否清除浊气。

【任氏曰第2段】"清而有气，则精神贯足；清而无气，则精神枯槁。精神枯即邪气入，邪气入则清气散，清气散则不贫即贱矣。"

→ 解读：清有两种：清而有精神（有气）、清而无精神（枯槁）。清而枯比浊更差！因为清枯意味着全局缺乏生机，邪气入侵时连抵御之力都没有。

【任氏曰第3段】"夫清浊者，八字皆有也，非正官一端而论也。"

→ 解读：清浊适用于所有格局类型，不仅仅是正官格。八格皆可论清浊。

【任氏曰第4段-宏观论清】"如正官格，身弱有印……即使有财，不可便作浊论，须要看其情势。"

→ 解读：任铁樵以正官格为例，展示清浊的分析方法。关键在于看各神的位置和"情势"（相互关系）：
- 身弱用印 → 财星如果排在官、印之后，且生官、官生印、印生身 → 这是清而非浊
- 如果有印但印无气、与官不通、或日主弱不受生 → 即使没有财星，也是"清而枯"而非真清

【任氏曰第5段-宏观论浊】"如正官格，身旺喜财……亦看情势……如伤官在天干，财星在地支，必须天干财运以解之……"

→ 解读：身旺用财的格局中，判断浊与清的视角不同。伤官化劫生财、财生官、官护财——这是好的清。如果伤官与财不呼应（一个在天干一个在地支），则需要天干的财运来"通"。这些都是澄浊求清的具体方法。

【任氏曰总结】"总之喜神宜得地逢生，与日主紧贴者佳；忌神宜失势临绝，与日主远隔者美。日主喜印，印星贴身，或坐下印绶，此即日主之精神也；官星贴印，或坐下官星，此即印绶之精神。"

→ 解读：任铁樵给出了清浊判断的终极标准：
- **喜神**：应当得地、逢生、与日主紧贴——这叫有"精神"
- **忌神**：应当失势、临绝、与日主远隔
- **精神传递链**：官星→印星→日主每个环节都紧贴，则精神贯足

## 三、核心义理深度解析

### 3.1 清的本质——各安其位而非五行单一

**原文依据**

【原注】"清者不徒一气成局之谓也。纵有比肩食神财煞印绶杂之，皆循序得所，有安顿，或作闲神，不来破局，乃为清奇。"

**义理解读**

很多人把"清"理解为五行纯一（如一个命局只有两三种五行），但原注明确指出：清不是五行数量问题。关键在于两个标准：
1. **循序得所**——各十神在生克链条中位置合理，生入有用、克入所需
2. **闲神不破局**——不参与核心生克的五行，只要不破坏整体格局，不影响清

举例：一个正官格，有财生官、有印护身、有比肩帮身，虽然十神种类很多，但只要各安其位、生克有序，就是清。

**命造验证**

乾造：癸酉 甲子 丙寅 乙未

此造丙生子月，官星当令（癸水正官透于年干），坐下长生（寅木印星），印透根深（甲乙木透干）。官生印、印生身，一气贯通。即使有癸水正官、甲乙木印星、未土等五行并出，但"一清到底有精神"——各神位置合理、生克有序、目标一致（木火通明）。金水运中登科发甲，名高翰苑。

### 3.2 澄浊求清——动态清除浊气的方法论

**原文依据**

【原注】"或得一神有力，或行运得所，以扫其浊气，冲其滞气，皆为澄浊以求清。"

【任氏曰】"如伤官在天干，财星在地支，必须天干财运以解之……或财官相贴，而财神被合神绊住，或被闲神劫占，亦须岁运冲其合神，制其闲神，皆为澄浊求清。"

**义理解读**

澄浊求清是清浊理论的实践核心。不是"浊就是坏"，而是要看：
1. 原局中是否有一神"有力"能扫除浊气
2. 岁运中能否清除浊气

具体方法包括：
- **天干地支不通** → 需天干财运或地支财运来通
- **财神被合** → 岁运冲其合神
- **闲神劫占** → 岁运制其闲神

**命造验证**

乾造：癸未 甲子 丙寅 丁酉

此造与前癸酉者大同小异——前则官坐财地（癸酉），此则官坐伤地（癸未）。兼之子未相贴，天干之官受克，地支之官亦伤。更嫌劫入财乡，所谓"财劫官伤"——这就是浊气。但辛酉庚申运干支皆财，财星扫除浊气——"澄浊求清清得去"，家业丰裕。而一交己未，财星失势，浊气复来，家业大破。此即」澄浊求清"依赖岁运的活例。

## 四、命造实例拆解

### 命造一：一清到底——官印相生，精神贯足

【命造一】乾造：癸酉 甲子 丙寅 乙未

大运：癸亥 → 壬戌 → 辛酉 → 庚申 → 己未 → 戊午
（出自《滴天髓阐微·清气》）

**格局**：正官格，**用神**取甲木印星（官生印、印生身），**喜**木火运，**忌**土运（泄火气）。

**逐字分析**：
- 年柱癸酉：癸水正官当令，酉金为正财生官
- 月柱甲子：甲木印星透干，子水正官当令
- 日柱丙寅：丙火日主坐寅木长生
- 时柱乙未：乙木印星再透，未为木库

**结论**：此造官当令（子月），印透根深（甲寅乙未），日主得生得地。官→印→身的链条完整，各神各安其位。年干癸水生月干甲木，甲木生日主丙火，时干乙木再助——生克链条一气呵成。任铁樵评价"一清到底有精神"，这正是清气的最好例证。惜中运火土，以致终老于词林。

### 命造二：澄浊求清——官受财伤，岁运来清

【命造二】乾造：癸未 甲子 丙寅 丁酉

大运：癸亥 → 壬戌 → 辛酉 → 庚申 → 己未 → 戊午
（出自《滴天髓阐微·清气》）

**格局**：正官格，但官坐伤地（癸未），官星受损。**用神**取财星（辛酉庚申运扫除浊气）。

**逐字分析**：
- 年柱癸未：癸水正官，未土伤官暗伤官星
- 月柱甲子：甲木印星，子水正官
- 日柱丙寅：丙火日主坐长生
- 时柱丁酉：丁火劫财，酉金正财

**结论**：此造的"浊"在于：子未相贴，官受土伤；更兼时柱丁火劫财占据财乡。官印链条断损——官星有损、印星被泄。但辛酉庚申运中干支皆财，财扫浊气，"澄浊求清清得去"，家业丰裕。己未运浊气复来而家破。此造的精妙在于展示了清浊的动态性——浊气可以暂去、也可以复来，关键在于岁运能否持续支持。

## 六、常见误解与辨析

### 误解一：清就是五行种类少、八字纯净

**错误说法**：很多人认为清气的命局就是五行的种类少，比如两行成象、三行成象等。

**经典原文反驳**：

【原注】"清者不徒一气成局之谓也。纵有比肩食神财煞印绶杂之，皆循序得所，有安顿，或作闲神，不来破局，乃为清奇。"

**正确理解**：原注明确指出"不徒一气成局"——清不仅仅是五行的纯净。即使十神种类繁多，只要各安其位、生克有序，也是清。反过来说，即使只有两三种五行，如果位置错乱、生克矛盾，也是浊。清的本质是秩序，而非单纯。

### 误解二：浊的命局一定贫贱

**错误说法**：见到命局有浊气就断定是贫贱之命。

**经典原文反驳**：

【原注】"或得一神有力，或行运得所，以扫其浊气，冲其滞气，皆为澄浊以求清，皆富贵命矣。"

**正确理解**：浊气可以通过"澄浊求清"来清除——原局有有力之神、或岁月清除浊气，都可以化浊为清。很多富贵命恰恰是经历了澄浊求清的过程。浊并不是终局判决，而是需要动态评估的中间状态。

### 误解三：清枯比浊好

**错误说法**：有人认为"清"总是好的，"浊"总是差的。所以清枯的命比浊命好。

**经典原文反驳**：

【任氏曰】"清而无气，则精神枯槁。精神枯即邪气入，邪气入则清气散，清气散则不贫即贱矣。"

**正确理解**：任铁樵明确指出：清而枯（清却没有精神气）比浊更差。因为清气枯槁意味着全局缺乏生机，一旦邪气入侵（岁运不吉），连抵抗之力都没有。浊命尚有发力清除的可能，清枯命连发力的基础都没有。宁可清中浊，不可清中枯。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"一清到底有精神，管取生平富贵真"**——有精神、一清到底才是真富贵。
2. **"澄浊求清清得去，时来寒谷也回春"**——浊可清除，时机到自然转吉。
3. **"清者不徒一气成局"**——清的本质是各神各安其位，而非五行单一。
4. **"命之最难辨者，清浊两字也"**——清浊是八字中最难判断的。
5. **"喜神宜得地逢生，与日主紧贴者佳；忌神宜失势临绝，与日主远隔者美"**——清浊判断的终极标准。
