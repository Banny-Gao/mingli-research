## 功能

在完成三元拆解（天道技能）和五行偏全评估（坤道技能）之后，对八字中天干地支之间的"顺"与"悖"关系进行逐个柱位与全局的分析。此技能的核心产出是：判断各柱之间的生克关系是有情（顺吉）还是无情（悖凶），并识别命局中是否存在"层层递减"式的救应路径。

## 输入

- **四柱八字**：年柱、月柱、日柱、时柱（字符串）
- **三元分析结果**（可选）：来自天道技能（tiandao）的三元分析输出
- **五行偏全评估**（可选）：来自坤道技能（kundao）的五气偏全分析输出
- **大运流年**（可选）：用于判断岁运引发的顺悖变化

## 处理逻辑

1. **单柱顺悖判定**（对每一柱执行）
   a. 判断天干与同柱地支之间的关系：
   - 天干生于地支 / 地支生于天干 / 天干同地支 / 天干克地支 / 地支克天干
     b. 判定标尺：
   - 天干弱 + 地支生天干 → 顺（有情），标注为"+"
   - 地支弱 + 天干生地支 → 顺（有情），标注为"++"
   - 天干弱 + 地支克天干 → 悖（无情），标注为"--"
   - 地支弱 + 天干克地支 → 悖（无情），标注为"-"
     c. **依据**：人道篇【任氏曰】"如天干气弱，地支生之……皆为有情而顺则吉；如天干气衰，地支抑之……皆为无情而悖则凶"

2. **全局顺悖扫描**
   a. 统计各柱的顺悖得分（顺 +1，悖 -1，中性 0）
   b. 识别直接的生克冲突（如甲木坐申金）
   c. 识别间接生克链条（如甲己合土→土生金→金克木的间接克）
   d. **依据**：人道篇【任氏曰】"若天干无庚辛，而反遇之以甲己……皦助甲己之金，母子无生意"

3. **救应路径筛查**
   a. 对标记为"悖"的柱位，按"层层递减法"四层筛查：
   - 第1层：是否有第三方五行化解（如金克木时，有水转化）
   - 第2层：是否有印绶通变（如正偏印提供生扶）
   - 第3层：是否有通根支撑（如地支有日主根基）
   - 第4层：无救应 → 直接判定为凶
     b. **依据**：人道篇【任氏曰】"假使干是木，畏金之克，地支有亥子水生之；支无亥子，无印绶化之；干无印绶，地支有淫箔以通根……余可类推"

4. **综合顺悖评分**
   a. 计算全局顺悖指数 = (顺的柱数 - 悖的柱数) / 总柱数 × 100
   - 指数 > 50 → 整体偏顺（吉）
   - 指数 -50 ~ 50 → 顺悖平衡（需结合大运）
   - 指数 < -50 → 整体偏悖（凶）
     b. **依据**：人道篇【原句】"顺则吉兮悖则凶"
