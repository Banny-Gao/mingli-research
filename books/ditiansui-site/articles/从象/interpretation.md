# 《滴天髓阐微·从象》解读

> 篇目编号：46 | 难度分级：专题级
> 原文来源：`articles/从象/source.md`

## 一、入门导读

### 1.1 这一篇在学什么

从象讲的是八字中"从格"的判断法则。当一个命局中日主极弱无根、满盘都是财官印食等强势五行时，日主无力自立，只能放弃自身立场、顺从强者之势，这种格局称为"从格"。本篇从日主有无根基这个核心出发，教你辨认什么情况下日主必须从、什么情况下是假从。

### 1.2 本篇在整个子平体系中的地位

从象是滴天髓下篇"象局"组（第46-57篇）的开端篇，与化象（第47篇）并列，分别论述"从"与"化"两种特殊格局。假从（第48篇）与假化（第49篇）是前两篇的延伸，讨论从格与化格不纯粹时的判定。四篇连读形成"真从→真化→假从→假化"的完整框架。此外，从格还与八格篇（第12篇）中任铁樵提到的"从财、从官杀、从食伤、从强、从弱、从势"六种变格相互印证。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **判定日主能否独立**——通过检查地支有无日主之根（禄、旺、余气），判断日主是否"无援"
2. **识别从格的基本条件**——日主无根 + 满局财官（或满局一种强势五行）= 从格
3. **区分真从与假从**——真从无条件顺从，假从有杂神干扰
4. **辨识从格的三大类型**——从财、从官杀、从食伤（以及从旺、从势）

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
从象者，日主无根，不得不从也。真从者贵，假从者贱，此从格之辨也。

【原注】
从象者，日主无援，不得不从也。从真则贵，从假则贱，不可不辨也。

【任氏曰】
从象之论，须观日主之根有无也。日主无根，而满局财官，不得不从，此从象也。真从者富贵，假从者凶夭。
```

### 2.2 原文逐句精解

【原句】"从象者，日主无根，不得不从也。真从者贵，假从者贱，此从格之辨也。"

→ 解读：开篇即点出"从"的核心——日主没有根基（地支中没有日主五行出现的禄位、旺位或余气位），在命局中完全弱势，没有能力与强势五行对抗，所以只能"从"——顺从、跟随强者。接着给出价值判断：真从的命格尊贵，假从的命格卑贱，这是判断从格的关键分界。

→ 这里"从"的含义你要理解为：在八字命局中，各种五行力量不可能和平共存，当某一种（或某几种）五行力量绝对压倒日主时，日主如果还硬撑，就是自取灭亡。所以从格的本质是"顺势而为"——日主放弃自我，完全跟随强势五行。

【原注】"从象者，日主无援，不得不从也。从真则贵，从假则贱，不可不辨也。"

→ 解读：原注把"无根"换成了"无援"——日主不仅没有根基，而且没有援助力量。这个"援"字很精妙，它包括了比劫帮身、印绶生身等所有可能帮助日主的力量。如果日主既无根又无援，那就真的"不得不从"了。原注特别强调：从真和从假，一贵一贱，"不可不辨"——一定要区分清楚。

【任氏曰】"从象之论，须观日主之根有无也。日主无根，而满局财官，不得不从，此从象也。真从者富贵，假从者凶夭。"

→ 解读：任铁樵把判断方法进一步具体化——"须观日主之根有无"，也就是先检查日主在地支中有没有根。他说"满局财官"——这是最常见的从格类型。但要注意，任氏说的是"例如财官"，并非只有从财官才算从格。从食伤、从旺、从势都是从格的子类型。

→ 任氏把"从真者贵"修正为"真从者富贵"（富贵兼备），把"从假者贱"修正为"假从者凶夭"（凶险短命），更准确地区分了两者的差距。注意，他的"凶夭"比原注的"贱"更严厉——假从不仅贫贱，还可能有凶祸、寿命不长。

## 三、核心义理深度解析

### 3.1 什么是"从"——日主无根，不得不从

**原文依据**

【原注】"从象者，日主无援，不得不从也。"

【任氏曰】"日主无根，而满局财官，不得不从，此从象也。"

**义理解读**

"从"的字面意思是跟从、顺从。在八字命理中，它描述的是这样一种格局状态：日主（代表命主自身）在命局中处于绝对弱势，地支中找不到日主五行的根基，同时又缺乏比劫、印绶的帮扶，而命局中的其他五行（财、官、食伤等）却极旺，形成压倒性的优势。

这时候，日主只有两种选择：
- **抗争**——但无根无援，抗争就是自取灭亡，对应的是凶命
- **顺从**——放弃自我，跟随最强的一方，对应的是从格

从格的主动态是"顺势"——不是日主想反抗但反抗不了，而是日主本身就没有独立意志，完全融入强势阵营。打个比方：一团棉花扔进水里，棉花不是"被迫"浸湿，而是它本身就是吸水的材质，天然与水融合。

**命造验证**

命造：庚戌 甲申 甲戌 乙丑（出自《滴天髓阐微·震兑》）

甲木生于申月（孟秋），天干虽有甲乙木三透，但地支全无木根——戌为火土、申为金、丑为湿土。所谓"天干三透甲乙，而地支不载"，甲木在地元全无根基。满局金旺当令（申金正当时令）、财杀并旺（庚金七杀、戌丑财星），甲木无根无援，只能从金杀之势。任铁樵断为"格成从杀"，这就是从格。

### 3.2 真从与假从——日主有无残根是关键

**原文依据**

【原注】"从真则贵，从假则贱，不可不辨也。"

【任氏曰】"真从者富贵，假从者凶夭。"

**义理解读**

真从与假从的分界只有一个条件：**日主在地支中有没有残根**。

- **真从**：日主在地支中完全无根（没有禄、旺、余气），也没有印绶生扶。日主放弃得彻底，反而能获得顺从之神的全部力量，所以"富贵"。
- **假从**：日主在地支中有一点点微根（比如余气墓库），或者有一两个杂神干扰，没有完全顺从。这种"不彻底"反而坏事——从得不够干净，既得不到强势阵营的全力支持，自身那点残根又不够自立，所以"凶夭"或"贱"。

任铁樵在原注基础上进一步区分：真从是"富贵"（既富且贵），假从是"凶夭"（凶险短命）。这个差异说明，从格讲究的是"纯粹"——要么坚决地从，干干净净获益；要么就不从。半从半不从反而最糟糕。

**命造验证**

真从之例：庚戌 甲申 甲戌 乙丑（同上命造）

此造甲木日主在地支申、戌、丑中全无木之禄旺，天干虽有甲乙透出，但地支无根——天干只是外显之象，真正的根基在地支。甲木的残根全无，只能彻底从金杀之势。任铁樵断为从杀格，此人"戌运武甲出身"——从格真者，行运助从神则发迹。

### 3.3 从格的几种类型

**原文依据**

【任氏曰】"满局财官，不得不从"——任氏以财官为例说明从格，但在八格篇中扩展为六种变格："曰从财，曰从官杀，曰从食伤，曰从强，曰从弱，曰从势。"

**义理解读**

从格不是只有"满局财官"一种，根据命局中强势五行的不同，可以分为：

| 类型 | 条件 | 特点 |
|------|------|------|
| **从财** | 日主无根 + 满局财星（土/金/水等） | 从财者富，但要看财星是否得令 |
| **从官杀** | 日主无根 + 满局官杀（金克木、火克金等） | 从杀者贵，官杀旺则权柄在手 |
| **从食伤** | 日主无根 + 满局食伤（木火、火土等） | 从食伤者以才艺闻名 |
| **从旺** | 日主极旺 + 全局归旺 | 实际是"顺局"而非严格意义的"从" |
| **从势** | 日主无根 + 两三种五行混杂皆旺 | 从势者灵活多变，视旺者而定 |

在这几种中，从财和从官杀最常见，也是最正统的从格。任铁樵说的"满局财官"正是这两类的代表。

## 四、命造实例拆解

### 命造一：从杀格——甲木无根从金杀

【命造一】乾造：庚戌 甲申 甲戌 乙丑

大运：乙酉 → 丙戌 → 丁亥 → 戊子 → 己丑 → 庚寅
（出自《滴天髓阐微·震兑》）

| 柱 | 干支 | 天元分析 | 地元分析 | 人元分析 |
|----|------|---------|---------|---------|
| 年 | 庚戌 | 庚金七杀攻身 | 戌土为财 | 辛丁戊（金伤官、火食神、土偏财） |
| 月 | 甲申 | 甲木比肩帮身 | 申金为杀之令 | 庚壬戊（金七杀、水偏印、土偏财） |
| 日 | 甲戌 | 甲木日主 | 戌土为财 | 辛丁戊 |
| 时 | 乙丑 | 乙木劫财帮身 | 丑土为财 | 己癸辛（土正财、水正印、金正官） |

**格局**：从杀格。日主甲木地支无根（申戌丑中全无木气），天干虽有三木透出，但在地元全无支撑，木虚浮无根。月令申金当权，庚金透干为杀，全局金土成势，日主不得不从。

**用神**：金（官杀）为用神，土（财）为喜神——从杀格以杀为用，财星生杀助势。

**喜忌**：喜金土运（助从神之势），忌水木运（生木助日主，破坏从格）。

**结论**：此造为真从格——甲木日主地支无一丝木根，天干虽透三木但全是虚浮之象。月令申金当权，庚金透干为杀，全局金土成势。任铁樵断"格成从杀"，戌运"武甲出身"，丁亥运"生木克金"反而不利，戊子己丑运"财生杀旺"又复起——从格之关键就在"从"得坚决，运助从神则发，运助日主则败。

### 命造二：从财格——壬水无根从财势

【命造二】坤造：戊申 甲寅 壬寅 丁未

大运：癸丑 → 壬子 → 辛亥 → 庚戌 → 己酉 → 戊申
（出自《滴天髓阐微·女命》）

| 柱 | 干支 | 天元分析 | 地元分析 | 人元分析 |
|----|------|---------|---------|---------|
| 年 | 戊申 | 戊土七杀 | 申金为印 | 庚壬戊（金偏印、水比肩、土七杀） |
| 月 | 甲寅 | 甲木食神 | 寅木为食神之根 | 甲丙戊（木食神、火偏财、土七杀） |
| 日 | 壬寅 | 壬水日主 | 寅木为食神 | 甲丙戊 |
| 时 | 丁未 | 丁火正财 | 未土为财 | 己丁乙（土正官、火正财、木伤官） |

**格局**：从财格。壬水生于寅月（孟春），地支寅、寅、未全无水木之根。天干透甲木食神、丁火正财，木火成势。戊土七杀透干，但食神制杀。全局木火土三行偏旺，日主壬水无根无援，"见其财星有势，自然从财而去"。

**用神**：木（食伤）生火（财），以火财为用。

**喜忌**：喜木火运（助财势），忌金水运（生日主破从格）。

**结论**：此造壬水无根，木火土三行成势。任铁樵断其"从财而去"——女子之命，背夫弃子，从财势而走。这正是从格的实践验证：从格不论吉凶（并非从格就一定是好命），而是论日主"从"了什么、后果如何。从财格财富丰厚，但在女命中以财为夫星，从财而去意味着夫星失位。

|:-----------:|:-----------:|:-----------:|

## 六、常见误解与辨析

### 误解一：身弱就是"从格"

**错误说法**：很多人看到日主身弱、财官旺，就直接说这是"从格"。

**经典原文反驳**：

【原注】"从象者，日主无援，不得不从也。"

【任氏曰】"日主无根，而满局财官，不得不从，此从象也。"

**正确理解**："身弱"和"从格"是完全不同的概念。身弱是日主衰弱但还有根（比如地支有禄、有印），需要印比扶助——这叫"扶抑格"。从格是日主无根（地支中完全没有日主五行的根基），满局都是其他五行，日主无力也无援。任铁樵的"根"字是关键——没有根就不能自立，不是弱不弱的问题，是有没有立足之地的问题。身弱有根是"弱而有位"，从格无根是"无位可从"。

### 误解二：从格一定是大富大贵的好命

**错误说法**：遇到从格命断语就是"大富大贵"。

**经典原文反驳**：

【任氏曰】"真从者富贵，假从者凶夭。"

**正确理解**：从格分真假，真假决定贵贱。真从格（日主彻底无根、纯粹顺从强者）确实富贵，但假从格（日主有残根、半从不从）反而凶夭。而且，从格的富贵需要行运配合——运助从神则发，运助日主则破。从格本身不代表终身富贵，要看行运是否能维持从格不被破坏。更关键的是，从格之人往往没有自我意志，"从"意味着放弃自主权——富贵背后是身不由己。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"从象者，日主无根，不得不从也"**——从格的根本条件：日主无根。
2. **"从真则贵，从假则贱"**——真从和假从的贵贱之分。
3. **"须观日主之根有无"**——判断从格的起点：检查日主有没有根。
4. **"满局财官，不得不从"**——最常见的从格类型：日主无根而财官满局。

### 7.2 与其他篇章的关联脉络

- **化象（47）**——从象的姊妹篇，"从"是顺从强者之势，"化"是天干合化变性，两者构成特殊格局的两大分支。
- **假从（48）**——直接从从象延伸，讨论日主有残根时的假从格局。
- **八格（12）**——任铁樵在八格中列出"从财、从官杀、从食伤、从强、从弱、从势"六种变格，与从象互相印证。
- **震兑（33）**——包含从杀命造的实例（庚戌 甲申 甲戌 乙丑），是从格在具体命造中的应用。
- **顺局（50）**——从旺的进一步展开，日主极旺时的"顺"而非"从"。

### 7.3 进阶学习路径建议

1. **化象（47）**——紧接从象之后，学习"化"的法则（专题级，难度与从象相当）
2. **假从（48）**——从象的进阶篇，学习假从与真从的分辨（专题级）
3. **假化（49）**——化象的进阶篇，学习假化与真化的分辨（专题级）
4. **八格（12-26）**——回到正格框架，理解从格在整个格局体系中的定位
