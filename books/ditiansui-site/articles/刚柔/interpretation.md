## 一、入门导读

### 1.1 这一篇在学什么

刚柔篇讲的是阴阳在八字中的平衡之道。阳干（甲丙戊庚壬）为刚、阴干（乙丁己辛癸）为柔——刚柔相济才是命局的"正道"。一个命局如果纯刚无柔（暴戾）或纯柔无刚（懦弱），都是失衡的表现。你要学会在八字中识别刚柔的状态，判断是否需要"以柔济刚"或"以刚济柔"。

### 1.2 本篇在整个子平体系中的地位

刚柔篇承接**真神篇（25）**和**假神篇（26）**的"用神辨析"主题，将阴阳五行的"性"纳入考量。前面的篇章侧重于力量的强弱和真假，而刚柔篇引入了一个新的分析维度——**性格的阴阳属性**。同为木，甲木（阳刚）和乙木（阴柔）的性情和在命局中的角色完全不同。这一点向下延伸到**顺逆篇（28）**（时令的顺逆）和**寒暖篇（29）**（气候的调候），共同构成了"干支心性=力量格局=时令顺逆=气候调候"的四维分析体系。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **识别天干的阴阳刚柔属性**——阳干为刚，阴干为柔
2. **判断命局的刚柔状态**——纯刚、纯柔还是刚柔相济
3. **理解刚柔相济的必要性**——过刚则暴，过柔则弱，平衡才是正道
4. **在具体八字中应用刚柔法则**——判断男女命、日主强弱之外的阴阳平衡维度

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
刚柔者，阴阳之分也。阳刚阴柔，刚柔相济，此命之正道也。
```

【原注】
刚柔者，阴阳之分也。甲丙戊庚壬为阳，刚也；乙丁己辛癸为阴，柔也。阴阳相济，此命之正道也。

【任氏曰】
刚柔之论，乃阴阳之辨也。刚者宜柔济之，柔者宜刚济之，此相济之道也。

### 2.2 原文逐句精解

【原句】"刚柔者，阴阳之分也。"

→ 解读：这就是全篇的立论基础——"刚柔"其实就是"阴阳"在另一个维度上的表达。阳为刚，阴为柔。不是两种不同的东西，而是同一个事物的两面。

【原句】"阳刚阴柔，刚柔相济，此命之正道也。"

→ 解读："相济"的意思不是谁压倒谁，而是互相补足、互相成就。阳刚需要阴柔来调和它的刚猛，阴柔需要阳刚来支撑它的柔弱。八字命局的最高境界不是极刚或极柔，而是刚柔平衡。这就是"命之正道"。

【原注】"甲丙戊庚壬为阳，刚也；乙丁己辛癸为阴，柔也。阴阳相济，此命之正道也。"

→ 解读：原注给出了天干刚柔的具体分类：

- **刚（阳）**：甲木（阳木）、丙火（阳火）、戊土（阳土）、庚金（阳金）、壬水（阳水）
- **柔（阴）**：乙木（阴木）、丁火（阴火）、己土（阴土）、辛金（阴金）、癸水（阴水）

"阴阳相济"的意思是：八字中阳干和阴干应当恰当配合，不要纯阳或纯阴。

【任氏曰】"刚柔之论，乃阴阳之辨也。刚者宜柔济之，柔者宜刚济之，此相济之道也。"

→ 解读：任铁樵进一步发挥——判断刚柔其实就是在判断阴阳。刚（阳）的一方适合用柔（阴）来补济，柔（阴）的一方适合用刚（阳）来补济。这不仅仅是八字本身的分析原则，也是一种人生智慧。任铁樵的"相济之道"强调的是动态平衡：不是各占一半就叫平衡，而是看场合、看需求——缺什么补什么。

## 三、核心义理深度解析

### 3.1 天干阴阳与刚柔的对应关系

**原文依据**

【原注】"甲丙戊庚壬为阳，刚也；乙丁己辛癸为阴，柔也。"

**义理解读**

天干的刚柔不是任意的分类，而是根源于干支理论的阴阳二分法。这种分类在命理分析中至少有以下三个层面的应用：

1. **性情层面**：甲木之刚表现为进取、开拓、担当；乙木之柔表现为灵活、适应、包容。同样是木，甲木和乙木的性情截然不同。
2. **格局层面**：阳日主（甲丙戊庚壬）和阴日主（乙丁己辛癸）在格局取用上有天然差异。阳日主喜用柔（阴干）来调和，阴日主喜用刚（阳干）来支撑。
3. **配置层面**：一个八字如果全是阳干——甲丙戊庚壬齐备——必然性格刚烈、处事激进，但缺少阴柔的调和则易折。反之，如果全是阴干——乙丁己辛癸齐备——则心思细腻、擅长应对，但缺少阳刚的支撑则易靡。

**命造验证**

【真神篇 山东刘中堂造】甲子 丙寅 己丑 甲子

日主己土为阴（柔），天干透出甲甲丙（三阳），地支子寅丑子（寅藏甲丙戊为阳，子藏癸、丑藏己癸辛为阴）。这是一个"以刚济柔"的典范——阴日主（己土）得到重重阳干的扶持（甲木官星为用、丙火印星为相），刚柔相济，所以命主既有阴柔的持重稳重，又有阳刚的进取担当，官至尚书。

### 3.2 相济之道——不是数量上的平均，而是功能上的互补

**原文依据**

【任氏曰】"刚者宜柔济之，柔者宜刚济之，此相济之道也。"

**义理解读**

"相济"这个词非常关键——它不是指"量化平分"，而是"缺什么补什么"。具体来说：

- **当命局偏刚时**：比如日主为阳干（庚金），全局又阳干多透、无阴干调和——此谓"过刚"。过刚则性格刚愎、行事暴戾、容易折断。此时需要阴柔的五行（如乙木、丁火、癸水）来"济"——乙木之柔能让庚金懂得收敛，丁火之暖能让刚金变韧，癸水之润能让燥金降温。
- **当命局偏柔时**：比如日主为阴干（乙木），全局又阴干多透、无阳干支撑——此谓"过柔"。过柔则性格懦弱、优柔寡断、缺乏主见。此时需要阳刚的五行（如甲木、丙火、庚金）来"济"——甲木之撑能让乙木有所依附，丙火之烈能让寒木得到温暖。

**命造验证**

【天道篇命造】庚申 甲申 甲申 庚午

此造甲木日主（阳刚）生于申月秋令，天元重重庚金七杀，全局金气极刚。命局中只有时支午火（丁火，阴柔）为唯一的柔济之力。这就是典型的"过刚"——刚（金）对刚（木）的压制，柔少难以调济。但这一丝午火之柔恰恰就是全盘的救应，也是"以柔济刚"的妙用。

## 四、命造实例拆解

### 命造一：以刚济柔——阴日主的刚柔平衡之道

【命造一】乾造：甲子 丙寅 己丑 甲子

大运：丁卯 → 戊辰 → 己巳 → 庚午 → 辛未 → 壬申
（出自任铁樵《滴天髓阐微·真神》）

| 柱  | 干支 | 天干阴阳   | 地支阴阳           | 刚柔分析 |
| --- | ---- | ---------- | ------------------ | -------- |
| 年  | 甲子 | 甲阳（刚） | 子藏癸阴（柔）     | 刚柔兼有 |
| 月  | 丙寅 | 丙阳（刚） | 寅藏甲丙戊（皆阳） | 纯刚     |
| 日  | 己丑 | 己阴（柔） | 丑藏己癸辛（皆阴） | 纯柔     |
| 时  | 甲子 | 甲阳（刚） | 子藏癸阴（柔）     | 刚柔兼有 |

**格局与刚柔分析**：己土日主（阴、柔）生于寅月，天干透出甲甲丙三个阳干，地支寅为纯阳。这是一个典型的"柔日主得刚扶"的格局——阴柔的日主被阳刚的官印包围，刚柔相济而平衡。天干三阳一阴，地支阴阳交错，整体呈现以刚济柔的态势。

**结论**：此造是"刚柔相济"的典范——阴日主（己土）得阳贵（甲木正官）之扶、得阳印（丙火正印）之生。阴柔为体而得阳刚为用，性格上既稳重内敛（己土之性）又果决担当（甲丙之助）。这正是"柔者宜刚济之"的最佳例证。

### 命造二：过刚失柔——纯阳命局的危机

【命造二】乾造：庚申 甲申 甲申 庚午

大运：乙酉 → 丙戌 → 丁亥 → 戊子 → 己丑 → 庚寅 → 辛卯
（出自任铁樵《滴天髓阐微·天道》）

| 柱  | 干支 | 天干阴阳   | 地支阴阳               | 刚柔分析 |
| --- | ---- | ---------- | ---------------------- | -------- |
| 年  | 庚申 | 庚阳（刚） | 申藏庚壬戊（二阳一阴） | 偏刚     |
| 月  | 甲申 | 甲阳（刚） | 申藏庚壬戊             | 偏刚     |
| 日  | 甲申 | 甲阳（刚） | 申藏庚壬戊             | 偏刚     |
| 时  | 庚午 | 庚阳（刚） | 午藏丁己（二阴）       | 偏柔     |

**格局与刚柔分析**：甲木日主（阳刚）被两重庚金（阳刚）所克，申金月令当权。天干三阳（庚甲庚），地元三申一午——全局阳刚之气极重，唯一的一丝阴柔来自时支午中的丁火。此为典型的"过刚"格局——日主阳刚，七杀亦阳刚，刚与刚相战，缺少阴柔调和。

**结论**：此造揭示了"过刚"的危机——甲木日主虽为阳刚，但被更刚的庚金七杀攻身，刚与刚不相济反相战。唯一的一线生机在时支午火（丁火），以一丁之柔济三申之刚。正因为刚柔严重失衡，此造若非午火救应，几无生理。

## 六、常见误解与辨析

### 误解一：刚柔相济就是阳干和阴干数量各半

**错误说法**：有人认为命局中阳干和阴干各两个就是"刚柔相济"。

**经典原文反驳**：

【任氏曰】"刚者宜柔济之，柔者宜刚济之，此相济之道也。"

**正确理解**：刚柔相济不是简单的数量对等，而是功能互补。关键看的是"谁需要谁"——如果日主是阳干且命局偏刚，需要阴柔来济；如果日主是阴干且命局偏柔，需要阳刚来济。一个八字三阳一阴，只要那一阴恰好是日主所需要的"济"，它就是完整的刚柔相济。反过来，二阳二阴如果阴阳相隔互不相济，反而不算相济。

### 误解二：刚就是好，柔就是不好（或反之）

**错误说法**：有人认为阳干多代表命局好（有气势），阴干多代表不好（格局小）。或者反过来认为纯阴为好（细腻）纯阳为差（粗鲁）。

**经典原文反驳**：

【原注】"阴阳相济，此命之正道也。"

**正确理解**：刚和柔本身没有好坏之分，都是"命之正道"的必要组成部分。关键在于"相济"——它们之间的关系是互补还是对立。纯阳无阴的八字性格刚烈有余而细致不足，纯阴无阳的八字细腻有余而魄力不足。最好的状态是刚中有柔（坚定而灵活）或柔中有刚（温和而坚定），这才是"相济"的真谛。

### 误解三：刚柔只关乎性格，不影响格局高低

**错误说法**：有人认为刚柔只是用来判断性格的，和分析格局富贵贫贱无关。

**经典原文反驳**：

【原注】"甲丙戊庚壬为阳，刚也；乙丁己辛癸为阴，柔也。阴阳相济，此命之正道也。"

**正确理解**：刚柔不仅仅关乎性格，它直接影响格局的成败。一个八字即使用神有力、格局清纯，但如果阴阳严重失衡——比如纯阳无阴，五行偏枯——也会在关键环节"掉链子"（大运或流年冲克时一溃千里）。刚柔平衡是格局稳定的基础，是决定一个命局能"走多远"的深层因素。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"刚柔者，阴阳之分也"**——刚柔就是阴阳在天干上的表现
2. **"甲丙戊庚壬为阳，刚也；乙丁己辛癸为阴，柔也"**——十天干刚柔的具体对照
3. **"刚者宜柔济之，柔者宜刚济之"**——相济之道：缺什么补什么
4. **"阴阳相济，此命之正道也"**——无论刚柔，平衡才是正道

### 7.2 与其他篇章的关联脉络

- **前序关系**：刚柔篇的"阴阳之辨"直接继承了**天道篇（01）**的阴阳理论
- **直接关联篇目**：
  - **天干（07）**——刚柔是十天干阴阳属性的具体运用
  - **顺逆（28）**——刚柔为体，顺逆为用，刚柔是心性，顺逆是时令应对
  - **寒暖（29）**——刚柔是"质"（属性），寒暖是"候"（气候），两者共同决定命局的调候需求

### 7.3 进阶学习路径建议

1. **顺逆篇（28）**——从"刚柔"进入"顺逆"，学习中令的顺从与违逆（基础级）
2. **寒暖篇（29）**——学习气候调候（基础级）
3. **天干篇（07）**——深入学习十天干的心性特征（基础级）
4. **实战练习**：拿到一个八字后，增加一个分析维度——先看刚柔，判断是否需要以柔济刚或以刚济柔
