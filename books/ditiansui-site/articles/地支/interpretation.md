## 一、入门导读

### 1.1 这一篇在学什么

地支篇系统讲解十二地支的藏干、三合三会、六合、六冲、三刑、六害等所有地支作用关系。核心要义是：地支的变化比天干更复杂，因为地支藏有人元（藏干），是命局中五行气的"仓库"和"根基"。学完本篇，你就能看懂八字中的地支之间如何相互作用。

### 1.2 本篇在整个子平体系中的地位

地支篇属于基础级难度。它与"天干"篇（讲十干性情）是姊妹篇——天干为天元、地支为地元、藏干为人元（隐藏在地支中的人元）。三者共同构成一个八字的完整结构。地支篇上承天干篇，下接"八格"篇（取格要看月令地支），是子平命理承上启下的关键篇章。

### 1.3 学习目标

学完本篇后，你应该能够：

1. 准确说出十二地支的五行属性、阴阳属性和方位
2. 记住每个地支的藏干内容及其含义
3. 区分三合局（长生帝旺墓库）、三会局（一方之气）、六合（合而化）、六冲、三刑、六害的作用形式
4. 理解地支作用关系的吉凶判断逻辑（谁冲谁、谁胜谁负）
5. 在分析命局时能结合地支关系判断五行气的来源和流向

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

地支之变化，较天干为多。地支藏干，地支会局，地支相冲，地支相刑，地支相害，其理皆精微，非可以一端论也。

地支有十二：子丑寅卯辰巳午未申酉戌亥。十二地支分阴阳：子寅辰午申戌为阳支，丑卯巳未酉亥为阴支。十二地支配五行：寅卯属木，巳午属火，申酉属金，亥子属水，辰戌丑未属土。十二地支配方位：寅卯东方，巳午南方，申酉西方，亥子北方，辰戌丑未居四角。十二地支配季节：寅卯辰为春季，巳午未为夏季，申酉戌为秋季，亥子丑为冬季。

子藏癸水，丑藏己癸辛，寅藏甲丙戊，卯藏乙木，辰藏戊乙癸，巳藏丙戊庚，午藏丁己土，未藏己丁乙，申藏庚壬戊，酉藏辛金，戌藏戊辛丁，亥藏壬甲木。地支藏干，为人元也。人之禀命，万有不齐，皆由地支藏干之不同。此子平命学之精微处，学者不可不深究也。

三合者，三个地支相合而成局也。申子辰合水局：申为长生，子为帝旺，辰为墓库。亥卯未合木局：亥为长生，卯为帝旺，未为墓库。寅午戌合火局：寅为长生，午为帝旺，戌为墓库。巳酉丑合金局：巳为长生，酉为帝旺，丑为墓库。三合局之力，胜于单支。柱中遇之，须看天干是否透出用神。若透出，则局显其用；若不透，则局藏其力，减半论之。

三会者，三个地支相会而成一方之气也。寅卯辰会东方木，巳午未会南方火，申酉戌会西方金，亥子丑会北方水。三会局之力，较三合局为专。三会者，气聚一方，不似三合者分散三方也。然三会局之成，须三字齐全，若缺其一，则不成为会也。

六合者，两个地支相合也。子丑合化土，寅亥合化木，卯戌合化火，辰酉合化金，巳申合化水，午未合化火。六合者，亲近之合也。其力较三合为弱，然亦不可忽视。命局中六合有情，合而化之，则吉；合而绊之，则凶。

地支相冲有六：子午冲，丑未冲，寅申冲，卯酉冲，辰戌冲，巳亥冲。相冲者，动也。冲则破，冲则变。命局中地支相冲，须看谁冲谁，谁胜谁负。冲旺者，冲则动，动则旺，旺者被冲，则旺气激发，此冲之吉也；冲衰者，冲则衰，衰者被冲，则衰气败散，此冲之凶也。又须看冲之远近、紧贴与遥隔。紧贴之冲，其力大；遥隔之冲，其力小。须细辨之。

地支相刑有三：恃势之刑，寅刑巳，巳刑申，申刑寅，申巳寅连环相刑，其力最盛；无恩之刑，丑刑戌，戌刑未，未刑丑，戌为火库，未为木库，相刑而战；无礼之刑，子刑卯，卯刑子，子卯相刑，水木相战；自刑，辰午酉亥自刑，辰为水库，午为火库，酉为金库，亥为木宫，自刑则本气受伤。相刑者，多主官讼、伤病、刑灾。命局中刑多者，不吉之兆也。

地支相害有六：子未害，丑午害，寅巳害，卯辰害，申亥害，酉戌害。相害者，两支相嫉，暗中相损也。害则绊，绊则互相牵制，不利六亲。命局中害多者，须防暗损暗伤。

地支之配合，较天干为繁。地支藏干、会局、相冲、相刑、相害，错综变化，非可以一端论也。任铁樵云："地支之变化，较天干为多。"学者当以地支藏干为根基，以三合三会为枢纽，以相冲相刑相害为变通，参透此理，方能入子平之门径也。

### 2.2 原文逐句精解

【原注】"地支之变化，较天干为多。地支藏干，地支会局，地支相冲，地支相刑，地支相害，其理皆精微，非可以一端论也。"

→ 解读：开篇点题。地支的变化比天干复杂得多，因为地支有藏干、三合三会、六冲三刑六害等多种作用关系，每一种都需要仔细辨析，不能用一个简单原则（一端）一概而论。

【任氏曰第1段】"地支有十二：子丑寅卯辰巳午未申酉戌亥。十二地支分阴阳：子寅辰午申戌为阳支，丑卯巳未酉亥为阴支。"

→ 解读：十二地支最基本的分类之一。阳支和阴支的排列是有规律的——从子开始，单数为阳（子1、寅3、辰5、午7、申9、戌11），双数为阴（丑2、卯4、巳6、未8、酉10、亥12）。

【任氏曰第2段】"十二地支配五行：寅卯属木，巳午属火，申酉属金，亥子属水，辰戌丑未属土。"

→ 解读：地支的五行归属——寅卯为春季故属木，巳午为夏季故属火，申酉为秋季故属金，亥子为冬季故属水，辰戌丑未为四季末月故属土（土居四维）。

【任氏曰第3段】"十二地支配方位：寅卯东方，巳午南方，申酉西方，亥子北方，辰戌丑未居四角。"

→ 解读：地支的方位对应——寅卯在正东，巳午在正南，申酉在正西，亥子在正北。辰在东南、戌在西北、丑在东北、未在西南，四角为土。

【任氏曰第4段】"子藏癸水，丑藏己癸辛，寅藏甲丙戊，卯藏乙木，辰藏戊乙癸，巳藏丙戊庚，午藏丁己土，未藏己丁乙，申藏庚壬戊，酉藏辛金，戌藏戊辛丁，亥藏壬甲木。地支藏干，为人元也。人之禀命，万有不齐，皆由地支藏干之不同。此子平命学之精微处，学者不可不深究也。"

→ 解读：**地支藏干是地支篇最核心的内容。** 每一个地支除了自身的五行属性外，内部还藏有天干之气，称为"人元"。比如子中藏癸水（子为水，藏癸是水中之精粹），丑中藏己癸辛（丑为土，藏己为本气、癸为余气、辛为中气）。为什么要藏干？因为五行之气不是纯粹的——每个季节末尾都包含不同五行气的过渡。藏干的不同决定了命局中"气"的丰富程度。原注说"万有不齐，皆由地支藏干之不同"，足见其重要性。

【辰戌丑未段】"三合者，三个地支相合而成局也。申子辰合水局：申为长生，子为帝旺，辰为墓库。亥卯未合木局：亥为长生，卯为帝旺，未为墓库。寅午戌合火局：寅为长生，午为帝旺，戌为墓库。巳酉丑合金局：巳为长生，酉为帝旺，丑为墓库。三合局之力，胜于单支。柱中遇之，须看天干是否透出用神。若透出，则局显其用；若不透，则局藏其力，减半论之。"

→ 解读：三合局是地支作用中最强的力量之一。长生（起点）→ 帝旺（顶峰）→ 墓库（收束），三个地支形成一个完整的五行气循环。例如申（金的长生）→ 子（水的帝旺）→ 辰（水的墓库）形成水局。关键原则："透则显用"——如果三合局对应的五行在天干上有透出，这个局的力量就全部显现；如果三合局的天干没有透出，则力量减半。这是子平论"有形与无形"的重要思想。

【三会部分】"三会局之力，较三合局为专。三会者，气聚一方，不似三合者分散三方也。然三会局之成，须三字齐全，若缺其一，则不成为会也。"

→ 解读：三会局（寅卯辰东方木等）比三合局力量更"专"——因为它汇聚的是一方之气（比如东方寅卯辰都是木气），不像三合局是分散在三个不同方位。但三会局必须三字齐全，缺一不可。三合局则灵活一些（半合局也有效力）。

【六合部分】"六合者，两个地支相合也。子丑合化土，寅亥合化木，卯戌合化火，辰酉合化金，巳申合化水，午未合化火。六合者，亲近之合也。其力较三合为弱，然亦不可忽视。命局中六合有情，合而化之，则吉；合而绊之，则凶。"

→ 解读：六合是"亲近之合"——就像两个人手拉手。六合的力量小于三合，但同样重要。"合而化之则吉"——如果合化成功（新五行对命局有利），就是好事；"合而绊之则凶"——如果合化束缚了某个重要的五行（比如合住了喜神），就是凶事。

【相冲部分】"地支相冲有六：子午冲，丑未冲，寅申冲，卯酉冲，辰戌冲，巳亥冲。相冲者，动也。冲则破，冲则变。命局中地支相冲，须看谁冲谁，谁胜谁负。冲旺者，冲则动，动则旺，旺者被冲，则旺气激发，此冲之吉也；冲衰者，冲则衰，衰者被冲，则衰气败散，此冲之凶也。又须看冲之远近、紧贴与遥隔。紧贴之冲，其力大；遥隔之冲，其力小。须细辨之。"

→ 解读：这一段的逻辑极为重要，请细读：

第一，相冲的本质是"动"——不一定是坏事。冲则破，但也冲则变。关键在于"旺衰"：旺方被冲，旺气被激发，反而更旺（如同烈火被风吹动，烧得更旺）；衰方被冲，衰气散败，更加衰弱（如同残烛被风吹灭）。所以冲旺为吉，冲衰为凶。

第二，冲的力度取决于远近。紧贴的两个地支相冲（如月日相邻），力量最大；隔位相冲（如年日相冲），力量中等；遥冲（如年时相冲），力量最小。

【相刑部分】"地支相刑有三：恃势之刑，寅刑巳，巳刑申，申刑寅，申巳寅连环相刑，其力最盛；无恩之刑，丑刑戌，戌刑未，未刑丑，戌为火库，未为木库，相刑而战；无礼之刑，子刑卯，卯刑子，子卯相刑，水木相战；自刑，辰午酉亥自刑，辰为水库，午为火库，酉为金库，亥为木宫，自刑则本气受伤。相刑者，多主官讼、伤病、刑灾。命局中刑多者，不吉之兆也。"

→ 解读：三刑分为四种：

1. **恃势之刑**（寅巳申）——三个地支连环相刑，力量最强，在命局中同时出现三者时极为激烈
2. **无恩之刑**（丑戌未）——三个土地支相刑，戌为火库、未为木库、丑为金库，三者相刑实质是三个"库"之间的交战
3. **无礼之刑**（子卯）——子水与卯木相刑，水木相战，子为水、卯为木，水生木本是母子关系，子卯相刑故称"无礼"
4. **自刑**（辰午酉亥）——同一个地支自身刑自己，说明该地支内部气机不纯、自我矛盾

【相害部分】"地支相害有六：子未害，丑午害，寅巳害，卯辰害，申亥害，酉戌害。相害者，两支相嫉，暗中相损也。害则绊，绊则互相牵制，不利六亲。命局中害多者，须防暗损暗伤。"

→ 解读：六害是暗中相损的关系，不像冲那样光明正大。"害"的力量是暗中的牵制，好比暗箭难防。六害的影响主要体现在六亲关系上——害多的人，六亲缘分往往不够好（不利六亲）。

## 三、核心义理深度解析

### 3.1 地支藏干——人元的根源

**论点名称：地支藏干是子平命学的精微所在**

**原文依据：**
【任氏曰】"地支藏干，为人元也。人之禀命，万有不齐，皆由地支藏干之不同。此子平命学之精微处，学者不可不深究也。"

**义理解读：**
地支藏干为什么这么重要？八字中每个地支都有暗藏的天干之气，称为"人元"。比如辰（龙）这个地支，表面五行是土，但它内部藏有"戊乙癸"三种天干之气——戊土是本气（与辰的五行一致）、乙木是余气（辰为三月木之余气）、癸水是墓库之气（辰为水库）。

这意味着八个字（四柱）的"表面"是四天干四地支，但每个地支下面还藏着多达三种天干之气，加起来总共可以多达二十多种气在命局中流动。人的命运为什么"万有不齐"？因为地支藏干的不同导致命局气的组合千差万别。

**如何运用：**

- 看一个地支的"本气"（第一个藏干）——与地支表面五行一致，力量最强
- 看"中气"（第二个藏干）——次要之气
- 看"余气"（第三个藏干）——最弱，但有时能起关键作用
- 月令地支的藏干透于天干，是取格的关键依据

### 3.2 三合局——力量最强的地支组合

**论点名称：长生-帝旺-墓库构成完整的气运循环**

**原文依据：**
【任氏曰】"申子辰合水局：申为长生，子为帝旺，辰为墓库。……三合局之力，胜于单支。柱中遇之，须看天干是否透出用神。若透出，则局显其用；若不透，则局藏其力，减半论之。"

**义理解读：**
三合局包含完整的气运循环：长生（气开始生发）→ 帝旺（气达到顶峰）→ 墓库（气收敛归藏）。以水局（申子辰）为例：

- 申：水的长生（金生水，申为金，是水的母亲位置）
- 子：水的帝旺（子为北方，水最旺时）
- 辰：水的墓库（辰为水库，水收藏之处）

三合局需要三个条件才能完整发力：

1. 三个地支齐全（或至少有两个形成半合）
2. 对应的五行在天干有透出（透则显用）
3. 不受严重冲刑破坏

### 3.3 六冲的吉凶取决于"旺衰"而非"冲就是坏"

**论点名称：冲之所以吉凶在旺衰不在冲本身**

**原文依据：**
【任氏曰】"冲旺者，冲则动，动则旺，旺者被冲，则旺气激发，此冲之吉也；冲衰者，冲则衰，衰者被冲，则衰气败散，此冲之凶也。"

**义理解读：**
很多初学者一看到命局中有子午冲、寅申冲就紧张，认为冲一定是不好的。但任铁樵明确指出：**冲的吉凶取决于被冲一方的旺衰，不取决于冲本身。**

- 旺方被冲：如同吹旺一堆篝火——气被激发，反而更旺
- 衰方被冲：如同吹灭一束残烛——气彻底散败

举例：子午冲。如果子水当令（冬天出生），午火失令，子水冲午火，午火衰散（凶）；如果午火当令（夏天出生），子水失令，午火被子水冲，午气的旺气反而被激发出来，子水被反制（吉）。

另外要检查冲的两个重要维度：

1. **远近**：紧贴（月日相邻）的冲力量最大，隔位次之，遥冲最弱
2. **势力**：联合力量——比如辰戌冲，如果辰方有子申相助（申子辰水局），戌方孤立，则辰方胜

## 四、命造实例拆解

由于地支篇原文未提供具体命造，以下命造选自《滴天髓阐微·八格篇》（同一作者任铁樵），用以验证地支理论的实际运用。

【命造一】乾造：己丑年 壬申月 丁未日 丙午时

| 年柱 | 月柱 | 日柱 | 时柱 |
| ---- | ---- | ---- | ---- |
| 己丑 | 壬申 | 丁未 | 丙午 |

**地支分析：**
丑（年支）→ 藏己癸辛。丑未逢冲（丑未冲），丑为金库，未为木库。

申（月支）→ 藏庚壬戊。申为壬水长生之地，壬水当令。

未（日支）→ 藏己丁乙。未午合（午未合化火），未丑冲。

午（时支）→ 藏丁己。午未合化火。

**结论：** 此造地支关系复杂：丑未冲（年日相冲）、午未合（日时六合）、申为壬水长生。官星壬水得申金长生，时柱午火帮身但合于未。任氏评"功名未遂，耗散资财"——丑未冲动了作为根基的两个地支，壬水逢申长生克火，一合（未午合）一冲（丑未冲）之间，功名未就。这正验证了地支篇"冲旺为吉、冲衰为凶"和"合而绊之为凶"的道理——冲未土，未衰散；合午于未，午火被绊住。

【命造二】乾造：癸未年 乙卯月 丙午日 辛卯时

| 年柱 | 月柱 | 日柱 | 时柱 |
| ---- | ---- | ---- | ---- |
| 癸未 | 乙卯 | 丙午 | 辛卯 |

**地支分析：**
未（年支）→ 藏己丁乙。未卯半合木局（亥卯未木局缺亥），但未对于卯来说是木库。

卯（月支）→ 藏乙。卯当令为春季二月。

午（日支）→ 藏丁己。午与卯无直接冲合。

卯（时支）→ 藏乙。双卯加一未，木气极盛。

**结论：** 此造地支卯未拱木（卯未半合木），且未藏乙木——全是木气。印星极旺（木为火之印），双卯加未，木势滔天。任氏评"品超群，才华卓越"是因为地支木火通明，光明磊落；但"印星太重"又"官星泄气"，导致功名蹭蹬。这正是地支"气聚一方"（双卯一未木势成林）的体现——地支的气机过于偏一方时，即使格局清正，也会有所偏废。

## 六、常见误解与辨析

### 6.1 误解一：六冲一定不好

**错误说法：** 只要命局中出现地支相冲，就代表有灾祸、变动或不利因素，冲就是坏的。

**经典原文反驳：**
【任氏曰】"冲旺者，冲则动，动则旺，旺者被冲，则旺气激发，此冲之吉也；冲衰者，冲则衰，衰者被冲，则衰气败散，此冲之凶也。"

→ 解读：任铁樵明确指出，冲有吉有凶，取决于被冲方的旺衰。旺方被冲是激发旺气（吉），衰方被冲是败散衰气（凶）。冲的本质是"动"，"动"不一定是坏事——命局需要气机流动时，冲反而是一种激活。

**正确理解：** 判断六冲的吉凶须看三个条件：第一，冲的双方谁旺谁衰；第二，冲的距离远近（紧贴力大，遥隔力小）；第三，冲的结果对格局用神是否有利。

### 6.2 误解二：三刑一定是凶的

**错误说法：** 命局中有三刑就是不好，代表官司、伤病、刑灾，见刑就是大凶。

**经典原文反驳：**
【任氏曰】"相刑者，多主官讼、伤病、刑灾。命局中刑多者，不吉之兆也。"

→ 解读：注意任氏原文用的是"多主"（大多情况主）和"刑多者"（刑多的情况下），不是绝对的"刑=凶"。他在八格篇等其他篇章中也处理过带刑的命造，有些是能通过组合化解的。

**正确理解：** 三刑确实比六冲、六害更倾向于凶的意味，但也不是绝对的。第一，要看刑的具体类型（恃势之刑力量最强，自刑力量最弱）；第二，要看刑是否破坏了对格局有利的地支；第三，要看是否有解救因素。不要看到刑就断凶，要结合全局分析。

### 6.3 误解三：三会局一定比三合局好

**错误说法：** 三会局力量比三合局更"专"，所以三会局一定比三合局好，遇到三会局就是好运。

**经典原文反驳：**
【任氏曰】"三会局之力，较三合局为专。三会者，气聚一方，不似三合者分散三方也。然三会局之成，须三字齐全，若缺其一，则不成为会也。"

→ 解读：原文说三会比较专（纯粹、集中），但并没有说"专就是好"。"专"意味着气完全偏向一方，如果这一方正好是命局所需要的用神，当然好；但如果这一方是忌神，那么"专"反而意味着问题更集中、更麻烦。就像地势过于倾斜——不一定是好事。

**正确理解：** 三会局和三合局是不同类型的力量集中方式，没有优劣之分。判断好坏要看：第一，会合的方向（是喜神方还是忌神方）；第二，是否会破坏原有的格局平衡；第三，天干是否有透出引化。会合局本身的专与散只是形式，吉凶取决于此气是否为命局所喜。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **地支藏干**——每个地支暗藏天干之气（本气/中气/余气），是人元的根源
2. **三合局**——长生→帝旺→墓库循环，力量胜于单支，透干则显用
3. **三会局**——气聚一方，比三合更专，但必须三字齐全
4. **六合**——亲近之合，合而化之则吉，合而绊之则凶
5. **六冲**——冲旺为吉（激发旺气），冲衰为凶（败散衰气），远近定力度
6. **相刑与相害**——刑多伤官讼，害多损六亲，但须结合全局判断

### 7.2 与其他篇章的关联脉络

- **天干篇**：天干为天元、地支为地元，两者须结合理解。天干决定性情气机，地支决定根基根基
- **八格篇**：取格以月令地支为纲，地支中透出的天干决定格局性质
- **配合篇**：配合篇讲干支总体的搭配原则，地支篇深入到地支内部的全部关系
- **干支总论篇**：是天干篇和地支篇的综合运用，进阶篇章

### 7.3 进阶学习路径建议

地支篇学完后，建议按以下顺序进阶：

1. **八格篇**（进阶级）——学习如何根据月令地支和透干来取格定局
2. **方局篇**（进阶级）——深入理解三合三会在命局中的运用
3. **干支总论篇**（基础级）——系统理解干支配合的全部原则

难度对比：天干（基础级）= 地支（基础级）< 八格（进阶级）。地支篇的信息量较大（藏干表、各种合冲刑害），但核心原理清晰，花时间记住藏干和合冲刑害的组合即可掌握。
