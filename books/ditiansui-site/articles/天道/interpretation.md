## 一、入门导读

### 1.1 这一篇在学什么

天道篇是《滴天髓》的总纲。它告诉你所有八字命理的根本法则都源于"三元"——天干（天元）、地支（地元）、地支藏干（人元）。并通过"帝载"（太极本源）和"神功"（五行四时运行）两个概念，揭示宇宙气运如何落实到人的命局之中。理解此篇，就抓住了拆解一切八字命运的钥匙。

### 1.2 本篇在整个子平体系中的地位

天道是《滴天髓》开篇第一论，与紧随其后的坤道（坤元合德）、人道（顺悖吉凶）构成"天道→地道→人道"的三才递进结构。天道讲宇宙的本源法则（太极生阴阳、阴阳化五行），坤道将此法则落实于地的方位与五气分布，人道则讲人如何在天地的框架中定吉凶。三篇构成了整个子平命理的宇宙论基石。此后的理气、配合、天干、地支等所有篇章，皆在此框架下展开细化。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **说清楚"三元"是什么**——用自己的话解释天元、地元、人元分别指什么，为什么三者缺一不可
2. **理解"帝载"与"神功"的含义**——明白太极是宇宙本源（帝载），五行在四时的轮转是它在现实世界的体现（神功）
3. **建立阅读全书的坐标系**——理解"三元万法宗"是全书所有论断的总出发点
4. **能在具体八字中拆解三元**——识别天干、地支、地支藏干各自的角色与相互关系
5. **预备后续篇目知识**——为学习坤道（五气偏全）、人道（顺悖吉凶）打好基础

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
欲识三元万法宗，先观帝载与神功。

【原注】
天有阴阳，故春木、夏火、秋金、冬水、季土，随时候显其神功，命于天地人三元之理，悉本于此。

【任氏曰】
干为天元，支为地元，支中所藏为人元。人之禀命，万有不齐，总不越三元之理，所谓万法宗也。阴阳本乎太极，是谓帝载，五行播于四时，是谓神功，乃三才之统系，万物之本原。《滴天髓》首明天道如此。
```

### 2.2 原文逐句精解

【原句】"欲识三元万法宗，先观帝载与神功"

→ 解读：这是《滴天髓》的开篇总纲。如果你想认识"三元"——万种命理法则的宗主，就必须先观察"帝载"和"神功"。拆解任何八字之前，必须先理解宇宙气运的本源（帝载）和它的运行表现（神功）。

【原注】"天有阴阳，故春木、夏火、秋金、冬水、季土，随时候显其神功"

→ 解读：原注指出"神功"的表现就是四季各自主事的五行——春木、夏火、秋金、冬水、季土。一年中五行轮流当令，这是"神功"在自然界的规律性显现。

【原注】"命于天地人三元之理，悉本于此"

→ 解读：人的命运（"命"）所包含的天地人三元之理，全部根源于此——即阴阳五行在四时中的运行。

【任氏曰第1段】"干为天元，支为地元，支中所藏为人元。人之禀命，万有不齐，总不越三元之理，所谓万法宗也。"

→ 解读：任铁樵给出了"三元"最明确的定义：

- **天元**：八字中的天干（年干、月干、日干、时干），代表外显之气
- **地元**：八字中的地支（年支、月支、日支、时支），代表根基之气
- **人元**：地支中藏着的天干（如子中藏癸水、午中藏丁己），代表深层潜藏之气

人的命局千差万别，但都在三元分析框架之内，所以称三元为"万法宗"。

【任氏曰第2段】"阴阳本乎太极，是谓帝载，五行播于四时，是谓神功，乃三才之统系，万物之本原。《滴天髓》首明天道如此。"

→ 解读："帝载"是宇宙的本源——太极，它产生了阴阳。"神功"是阴阳的具体运作——五行在四季中的运行。这两者统摄"三才"（天、地、人）的系统，是万物的本原。任铁樵总结：《滴天髓》第一篇阐明天道，就是为了说明这个根本原理。

## 三、核心义理深度解析

### 3.1 三元分析法——八字拆解的根本框架

**原文依据**

【任氏曰】"干为天元，支为地元，支中所藏为人元。人之禀命，万有不齐，总不越三元之理，所谓万法宗也。"

**义理解读**

"三元"是八字分析的根本框架。任何一个八字都可以从三个层次来读取：

- **天元（天干）**——代表外在的、显性的能量。好比一棵树的树冠，外露可见，代表个人的外部表现和社会角色。
- **地元（地支）**——代表内在的、根基性的能量。好比树的根系，扎在土里，代表个人的内在禀赋和家庭根基。
- **人元（地支藏干）**——代表更深层的、潜在的运作能量。好比树的年轮，记录着成长的深层信息。人元在岁运引发时才会"透出"发挥作用。

三元不是三个独立体系，而是同一命局的不同层次。三者通过生克制化相互作用，缺一不可。

**命造验证**

【任氏曰命造】乾造 癸卯 乙卯 甲寅 乙亥

此造天元三木（甲、乙、乙）透出，地元双卯一寅一亥，人元亥水所藏壬甲暗中生木。三元一气贯通——天元显用、地元为根、人元暗助，形成木气专旺之势。任铁樵以此例说明：三元之间生扶贯通则格局清贵，这正印证了"总不越三元之理"的总纲。

### 3.2 帝载与神功——从宇宙本源到命局呈现

**原文依据**

【原注】"天有阴阳，故春木、夏火、秋金、冬水、季土，随时候显其神功。"

【任氏曰】"阴阳本乎太极，是谓帝载，五行播于四时，是谓神功。"

**义理解读**

"帝载"和"神功"是一体两面的关系：

- **帝载**是本体——宇宙的终极本源（太极），它产生阴阳两种基本力量
- **神功**是作用——阴阳力量在现实世界的具体表现，即五行在四季中的轮转

打个比方：帝载像是电力的来源（发电厂），神功像是电流通过不同设备产生的效果（灯发光、炉发热）。太极是同一个，但在不同季节"显示"为不同的五行当令。

这对命理分析的实践意义：看八字不能只看表面五行，还要看五行所处的季节（月令）。同样一个甲木，生在春天（寅卯月）和生在秋天（申酉月），其"神功"完全不同——春木当令则旺，秋金当令则木受克。

**命造验证**

【任氏曰命造】乾造 庚申 甲申 甲申 庚午

甲木生于申月（秋金当令），天元两重庚金透出，地元三申当权，人元庚金本气为杀。此造中"帝载"（太极阴阳）通过"神功"（秋季金旺当令）表现为七杀攻身的局面——秋季金气当权，木气衰绝，这是天道（帝载与神功）在命局中的直接映射。唯一的转机在时支午火（人元丁火），以火制金，此为险中救应。

### 3.3 三元在命局中的层次关系

**原文依据**

【任氏曰】"人之禀命，万有不齐，总不越三元之理。"

**义理解读**

三元在命局中不是并列关系，而是嵌套关系。天元居最表层，地元居中，人元藏于地元内部。这种三层结构决定了八字分析的复杂性——同一天干因下坐的地支不同，吉凶可能截然相反。例如甲木坐寅（甲木的禄地，得根而旺）与甲木坐申（甲木的绝地，受克而衰），虽然天元同为甲木，但因地元不同而吉凶迥异。

## 四、命造实例拆解

### 命造一：三元贯通——木气专旺

【命造一】乾造：癸卯 乙卯 甲寅 乙亥

大运：癸丑 → 壬子 → 辛亥 → 庚戌 → 己酉 → 戊申 → 丁未
（出自任铁樵《滴天髓阐微·天道》）

| 柱  | 干支 | 天元分析     | 地元分析       | 人元分析                         |
| --- | ---- | ------------ | -------------- | -------------------------------- |
| 年  | 癸卯 | 癸水印星生木 | 卯木为木之根   | 乙木暗助日主                     |
| 月  | 乙卯 | 乙木比劫帮身 | 卯木月令当权   | 乙木与本柱天元同气               |
| 日  | 甲寅 | 甲木日主自身 | 寅木为日主强根 | 甲丙戊（木主气、火食神、土偏财） |
| 时  | 乙亥 | 乙木比劫相扶 | 亥水印星生木   | 壬甲（水正印、木比肩）           |

**格局**：木专旺格。**用神**取水木（印比），水生木旺、比劫帮身。**喜**水木运，**忌**金运（金克木破局）。

**结论**：此造四柱木气极旺——天元三木、地支双卯一寅一亥，水木相生之气贯通全局。癸水印星在年、亥水印星在时，源流不绝。任铁樵以此例说明"三元得助，一气贯通"的理法——天元甲木为尊，地元寅卯为根基，人元亥水暗生木气，三元互相扶持而非交战，故为贵格。

### 命造二：三元交战——七杀攻身

【命造二】乾造：庚申 甲申 甲申 庚午

大运：乙酉 → 丙戌 → 丁亥 → 戊子 → 己丑 → 庚寅 → 辛卯
（出自任铁樵《滴天髓阐微·天道》）

| 柱  | 干支 | 天元分析     | 地元分析         | 人元分析                         |
| --- | ---- | ------------ | ---------------- | -------------------------------- |
| 年  | 庚申 | 庚金七杀克身 | 申金为杀之强根   | 庚壬戊（金本气、水食神、土偏财） |
| 月  | 甲申 | 甲木比肩助身 | 申金月令当权     | 庚壬戊（金旺木囚）               |
| 日  | 甲申 | 甲木日主     | 申金坐下七杀贴身 | 庚壬戊                           |
| 时  | 庚午 | 庚金七杀再透 | 午火食神制杀     | 丁己（火伤官、土正财）           |

**格局**：七杀格（杀旺身弱）。**用神**取火（食神制杀），**喜**火土运，**忌**金水运（金助杀势、水灭火源）。

**结论**：此造七杀极旺——天元两庚透出，地元三申当令，人元庚金本气为杀。日主甲木在天元被庚金克，在地元坐下申金贴身克，在人元申中庚金暗克。三元之中，金占主导、木受制过重。唯一生路在时支午火，以丁火制申金之杀。任铁樵以此例说明：三元交战、日主受困的险局，若非午火救应，木无生理。这正是"万变不离三元"的实践例证。

## 六、常见误解与辨析

### 误解一：三元是天干、地支、藏干三套独立的体系

**错误说法**：有人看八字时只看天干、或只看地支，认为三元互不相关。

**经典原文反驳**：

【任氏曰】"干为天元，支为地元，支中所藏为人元。人之禀命，万有不齐，总不越三元之理，所谓万法宗也。"

**正确理解**：三元不是三个独立体系，而是同一命局的三个层次。天干在外、地支在内、藏干在更深层。三者通过生克制化相互作用——天干需要地支为根、地支需要天干显用、人元在岁运引发时透出发挥作用。看八字必须把三元作为一个整体来分析，缺一不可。

### 误解二："帝载"和"神功"是两个互不相关的概念

**错误说法**：有人认为"帝载"和"神功"各说各的，可以分开理解。

**经典原文反驳**：

【任氏曰】"阴阳本乎太极，是谓帝载，五行播于四时，是谓神功，乃三才之统系，万物之本原。"

**正确理解**：帝载和神功是一体两面的关系。帝载是"体"（本源），神功是"用"（表现）。同一太极，在帝载层面是静态源，在神功层面是动态用。缺一不可——没有帝载则神功无源，没有神功则帝载无显。实际看八字时，既要看五行的本源（帝载），也要看五行在四时的当令状态（神功）。

### 误解三：天道篇只有理论意义，缺乏实用性

**错误说法**：天道篇太抽象，对实际看八字帮助不大。

**经典原文反驳**：

【任氏曰】"人之禀命，万有不齐，总不越三元之理。"

**正确理解**：天道篇是整个《滴天髓》的"操作系统内核"，它提出的三元分析法是后续所有篇章的分析基础。看八字第一步就是区分三元——天干是什么（外在表现），地支是什么（内在根基），地支藏干是什么（潜在暗力）。后面的理气、配合、天干、地支等篇章，都是在三元框架下的具体展开。没有天道篇的框架，后续分析就失去了坐标系。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"欲识三元万法宗，先观帝载与神功"**——八字分析的总出发点：先看三元，先明本源。
2. **"干为天元，支为地元，支中所藏为人元"**——三元的具体定义。
3. **"阴阳本乎太极，是谓帝载，五行播于四时，是谓神功"**——帝载是本源（太极），神功是表现（五行四时）。
4. **"人之禀命，万有不齐，总不越三元之理"**——所有命局都在三元框架内分析。

### 7.2 与其他篇章的关联脉络

- **前序关系**：天道是全书首篇，为所有篇章提供总纲。
- **直接关联篇目**：
  - **坤道（02）**——将天道中的"神功"落实于地的方位（五气偏全定吉凶）
  - **人道（03）**——在天道"三元"和坤道"坤元合德"基础上，论人的顺悖吉凶
  - **天干（07）**——细化"天元"的具体用法
  - **地支（08）**——细化"地元"和"人元"（地支藏干）的具体用法
  - **月令（15）**——细化"神功"在月令上的运用（旺相休囚死）

### 7.3 进阶学习路径建议

天道篇之后的推荐学习顺序：

1. **坤道（02）**——与天道体用互补，篇幅同样简短，难度一致（入门级）
2. **人道（03）**——完成"三才"统一框架（入门级）
3. **理气（05）**——开始进入具体分析方法（基础级）

天道 + 坤道 + 人道三篇连读，可以建立起《滴天髓》的完整宇宙论框架，为后续篇章打下坚实基础。
