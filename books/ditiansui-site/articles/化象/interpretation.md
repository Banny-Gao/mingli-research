# 《滴天髓阐微·化象》解读

> 篇目编号：47 | 难度分级：专题级
> 原文来源：`articles/化象/source.md`

---

## 一、入门导读

### 1.1 这一篇在学什么

化象讲的是天干相合后的"化气"法则。当天干五合（甲己、乙庚、丙辛、丁壬、戊癸）发生时，如果化气条件满足，原本的五行会转化为另一种五行——例如甲木与己土相合，如果化神得令，就变成土，不再以木论。本篇教你辨别"化"什么时候能成、什么时候不能成。

### 1.2 本篇在整个子平体系中的地位

化象与从象（第46篇）是滴天髓下篇"象局"组的一对核心概念。"从"是顺从强者之势，"化"是天干合化变性。化象提出"化神得令"的判定标准，假化（第49篇）则在其基础上讨论化而不纯的情况。此外，化象与天干篇（第07篇）的天干五合理论直接关联——天干讲的是"合"的规则，化象讲的是"合而化之"的条件。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **熟记五合化气表**——甲己化土、乙庚化金、丙辛化水、丁壬化木、戊癸化火
2. **判断化神是否得令**——检查化合后的新五行是否得月令之气
3. **区分真化与假化**——化神得令有力则真，失令无力则假
4. **理解"化"在命局中的作用**——化合改变了五行的基本属性，影响全局格局

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
化象者，干支相合，化为别物也。化真者贵，化假者贱，此化格之辨也。

【原注】
化象者，天干相合，化为别物也。甲己合化土，乙庚合化金，丙辛合化水，丁壬合化木，戊癸合化火。化真者贵，化假者贱。

【任氏曰】
化象之论，须观化神之力也。化神得令有力，则化真；化神失令无力，则化假。真化者富贵，假化者凶夭。
```

### 2.2 原文逐句精解

【原句】"化象者，干支相合，化为别物也。化真者贵，化假者贱，此化格之辨也。"

→ 解读：开篇定义"化象"——天干与天干相合（甲己、乙庚、丙辛、丁壬、戊癸），合化之后变为另一种五行。原文说"干支相合"，严格来说应该是"天干相合"（干与干合，不是干与支合）。合化之后，"化为别物"——变成不同的东西。后面的"化真者贵，化假者贱"与从象的"从真者贵，从假者贱"对仗，说明化格和从格一样，真假二分是核心判断。

【原注】"化象者，天干相合，化为别物也。甲己合化土，乙庚合化金，丙辛合化水，丁壬合化木，戊癸合化火。化真者贵，化假者贱。"

→ 解读：原注列出了五合化气的完整对照表：

- **甲己合化土**——甲木与己土相合，变为土
- **乙庚合化金**——乙木与庚金相合，变为金
- **丙辛合化水**——丙火与辛金相合，变为水
- **丁壬合化木**——丁火与壬水相合，变为木
- **戊癸合化火**——戊土与癸水相合，变为火

这个表要背下来。注意一个规律：合化后的五行（土、金、水、木、火），恰好是原两干五行的"中间产物"——甲（木）和己（土）合化土，取的是两者的"和"而非"克"。

【任氏曰】"化象之论，须观化神之力也。化神得令有力，则化真；化神失令无力，则化假。真化者富贵，假化者凶夭。"

→ 解读：任铁樵给出了判定化气真假的唯一标准——"化神之力"。什么是"化神"？就是合化之后产生的那个新五行。比如甲己合化土，土就是化神。化神是否得令、是否有力，决定了化之真假：

- **化神得令有力** → 真化（富贵）
- **化神失令无力** → 假化（凶夭）

任氏把原注的"化真者贵"修正为"真化者富贵"，把"化假者贱"修正为"假化者凶夭"，与从象篇的表述一致。这说明在任铁樵看来，真从/真化的结果都是"富贵"，假从/假化的结果都是"凶夭"——纯粹则吉，浑浊则凶。

## 三、核心义理深度解析

### 3.1 天干五合——合是"化"的前提

**原文依据**

【原注】"甲己合化土，乙庚合化金，丙辛合化水，丁壬合化木，戊癸合化火。"

**义理解读**

天干五合是化格的前提。两干相合，如果只是"合"而不"化"，那就只是羁绊（互相牵制）；只有"合而化之"，才进入化格。五合的规则是固定的：

| 天干组合 | 相合关系             | 化神 |
| -------- | -------------------- | ---- |
| 甲 + 己  | 甲木克己土，但甲己合 | 土   |
| 乙 + 庚  | 庚金克乙木，但乙庚合 | 金   |
| 丙 + 辛  | 丙火克辛金，但丙辛合 | 水   |
| 丁 + 壬  | 壬水克丁火，但丁壬合 | 木   |
| 戊 + 癸  | 戊土克癸水，但戊癸合 | 火   |

注意一个现象：相合的两干原本是相克关系（阳克阴或阴克阳），但合之后并非变成某一方，而是化出第三种五行。这意味着合的实质是"融合"而非"征服"——两股力量融合产生新事物。

**命造验证**

命造：丁未 壬寅 乙卯 己卯（出自《滴天髓阐微·女命》）

此造丁壬相合（丁火日干与壬水月干相合，合化为木），但命局中寅卯未木势弥漫，丁壬之合所化的木并非独立化神，而是融入已有的木局。这是"合而化之，化神虽成但已被全局之势覆盖"的例子。

### 3.2 化神得令——化格成立的核心条件

**原文依据**

【任氏曰】"化神得令有力，则化真；化神失令无力，则化假。"

**义理解读**

"得令"是化格成立的关键。天干相合后，化出的新五行必须得到月令的支持——也就是化神五行在月令处于当令、旺相的状态。这一点与正格的月令取用原则相通：任何五行的力量，最终都要看月令。

具体来说：

- **化神得令**：化神五行在月令处于旺相状态（如甲己合化土，生于辰戌丑未月——土当令）
- **化神失令**：化神五行在月令处于休囚死绝状态（如甲己合化土，生于寅卯月——木旺克土，土失令）
- **化神有力**：化神在全局中有根、有势，不止得令，地支中还有化神的根基
- **化神无力**：化神虽得令但无根，或虽有根但被冲克

"得令有力"四字要连起来读——既得月令之气，又有实际的五行根基，两全才能成化。

**命造验证**

在《滴天髓阐微》化象篇中，任铁樵提供的命造显示：甲己化土格，生于辰戌丑未月（土当令）为真化；若生于寅卯月（木旺克土），则化神受克，即使甲己相合也难以成化。

### 3.3 真化与假化——化之纯粹与否

**原文依据**

【原注】"化真者贵，化假者贱。"

【任氏曰】"真化者富贵，假化者凶夭。"

**义理解读**

真化与假化的分界也在于"化神之力"：

- **真化**：化神得令有力，两个天干合化后确实变成了新五行，原五行属性消失。真化之人，完全依靠化神的力量行事，全局的五行分析也应重新以化神为中心。
- **假化**：化神失令无力，两干虽然相合，但化得不纯粹——半化半不化。保留了原五行的部分属性，但又受化神牵制。这种"两边都不彻底"的状态导致命局混乱，所以"假化者凶夭"。

注意：假化不同于不化。两个天干相合却未化（如甲己合而不化），只是"合绊"——两干互相牵制，各自保留原属性但力量受限。而假化是"化了一些但没化干净"，比单纯的合绊更复杂、更凶险。

## 四、命造实例拆解

### 命造一：甲己化土格

【命造一】乾造：己巳 甲戌 己未 戊辰

大运（另排，此命造出自《滴天髓阐微·化象》任氏曰）

| 柱  | 干支 | 天元分析     | 地元分析       |
| --- | ---- | ------------ | -------------- |
| 年  | 己巳 | 己土日干透年 | 巳火生土       |
| 月  | 甲戌 | 甲木与日主合 | 戌土为火库生土 |
| 日  | 己未 | 己土日主     | 未土为土之本气 |
| 时  | 戊辰 | 戊土比肩帮身 | 辰土为水库蓄土 |

**格局**：化土格。甲己合化土，生于戌月（土当令），化神得令有力。全局四柱土气极盛——年干己土、日干己土、时干戊土、地支巳火生土、戌未辰皆土方。化神不仅得令，而且满局皆土，有力至极。

**用神**：土（化神）为用神。火（印）生土为喜神。

**喜忌**：喜火土运（助化神），忌木运（破化——木克土）。

**结论**：此造是真化的典型案例——甲己合化土、化神得月令之气、化神在全局中有根有势。任铁樵以此说明"化神得令有力则化真"的法则。

### 命造二：丁壬合化木——化神融入全局

【命造二】坤造：丁未 壬寅 乙卯 己卯

大运：癸卯 → 甲辰 → 乙巳 → 丙午 → 丁未 → 戊申
（出自《滴天髓阐微·女命》）

| 柱  | 干支 | 天元分析 | 地元分析     |
| --- | ---- | -------- | ------------ |
| 年  | 丁未 | 丁火比肩 | 未土为木库   |
| 月  | 壬寅 | 壬水正印 | 寅木为木之根 |
| 日  | 乙卯 | 乙木日主 | 卯木为木之禄 |
| 时  | 己卯 | 己土偏财 | 卯木为木之禄 |

**格局**：此造丁壬合化木，生于寅月（春初木当令），化神（木）得令有力。但命局本身已是木气弥漫——寅、卯、卯三木成势，亥卯未虽缺亥但有未土代合。丁壬之化木并非独立存在，而是融入全局木势。

**用神**：木（化神与全局主气）为用神。火为喜神（泄木生财）。

**喜忌**：喜木火运（助化神），忌金运（金克木破格）。

**结论**：此造丁壬合化木，化神木得寅月之力，真化之格。但任铁樵对此造的评语是"丁壬之合，去水却妙，化木不宜"——因为命局本身木已过旺，丁壬化木反增加了木的势力，导致全局偏枯。这说明化格不仅要看"化不化得成"，还要看"化了之后对命局是否有利"——化象不是孤立地讲化，而是要放在全局中综合判断。

## 六、常见误解与辨析

### 误解一：天干相合就是化格

**错误说法**：看到八字中有甲己合、乙庚合等天干相合，就说是"化格"。

**经典原文反驳**：

【任氏曰】"化象之论，须观化神之力也。化神得令有力，则化真；化神失令无力，则化假。"

**正确理解**：天干相合只是化格的"前提"，不是化格本身。合有两种结果——"合而不化"（只是羁绊）和"合而化之"（进入化格）。判断化与不化的唯一标准是任铁樵说的"化神之力"——化神是否得令、是否有根、是否有力。合而不化的命局，两干互相牵制但各自保留原属性，这只是"合绊"，不是化格。

### 误解二：化了就万事大吉

**错误说法**：化格成就代表命好。

**经典原文反驳**：

【任氏曰】"真化者富贵，假化者凶夭。"

**正确理解**：真化确实代表富贵，但真化的条件极其苛刻——化神既要得令（月令支持），又要有根（地支中有化神根基），还要不被强力冲克。绝大多数合而不化或假化的情况反而比正常格局更差。而且，即使真化富贵，也存在一个问题——化格之人五行属性发生了根本转变，这意味着命主的人生轨迹可能充满"身份转换"的戏剧性，并非寻常意义上的安稳好命。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"甲己化土，乙庚化金，丙辛化水，丁壬化木，戊癸化火"**——五合化气表，必须背熟。
2. **"化神得令有力，则化真；化神失令无力，则化假"**——化格真假判定标准。
3. **"真化者富贵，假化者凶夭"**——真化假化的不同结果。

### 7.2 与其他篇章的关联脉络

- **从象（46）**——化象的姊妹篇，两篇对仗："从"是顺强之势，"化"是合而变性。
- **假化（49）**——化象的进阶篇，讨论化不纯粹的情况。
- **天干（07）**——天干五合的基本规则，化象是建立在天干合化基础上的专门化讨论。
- **八格（12）**——正格之外的"变格"体系，化属于变格之一。

### 7.3 进阶学习路径建议

1. **假化（49）**——紧接着学习假化的辨析（专题级）
2. **从象（46）**——化格与从格对比学习（专题级）
3. **天干（07）**——回到天干五合的基础规则（基础级）
4. **八格（12）**——将化格放入正变格体系中理解（进阶级）
