## 一、入门导读

### 1.1 这一篇在学什么

八格篇是《滴天髓阐微》中格局理论的核心篇章。它讲的是如何通过月令地支和透干来确定命局的"格局"——八字有了格局，才有了评判富贵贫贱的基本框架。同时，本篇犀利批判了大量流行但错误的"格局说"（飞天禄马、壬骑龙背等），帮学命者去伪存真。

### 1.2 本篇在整个子平体系中的地位

八格篇属于进阶级难度，是前五篇（天道、坤道、人道、知命、理气、配合、天干、地支）理论知识的综合运用。天干篇给了你日主的性情，地支篇给了你地支的气机根基，而八格篇教你如何"定格局"——格局定了，用神取向就有了基本框架。这是从"知道"到"会用"的关键一跃。任氏在篇中反复强调：格局不可执一而论，必须结合旺衰喜忌综合判断。

### 1.3 学习目标

学完本篇后，你应该能够：

1. 理解八格（财、官、印、食、伤、煞等）的定义和取用方法
2. 掌握月令取格的正统流程——先看月令地支，再看透干，再究司令
3. 辨别哪些"格局"是正统（五行正理之格）、哪些是错误的（飞天禄马、壬骑龙背等）
4. 理解任铁樵的格局观：格正用真行运不悖则名利自如；格破用损则是"有病"，行运得所则可愈
5. 通过命造分析，学会格局、用神、喜忌的完整分析流程

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

财官印绶发偏正，兼论食伤八格定。

影响遥系既为虚，杂气财官不可拘。

### 2.2 原文逐句精解

#### 第一段：八格总论

【原注】"自形象气局之外，而格为最。格之真者，月支之神，透于天干也。以散乱之天干，而寻其得所附于提纲，非格也。自八格之外，若曲直五格皆为格，而方已局气象定之者，不可言格也。五格之外，飞天合禄虽为格，而可以破害刑冲论之者，亦不可言格了也。"

→ 解读：原注定义了什么才是"真格局"——月令地支中透到天干上的五行之神，才是真正的格局。如果你的八字中，天干散乱不成体系，硬要从天干中找一个"附着"在月令上的东西，那不是真格局。原注还区分了三种"格"：

1. **正格（八格）**：财官印食等八种，按月令地支取用
2. **变格（曲直等五格）**：从方局气象定的特殊格局，不能按一般八格论
3. **伪格（飞天合禄等）**：是"以破害刑冲论之"的，不能算格局

【任氏曰】"八格者，命中之正理也。先观月令所得何支，次看天干透出何神，再究司令以定真假，然后取用，以分清浊，此实依经顺理，若月逢禄刃，无格可取，须审日主之喜忌另寻别支透出天干者，借以为用。"

→ 解读：这一段是八格取用的"标准流程"：

1. **先观月令**——看月支是什么（比如卯月、午月、酉月）
2. **次看透干**——看月支中哪个藏干透到了天干上（比如月支卯，藏乙，天干有乙透出）
3. **再究司令**——看月令中哪一"神"在当令掌权
4. **然后取用**——确定格局后取用神
5. **分清浊**——判断格局是"真/清"还是"假/浊"

如果月令是禄（如甲日生于寅月，寅为甲之禄）或刃（如丙日生于午月，午为丙之刃），没有可取的正格，就要从其他地支中找透出的有用之神借以取用。

【任氏曰续段】"然格局有正有变，正者必兼五行之常礼也，曰官印，曰煞印，曰财煞，曰食神制杀，曰食神生财，曰伤官佩印，曰伤官生财；变者，必从五行之气势也，曰从财，曰从官杀，曰从食伤，曰从强，曰从弱，曰从势，曰一行得气，曰两气成形。"

→ 解读：任氏将格局分为"正"和"变"两类：
**正格**：遵循五行常理——官印相生、煞印相生、财煞相济、食神制杀、食神生财、伤官佩印、伤官生财。这些都是八字中常见的组合关系。
**变格**：从五行气势——从财、从官杀、从食伤、从强、从弱、从势等。这些是五行偏旺或偏弱的特殊情况，日主不坚持自己的五行，而是顺从全盘气势。

【任氏曰续段】"其余外格我端，余务考群书，俱不从五行正理，尽属谬谈；至于《兰台妙选》，所定一切奇格异局，纳音诸法，尤属不经，不待辩而知其荒唐也。"

→ 解读：这是任铁樵非常激烈的批判。他说其他那些"外格"——他不厌其烦地考证了大量书籍——都不遵循五行正理，"尽属谬谈"（全是胡说）。尤其是《兰台妙选》中所有的奇格异局、纳音诸法，更是"不经"（不经之谈），"不待辩而知其荒唐也"（不用辩论都知道是荒唐的）。

【任氏曰严批伪格】"即如壬辰日为'壬骑龙背'，壬寅日为'王骑虎背'，何不再取壬午、壬申、壬戌、壬子，谓骑猴马犬鼠之背乎？又如六辛日逢子时，谓'六阴朝阳'，夫五阴皆阴，何独辛金可朝阳，余干不可朝阳乎？且子乃体阳用阴，子中癸水，六阴之至，何谓阳也？又如六乙日逢子时，谓'鼠贵格'，夫鼠者，耗也，何以为贵？且十干之贵，时支皆有之者，岂余干不可贵乎？不待辩而知其谬也。"

→ 解读：任氏用一连串犀利的反问，击破三个流行的伪格：

1. **壬骑龙背格**（壬辰日）——如果壬辰叫骑龙背，壬寅叫骑虎背，那壬午是不是骑马背？壬申骑猴背？壬戌骑狗背？壬子骑鼠背？显然荒谬。
2. **六阴朝阳格**（六辛日逢子时）——如果只有辛金可以朝阳，其他阴干为什么不行？再说子中癸水是至阴之水，怎么叫"阳"？
3. **鼠贵格**（六乙日逢子时）——老鼠本来是耗损之物的象征，怎么能叫"贵"？而且每个天干在时支都会有贵人，凭什么只有乙日子的时支是"鼠贵"？

这一连串反问逻辑清晰、言辞犀利，充分体现了任铁樵的求真精神。

#### 第二段：影响遥系与杂气财官

【原注】"飞天合禄之类，固为影响遥系而非格矣。如四季月生人，只当取土为格，不可言杂气财官；戊己日生于四季月者，当看人元透出天干者取格，不可概以杂气财官论之；至于建禄月劫羊无能为刃，亦当看月令中人元透于天干者取格，若不合气象形局，则又无格矣。只取用神，用神又无所取，只得看其大势，以皮面上断其穷通。不可执格论也。"

→ 解读：原注指出几点：

1. **飞天合禄类都是"影响遥系"**——远距离的、间接的关系，不是真正的格局
2. **四季月（辰戌丑未）生人**——应该取土为格（因为辰戌丑未本身五行属土），不要硬套"杂气财官"
3. **戊己日生于四季月**——要看地支藏干（人元）中哪个透出于天干，按透干取格
4. **建禄月劫（比劫当令）**——没有格局可取时，只看用神；用神也没法取，就看整体气势
5. **不可执格论也**——不要执着于"格局"这个形式

【任氏曰】"影响遥系者，即暗冲暗合之格也。俗书所谓飞天禄马是也。如丙午日支全三午，癸酉日支三酉，逢三则冲，午去暗冲子水为官，酉去暗合辰土为官……又云，先要四柱不要财官，为真方可冲合。夫冲者，散也，合者，化也，何能为我用乎？四柱原有财官，不宜冲合，尚有喜与不喜，何况四柱无财官乎？"

→ 解读：任氏一针见血地指出"暗冲暗合"的逻辑缺陷：如果四柱本来就没有财官，依靠地支之间远距离的暗冲暗合来"制造"财官——这怎么可能有用？冲的本质是散，合的本质是化——无论是散还是化，都不能凭空产生财官。四柱原本就有财官的，都不一定敢说不怕冲合，何况原本就没有财官的？这个逻辑非常有力。

【任氏曰续段】"诸书所载，禄分四种，年为背禄，月为建禄，日为专禄，时为归禄。又云建禄喜官，归禄忌官，则又遗背禄专禄矣……大凡格局真实而纯粹者，百无一二，破坏而杂气者，十有八九，无格可取者甚多，无用可寻者不少。格正用真，行运不悖，名利自如；格破用损，谓之有病，忧多乐少。倘行运得所，去其破损之物，扶其喜用之神，譬如人染沉疴，得良剂以生也，不贵亦富。"

→ 解读：任氏这一大段非常重要，揭示了命理的现实：

- 真正纯粹的好格局，一百个八字里不一定有一两个
- 格局有破损的、混杂的，十有八九
- 没有格局可取、没有用神可寻的，也大有人在
- **关键是运**：格正用真、行运不悖——名利双全
- **格破用损叫做"有病"**——好比人得了病
- **行运能去病扶喜**——好比吃对了药，不贵也能富

这是任氏格局理论中最重要的"病药说"：格局本身不可能完美，人生也不可能一帆风顺，关键是看大运是否能"治病"（去除格局中的破损之物）、"扶药"（生扶用喜之神）。

#### 命造分析（原文中的八个命造）

**命造一：正官格（乙木日主）**

甲申 乙酉 丙戌 丁亥 戊子 己丑

→ 解读：乙木日主，地支三未（木库），天干双癸水印星贴身生扶，庚金正官独发。印星不旺不弱，官星清透不被混杂，财（未中己土财星）旺生官，格局"中和纯粹"，所以仕途顺利。

辛未 庚午 己巳 戊辰 丁卯 丙寅

【任氏曰】"此造以大势观之，官得清地彼，何彼则富贵，此则困穷？不知此造无印，官紧克，午未虽是余气禄旺，丑中蓄水，暗伤午未之火，壬水逢生，又克丙火；更嫌己土一透，不能制水，反能晦火；兼之中运逢土，又泄火气，谓克泄交加。因之功名未遂，耗散资财……"

→ 解读：和命造一对比学习：命造一的乙木有印星生扶，此造的丁火无印星护身。壬水官星直接克丁火（官杀克身），己土食神透出不但不能制水，反而晦火。地支丑未冲（丑中癸水暗伤午未之火）。这就是"格破用损"——格局看似官格，但官星克身、食神晦火、地支冲战，"克泄交加"，所以功名未遂。

→ 解读：丙火日主，乙卯正印当令，癸水正官透年干。格局清正，但因为印星太重（双卯一未，木势滔天），正官癸水被印星泄气（水生木），导致"神有余而精不足"——印太旺泄官，格局虽正面但有偏颇，所以"大材小用"。

**命造四：印绶格（有药）**

辛卯 丙申 癸卯 壬戌
乙未 甲午 癸巳 壬辰 辛卯 庚寅

【任氏曰】"此印绶格，以申金为用，以丙火为病，以壬水为药，中和纯粹，秋水通源。运至癸巳，金水逢生得助，科甲联登；壬辰药病相济，由部属出为郡守。"

→ 解读：癸水日主，申金正印当令。申金生癸水（金生水），丙火正财是"病"（丙火耗癸水），壬水劫财是"药"（壬水帮身）。运至癸巳、壬辰，金水旺地帮身，"病药相济"，科甲联登。这是典型的"格正用真、行运不悖则名利俱成"的案例。

**命造五：与命造四对比（无药）**

辛卯 丙申 癸卯 甲寅
乙未 甲午 癸巳 壬辰 辛卯 庚寅

【任氏曰】"此亦以申金为用，以丙火为病，与前只换一寅字，不但有病无药，而且生助病神。彼则青钱万选，名利两全，此则机杼空抛，守株待兔。更嫌寅申遥冲，卯木助之，印绶反伤，木旺金缺，且月建乃六亲之位，未免分荆破斧，资财耗散。"

→ 解读：和命造四对比惊人——只换了一个字"壬"变"甲"，结果天差地别。命造四有壬水劫财为"药"帮身，命造五的"甲"是伤官，伤官生财（甲木生丙火），加重了丙火的为害之势；而且寅申冲（日支与时支），申金印绶被冲破，木旺金缺，格局破坏。这就是任氏"病药说"的经典对比案例——**一字之差，天壤之别。**

**命造六：伤官生财格（非飞天禄马）**

己巳 庚午 丙午 甲午
己巳 戊辰 丁卯 丙寅 乙丑 甲子 癸亥

【任氏曰】"此造俗论，丙午日支全三午，四柱滴水全无，中年又无水运，必作飞天禄马，名利双辉，不知此造午中己土，巳中庚金，元神透出年月两干，真火土伤官生财格。"

→ 解读：这是用实证反驳"飞天禄马格"的典型案例。俗论看到丙午日、地支三午、滴水全无，就套用"飞天禄马格"，说应该名利双辉。但任氏指出这是**伤官生财格**（己土伤官透年干，庚金偏财透月干，火生土、土生金）。如果按飞天禄马论，应该大忌水运，但实际上此造走甲子、癸亥水运时发财数万——所以飞天禄马的判断是错的。任氏用事实说话：理论要与实际吻合，否则就是错的。

**命造八与九：杂气财官辩**

丁未 癸丑 甲辰 甲戌
壬子 辛亥 庚戌 己酉 戊申 丁未

丁亥 癸丑 甲子 辛未
壬子 辛亥 庚戌 己酉 戊申 丁未

【任氏曰】"此造支全四库逢冲，俗作杂气财官也，不知丑未逢冲，不特官星受伤，而且冲去库根……前则逢冲，官印两伤，名利无成，此则不动，名成利遂。可知墓库逢冲必发者，谬也。"

→ 解读：两个命造对比，一个地支全库逢冲（丑未辰戌四库全冲），一个地支只见丑未但遥隔不冲。俗论以为"墓库逢冲必发"，但命造八四库全冲的结果是"家业破尽，无子而亡"；命造九地支不冲的结果却是"名成利遂"。任氏以此铁证推翻"墓库逢冲必发"的谬论——墓库逢不逢冲，不是关键，关键在格局是否被破坏、用神是否受损。

## 三、核心义理深度解析

### 3.1 月令取格的正确流程

**论点名称：格局三步骤——先月令，再透干，再司令**

**原文依据：**
【任氏曰】"先观月令所得何支，次看天干透出何神，再究司令以定真假，然后取用，以分清浊。"

**义理解读：**
任铁樵给出了清晰的格局取用四步流程：

**第一步：观月令何支。** 月柱的地支是"提纲"，是整个八字中力量最集中的地方。先看这个地支是什么。

**第二步：看天干透出何神。** 月支中藏有多个天干（地支藏干），看哪个藏干在天干上出现了（透出）。比如月支是卯，卯藏乙木，如果天干有乙木出现，即为透出。

**第三步：究司令定真假。** 月令中的节气和当令之神需要查节气来确定（比如卯月中，前半为春分前乙木司令，后半为春分后卯木司令）。

**第四步：取用分清浊。** 定了格局之后，要看格局是真清（纯正不杂）还是假浊（混杂破损）。

注意：如果月令是禄（甲日寅月）或刃（丙日午月），没有格局可取，就要"另寻别支透出天干者，借以为用"。

### 3.2 病药说——格局没有完美的

**论点名称：格破用损谓之"有病"，行运得所如同"吃药"**

**原文依据：**
【任氏曰】"大凡格局真实而纯粹者，百无一二，破坏而杂气者，十有八九……格破用损，谓之有病，忧多乐少。倘行运得所，去其破损之物，扶其喜用之神，譬如人染沉疴，得良剂以生也，不贵亦富。"

**义理解读：**
这一段是任铁樵格局理论的灵魂所在。以下几点必须深刻理解：

1. **格局完美的八字极少**——"百无一二"，一百个里找不到一两个。绝大多数八字都是有问题的。

2. **有问题的八字叫"有病"**——格局破损、用神受损，就是"有病"。

3. **大运就是"药"**——行运如果能"去其破损之物"（去除不好的因素）、"扶其喜用之神"（帮助有利的因素），就是吃对了药，不贵也能富。

4. **这个比喻非常重要**——就像人生病需要用药物调理，八字的格局问题也需要大运来弥补。这就解释了为什么有些人八字看似很好但一生坎坷，而有些人八字并不完美却能富贵双全——关键是"运"这个药是否对症。

命造四和命造五的对比就是最经典的例子：两个八字几乎一样，唯一的区别是命造五的"壬"被换成了"甲"。命造四有壬水（药），命造五没有。结果命造四科甲联登，命造五守株待兔。

### 3.3 破除伪格——任铁樵的批判精神

**论点名称：凡不遵循五行正理的格局都是谬谈**

**原文依据：**
【任氏曰】"其余外格我端，余务考群书，俱不从五行正理，尽属谬谈；至于《兰台妙选》，所定一切奇格异局，纳音诸法，尤属不经，不待辩而知其荒唐也。"

→ 解读：任铁樵以一人之力批判了大量流行的"格局"说。他的核心逻辑很简单：**凡是不从五行正理（生克制化）出发的格局之说，都是错误的。** 因为八字论命的本质是五行气运的平衡与配合，而不是某种特殊的"格局名称"。

**任氏批判的主要伪格：**

1. **飞天禄马格**（暗冲暗合）——四柱没有财官，靠暗冲暗合制造财官，逻辑不通（"冲合者，何能为我用乎"）
2. **壬骑龙背格**（壬辰日为龙背）——类推下去壬午壬申壬戌都可以叫"骑某背"，显然荒谬
3. **六阴朝阳格**（六辛日逢子时）——如果只有辛可以朝阳，为什么其他阴干不行？逻辑矛盾
4. **鼠贵格**（六乙日逢子时）——老鼠是耗物，何以为贵？而且贵人众多，为何只此一种
5. **墓库逢冲必发**——命造八和九的对比证明"必发"是错的

## 四、命造实例拆解

以下命造均选自《滴天髓阐微·八格篇》原文。

【命造一】乾造：庚辰年 癸未月 乙未日 癸未时

| 年柱 | 月柱 | 日柱 | 时柱 |
| ---- | ---- | ---- | ---- |
| 庚辰 | 癸未 | 乙未 | 癸未 |

**格局**：正官格（月令未土藏己土，天干透庚金正官）

**用神分析**：

- 日主乙木，地支三未（木之余气），天干双癸水（偏印）贴身生扶
- 庚金正官独透于年干，清而不杂
- 癸水润土养金，财（土）旺生官

**大运**：甲申 → 乙酉 → 丙戌 → 丁亥 → 戊子 → 己丑

**喜忌**：

- 喜：水（印星生身）、金（官星清透）
- 忌：火（伤官克官）、土过旺（财多生官虽好但土重木折）

**结论**：格局"中和纯粹"——官星清、印星润、财星旺而不过。任氏评"科甲出身，仕至藩臬，官境安和"。

【命造二】乾造：辛卯年 丙申月 癸卯日 壬戌时

| 年柱 | 月柱 | 日柱 | 时柱 |
| ---- | ---- | ---- | ---- |
| 辛卯 | 丙申 | 癸卯 | 壬戌 |

**格局**：正印格（月令申金为正印，生癸水日主）

**用神分析**：

- 癸水日主，申金当令正印生身
- 丙火正财为"病"（耗日主之气）
- 壬水劫财为"药"（帮身抗财耗）

**大运**：乙未 → 甲午 → 癸巳 → 壬辰 → 辛卯 → 庚寅

**喜忌**：

- 喜：水（帮身）、金（印星生身）
- 忌：火（财星耗身）、木（食伤生财加重耗身）

**结论**：任氏评"运至癸巳，金水逢生得助，科甲联登；壬辰药病相济，由部属出为郡守"。此造是"格正用真、行运不悖"的典范。

【命造三（对比）】乾造：辛卯年 丙申月 癸卯日 甲寅时

| 年柱 | 月柱 | 日柱 | 时柱 |
| ---- | ---- | ---- | ---- |
| 辛卯 | 丙申 | 癸卯 | 甲寅 |

**格局**：正印格（印星申金当令），但与命造二相比换时柱壬戌为甲寅

**用神分析**：

- 与命造二基本相同，但时干甲木（伤官）代替壬水（劫财）
- 丙火财星本来就耗身，甲木伤官又生财（木生火），加重了耗身之势
- 甲寅与月支申寅遥冲，印绶受伤

**大运**：乙未 → 甲午 → 癸巳 → 壬辰 → 辛卯 → 庚寅

**喜忌**：

- 同理，喜金水、忌火木
- 但命局中既无金（印星申金被冲伤）又无水（无壬癸帮身），反而有木（伤官）助火

**结论**：一字之差（壬→甲、戌→寅），天壤之别。命造二名利两全，命造五守株待兔。这就是"病药说"的经典验证——命造二有病有药（丙火为病、壬水为药），命造五有病无药（丙火为病，甲木反助火势）。

## 六、常见误解与辨析

### 6.1 误解一：格局越多越好

**错误说法：** 格局越多，命越贵。一个八字同时有多个格局（比如既是财格又是官格还是印格），说明这个命非常好。

**经典原文反驳：**
【任氏曰】"大凡格局真实而纯粹者，百无一二，破坏而杂气者，十有八九。"

→ 解读：任氏强调"纯粹"二字——格局越纯越好，越杂越差。格局"清"不是指格局多，而是指气纯不杂。一个八字越少混杂越可贵，而不是越多越好。

**正确理解：** 格局贵在清纯不杂。一个官星透出的八字，如果又有其他杂气干扰（比如食神混杂、伤官混杂），反而会降低格局的质量。格局清比格局多重要得多。

### 6.2 误解二：飞天禄马格、壬骑龙背格是上等格局

**错误说法：** 民间命理书和网上的很多文章都在讲"飞天禄马格""壬骑龙背格"等特殊格局，认为遇到这些格局就是大富大贵之命。

**经典原文反驳：**
【任氏曰】"其余外格我端，余务考群书，俱不从五行正理，尽属谬谈。"

【任氏曰】"壬辰日为'壬骑龙背'……何不再取壬午、壬申、壬戌、壬子，谓骑猴马犬鼠之背乎？"

→ 解读：任铁樵用自己的考证得出结论：这些不从五行正理的"外格"全是谬谈。他的反问非常精彩——如果壬辰是骑龙背，那壬午（骑马背）、壬申（骑猴背）、壬戌（骑狗背）、壬子（骑鼠背）怎么就不成立了？逻辑显然说不通。

**正确理解：** 格局判断应以五行生克制化为核心，而不是靠名字听起来好听的"特殊格局"。任氏严批的这些伪格在清代就已经存在，到今天网上仍然流传，你应当擦亮眼睛，以五行正理为准绳。

### 6.3 误解三：墓库逢冲必发

**错误说法：** 辰戌丑未这四个地支在命局中如果互相冲撞，就能"冲开墓库"，发大财、升大官。"墓库逢冲必发"是命理界的经典口诀。

**经典原文反驳：**
【任氏曰】"此造支全四库逢冲，俗作杂气财官也，不知丑未逢冲，不特官星受伤，而且冲去库根……前则逢冲，官印两伤，名利无成，此则不动，名成利遂。可知墓库逢冲必发者，谬也。"

→ 解读：任氏用两个命造的对比（命造八和命造九）直接证明了"墓库逢冲必发"是错误的：

- 命造八：地支丑未辰戌四库全冲 → 结果是"家业破尽，无子而亡"
- 命造九：地支丑未遥隔不相冲 → 结果是"名成利遂"

同样是墓库，冲的反而败了、不冲的反而成了。

**正确理解：** 墓库逢冲要看具体情况：

1. 冲的双方旺衰如何？
2. 冲是否破坏了格局中的用神？
3. 冲是否动了命局的核心根基？
4. 整体格局是喜冲还是不喜冲？

一概而论"逢冲必发"是不符合五行正理的。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **八格取用流程**——先观月令，次看透干，再究司令，然后取用，分清浊
2. **格正用真行运不悖则名利自来**——这是命局的上限
3. **格破用损谓之有病**——格局不完美是常态，关键是看运能否补救
4. **病药说**——格局缺陷如疾病，大运如药物，药对则愈
5. **伪格三例**——飞天禄马、壬骑龙背、六阴朝阳均为五行正理之外的谬谈
6. **墓库逢冲不一定发**——要结合格局用神判断

### 7.2 与其他篇章的关联脉络

- **天干篇**：决定了日主和各种天干的性情，是格局分析的基础
- **地支篇**：决定了地支藏干和合冲刑害，是格局的气机根基
- **配合篇**：格局中的"用神"与"喜忌"配合
- **理气篇**：格局的旺衰来源于理气
- **知命篇**：格局的最终目的是"知天命"

### 7.3 进阶学习路径建议

八格篇是进阶级篇章，学完后建议：

1. **回顾基础**：如果发现自己对天干、地支的理论还不够熟悉，重新复习这两篇
2. **学习方局篇**：深入理解三合三会对格局的影响
3. **学习形象篇**：特殊格局的综合判断
4. **大量看命造**：找《滴天髓阐微》中其他篇章的命造来自行分析，验证所学

难度对比：天干=地支（基础级）< 八格（进阶级）。八格篇的难度在于综合运用——它需要你同时掌握天干、地支、配合等前几篇的理论，并且在命造中活学活用。
