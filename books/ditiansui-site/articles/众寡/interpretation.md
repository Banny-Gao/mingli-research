## 一、入门导读

### 1.1 这一篇在学什么

众寡篇讲八字中"强众"与"强寡"两种对立势力的处理原则。当强势的多数派对抗弱势的少数派时，趋势在于消灭少数派（扶众抑寡）；当强势的少数派对抗弱势的多数派时，趋势在于成全少数派（扶寡抑众）。本篇教你如何在强弱对抗中找到正确的站队方向。

### 1.2 本篇在整个子平体系中的地位

众寡篇是通神论第32篇，紧接"隐显"（31篇）之后。隐显讲的是吉凶神的藏露策略，众寡则进一步讲强弱势力对抗时如何选择站队。两篇构成"藏露策略→对抗策略"的递进关系。之后再进入震兑（33篇）和坎离（34篇）的具体五行对抗场景。本篇属于**专题级**难度，需要对日主强弱判断有基础认识。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **判断"强众"与"强寡"**——区别多数派强势和少数派强势两种局面
2. **掌握正确的站队原则**——强众欲去寡时顺势去掉寡；强寡欲成众时借势成就寡
3. **区分两条分析路径**——以日主为中心还是以四柱全局为中心来判断众寡
4. **识别财官众寡的实战案例**——能拆解任铁樵给出的命造

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
强众而敌寡者，势在去其寡；强寡而敌众者，势在成乎众。

【原注】
强寡而敌众者，喜强而助强者吉；
强众而敌寡者，恶敌而敌众者滞。

【任氏曰】
众寡之说，强弱之意也，须分日主四柱两端而论也。
如以日主分众寡，如日主是火，生于寅、卯、巳、午月，官星是水，
四柱无财，反有土之食伤，即使有财，财无根气，不能生官，
此日主之党众，敌官星之寡，势在尽去其官，岁运宜扶众抑寡则吉。
如以四柱分众寡，则分四柱之强弱，然又要与日主符合，弗反背为妙。
假如水星是官星，休囚无气，土是伤官，当令得时，其势以去其官星，岁运亦宜制官为美；
日主是火，亦要通根得气，则能生土，或有木而克土，则日主自能化木，转转相生，
所谓日主符合者也。
官星是水，虽不及时，却有财生助，或财星当令，或成财局，
此官星虽寡，得财星扶则强，岁运宜扶寡而抑众者吉。
虽举财官而论，其余皆同此论。
```

### 2.2 原文逐句精解

【原句】"强众而敌寡者，势在去其寡；强寡而敌众者，势在成乎众"

→ 解读：开篇点题。如果强势的多数派（强众）与弱势的少数派（寡）对立，趋势在于去掉这个少数派；如果强势的少数派（强寡）在对抗弱势的多数派（众），趋势在于成全这个少数派（让它壮大以对抗众）。核心是：**跟强的一方站在一起**，不要逆势而为。

【原注第1句】"强寡而敌众者，喜强而助强者吉"

→ 解读：当少数派是强者时（强寡对抗众），应该喜欢这个强者并帮助它，这样做吉利。也就是说，少数派虽然人少但力量大，你要站在它这边。

【原注第2句】"强众而敌寡者，恶敌而敌众者滞"

→ 解读：当多数派是强者时（强众对抗寡），你应该厌恶（去）那个少数派，如果反而与多数派为敌就会阻滞不顺。"恶敌"意为"厌恶并去除敌人（寡）"，"敌众"意为"与多数派为敌"。

【任氏曰第1段】"众寡之说，强弱之意也，须分日主四柱两端而论也"

→ 解读：任铁樵首先强调，众寡的核心其实是强弱，并且需要从"日主"和"四柱"两个角度来分析。不是你人数多就一定强，也不是你人数少就一定弱。

【任氏曰第2段】"如以日主分众寡，如日主是火，生于寅、卯、巳、午月，官星是水，四柱无财，反有土之食伤，即使有财，财无根气，不能生官，此日主之党众，敌官星之寡，势在尽去其官，岁运宜扶众抑寡则吉"

→ 解读：以日主为中心来分析众寡。举例：日主是火（丙丁），生在寅卯巳午月——木火旺相，日主强。官星是水（壬癸），这是日主的敌人。如果四柱没有财星（金）来生水助官，反而有土（食伤）来克水，那么日主一方"党众"（帮派大），官星一方"党寡"（势力小），趋势应该"尽去其官"——彻底去除官星，岁运也帮助日主一方（扶众），抑制官星（抑寡）。

【任氏曰第3段】"如以四柱分众寡，则分四柱之强弱，然又要与日主符合，弗反背为妙"

→ 解读：以四柱全局分析众寡。这时候要看四柱整体哪一方强、哪一方弱，同时要与日主的方向保持一致，不能背离。

【任氏曰第4段】"假如水星是官星，休囚无气，土是伤官，当令得时，其势以去其官星，岁运亦宜制官为美；日主是火，亦要通根得气，则能生土，或有木而克土，则日主自能化木，转转相生，所谓日主符合者也"

→ 解读：以四柱全局分析：官星（水）休囚无力，而伤官（土）当令得时——伤官的气势要去掉官星。如果日主是火（火能生土），并且通根有气，就能生土助伤官之势。如果还有木来克土，日主火又能化木生土（木生火、火生土），这就是"转转相生"——日主与四柱的势力方向一致，即"日主符合"。

【任氏曰第5段】"官星是水，虽不及时，却有财生助，或财星当令，或成财局，此官星虽寡，得财星扶则强，岁运宜扶寡而抑众者吉"

→ 解读：反过来，官星虽然是水（休囚），但如果四柱有财星（金）生助它，或财星当令、或成财局（金局），那么官星虽然表面"寡"（人少），得到财星的扶助就成了"强寡"（少数派但力量强）。这时岁运应该帮助这个"强寡"（扶寡），抑制"众"（抑众）。

## 三、核心义理深度解析

### 3.1 众寡的本质——强者为尊，数不胜力

**原文依据**

【任氏曰】"众寡之说，强弱之意也"

**义理解读**

众寡的关键不是人数多少，而是**力量强弱**。所谓"众"不是人多，而是"势众"（力量大的一方）；所谓"寡"不是人少，而是"势寡"（力量小的一方）。核心原则只有一个：**顺应强者，不要逆强者之势**。

这个原则在实际运用中分为两种情况：

- **强众（强者且人多） + 寡（弱者）** → 去除这个弱者，因为强者要消灭它
- **强寡（强者但人少） + 众（弱者且人多）** → 帮助这个强者去成事，因为强者虽然人少但力量大

这不仅仅是"以多欺少"那么简单——有时少数派力量反而更强（如以毒蛇对抗绵羊群）。关键是看谁的力量真正占主导地位。

**命造验证**

【命造一】戊辰 乙丑 戊戌 辛酉

此造四柱全土（戊辰、戊戌、丑、辰），土势极旺，是典型的"强众"（土之党众多）。乙木官星孤悬无根（乙木在土旺之地休囚至极），为"寡"。此局是"强众而敌寡"——土势要去掉乙木这个"寡"。初运丙寅丁卯，木运助官，反而引起土与木的交战，刑耗多端。到戊辰、己巳运，土旺扶众去寡，捐纳出仕，一路顺遂。

### 3.2 两条分析路径——日主中心与四柱全局

**原文依据**

【任氏曰】"须分日主四柱两端而论也"

**义理解读**

任铁樵提出了两条分析路径：

**路径一：以日主为中心**
以日主为一方，以官（或杀、财等）为敌方，比较多方的力量。日主党众（帮派多）就支持日主去官；官星党众就支持官星制日主。这条路径适合分析日主与用神/忌神之间的对抗关系。

**路径二：以四柱全局为中心**
以四柱中两支对抗势力来分析——哪方强就支持哪方，同时要检查日主是否与强势方"符合"（日主的五行是否生扶强势方）。如果日主与强势方方向一致，就是"符合"，格局清贵；如果日主与强势方方向相悖（如强方要克官，日主却生官），就会出现背离，格局不顺。

**命造验证**

【命造二】戊午 壬戌 丁卯 癸卯

此造戌土当令（九月），戊土伤官透干当权。官星癸水无根（被戊土克、被戌土伤）。四柱全局来看：伤官（土）是强众，官星（水）是弱寡。日主丁火，通根午火、得卯木生，能生土助伤官之势——日主与强方"符合"。

初运北方水地（亥子丑），官星得势，但这是扶寡抗众（违背强方意志），所以一事无成。至丙寅丁卯，木火运助日主生土，经营发财；至戊辰己巳，去尽官煞，一子登科。

## 四、命造实例拆解

### 命造一：土众敌木寡——舍官从伤

【命造一】乾造：戊辰 乙丑 戊戌 辛酉

大运：丙寅 → 丁卯 → 戊辰 → 己巳 → 庚午 → 辛未
（出自任铁樵《滴天髓阐微·众寡》）

| 柱  | 干支 | 分析                           |
| --- | ---- | ------------------------------ |
| 年  | 戊辰 | 戊土比肩，辰为湿土             |
| 月  | 乙丑 | 乙木正官（孤悬无力），丑为湿土 |
| 日  | 戊戌 | 戊土日主，戌为燥土             |
| 时  | 辛酉 | 辛金伤官，酉金泄土之秀气       |

**格局**：从强格（土势专旺）。**用神**：火土（助土势）。**喜**：火土运。**忌**：木运（官星逆土势）。

**结论**：此造全局土势极旺——四柱中天元戊土两透，地元辰丑戌酉皆与土相关。乙木官星孤悬在月干，无半点根气（辰丑中虽有乙木余气但被土掩），是典型的"强众敌寡"。大势要去乙木这个官星（去寡）。初运丙寅丁卯（木火），木运助官，等于扶寡抗众——结果刑耗多端。到戊辰、己巳土运，扶众去寡，出仕为官。庚午辛未火土运，一路顺遂。

### 命造二：火旺成强寡——从儿格

【命造二】乾造：癸丑 壬戌 丙午 庚寅

大运：辛酉 → 庚申 → 己未 → 戊午 → 丁巳 → 丙辰
（出自任铁樵《滴天髓阐微·众寡》）

| 柱  | 干支 | 分析                       |
| --- | ---- | -------------------------- |
| 年  | 癸丑 | 癸水官星，丑土晦火         |
| 月  | 壬戌 | 壬水七杀，戌土燥火         |
| 日  | 丙午 | 丙火日主坐午火阳刃（强根） |
| 时  | 庚寅 | 庚金偏财，寅午戌合火局     |

**格局**：从旺格（火势成局）。**用神**：火（强寡）。**喜**：金水运（成众）？**忌**：火土运。

**结论**：此造是"强寡而敌众者，势在成乎众"的典型案例。丙火日主坐午火阳刃，寅午戌三合火局——这是"强寡"（火虽然只有一个丙午日柱，但三合火局力量极强）。年干癸水进气、月干壬水透出，还有庚金生水——这是"众"（表面人多势众）。但局中火势已成强寡，趋势在于"成乎众"——即让水（众）的力量成全火（强寡）的格局，而非水火交战。辛酉庚申运，金生水旺，水势得成（成乎众），遗业丰盈。己未、戊午火土运，助火克水，等于扶寡抗众——结果火土交战，父母双亡、破家败业。

## 六、常见误解与辨析

### 误解一：众寡就是比人数多少

**错误说法**：有人认为"众"就是天干地支中出现次数多的五行，"寡"就是出现次数少的五行。

**经典原文反驳**：

【任氏曰】"众寡之说，强弱之意也。……此官星虽寡，得财星扶则强。"

**正确理解**：众寡的本质是"强弱"，不是简单的计数。一个官星虽然只出现一次，但如果它有财星（金）生助，力量就很强，成了"强寡"——虽然人少但实力强。反过来，日主虽然天干地支出现很多次，如果无根无气，也只是"弱众"。数个数是入门，判断力量才是关键。

### 误解二：强众去寡就意味着一定要直接消灭弱方

**错误说法**：认为"去其寡"就是大运流年一定要直接克掉那个弱势的五行。

**经典原文反驳**：

【任氏曰】"势在去其寡……岁运宜扶众抑寡则吉。"

**正确理解**："去其寡"不是必须直接消灭，而是"扶众抑寡"——通过强化强方的力量，自然达到抑制弱方的效果。好比一个国家强大后，边境小国的威胁自然解除，不一定需要派兵讨伐。实战中，"去"的含义更接近"使其不能为害"，而非物理消灭。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"强众而敌寡者，势在去其寡；强寡而敌众者，势在成乎众"**——全篇总纲，众寡判断的核心原则。
2. **"众寡之说，强弱之意也"**——众寡的本质是强弱，不是数量。
3. **"须分日主四柱两端而论"**——既要看日主立场，也要看全局立场。
4. **"此官星虽寡，得财星扶则强"**——孤木不成林，但寡可借势成强。
