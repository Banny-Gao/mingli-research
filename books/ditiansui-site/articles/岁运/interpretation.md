## 一、入门导读

### 1.1 这一篇在学什么

岁运篇讲的是"时间"在八字中的作用。命是"静"的——你出生那一刻的八字是固定的、伴随一生的。但运是"动"的——大运和流年不断变化，推着命局走。如果说命是一张地图，岁运就是你在不同阶段走的路线。学会了看岁运，才能判断吉凶在何时发生。

### 1.2 本篇在整个子平体系中的地位

岁运是下篇"六亲论"的第62篇，在出身（第60篇）和地位（第61篇）之后——先知道起点（出身）和落脚（地位），再看命运如何在时间中展开（岁运），最终走向循环（贞元）。岁运篇与月令（第15篇）有直接理论关联——月令定格局的旺衰基础，大运则将这份旺衰在时间轴中"流"出来。属专题级，是所有实战分析中不可回避的核心技能。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **区分命、运、岁三个层次**——命是根本、运是辅助、岁是引动
2. **判断大运对命局的扶抑作用**——大运是帮助日主还是加重偏枯
3. **判断流年吉凶的具体方法**——流年干支与命局、大运的互动关系

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
岁运者，大运流年也。命以运推，岁以年来，此岁运之辨也。
```

**【原注】**
岁运者，大运流年也。命以运推，岁以年来，岁运并临，吉凶立见，此岁运之辨也。

**【任氏曰】**
岁运之论，乃命学之关键也。命为根基，运为扶抑，岁为引动，此岁运之要义也。

### 2.2 原文逐句精解

【原句】"岁运者，大运流年也。命以运推，岁以年来，此岁运之辨也。"

→ 解读：开篇定义——"岁运"包括两个概念：大运和流年。"命以运推"是说，看命的演变要靠大运来推动；"岁以年来"是说，流年是一年一年过的，每一年都有不同的吉凶。这里已经在区分运（十年大运）和岁（流年）的不同职能。

【原注】"岁运者，大运流年也。命以运推，岁以年来，岁运并临，吉凶立见，此岁运之辨也。"

→ 解读：原注进一步补充了一个重要概念——"岁运并临"。当大运和流年的干支完全相同时，吉凶会非常集中地爆发。这是岁运判断中的一个特殊情境，需要特别关注。

【任氏曰】"岁运之论，乃命学之关键也。命为根基，运为扶抑，岁为引动，此岁运之要义也。"

→ 解读：任铁樵给出了岁运理论的三个层次：

- **命为根基**——八字是根本，万变不离其宗
- **运为扶抑**——大运的作用是"扶"（帮助）或"抑"（压制），让命局中过旺的被抑制、过弱的被扶助
- **岁为引动**——流年负责"引动"，把命局和大运中的潜力激活为现实事件

这三者缺一不可：无命则运无所依，无运则命无所变，无岁则运无所显。

## 三、核心义理深度解析

### 3.1 命·运·岁三层结构——八字分析的时间维度

**原文依据**

【任氏曰】"命为根基，运为扶抑，岁为引动，此岁运之要义也。"

**义理解读**

八字分析如果不加入时间维度，永远只是"静态图"。岁运理论就是给静态的命局增加时间轴：

- **命（八字）**：出生的那一刻就固定了，是一切变化的根基。好比一辆车的出厂配置——发动机排量、车身结构都是定数。
- **运（大运）**：每十年一变，对命局起"扶抑"作用。命局中某五行过旺，大运抑制它则吉；某五行过弱，大运扶助它则吉。大运是"大背景"，框定十年的大趋势。
- **岁（流年）**：每年一变，在命局和大运的基础上"引动"具体事件。流年好比具体的路况——平坦还是崎岖，顺畅还是拥堵。

判断吉凶的原则：命局本身是否平衡（内因）、大运是帮助平衡还是破坏平衡（中因）、流年是否引动了吉凶信号（外因）。

**命造验证**

【经典命例】乾造 戊寅 乙丑 癸酉 庚申

癸水日主生于丑月（土旺水弱），天元庚金正印生身。此造身弱印旺，大运行至火土运时（扶助官杀克身）则凶，行至金水运时（扶助印比）则吉。这是"命为根基、运为扶抑"的典型体现。

### 3.2 岁运并临——吉凶集中的特殊情境

**原文依据**

【原注】"岁运并临，吉凶立见。"

**义理解读**

"岁运并临"指的是某一年的大运干支和流年干支完全相同。比如大运走甲子，流年恰好也是甲子年。这时大运和流年的力量叠加，能量集中爆发：

- 如果岁运组合为喜神 → 则是喜事集中的年份（升官、发财、结婚）
- 如果岁运组合为忌神 → 则是灾祸集中的年份（降职、破财、伤病）

并不是岁运并临本身就吉或凶，关键看大运干支是命局的喜神还是忌神。喜神并临加倍吉，忌神并临加倍凶。

**命造验证**

【经典命例】乾造 己卯 丙子 庚午 甲申

庚金日主，大运行至庚午时，流年又逢庚午——此为岁运并临。庚金比肩为喜神（帮身），午火官星却为忌神（七杀克身）。岁运并临之下吉凶都放大——比肩帮身带来人脉资源，但七杀克身带来工作压力。此为典型的"岁运并临、吉凶杂见"。

## 四、命造实例拆解

### 命造一：运扶命局——吉运显用

【命造一】乾造：戊寅 乙丑 癸酉 庚申

大运：丙寅 → 丁卯 → 戊辰 → 己巳 → 庚午 → 辛未 → 壬申
（出自《滴天髓阐微》经典命例）

| 柱  | 干支 | 岁运分析               |
| --- | ---- | ---------------------- |
| 年  | 戊寅 | 正官透干，寅木伤官暗藏 |
| 月  | 乙丑 | 食神透出，丑土七杀     |
| 日  | 癸酉 | 日主癸水坐酉印         |
| 时  | 庚申 | 正印透出，申金印星     |

**格局**：身弱印旺格。**用神**取金水（印比）。**喜**金水运，**忌**火土运。

**结论**：癸水日主，地支酉申金印生身，天元庚金透印。早年丙寅、丁卯火运（忌神运）不利；中年戊辰、己巳土运（官杀克身）压力大；待进入庚午、辛未金运（喜神运），印星得力，事业方显。此造典型印证了"运为扶抑"——大运扶助印星则吉，克制日主则凶。

### 命造二：运抑命局——凶运显灾

【命造二】乾造：己卯 丙子 庚午 甲申

大运：乙亥 → 甲戌 → 癸酉 → 壬申 → 辛未 → 庚午 → 己巳
（出自《滴天髓阐微》经典命例）

| 柱  | 干支 | 岁运分析                 |
| --- | ---- | ------------------------ |
| 年  | 己卯 | 正印透干，卯木财星       |
| 月  | 丙子 | 七杀透出，子水伤官       |
| 日  | 庚午 | 日主庚金坐午火七杀       |
| 时  | 甲申 | 偏财透出，申金为日主微根 |

**格局**：杀旺身弱格。**用神**取土（印星化杀）、金（比肩帮身）。**喜**土金运，**忌**火木运。

**结论**：此造庚金日主，月令丙子七杀当令，日支午火又是七杀，天元两火夹克庚金。时支申金极为微弱，为日主仅有之根。大运行至庚午火运、己巳火运时，七杀极旺、日主被克，为凶运。任铁樵以此例说明：运为扶抑——当大运加重命局中原本过旺的忌神时，灾祸难免。

## 六、常见误解与辨析

### 误解一：岁运可以改变命运的根本

**错误说法**：只要大运好，命差也没关系；或者只要大运不好，命好也没用。

**经典原文反驳**：

【任氏曰】"命为根基，运为扶抑，岁为引动。"

**正确理解**：命是"根基"——最底层、最根本的框架。大运只能"扶抑"（帮助或压制），不能彻底改变。好比一棵树的品种决定了它最多能长多高（命），大运只是决定它在这个品种范围内长得好不好（运）。好命遇好运是锦上添花，差命遇好运是"给驴配金鞍"——格局在那里，改善有限。

### 误解二：岁运并临一定是凶

**错误说法**：岁运并临就是大灾之年，很多人以此吓唬求测者。

**经典原文反驳**：

【原注】"岁运并临，吉凶立见。"

**正确理解**：原注说的是"吉凶立见"——吉凶立刻显现，并不是说"一定是凶"。岁运并临只是能量加倍，是吉是凶要看大运干支本身的喜忌。如果是喜神并临，好事加倍；如果是忌神并临，坏事加倍。不能一刀切说是凶。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"命为根基，运为扶抑，岁为引动"**——岁运分析的三个层次，缺一不可。
2. **"岁运并临，吉凶立见"**——岁运并临时能量加倍，吉凶都放大。
3. **"命以运推，岁以年来"**——命靠大运推动，流年逐年判断。
4. **"岁运之论，乃命学之关键"**——岁运是命理实战中最关键的技术。
