## 功能

在八字格局分析中，当发现伤官与官星同时出现（或伤官单独构成格局），调用此技能进行伤官格局的专项分析。此技能判断"伤官见官"是否成立、伤官的格局类型（五格）、以及对应的用神与喜忌。

## 输入

- **四柱八字**：年柱、月柱、日柱、时柱对应的天干地支（字符串）
- **日主强弱评估**：由上游分析提供的日主旺衰判断（身旺/身弱/中和）
- **官星标记**：已标记出的正官和七杀所在位置（可选）

## 处理逻辑

1. **判断伤官见官是否可"见"**
   a. 检查原局是否有财星：
      - 有财 → 基本可见官（财能泄伤生官）
      - 无财 → 基本不可见官
   b. 检查日主强弱与伤官轻重组合：
      - 身弱伤旺有印 → 可见官
      - 身旺伤旺 → 见财而不见官
      - 伤旺无财 → 遇官有祸
   c. **依据**：伤官篇【原注】"大率伤官有财，皆可见官；伤官无财，皆不可见官"

2. **定位伤官格局**
   a. 扫描配置，匹配五种格局：
      - 日主弱、伤官旺、有印 → 伤官用印
      - 日主旺、伤官旺 → 伤官用财
      - 日主弱、伤官旺、无印 → 伤官用劫
      - 日主旺、无财官、有伤 → 伤官用伤
      - 日主旺、比劫多、财衰伤轻 → 伤官用官
   b. **依据**：伤官篇【任氏曰】"有伤官用印，伤官用财，伤官用劫，伤官用伤，伤官用官"

3. **解析"为祸百端"适用性**
   a. 检查是否满足"身弱用比劫帮身"
   b. 如是 → 伤官见官真为祸
   c. 如有印 → 伤官见官反为福
   d. **依据**：伤官篇【任氏曰】"所谓伤官见官为祸百端者，皆日主衰弱……若局中有印，见官不但无祸，而且有福也"

4. **确定用神与喜忌**
   a. 根据格局类型确定：
      - 用印格：喜印旺身旺，忌财旺伤旺
      - 用财格：喜财旺伤旺，忌印旺劫旺
      - 用劫格：喜印旺，忌财官
      - 用伤格：喜财伤，忌官印
      - 用官格：喜财官，忌伤印
   b. **依据**：伤官篇【任氏曰】各格喜忌论述
