## 功能

在收到八字四柱后，根据出生月份和全局五行组合，评估命局的整体寒暖状态（极寒、偏寒、适中、偏暖、极热），判断是否需要调候，以及调候用神是否有力。此技能应在格局分析和用神判断之前调用——调候需求会直接影响用神的选取。

## 输入

- **四柱八字**：年柱、月柱、日柱、时柱对应的天干地支（字符串，格式如"丙子 己亥 辛酉 己亥"）
- **出生季节**：月令地支（已包含在四柱中）
- **用神分析结果**：已确定的格局用神（可选，用于调候与格局的冲突判断）

## 处理逻辑

1. **月令气候判定**
   a. 根据月支确定出生季节的基本气候：
      - 亥子丑月 → 冬（寒）
      - 寅卯辰月 → 春（寒暖过渡）
      - 巳午未月 → 夏（热）
      - 申酉戌月 → 秋（凉燥过渡）
   b. **依据**：寒暖篇【原注】"冬令水冷，须火调之；夏令火炎，须水润之"

2. **全局寒暖综合评估**
   a. 统计全局五行中"暖"的要素：
      - 火的数量和力量（丙丁巳午）
      - 木能生火（木多间接增暖）
   b. 统计全局五行中"寒"的要素：
      - 水的数量和力量（壬癸亥子）
      - 金能生水（金多间接增寒）
   c. 结合月令和全局五行，综合评分命局寒暖（-10极寒到+10极热）
   d. **依据**：寒暖篇【任氏曰】"寒暖之论，乃气候之辨也"

3. **调候需求判定**
   a. 综合评分 ≤ -6 → "极寒"，急需火调候
   b. 综合评分 -5 ~ -3 → "偏寒"，建议火调候
   c. 综合评分 -2 ~ +2 → "寒暖适中"，无需专门调候
   d. 综合评分 +3 ~ +5 → "偏暖"，建议水调候
   e. 综合评分 ≥ +6 → "极热"，急需水调候
   f. **依据**：寒暖篇【原注】"调候为急，不可不知也"

4. **调候用神有效性检查**
   a. 需火调候 → 检查丙丁巳午是否出现且有力
   b. 需水调候 → 检查壬癸亥子是否出现且有力
   c. 调候之神无根或受克 → "调候无力"，命局危险
   d. **依据**：寒暖篇【任氏曰】"寒命宜暖，暖命宜寒，此调候之道也"

5. **调候与格局整合**
   a. 调候用神 == 格局用神 → "调候与格局一致"（最佳）
   b. 调候用神 ≠ 格局用神 → "调候与格局冲突"（优先调候，行运再补格局）
   c. 无需调候 → "仅以格局为准"

## 输出

```typescript
interface HanNuanReport {
  /** 命局气候基础 */
  climate: {
    birthSeason: "春" | "夏" | "秋" | "冬";
    coldWarmScore: number;           // -10 ~ +10
    condition: "极寒" | "偏寒" | "适中" | "偏暖" | "极热";
  };
  
  /** 寒暖要素统计 */
  elements: {
    warmFactors: {
      fireCount: number;             // 火的数量
      woodCount: number;             // 木的数量（生火）
      description: string;
    };
    coldFactors: {
      waterCount: number;            // 水的数量
      metalCount: number;            // 金的数量（生水）
      description: string;
    };
  };
  
  /** 调候需求 */
  adjustmentNeeded: {
    needsAdjustment: boolean;
    priority: "极高（急需）" | "中度（需要）" | "无（适中）";
    recommendedElement: string | null;  // "火" 或 "水" 或 null
  };
  
  /** 调候用神检查 */
  adjustmentCandidate: {
    element: string | null;
    strength: "有力" | "有根但弱" | "无根或受克" | "不存在";
    details: string;
  };
  
  /** 调候与格局整合 */
  integration: {
    relation: "调候与格局一致" | "调候与格局冲突" | "无需调候";
    advice: string;
    finalDecision: string;           // 最终取用建议
  };
}
```
