# 配合 解读

> 《滴天髓阐微》通神论第六篇
> 难度分级：基础级

---

## 一、入门导读

### 1.1 这一篇在学什么

配合篇讲的是干支之间的配合取舍之道——天干和地支如何互相配合、各十神（财、官、印、比劫、食伤等）皆可为用神，关键在于判断日主衰旺和用神喜忌之后进行"去留舒配"（取舍安排）。

### 1.2 本篇在整个子平体系中的地位

配合篇位于知命（第四篇）和理气（第五篇）之后，是通神论方法论部分的总结性篇章。知命篇建立了"顺逆之机"的辩证思维，理气篇落实到气的进退判断，而配合篇则讲述如何将这些判断综合运用——将四柱干支进行通盘配合、精细取舍，最终确定用神。本篇属于**基础级**难度，但在方法论上是前三篇（知命、理气、配合）的集大成者。

### 1.3 学习目标

学完本篇后，你应能：

1. 说清楚"配合干支"的具体含义——不只是排列组合，而是审量日主衰旺和用神喜忌后的取舍安排
2. 理解"不拘财官印绶比劫伤官，皆可为用"的用神观念——用神不限于财官
3. 掌握"去留舒配"的基本方法——判断哪些因素该留、哪些该去
4. 将配合、知命、理气三篇的方法论串连成一条完整的分析链条

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
配合干支仔细详，定人福祸与灾祥。

【原注】天干地支，相为配合，仔细推详其进退之机，则可以断人之福祸灾祥。

【任氏曰】此章乃谈命之要领也。配合干支，必须正理搜微推详，与衰旺喜忌之理，不可将四柱干支蓦论，专从奇格、异局、神煞等类妄谈，以致祸福无凭，吉凶不验。

命中之理，只存用神，不拘财、官、印绶、比劫、伤官、六亲，皆可为用，勿以名之美者恶之，恶者为佳。果能审日主之衰旺，用神之喜忌，当抑则抑，当扶则扶，所谓去留舒配，取裁确当，则运途否泰，赫然明白，祸福灾祥，无不验矣。
```

### 2.2 原文逐句精解

**【原文首句】** "配合干支仔细详，定人福祸与灾祥。"

→ 解读：开篇点明配合的重要性——干支配合必须仔细推详，因为这是判断一个人福祸灾祥的关键。"仔细详"三个字强调配合不是粗看，需要精细入微的功夫。

**【原注】** "天干地支，相为配合，仔细推详其进退之机，则可以断人之福祸灾祥。"

→ 解读：原注进一步明确：天干和地支是互相配合的关系，而且需要"仔细推详"。注意这里提到了"进退之机"——呼应了理气篇的概念，说明配合不是机械的排列，而是要结合气的进退之势来通盘考虑。

**【任氏曰第一段】** "此章乃谈命之要领也。配合干支，必须正理搜微推详，与衰旺喜忌之理，不可将四柱干支蓦论，专从奇格、异局、神煞等类妄谈，以致祸福无凭，吉凶不验。"

→ 解读：任铁樵开门见山指出：配合篇是"谈命之要领"——命理分析的关键所在。他强调配合干支必须遵循"正理"（正确的理论），要"搜微推详"（推敲到细微处），必须结合**衰旺喜忌之理**。

他再次批评了不看衰旺喜忌、只看奇格异局神煞的错误做法——这正是他在知命篇中也批评过的流弊。任氏在多个篇章反复批判同一种错误，说明这是他深感当时命学界普遍存在的弊病。

**【任氏曰第二段】** "命中之理，只存用神，不拘财、官、印绶、比劫、伤官、六亲，皆可为用，勿以名之美者恶之，恶者为佳。果能审日主之衰旺，用神之喜忌，当抑则抑，当扶则扶，所谓去留舒配，取裁确当，则运途否泰，赫然明白，祸福灾祥，无不验矣。"

→ 解读：这一段是全篇的精髓，分三个层次：

第一层——"命中之理，只存用神"：命理的核心就在于用神。用神是命局的调节中枢，抓住了用神就抓住了命理分析的关键。

第二层——"不拘财、官、印绶、比劫、伤官、六亲，皆可为用，勿以名之美者恶之，恶者为佳"：所有十神都可以作为用神，不要因为名称好听（如财、官）就偏爱，也不要因为名称不好听（如**伤官**、**劫财**）就厌弃。这打破了普通人对财官吉、伤官凶的机械认知。

第三层——"果能审日主之衰旺，用神之喜忌，当抑则抑，当扶则扶，所谓去留舒配，取裁确当"：配合的具体方法——审查日主的衰旺，确定用神的喜忌，该抑制的抑制，该扶助的扶助。这就是"去留舒配"（去除多余的、保留需要的、舒展得当的、搭配合理的）。

## 三、核心义理深度解析

### 3.1 用神的普适性——不拘一格，皆可为用

**论点名称**：用神的广泛性

**原文依据**：
【任氏曰】"命中之理，只存用神，不拘财、官、印绶、比劫、伤官、六亲，皆可为用"

**义理解读**：
这是配合篇最核心的观点：用神不应该局限于财官。普通人对命理的常见理解是"财官好，伤官差"，但任铁樵明确指出——这是错误的。

用神的本质是调节命局平衡的力量，任何十神都可以承担这个角色：
- 日主过弱，**印星**或**比劫**可能就是最好的用神
- 日主过旺，财星、官星、**食神**、伤官都可以是用神
- 格局特殊时，伤官、劫财等"凶神"反而能救局

关键在于"当抑则抑，当扶则扶"——该克制的时候克制，该扶助的时候扶助。用神没有固定的"好"与"坏"，只看它是否适合当前的命局。

**命造验证**：
任铁樵在其他篇章中曾以用伤官为用的格局为例：日主旺而财星弱时，可以用伤官生财——伤官虽然是"凶神"，但在此格局中恰恰是推动财富的关键。反之，如果日主弱而伤官旺，则伤官成为忌神。这验证了"勿以名之美者恶之，恶者为佳"的道理。

### 3.2 去留舒配——命局调整的核心方法

**论点名称**：去留舒配

**原文依据**：
【任氏曰】"果能审日主之衰旺，用神之喜忌，当抑则抑，当扶则扶，所谓去留舒配，取裁确当"

**义理解读**：
"去留舒配"四个字各自有含义：
- **去**：去掉命局中多余、有害的因素——通过克制、冲合等方式减轻其影响
- **留**：保留对命局有利的因素——保护用神不受伤害
- **舒**：舒展不畅的因素——通关、引化，使气的流通顺畅
- **配**：搭配恰当的因素——使各十神之间形成协调的关系

通俗地说：命局就是一个团队，有的成员能力太强需要约束，有的成员太弱需要扶持，有的成员之间有矛盾需要调解，有的成员空缺需要补齐。"去留舒配"就是做团队管理的功夫。

**命造验证**：
配合篇虽然没有给出具体命造，但任铁樵在其他篇章（如后续的天干、地支、八格等篇）中对大量命造的分析，本质上都是"去留舒配"的具体应用——审查日主衰旺，确定用神喜忌，决定哪个因素该去、哪个该留、哪个该舒、哪个该配。

## 四、命造实例拆解

### 【命造一】伤官为用——以凶为用

**格局**：日主强旺，伤官有力，伤官泄秀生财为用。

**任氏曰原文依据**："不拘财、官、印绶、比劫、伤官、六亲，皆可为用，勿以名之美者恶之，恶者为佳"

**取用依据**：日主过旺需要宣泄，财星需要源头生发，此时伤官泄秀生财，一举两得。伤官虽然是"凶神"（名称不好听），但在此格局中恰恰是最合适的选择。

**逐项分析**：
- 核心矛盾：日主过旺，财富不足
- 用神选择：以伤官泄日主之旺气，生发财星
- "恶者为佳"：伤官在此是"恶名"却起"佳用"
- 验证逻辑：若用财官反而会与旺日主冲突，不如用伤官转圜

**结论**：此造显示配合的本质是看实际效果而非名称好坏。伤官在常规理解中是"凶神"，但在恰当格局中可从"凶"转为"佳用"。

### 【命造二】印星为用——以扶为用

**格局**：日主衰弱，印星生扶日主为用。

**任氏曰原文依据**："果能审日主之衰旺，用神之喜忌，当抑则抑，当扶则扶"

**取用依据**：日主过弱，需要生扶。印星生扶日主是最直接的方式。

**逐项分析**：
- 核心矛盾：日主衰弱
- 用神选择：以印星生扶日主、以比劫辅助日主
- "当扶则扶"：日主弱则扶，这是配合的基本原则
- 辅助配合：印星之外，视命局情况可配比劫共同扶助日主

**结论**：此造展示了配合中最基本的"扶抑"逻辑——日主弱则扶，日主旺则抑。印星作为扶助日主的首选，与伤官为用形成对照，说明同一理论在不同格局中的灵活应用。

## 六、常见误解与辨析

### 误解一：用神一定得是财官

**错误说法**：看命主要看财官，财官好就是好命，财官不好就是坏命。

**经典原文反驳**：
【任氏曰】"不拘财、官、印绶、比劫、伤官、六亲，皆可为用，勿以名之美者恶之，恶者为佳。"

**正确理解**：任铁樵明确否定了这种观点。用神的选择取决于日主的衰旺和命局的需要，而不是十神名称的好听与否。日主弱时，比劫可以是用神；日主旺而财弱时，伤官生财可以是用神。反之，如果日主太弱，财官反而可能是忌神。

### 误解二：配合就是简单的五行生克

**错误说法**：配合篇就是讲干支之间的相生相克关系。

**经典原文反驳**：
【任氏曰】"配合干支，必须正理搜微推详，与衰旺喜忌之理，不可将四柱干支蓦论"

**正确理解**：配合不是简单的五行生克公式，而是"与衰旺喜忌之理"结合的精密分析。"搜微推详"要求推敲到细微处——不仅要看表面的干支关系，还要结合日主衰旺、用神喜忌、气的进退之势来综合判断。"蓦论"二字指出了粗暴套用公式的错误做法。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"命中之理，只存用神"**——命理分析的核心就是抓用神
2. **"不拘财、官、印绶、比劫、伤官、六亲，皆可为用"**——所有十神都能做用神，不限于财官
3. **"勿以名之美者恶之，恶者为佳"**——不要因名称好听与否判断十神的好坏
4. **"当抑则抑，当扶则扶"**——配合的八字方针
5. **"去留舒配，取裁确当"**——配合四步法：去多余、留必要、舒不畅、配得当

### 7.2 与其他篇章的关联脉络

- **知命（第四篇）**：配合篇的前提方法论，顺逆之机决定了配合的方向
- **理气（第五篇）**：配合篇中的"进退之机"呼应了理气篇的核心概念
- **八格（第十二篇）**：配合篇的用神理论在八格篇中得到具体展开
- **衰旺（第十七篇）**：配合篇的"审日主之衰旺"需要衰旺篇的技术支撑

### 7.3 进阶学习路径建议

- **继续**：天干（第七篇）——在掌握配合方法论后，学习具体天干的特性与配合
- **辅助阅读**：配合理解配合篇，可参考《子平真诠》中"论用神"及"论用神成败"章节
- **难度对比**：配合篇为基础级，但方法论的系统性强，建议与知命、理气连续学习后进入进阶级篇目
- **学习建议**：学完配合篇后，应尝试用"去留舒配"四步法分析至少一两个命造，检验理解程度
