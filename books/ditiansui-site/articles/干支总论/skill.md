## 功能

在收到八字四柱后，先分析天干之间的相合关系（甲己合土、乙庚合金等五种合化），再分析地支之间的六合关系（子丑合、寅亥合等六种），然后判断天干是否通根于地支、干支配合是否得当，最后以日主为中心评估命局的整体协调性。此技能在完成了天道篇三元分析之后、进入八格判断之前使用。

## 输入

- **四柱八字**：年柱、月柱、日柱、时柱的天干和地支（字符串，如"甲子 己巳 甲午 甲子"）
- **月令**：出生月份地支（用于判断合化条件的成败）
- **性别**：乾造或坤造（可选）

## 处理逻辑

1. **天干相合分析**
   a. 检测四柱中是否存在以下对子：
      - 甲 + 己 → 合土
      - 乙 + 庚 → 合金
      - 丙 + 辛 → 合水
      - 丁 + 壬 → 合木
      - 戊 + 癸 → 合火
   b. 判断合化是否成功：
      - 月令支持化出五行 → 标记为"化"
      - 月令不支持 → 标记为"绊"
   c. **依据**：干支总论篇【原句】"甲己合土，中正之合；乙庚合金，仁义之合..."

2. **地支六合分析**
   a. 检测四柱中是否存在以下对子：
      - 子 + 丑 → 合土
      - 寅 + 亥 → 合木
      - 卯 + 戌 → 合火
      - 辰 + 酉 → 合金
      - 巳 + 申 → 合水
      - 午 + 未 → 合火
   b. 判断合化还是绊住
   c. **依据**：干支总论篇【原句】"子丑合土，寅亥合木，卯戌合火，辰酉合金，巳申合水，午未合火"

3. **干支通根分析**
   a. 对每个天干，检查其在四个地支中是否有通根：
      - 甲 → 寅（禄）、卯（旺）、辰（余气）
      - 乙 → 卯（禄）、寅（旺）、辰（余气）
      - 丙/丁 → 巳午未
      - 戊/己 → 辰戌丑未
      - 庚/辛 → 申酉戌
      - 壬/癸 → 亥子丑
   b. **依据**：干支总论篇【原注】"天干地支，相为表里"

4. **日主强势分析**
   a. 定位日柱天干（日主）
   b. 判断日主是否通根于地支
   c. 判断日主在月令中的旺相休囚死状态
   d. 评估日主总体强弱
   e. **依据**：干支总论篇【原句】"日干为身，身旺则可以任财，身弱则不能敌官"

5. **配合评估**
   a. 检查干支之间是否存在相克关系（如甲申——甲木坐申金被克）
   b. 检查天干与地支的五行趋势是否一致
   c. **依据**：干支总论篇【原句】"干支配合，最宜辨其喜忌"

## 输出

```typescript
interface GanZhiReport {
  /** 天干相合分析 */
  tianGanHe: {
    pairs: Array<{
      gan1: string;
      gan2: string;
      heType: string;         // 合土/合金/合水/合木/合火
      status: "化" | "绊";
      heName: string;         // 如"中正之合"
    }>;
  };
  /** 地支六合分析 */
  diZhiHe: {
    pairs: Array<{
      zhi1: string;
      zhi2: string;
      heType: string;
      status: "化" | "绊";
    }>;
  };
  /** 干支通根 */
  tongGen: {
    hasRoot: boolean;
    rootBranches: Record<string, string[]>;  // 每个天干通根的地支
    rootQuality: "强根" | "中根" | "弱根" | "无根";
  };
  /** 日主分析 */
  riZhu: {
    stem: string;
    strength: "旺" | "中" | "弱";
    analysis: string;
  };
  /** 总体评估 */
  assessment: string;
}
```
