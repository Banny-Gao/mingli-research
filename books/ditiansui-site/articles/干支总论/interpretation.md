## 一、入门导读

### 1.1 这一篇在学什么

干支总论篇讲的是天干与地支之间的配合关系——天干如何通过地支得根、干支如何相合化气、以及天干地支各自的性情如何影响命局判断。它是天干篇和地支篇的"综合应用篇"，教你如何把干支知识活用在命局之中。

### 1.2 本篇在整个子平体系中的地位

干支总论位于天干篇（第7篇）和地支篇（第8篇）之后，是干支基础理论的综合与提升。天干篇讲了十天干的性情，地支篇讲了十二地支的关系，干支总论则将两者结合起来，讲干支如何配合、如何互动。它同时是后续形象（第10篇）、八格（第12篇）等综合格局篇章的理论基础。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **辨识天干相合**——区分甲己中正之合、乙庚仁义之合等十种天干合化
2. **辨识地支相合**——理解子丑合、寅亥合等六种地支合化
3. **判断干支配合**——看懂天干是否通根于地支、干支之间的生克关系
4. **把握四柱结构**——理解年柱为祖基、月柱为提纲、日柱为命主、时柱为归宿的四柱框架
5. **以日主为中心分析命局**——学会以日干为核心断身旺身弱和用神方向

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
干支者，天干地支也。天干有十，地支有十二。六十甲子，干支配合而成。

【原注】天干地支，相为表里，天干主动，地支主静，动静相感，上下相临，而命于是乎定。

【任氏曰】干支总论，乃万法之宗也。天干有甲乙丙丁戊己庚辛壬癸十数，地支有子丑寅卯辰巳午未申酉戌亥十二数。干支相配，成六十甲子，以纪年月日时，而命之富贵贫贱吉凶寿夭，皆由此定焉。

干支配合，最宜辨其喜忌。干与干合，支与支合，干与支合，其情各异。相生相克，相冲相刑，相害相破，变化不一。学者须先明干支之性情，次辨配合之喜忌，则命理思过半矣。

甲己合土，中正之合；乙庚合金，仁义之合；丙辛合水，威制之合；丁壬合木，淫慝之合；戊癸合火，无情之合。此十天干相合也。

地支相合者，子丑合土，寅亥合木，卯戌合火，辰酉合金，巳申合水，午未合火。相合则亲，相破则离，相冲则动，相刑则伤。命局之中，须观其合而化之，抑或绊之，以为用也。

干支并用，最重日主。日主者，日干也。日干为身，身旺则可以任财，身弱则不能敌官。年月日时，四柱配合，而贵贱贫富判矣。年为祖基，月为提纲，日为命主，时为归宿。此四柱之大略也。学者当循序渐进，先明干支性情，次辨四柱配合，然后可以言命也。
```

### 2.2 原文逐句精解

【原句第1句】"干支者，天干地支也。天干有十，地支有十二。六十甲子，干支配合而成。"

→ 解读：开篇点明主题。天干有十个（甲、乙、丙、丁、戊、己、庚、辛、壬、癸），地支有十二个（子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥）。天干和地支按照阳配阳、阴配阴的规则，组成六十甲子（60种不同的干支组合）。

【原注】"天干地支，相为表里，天干主动，地支主静，动静相感，上下相临，而命于是乎定。"

→ 解读：原注点出了干支关系的核心——"表里"关系。天干好比一个人的外表（显露在外的部分），地支好比一个人的内在（深藏的部分）。天干代表外在的主动力量，地支代表内在的被动根基。两者动静相感、上下相应，共同构成了命运的判断依据。

【任氏曰第1段】"干支总论，乃万法之宗也。天干有甲乙丙丁戊己庚辛壬癸十数，地支有子丑寅卯辰巳午未申酉戌亥十二数。干支相配，成六十甲子，以纪年月日时，而命之富贵贫贱吉凶寿夭，皆由此定焉。"

→ 解读：任铁樵将干支总论提升到"万法之宗"的高度。所有命运判断——富贵贫穷、吉凶祸福、寿命长短——都源自干支配合。这里呼应了天道篇"三元万法宗"的说法：天元（天干）和地元（地支）的配合，是万种命理方法的根本。

【原句第2段】"干支配合，最宜辨其喜忌。干与干合，支与支合，干与支合，其情各异。相生相克，相冲相刑，相害相破，变化不一。学者须先明干支之性情，次辨配合之喜忌，则命理思过半矣。"

→ 解读：这一段提出了干支学习的总方法——分三步走：第一步，明干支之性情（天干篇、地支篇已完成的任务）；第二步，辨配合之喜忌（本篇的核心任务）；第三步，在此基础上判断命理。任铁樵说走完前两步，"命理思过半矣"——你已经掌握一半多了。

【原句第3段】"甲己合土，中正之合；乙庚合金，仁义之合；丙辛合水，威制之合；丁壬合木，淫慝之合；戊癸合火，无情之合。此十天干相合也。"

→ 解读：这一段列出了十天干相合的完整列表。每种合都有自己的性情特征：
- 甲己合土 → 中正之合（甲为阳木、己为阴土，甲己合化土，性情中正）
- 乙庚合金 → 仁义之合（乙木为仁、庚金为义，合化为金，仁义兼备）
- 丙辛合水 → 威制之合（丙火威猛、辛金柔韧，合化为水，以威制胜）
- 丁壬合木 → 淫慝之合（丁火阴柔、壬水阳刚，合化为木，情欲交织）
- 戊癸合火 → 无情之合（戊土老阳、癸水少阴，合化为火，少情寡义）

需要注意的是，合化是有条件的——需要月令提携或其他条件配合才能真正化气。

【原句第4段】"地支相合者，子丑合土，寅亥合木，卯戌合火，辰酉合金，巳申合水，午未合火。相合则亲，相破则离，相冲则动，相刑则伤。命局之中，须观其合而化之，抑或绊之，以为用也。"

→ 解读：这一段列出地支六合。与天干合化一样，地支合也分"化"和"绊"两种情况：合而成化，则化出新的五行力量；合而绊住，则双方相互牵制，不能与其他干支发生作用。判断是化还是绊，需要结合月令和其他地支的情况。

【原句第5段】"干支并用，最重日主。日主者，日干也。日干为身，身旺则可以任财，身弱则不能敌官。年月日时，四柱配合，而贵贱贫富判矣。年为祖基，月为提纲，日为命主，时为归宿。此四柱之大略也。学者当循序渐进，先明干支性情，次辨四柱配合，然后可以言命也。"

→ 解读：最后一句话将全篇落在"日主"上。日主（日干）是八字的核心——生辰那天的天干就代表你自己。身旺（日主力量强），才能驾驭财星；身弱（日主力量弱），难以抵挡官杀克制。四柱各有含义：年柱代表祖上基业，月柱（提纲）代表大局气候，日柱代表自身命运，时柱代表归宿终结。最后再次强调学习方法——先明干支性情，再辨四柱配合，然后才谈得上真正的命理分析。

## 三、核心义理深度解析

### 3.1 天干五合——性情与条件

**原文依据**

【原句】"甲己合土，中正之合；乙庚合金，仁义之合；丙辛合水，威制之合；丁壬合木，淫慝之合；戊癸合火，无情之合。"

**义理解读**

天干相合是干支关系中最重要的互动形式之一。合化分两种情形：
- **合而成化**：在月令推动下，被合的两种天干转化为另一种五行（如甲己合化土）
- **合而不化（绊住）**：双方相互牵制，各自保留本气但无法自由行动

判断合化的条件：月令必须支持化出的五行；或者原局中化出的五行已经旺盛。如果甲己合土，月令是辰戌丑未（土月），则化土可能性大；如果月令在春秋，甲乙木旺的季节，则甲己多半被绊住而不化。

**命造验证**

【原注参考命造】

任铁樵在《滴天髓阐微》中多次以天干合化断吉凶。如甲木日主见己土在月干或时干，形成甲己合，日主被合牵绊，会对个人意志产生影响——甲木的正直被己土的柔顺所化，性格和运势都会因此转向。

### 3.2 地支六合——化与绊的分别

**原文依据**

【原句】"地支相合者，子丑合土，寅亥合木，卯戌合火，辰酉合金，巳申合水，午未合火。"

**义理解读**

地支六合与天干五合的逻辑类似。地支六合的本质是一个地支中的余气与另一个地支的本气发生感应：

- 子丑合土：子中癸水与丑中己土相合
- 寅亥合木：寅中甲木与亥中壬水（水生木）相合
- 卯戌合火：卯中乙木与戌中丁火相合
- 辰酉合金：辰中戊土生辛金相合
- 巳申合水：巳中丙火与申中壬水相克激荡成水
- 午未合火：午中丁火与未中己土相合

在实际分析中，被合的双方如同"牵手"——力量被绑定在一起，无法再与其他地支发生关系。这在判断吉凶时很关键。

### 3.3 日主为核心——四柱职能体系

**原文依据**

【原句】"年月日时，四柱配合，而贵贱贫富判矣。年为祖基，月为提纲，日为命主，时为归宿。"

**义理解读**

四柱各自有不同的职能：
- **年柱**：代表祖荫、遗传、早年环境。看一个人的根基，先看年柱的干支配合
- **月柱**：代表大环境、命局的"气候"。月令（月支）是判断五行旺衰的核心
- **日柱**：代表自身（日干为命主，日支为配偶宫），是整个八字的中心
- **时柱**：代表晚运、归宿、子女。看一个人最终结局，时柱是关键

## 四、命造实例拆解

### 命造一：甲己合土——日主被合

【命造一】乾造：甲子 己巳 甲午 甲子

大运：庚午 → 辛未 → 壬申 → 癸酉 → 甲戌 → 乙亥
（出自任铁樵《滴天髓阐微·配合》）

**格局**：建禄格（甲木生于巳月）。**用神**取印星（壬癸水）。**喜**金水运生扶日主，**忌**火土运加重消耗。

**逐字分析**：日主甲木，月干己土与之相合（甲己合土）。虽然日主是甲木，但被己土合住，日主的自由意志受到影响——甲木本来想向上升发，却被己土"绊住"转而为土。年干和时干也有甲木比肩帮扶，但巳月火土当令，甲木的根基弱。这种"日主被合"的情况，意味着命主的命运很大程度上受到外界环境的牵制。

**结论**：此造正是"甲己合土，中正之合"的实例。日主甲木被月干己土合住，命主行事中正温和，但也因此失去自主性，需要大运印星生助日主才能得志。

### 命造二：地支六合——辰酉合金

【命造二】乾造：庚辰 乙酉 庚申 丙子

大运：丙戌 → 丁亥 → 戊子 → 己丑 → 庚寅 → 辛卯
（出自任铁樵《滴天髓阐微·地支》）

**格局**：阳刃格（庚金生于酉月）。**用神**取火（官杀制刃）。**喜**火土运，**忌**金水运（助旺金势）。

**逐字分析**：辰酉合金——年支辰与月支酉相合，辰为湿土，酉为金旺之地，辰酉合化金，使金气更旺。天元三金（庚、庚、庚），地支双申一酉，加辰酉合金之势，金旺极。时支子水泄金秀气。此造中辰酉合金是关键——如果没有这个合，金气虽旺但不至于"专旺"；有了辰酉合金，金势达到顶峰。

**结论**：此造展示了地支六合的威力。辰酉合金使得金气更加专旺，命局走向"从革格"方向。这正是地支相合"其情各异"的体现——同样的辰酉，在命局中既可能是"化金"也可能是"绊住"，需要结合其他条件判断。

## 六、常见误解与辨析

### 误解一：天干合化一定会改变五行的本性

**错误说法**：有人认为天干一旦相合，就完全变成化出的五行，原来的五行不再起作用。

**经典原文反驳**：

【原句】"干与干合，支与支合，干与支合，其情各异。"

**正确理解**：天干相合的结果不是单一的，而是"其情各异"的——可能是合化成功（变成新五行），也可能是合而不化（绊住）。即使合化成功，原来的五行也不是完全消失，只是被压制或转化了部分力量。任铁樵在论述中强调"观其合而化之，抑或绊之，以为用也"，明确说需要判断是化还是绊。

### 误解二：日主一定要旺才是好命

**错误说法**：很多人认为身旺就是好命，身弱就是差命。

**经典原文反驳**：

【原句】"日干为身，身旺则可以任财，身弱则不能敌官。"

**正确理解**：这句话常被误解。身旺"可以"任财，不等于身旺就富贵。身旺身弱只是判断用神方向的条件——身旺的人需要看是否能驾驭财官；身弱的人需要用印比帮扶。富贵贫贱看的是整个命局的配合，而非单一的身旺身弱。很多身弱的命局，大运一路帮扶，反而成就高于身旺无依之命。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"天干主动，地支主静，动静相感，上下相临"**——干支关系的基本性质。
2. **"甲己合土，乙庚合金，丙辛合水，丁壬合木，戊癸合火"**——十天干相合口诀。
3. **"子丑合土，寅亥合木，卯戌合火，辰酉合金，巳申合水，午未合火"**——地支六合口诀。
4. **"年为祖基，月为提纲，日为命主，时为归宿"**——四柱职能定位。
5. **"先明干支性情，次辨四柱配合，然后可以言命"**——学习方法总纲。

### 7.2 与其他篇章的关联脉络

- **前序关系**：完全依赖天干篇（第7篇）和地支篇（第8篇）的知识。
- **直接关联篇目**：
  - **天干（07）**——提供十天干性情的具体知识
  - **地支（08）**——提供十二地支刑冲合害的具体知识
  - **形象（10）**——干支合化后的成象成局是形象篇的基础
  - **配合（06）**——干支配合的核心思想与配合篇一脉相承

### 7.3 进阶学习路径建议

干支总论之后的推荐学习顺序：

1. **八格（12）**——将干支配合知识应用到八格体系
2. **形象（10）**——干支合化后的形象判断
3. **源流（20）**——干支的源流关系

必修天干 + 地支 + 干支总论三篇连读，这是《滴天髓》命理分析的工具箱。
