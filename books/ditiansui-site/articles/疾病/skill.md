## 功能

通过分析八字命局中五行力量的偏枯（过旺或过弱）情况，推断命主容易出现的健康问题倾向和对应的身体脏腑部位。AI 执行者应在需要分析命局的健康倾向时调用本技能。注意：此技能提供的是命理层面的健康预警，不能替代医学诊断。

## 输入

- **四柱八字**：年柱、月柱、日柱、时柱的天干地支（字符串格式）
- **日主**：日柱天干的五行属性（自动提取）
- **五行力量分布**：可选，未提供时本技能默认按四柱五行统计
- **当前大运**（可选）：用于判断大运是否引动了偏枯的五行
- **关注部位**（可选）：如"肝胆""脾胃"等特定关注部位

## 处理逻辑

1. **统计五行力量**
   - 统计四柱天干和地支藏干的五行出现频次
   - 按月令加权计算五行力量
   - 得出五行力量排名，识别最强和最弱的五行
   - 依据：【正文】"五行偏枯，疾病随之"

2. **过旺五行分析**
   - 找出过旺的五行（显著高于均值）
   - 判断该五行所克的五行对应的脏腑
   - 木旺 → 克土 → 脾胃易出问题
   - 火旺 → 克金 → 肺、皮肤易出问题
   - 土旺 → 克水 → 肾、泌尿系统易出问题
   - 金旺 → 克木 → 肝、胆、筋骨易出问题
   - 水旺 → 克火 → 心脏、循环系统易出问题

3. **过弱五行分析**
   - 找出过弱的五行（显著低于均值）
   - 过弱五行本身对应的脏腑功能偏弱
   - 木弱 → 肝胆功能不足
   - 火弱 → 心脏功能偏弱
   - 土弱 → 脾胃消化弱
   - 金弱 → 肺气弱、免疫力低
   - 水弱 → 肾功能偏弱
   - 依据：【任氏曰】"五行各有偏枯，疾病随之"

4. **大运引动检查**
   - 若有当前大运信息，检查大运是否引动了偏枯的五行
   - 大运助旺过旺五行 → 加剧失衡，发病风险增加
   - 大运补足过弱五行 → 缓解失衡，健康改善

5. **输出报告**
   - 列出各五行偏枯类型（过旺/过弱）
   - 列出对应的可能健康问题
   - 标注严重程度（高/中/低）
   - 给出大运引动的评估
