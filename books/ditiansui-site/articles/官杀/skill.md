## 功能

在八字格局分析中，当发现正官与七杀同时出现（或单独七杀制化复杂），调用此技能进行官杀格局的专项分析。此技能输出官杀关系的判定结果（可混/不可混、格局类型定位），不重复基础格局判断。

## 输入

- **四柱八字**：年柱、月柱、日柱、时柱对应的天干地支（字符串）
- **日主强弱评估**：由上游分析提供的日主旺衰判断（身旺/身弱/中和）
- **官杀标记**：已标记出的正官和七杀所在位置（可选）

## 处理逻辑

1. **判定官杀性质**
   a. 根据日主强弱判定官杀的实际性质：
   - 日主旺 → 七杀可视为官（身旺者以杀为官）
   - 日主弱 → 正官也可能构成压力（身弱者以官为杀）
     b. **依据**：官杀篇【任氏曰】"杀即官也，身旺者以杀为官；官即杀也，身弱者以官为杀"

2. **判断官杀可混性**
   a. 日主旺相 → 官杀可混（有可）
   b. 日主休囚 → 官杀不可混（有不可）
   c. 检查是否"势在于官"或"势在于杀"：
   - 势在官且官有根 → 依附之杀岁运助官为真混
   - 势在杀且杀有权 → 依附之官岁运助杀为真混
     d. **依据**：官杀篇【原注】"势在于官，官有根……势在于杀，杀有权……"

3. **定位官杀格局类型**
   a. 扫描原局配置，按顺序匹配六种格局：
   - 制杀太过（杀弱被众多食伤克制）→ 制杀太过格
   - 官杀并见且日主能担 → 官杀混杂格
   - 有合官留杀或合杀留官 → 合官留杀/合杀留官格
   - 有食神制杀 → 食神制杀格
   - 有印化杀 → 杀重用印格
   - 有财生杀，杀势不重 → 财滋弱杀格
     b. **依据**：官杀篇【任氏曰】"今将杀分六等"

4. **确定用神与喜忌**
   a. 根据格局类型确定用神方向：
   - 财滋弱杀 → 用财杀
   - 杀重用印 → 用印星
   - 食神制杀 → 用食神
   - 合官留杀 → 留杀为主
   - 官杀混杂 → 视身强弱定
   - 制杀太过 → 扶杀
     b. **依据**：官杀篇各格任氏曰分析

interface GuanShaReport {

  basic: {
    officialPosition: string[] // 正官所在位置

  /** 格局定位 */

    type: '财滋弱杀' | '杀重用印' | '食神制杀' | '合官留杀' | '官杀混杂' | '制杀太过'

    yongShen: string // 用神
    likes: string[] // 喜
    dislikes: string[] // 忌
  }
  /** 处理建议 */
  suggestion: {
    action: '去官留杀' | '去杀留官' | '用印通关' | '食神制杀' | '扶抑并用' | '顺其势'
    detail: string // 详细建议
  }
}
