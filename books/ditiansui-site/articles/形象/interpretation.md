# 《滴天髓阐微·形象》解读

## 一、入门导读

### 1.1 这一篇在学什么

形象篇讲八字中"像"（五行聚合后形成的气象形态）的分类和判断方法。它教你识别三种特殊的命局形态——两气成象（两种五行均衡共存）、独象（一种五行独旺）、全象（三种及以上五行并存）——以及如何根据这些形态选取用神和判断运势。

### 1.2 本篇在整个子平体系中的地位

形象篇位于天干（第7篇）、地支（第8篇）、干支总论（第9篇）之后，是与方局篇（第11篇）并列的"特殊格局"篇章。形象篇讲抽象的"气象"分类（两气、独象、全象），方局篇讲具体的"方局"组合（寅卯辰、亥子丑等）。形象篇是方局篇的理论基础，方局篇是形象篇的实战细化。两篇连读，才能全面掌握特殊格局的判断方法。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **区分两气成象、独象、全象**——理解三种形象的定义和特征
2. **判断两气成象的喜忌**——知道哪些五行可以混入，哪些不能
3. **理解独象行化地的原理**——专旺格为何需要食伤泄秀
4. **判断形全形缺**——学会"旺则宜泄宜伤，衰则喜帮喜助"的灵活应用
5. **避免"吉凶颠倒"**——理解任铁樵反复强调的不可执一而论

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
两气合而成象，象不可破也。

【原注】天干属木，地支属火，天干属火，地支属木，其象则一。若见金水则破，余仿此。

【任氏曰】两气双清，非独木火二形也，如土金、金水、水木，木火、火土，相生各半五局。即相克之五局亦是也，如木土、土水、水火、火金、金木之各半用敌也。相克务须均敌，切忌偏重偏轻。若用金水，则火土不宜夹杂；如取水木，则火金不可交争。木火成象者，最怕金水破局。水火既济者，尤忌土来止水。各既如此，取运亦仿此而行。一路澄清，必位高而禄重；中途混乱，恐职弃而家倾。故此格最难全美，而看法贵在至精。若生而复生，乃是流通之妙；倘克而遇化，亦为和合之情。或谓理仅两神，似嫌狭少，不知格分十种，尽费推详。

（命造：甲午丁卯甲午丁卯、丁卯乙巳丁卯乙巳、丙午戊戌丙午戊戌、戊戌辛酉戊戌辛酉、戊戌癸亥戊戌癸亥、癸亥己未癸亥己未）

五气聚而成形，形不可害也。

【原注】木必得水以生之，火以行之，土以培之，金以成之。是以成形于要紧之地，或过或缺，则害。余皆仿之。

【任氏曰】木之成形，食伤泄气，水以生之；官杀交加，火以行之；印绶重叠，土以培之；财轻劫重，金以成之。成形于得用之地，庶无偏枯之病，何患名利不遂乎？即举木论，五行皆可成形，变仿此而推。若四柱无成，成之于岁运又无成处，则终身碌碌，凶多吉少，有志难伸矣。

（命造：壬戌壬子甲子戊辰、戊寅乙卯甲辰辛未、癸未乙卯甲戌乙亥）

独象喜行化地，而化神要昌。

【原注】一者为独，曲直炎上之类也。所生者为化神，化神宜旺，则其气流行，然后行财官之地方可。

【任氏曰】权在一人，曲直炎上之类是也。化者，食伤也，局中化神昌旺，岁运行化神之地，名利皆遂也。八字五行全备，固为合宜，而独象乘权，亦主光亨。木日，或方或局全，不杂金为曲直；火日，或方或局全，不杂水为炎上；土日，四库皆全，不杂木为稼穑；金日，或方或局全，不杂火为从革；水日，或方或局全，不杂土为润下。皆从一方之秀气，不同六格之常情。必要得时当令，遇旺逢生。但体质过于自强，须以引通为妙，而气势必有所关，务须审察其情。

（命造：甲寅丁卯甲辰丙寅、己未丁丑戊子己未、丙寅甲午丙戌乙未、庚申乙酉庚戌庚辰、壬子辛亥癸丑壬子）

全象喜行财地，而财神要旺。

【原注】三者为一，有伤官而又有财也，主旺喜财旺，而不行官杀之地方可。

【任氏曰】三者为全，非专论伤官与财也。伤官生财，固为全矣，而官印相生，财官并见，岂非全乎？伤官生财，日主旺相，固宜财运，倘四柱比劫多见，财星被劫，官运必佳，伤官运更美。须观局中意向为是。

（命造：戊申丙辰丁卯甲辰、己巳辛未丙午丁酉）

形全者宜损其有余，形缺者宜补其不足。

【原注】如甲木生于寅、卯、辰月，丙火生于巳、午、未月，皆为形全；戊土生于寅、卯、辰月，庚金生于巳、午、未月，缺。余仿此。

【任氏曰】形全宜损，形缺宜补之说，即子平"旺则宜泄宜伤，衰则喜帮喜助"之谓也。命书万卷，总不外此二句...须将四字分用为是，通变在一"宜"字。宜泄则泄之为妙，宜伤则伤之有功。泄者食伤也，伤者官杀也。均是旺也，或泄之有害，而伤之有利；或泄之有利，而伤之有害，所以泄伤两字，宜分而用之也。宜帮则帮之为切，宜助则助之为佳。帮者比劫也，助者印绶也。均是衰也，帮之则凶，而助之则吉；或帮之则吉，而助之则凶，所以帮助两字，亦宜分而用之也。

（命造：丁丑庚戌庚子甲申、戊申壬戌庚申乙酉、庚申辛巳丙辰乙未、壬子癸丑丙午壬辰）
```

### 2.2 原文逐句精解

【原句第1段】"两气合而成象，象不可破也。"

→ 解读：当命局中只有两种五行（如木火、土金等），而且两者的力量均衡配合，就形成了"两气成象"。这种"象"一旦形成就非常稳定——最忌讳的就是其他五行（破局的五行）混入，破坏这种平衡。

【原注第1段】"天干属木，地支属火，天干属火，地支属木，其象则一。若见金水则破，余仿此。"

→ 解读：原注以木火为例：天干全是木、地支全是火（或者反过来），这就形成了统一的"象"。如果出现了金或水（木火成象的忌神），就破局了。其他组合类推。

【任氏曰第1段】"两气双清，非独木火二形也，如土金、金水、水木，木火、火土，相生各半五局。即相克之五局亦是也，如木土、土水、水火、火金、金木之各半用敌也。相克务须均敌，切忌偏重偏轻。"

→ 解读：任铁樵大大扩展了两气成象的范围——不仅是木火，还有土金、金水、水木、火土共五种相生组合，以及木土、土水、水火、火金、金木共五种相克组合，合计十种。关键要求：相克的两气必须力量均等（各半），不能有偏重。

【任氏曰第2段】"若生而复生，乃是流通之妙；倘克而遇化，亦为和合之情。"

→ 解读：两气成象中，如果两气是相生关系（如木→火），且在格局中流通顺畅，非常妙；如果两气是相克关系（如木→土），但能通过合化来调和（如甲己合土），也是可以的。

【原句第2段】"五气聚而成形，形不可害也。"

→ 解读：当命局中五行俱全（五种气都汇聚了），就形成了"形"。这种"形"需要保全，不能被"害"——即每个五行都得到应有的生扶和调节。

【原注第2段】"木必得水以生之，火以行之，土以培之，金以成之。是以成形于要紧之地，或过或缺，则害。"

→ 解读：以木为例，木要成形需要水的滋润、火的疏导、土的培植、金的成器。任何一个环节"过"或"缺"，都会伤害木的"形"。其他五行同理。

【原句第3段】"独象喜行化地，而化神要昌。"

→ 解读：当命局中只有一种五行独旺（如全盘木、全盘火等），这就是"独象"。独象最喜欢走的是"化地"——即能将其旺盛的力量转化出去的五行方向。这种转化的力量（食伤）称为"化神"，化神必须昌盛有力。

【任氏曰第3段】"权在一人，曲直炎上之类是也。化者，食伤也，局中化神昌旺，岁运行化神之地，名利皆遂也。"

→ 解读：任铁樵将五种独象完整列出：

- **曲直**：木日主，方局全，不杂金
- **炎上**：火日主，方局全，不杂水
- **稼穑**：土日主，四库全，不杂木
- **从革**：金日主，方局全，不杂火
- **润下**：水日主，方局全，不杂土

"独象"的用神方向：化神（食伤）昌盛是最佳状态。木独象喜欢火（食伤），火独象喜欢土（食伤），等等。

【原句第4段】"全象喜行财地，而财神要旺。"

→ 解读：当命局中三种及以上五行并存，日主又旺，这就是"全象"。全象最喜欢走财运——财星（日主所克之五行）要旺。

【任氏曰第4段】"全象者，伤官生财，固为全矣，而官印相生，财官并见，岂非全乎？"

→ 解读：任铁樵纠正了原注"全象就是伤官生财"的偏狭说法。全象的范围更广——伤官生财、官印相生、财官并见等，只要是三种以上五行并存的都算。关键还是"观局中意向"——看命局的实际倾向。

【原句第5段】"形全者宜损其有余，形缺者宜补其不足。"

→ 解读：五行形全的（得月令之助，如木生于寅卯辰月）应当抑制其过旺的部分；五行形缺的（不得月令之助，如土生于寅卯辰月）应当补充其不足的部分。

【任氏曰第5段】"宜泄则泄之为妙，宜伤则伤之有功。泄者食伤也，伤者官杀也。均是旺也，或泄之有害，而伤之有利；或泄之有利，而伤之有害，所以泄伤两字，宜分而用之也。"

→ 解读：任铁樵将"损有余"细化为"泄"（食伤）和"伤"（官杀）两种不同的抑制方式。同样的旺势，有时泄比伤好，有时伤比泄好——关键在于"宜"字。这是形象篇最精微的心法：同样是旺，需要判断是"泄之"有利还是"伤之"有利。

## 三、核心义理深度解析

### 3.1 两气成象——十种组合与破局

**原文依据**

【任氏曰】"两气双清，非独木火二形也，如土金、金水、水木，木火、火土，相生各半五局。即相克之五局亦是也，如木土、土水、水火、火金、金木之各半用敌也。"

**义理解读**

两气成象共有十种可能：

**相生五局**（两气为相生关系）：

- 木火成象（木生火）
- 火土成象（火生土）
- 土金成象（土生金）
- 金水成象（金生水）
- 水木成象（水生木）

**相克五局**（两气为相克关系）：

- 木土成象（木克土）
- 土水成象（土克水）
- 水火成象（水克火）
- 火金成象（火克金）
- 金木成象（金克木）

相克的关键要求：两气必须势均力敌。一旦偏重，克的一方就变成了"欺凌"，格局就破了。

**命造验证**

例一（甲午丁卯甲午丁卯，木火各半，相生成象）：命局中木火两气循环相生——甲木生丁火，丁火生土（但不透出），地支午卯午卯，天干甲丁甲丁。金水全无，纯粹可观。任铁樵评"巳运丁火临官，南宫奏捷"——因巳为火之禄地，化神昌盛。庚运金入，官杀混局——金木相战，破象，降职。这是"象不可破"的铁证。

例二（丙午戊戌丙午戊戌，火土各半，相生成象）：火土两气，丙戊俱旺。辛丑运湿土晦火，"秀气流行"——土为化神，湿土调节了火的燥烈。壬运壬年，水来激火——水火相战，破象，死于都中。

### 3.2 独象与化神——专旺格的用神逻辑

**原文依据**

【任氏曰】"权在一人，曲直炎上之类是也。化者，食伤也，局中化神昌旺，岁运行化神之地，名利皆遂也。"

**义理解读**

独象（专旺格）的核心在于"化神"——用食伤把旺盛的五行之气"转化"出去。独象的特点是"体质过于自强"，如同一头精力过剩的猛兽，需要给它一条"出路"——这就是食伤泄秀的意义。

五种独象的用神选取：

- **曲直格**（木独象）→ 喜火（食伤泄秀），忌金（官杀破局）
- **炎上格**（火独象）→ 喜土（食伤泄秀），忌水（官杀破局）
- **稼穑格**（土独象）→ 喜金（食伤泄秀），忌木（官杀破局）
- **从革格**（金独象）→ 喜水（食伤泄秀），忌火（官杀破局）
- **润下格**（水独象）→ 喜木（食伤泄秀），忌土（官杀破局）

**命造验证**

例三（甲寅丁卯甲辰丙寅，曲直格）：地支寅卯辰全，东方木方已成。天元双甲比肩，丁丙食伤泄秀。任铁樵评"少年科甲，早遂仕路之光"——化神（丙丁火）昌盛是关键。庚辛金运有丙丁回克，故能化险为夷。壬运破局伤秀——壬水印星生木，反而压制了化神，"降职归田不禄"。

### 3.3 形全形缺——"泄伤帮助"四字分用

**原文依据**

【任氏曰】"宜泄则泄之为妙，宜伤则伤之有功。泄者食伤也，伤者官杀也。均是旺也，或泄之有害，而伤之有利；或泄之有利，而伤之有害，所以泄伤两字，宜分而用之也。"

**义理解读**

这是形象篇最精妙的心法。同样是一个"旺"字，需要区分两种处理方式：

- **泄**（食伤泄秀）——适合"旺而无制"的情况，用食伤温和地导出旺盛之气
- **伤**（官杀克制）——适合"旺而有制"的情况，用官杀有力地约束旺势

同样是"衰"，也分两种处理方式：

- **帮**（比劫帮扶）——适合"衰而需要同气相助"的情况
- **助**（印绶生扶）——适合"衰而需要源头生助"的情况

任铁樵特别指出，两者不可混淆——"伤之有利，而泄之有害"和"泄之有利，而伤之有害"两种情况都存在，务必区分。

**命造验证**

例四（丁丑庚戌庚子甲申）：此造庚金秋生，金旺极。官星（丁火）虚脱无根，财星（甲木）临绝。初运土金（戊申、己酉）更加助金势，形伤破耗。丁未、丙午运，"助起官星"——火的官杀克制金的旺势，"家业鼎新"。这正是"伤之有功"——用官杀（伤）来制约过旺的金，有效果。

例五（戊申壬戌庚申乙酉）：此造金旺极（乙从庚化，地支申酉戌西方金方）。虽然有壬水泄秀，但有戊土紧克壬水，"不能引通泄其杀气"。行至丙寅运，"触其旺神"——火来克金，金旺极而抗，一败如灰。这正是"泄之有益，伤之有害"——用食伤（水）泄金才是正确路径，用官杀（火）克金反而激起反噬。

## 四、命造实例拆解

### 命造一：木火两气成象——"象不可破"

【命造一】乾造：甲午 丁卯 甲午 丁卯

大运：戊辰 → 己巳 → 庚午 → 辛未 → 壬申 → 癸酉
（出自任铁樵《滴天髓阐微·形象》）

**格局**：两气成象（木火各半）。**用神**取丁火伤官（秀气）。**喜**木火运，**忌**金水运（破局）。

**逐字分析**：

- 年柱甲午：甲木生丁火
- 月柱丁卯：丁火泄甲木
- 日柱甲午：甲木日主坐午火
- 时柱丁卯：丁火再泄甲木

四柱木火两气循环，金水全无，"纯粹可观"。丁火伤官为秀气（用神），巳运（火的禄地）丁火临官——"南宫奏捷，名高翰苑"。庚运金入——官杀混局，破两气之象——降职。"西方之水"行运进一步破象。

**结论**：此造是"两气成象，象不可破"的教科书案例。纯粹的两气格局，一旦破象（金来克木或水来克火）即刻应凶。

### 命造二：火土两气成象——破象应灾

【命造二】乾造：丙午 戊戌 丙午 戊戌

大运：己亥 → 庚子 → 辛丑 → 壬寅 → 癸卯 → 甲辰
（出自任铁樵《滴天髓阐微·形象》）

**格局**：两气成象（火土各半）。**用神**取戊土食神（秀气）。**喜**火土运或湿土运，**忌**水运（激火破局）。

**逐字分析**：

- 年柱丙午：丙火比肩
- 月柱戊戌：戊土食神
- 日柱丙午：丙火日主
- 时柱戊戌：戊土食神

火土各半，戊土食神为秀气。辛丑运湿土晦火——"秀气流行"——登乡榜。壬运壬年，水来激火——"水激丙火，则火灭"——死于都中。

**结论**：任铁樵特别指出，如果两戌换两辰（辰为湿土而非燥土），就不会燥烈到极点，"虽逢水运，亦不至大凶"。这说明两气成象中，用神的配合也影响命局的抗风险能力。

### 命造三：独象曲直格——"化神要昌"

【命造三】乾造：甲寅 丁卯 甲辰 丙寅

大运：戊辰 → 己巳 → 庚午 → 辛未 → 壬申 → 癸酉
（出自任铁樵《滴天髓阐微·形象》）

**格局**：曲直格（木独象）。**用神**取丙丁火（食伤化神）。**喜**火土运（泄秀生财），**忌**金水运（破局伤秀）。

**逐字分析**：

- 年柱甲寅：甲木得禄
- 月柱丁卯：丁火泄秀，卯木帮身
- 日柱甲辰：甲木日主，辰土培木
- 时柱丙寅：丙火食神泄秀

地支寅卯辰东方木方，天元甲丙丁火泄秀。"化神者，丙丁也"——独象的关键在于化神昌盛。庚辛金运有丙丁回克，故无大咎。壬运水来，破局伤秀——"降职归田不禄"。

**结论**：独象如同一条奔腾的大河——化神（食伤）就是分流的渠道。渠道通畅（化神昌），大河就能造福人间（名利双收）；渠道堵了（化神被破），大河就会泛滥成灾。

### 命造四：金独象从革格——"泄之有益，伤之有害"

【命造四】乾造：庚申 乙酉 庚戌 庚辰

大运：丙戌 → 丁亥 → 戊子 → 己丑 → 庚寅 → 辛卯
（出自任铁樵《滴天髓阐微·形象》）

**格局**：从革格（金独象）。**用神**取水（食伤泄秀）。**喜**水金运，**忌**火运（官杀触其旺神）。

**逐字分析**：

- 年柱庚申：庚金得禄
- 月柱乙酉：乙庚化金
- 日柱庚戌：庚金日主
- 时柱庚辰：庚金比肩

地支申酉戌金方，天元三庚加乙庚化金。"局无食伤之故"——没有水（食伤）泄秀，金气过于肃杀。丁亥戊子（水运）尚可，庚寅辛卯运——寅木暗藏火气，卯木破金——"一交寅运，阵亡"。因为局中无食伤化金，火来破金（官杀触旺神）则大凶。

**结论**：此造与命造三形成对比——命造三有丙丁火化神泄木秀气，此造无水泄金秀气，"肃杀之气太锐"。化神的有无，直接决定了独象命格的吉凶高度。

## 六、常见误解与辨析

### 误解一：两气成象必须是相生关系

**错误说法**：有人认为两气成象只能是相生的两气（如木火、土金），相克的不算。

**经典原文反驳**：

【任氏曰】"相克之五局亦是也，如木土、土水、水火、火金、金木之各半用敌也。相克务须均敌，切忌偏重偏轻。"

**正确理解**：任铁樵明确给出了十种两气成象——五种相生、五种相克。相克成象的核心要求是"各半用敌"——两气力量必须均等。只要力量均衡，即使相克也能形成稳定的"象"。

### 误解二：独象就是身旺，不需要用神

**错误说法**：独象（专旺格）既然日主极旺，就不需要再用神了，行任何运都可以。

**经典原文反驳**：

【任氏曰】"独象虽美，只怕运途破局。"

【任氏曰】"独象喜行化地，而化神要昌。"

**正确理解**：独象不但有用神，而且用神很明确——就是化神（食伤）。独象最怕的不是官杀，而是"运途破局"——在行运中遇到克破独象的五行。独象的用神选取比正格更严格：必须用食伤泄秀，不能用官杀克身。

### 误解三：形全就损，形缺就补——机械套用

**错误说法**：看到木生于寅卯辰月就"损"（用金克），看到土生于寅卯辰月就"补"（用火生）。

**经典原文反驳**：

【任氏曰】"实似所当损者，而损之反有害，实似所当补者，而补之反无功，须说察焉。"

**正确理解**：形全形缺不是机械的公式。如果一个八字虽然没有当令（"形缺"），但天干透出旺助之神，实际上并不缺；反之，虽然当令（"形全"），但被其他力量严重牵制，实际上并不全。最重要的是"通变在一宜字"——看实际需要，而非机械套用。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"两气合而成象，象不可破也"**——两气均衡共存，忌破局五行。
2. **"独象喜行化地，而化神要昌"**——专旺格的核心：化神（食伤）必须昌盛。
3. **"全象喜行财地，而财神要旺"**——多行并存时，财星为用神方向。
4. **"形全者宜损其有余，形缺者宜补其不足"**——旺则适当损之，衰则适当补之。
5. **"通变在一'宜'字"**——所有规则的核心：灵活判断，一个"宜"字。

### 7.2 与其他篇章的关联脉络

- **前序关系**：形象以天干（第7篇）和地支（第8篇）的知识为基础。
- **直接关联篇目**：
  - **方局（11）**——形象的"独象"与方局的"成方成局"直接对应。独象中的曲直格必然涉及寅卯辰方或者亥卯未局
  - **八格（12）**——形象的"形全形缺"与八格的正格判断相辅相成
  - **体用（13）**——形象的"象不可破"与体用的"扶之抑之得其宜"一脉相承

### 7.3 进阶学习路径建议

形象篇之后的推荐学习顺序：

1. **方局（11）**——形象篇讲述抽象分类，方局篇提供方局组合的具体判断
2. **八格（12）**——在形象基础上进入八格体系，学习正格判断
3. **体用（13）**——形象的"形全形缺"与体用的用神选取相辅相成

形象 + 方局 + 八格三篇连读，可以建立起完整的格局认知体系。
