## 功能

在接收到儿童（16岁以下）八字时，优先按小儿篇的简化分析法——跳过复杂的格局拆解，聚焦于日主旺衰、印比有无两个核心维度，判断儿童的先天禀赋强弱与养育风险。此技能应在命主为儿童时自动激活，替代常规的成人论命流程。

## 输入

- **四柱八字**：年柱、月柱、日柱、时柱的天干地支（字符串）
- **命主年龄**：数字（以判断是否适用小儿论法；通常适用于童限之前，即年满16岁前）
- **性别**：乾造或坤造（可选，影响大运排法）

## 处理逻辑

1. **确认适用范围**
   a. 若命主年龄 >= 16：提示"此命主已超出小儿论命范围，建议使用成人论命方法"
   b. 若命主年龄 < 16：继续执行小儿论法

2. **判定日主旺衰**
   a. 按月令判定日主是否得令
   b. 统计四柱中帮扶日主的干支数量（印星、比劫）
   c. 统计四柱中克耗日主的干支数量（官杀、财星、食伤）
   d. **依据**：小儿篇【任氏曰】"须观日主之强弱"

3. **判定印比状态**
   a. 列表：所有印星（正印、偏印）的位置与力量
   b. 列表：所有比劫（比肩、劫财）的位置与力量
   c. **依据**：小儿篇【任氏曰】"印比之有无也"

4. **吉凶判定**
   a. 日主健旺 + 印比帮身 → "无虞"（吉）
   b. 日主衰颓 + 印比无援 → "有灾"（凶）
   c. 日主健旺 + 印比无力 → 中平（需结合具体大运）
   d. 日主衰颓 + 印比存在 → 中平（有救应，需看印比力量是否足够）
   e. **依据**：小儿篇【任氏曰】"日主健旺，印比帮身，此小儿无虞也；日主衰颓，印比无援，此小儿有灾也"

5. **输出评估报告**

## 输出

interface XiaoErReport {

  applicable: {
    isChild: boolean
    message: string

  /** 日主分析 */

    ganZhi: string
    wuXing: string
    status: '健旺' | '衰颓' | '中和'
    basis: string // 月令等判定依据
  }
  /** 印比分析 */
  yinBi: {
    yin: Array<{
      // 印星列表
      ganZhi: string
      type: '正印' | '偏印'
      strength: '强' | '中' | '弱'
    }>
    bi: Array<{
      // 比劫列表
      ganZhi: string
      type: '比肩' | '劫财'
      strength: '强' | '中' | '弱'
    }>
    summary: '帮身有力' | '帮身不足' | '无援'
  }
  /** 综合评估 */
  assessment: {
    level: '无虞' | '有灾' | '中平'
    description: string // 基于小儿篇的中文评估
    recommendation: string // 养育建议（基于五行禀赋）
  }
}
