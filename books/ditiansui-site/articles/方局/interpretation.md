## 一、入门导读

### 1.1 这一篇在学什么

方局篇教你识别八字中"方"（三方会合，如寅卯辰东方一气）和"局"（三合局，如亥子丑会水局）两种特殊的五行聚合形态，以及它们形成后命局气势的判断方法。理解了方局，就能看懂八字中那种"一方之气独旺"的格局。

### 1.2 本篇在整个子平体系中的地位

方局篇与形象篇（第10篇）、八格局（第12篇）构成「特殊格局」三连篇。形象篇讲"两气成象""五气成形"，方局篇细化到具体的"方"（方向性会合）和"局"（三合局）这两种五行聚合形态。此前的天干（第7篇）、地支（第8篇）分别讲述了干支的单个特性，而方局篇则将地支的聚合效应系统化。方局是实际断命中最常见的特殊格局来源之一。

### 1.3 学习目标

学完本篇后，你应该能够：

1. **区分"方"和"局"**——说清寅卯辰（三方会方）、亥子丑（三合局）等组合的异同
2. **判断方局气势**——看懂命局中一方之气独旺时，如何顺势而为
3. **理解"不可逆也"的含义**——方局成后，运岁不可逆其气势，否则灾祸立见
4. **在实际八字中识别方局**——从四柱中找出五行聚合的迹象

## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

```
方局者，三方四正也。东方木、南方火、西方金、北方水、辰戌丑未为土。合方成局，气势磅礴。

【原注】方局者，三方四正也。寅卯东方木，巳午南方火，申酉西方金，亥子北方水，辰戌丑未四季土也。

【任氏曰】方局之论，须观其气势也。成方者，如甲乙见寅卯，则木盛而从其方；成局者，如亥子丑会水，则水盛而从其局。方局既成，不可逆也。
```

### 2.2 原文逐句精解

【原句第1句】"方局者，三方四正也。东方木、南方火、西方金、北方水、辰戌丑未为土。合方成局，气势磅礴。"

→ 解读：开篇直接给出定义。"方"指三方会合——寅卯辰会东方木、巳午未会南方火、申酉戌会西方金、亥子丑会北方水。"局"指三合局——申子辰合水、亥卯未合木、寅午戌合火、巳酉丑合金，以及辰戌丑未四季土。方和局一旦形成，五行气势会极度集中，这是判断命局格局的关键信号。

【原注】"方局者，三方四正也。寅卯东方木，巳午南方火，申酉西方金，亥子北方水，辰戌丑未四季土也。"

→ 解读：原注补充说明了"方"的具体内容——寅卯辰是东方木气汇聚，巳午未是南方火气汇聚，申酉戌是西方金气汇聚，亥子丑是北方水气汇聚，辰戌丑未是中央土气汇聚。

【任氏曰第1段】"方局之论，须观其气势也。成方者，如甲乙见寅卯，则木盛而从其方；成局者，如亥子丑会水，则水盛而从其局。方局既成，不可逆也。"

→ 解读：任铁樵强调方局的核心在于"观气势"——气势一旦形成，就必须顺势而为。成方的例子：甲乙天干配上寅卯地支，木势力不可挡，只能顺从方向。成局的例子：亥子丑三支会合，水势滔天，也是顺从三合之局。最后的"不可逆也"是整篇的定调句——方局气势已成，大运流年若逆此气势，灾难必至。

## 三、核心义理深度解析

### 3.1 "方"与"局"的本质区别

**原文依据**

【任氏曰】"成方者，如甲乙见寅卯，则木盛而从其方；成局者，如亥子丑会水，则水盛而从其局。"

**义理解读**

"方"和"局"虽然都是五行聚合的形态，但本质上有所不同：

- **方**是同一五行三个地支连成一气（寅卯辰、巳午未、申酉戌、亥子丑），强调方向的连续性和气势的贯穿。你可以理解为"三个同属性的地支排在一起，力量连绵不断"。
- **局**是三个地支通过三合关系形成一个五行局（申子辰合水、亥卯未合木、寅午戌合火、巳酉丑合金），强调三个地支之间的感应和凝聚。

一个形象的对比：方像是一条直线上的三股力量叠加，局像是三角形的三股力量汇聚于一个中心点。

**命造验证**

来自形象篇的参考命造：甲寅 丁卯 甲辰 丙寅（寅卯辰三支，东方一气成方）。此造地支寅卯辰全现，东方木方一气贯通，天元甲木透出，木气极旺。任铁樵以食伤（丙丁火）为化神泄秀。这正是"成方者，木盛而从其方"的典型体现。

### 3.2 "方局既成，不可逆也"——顺逆之理

**原文依据**

【任氏曰】"方局既成，不可逆也。"

**义理解读**

这是方局篇最核心的原则，也是八字格局判断的关键准则。当命局中某一五行之力通过"方"或"局"的方式极度集中时，命局的气势就已定向。所谓的"不可逆"，指的是：

1. **大运流年不可克破方局**——如果命局已成木方，大运来金运克木，就是逆势，主凶灾
2. **用神选取必须顺势**——木方已成，用神当取火（食伤泄秀）或水（印星生助），不宜取金（官杀克身）

这"不可逆"的原则与天道篇"顺则吉，悖则凶"一脉相承，是子平命理"顺势"思想的集中体现。

## 四、命造实例拆解

### 命造一：东方木方——寅卯辰一气

参考自形象篇任氏曰命造：

【命造一】乾造：甲寅 丁卯 甲辰 丙寅

大运：戊辰 → 己巳 → 庚午 → 辛未 → 壬申 → 癸酉
（出自任铁樵《滴天髓阐微·形象》）

**格局**：地支寅卯辰全，东方木方已成，为木专旺格（曲直格）。**用神**取火（丙丁火食伤，为化神泄秀）。**喜**木火运（顺势生扶），**忌**金运（金克木破方）。

**结论**：此造地支寅卯辰三支连方，天元甲木坐寅得禄，月干丁火、时干丙火泄秀。木方气势贯穿，化神（火）昌盛，所以任氏称其"发泄菁华，少年科甲"。行金运时，有丙丁火回克之能，故无大咎。这正是"成方者，木盛而从其方"的典型——方成则顺势，逆则有灾。

### 命造二：北方水局——亥子丑会水

参考自形象篇任氏曰命造：

【命造二】乾造：壬子 辛亥 癸丑 壬子

大运：壬子 → 癸丑 → 甲寅 → 乙卯 → 丙辰 → 丁巳
（出自任铁樵《滴天髓阐微·形象》）

**格局**：地支亥子丑全，北方水局形成，为水专旺格（润下格）。**用神**取木（食伤泄秀）。**喜**水木运，**忌**土运（土克水破局）。

**结论**：此造地支亥子丑三支会水，天元壬癸水透出，辛金生水，水势滔天。任铁樵称其"局成润下，喜行运不背"。甲寅、乙卯运木气泄水秀气，所以"登科发甲，宦途平坦"。丙辰运土克水，加之"群劫争财"而亡。这正是"方局既成，不可逆也"的例证——水局已成，土运逆势则凶。

## 六、常见误解与辨析

### 误解一：成方成局就一定富贵

**错误说法**：有人认为八字中只要有方局就是好命，富贵无疑。

**经典原文反驳**：

【任氏曰】"方局既成，不可逆也。"——但方局成后还需岁运相顺。形象篇任氏曰同样"独象虽美，只怕运途破局"。

**正确理解**：方局不等于富贵，关键在于：

1. 方局是否得月令之助（如木方在寅卯月才是真旺）
2. 行运是否顺势（不可逆方局之气）
3. 化神是否得力（需要食伤泄秀或印星生扶）

方局只是五行气势的聚合形态，真正的吉凶取决于全局配合和运岁走向。

### 误解二：方和局是一回事

**错误说法**：有人将"方"和"局"混为一谈，认为寅卯辰和亥卯未都是"木局"。

**经典原文反驳**：

【任氏曰】"成方者，如甲乙见寅卯，则木盛而从其方；成局者，如亥子丑会水，则水盛而从其局。"

**正确理解**：任铁樵明确区分了"成方"和"成局"两个概念。方的本质是三支在同一方向连成一气（寅卯辰是连续的三个月），局的本质是三支通过合化关系形成一体（亥卯未是三个不同的支通过合木的关系连接）。在实战中，方的力量更集中更线型，局的力量更凝聚更中心化。

## 七、本篇学习总结

### 7.1 核心必背内容

1. **"方局者，三方四正也"**——方指三支连方（寅卯辰等），局指三支合局（亥卯未等）。
2. **"方局既成，不可逆也"**——方局气势已成，运岁不可克破，否则凶灾。
3. **"成方者，如甲乙见寅卯，则木盛而从其方"**——方强调方向性连线，需要天干透出。
4. **"成局者，如亥子丑会水，则水盛而从其局"**——局强调三支合化，形成新的五行力量。

### 7.2 与其他篇章的关联脉络

- **前序关系**：方局以天干（第7篇）、地支（第8篇）为基础，是在地支知识基础上的聚合效应分析。
- **直接关联篇目**：
  - **形象（10）**——方局的"成方成局"与形象的"两气成象"有高度关联，都是讲特殊气势格局
  - **八格（12）**——方局之后进入八格体系，方局是一种特殊格局，八格是正格体系
  - **干支总论（09）**——干支配合是方局判断的基础

### 7.3 进阶学习路径建议

方局篇之后的推荐学习顺序：

1. **八格（12）**——从方局的特殊格局进入八格的正格体系，建立全面的格局观
2. **源流（20）**——学习命局中五行源流关系，理解气势的来龙去脉
3. **形象（10）**——如果还没有先读形象篇，建议补齐，两者互为补充

方局 + 形象两篇连读，可以全面掌握特殊气势格局的判断方法。
