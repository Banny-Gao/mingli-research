## 二、原文逐段精读

### 2.1 原文全文（带标点整理）

> **【原注·刘氏开篇】**
> 自形象气局之外，而格为最。格之真者，月支之神，透于天干也。以散乱之天干，而寻其所得所附于提纲，非格也。自八格之外，若曲直五格皆为格，而方已局气象定之者，不可言格也。五行之外，飞天合禄虽为格，而可以破害冲论之者，亦不可以言格也已。

> **【任氏曰·核心注疏】**
> 八格者，命中之真理也。先观月令所得何支，次看天干透出何神，再究司令以定真假，然后取用，以分清浊，此实依经顺理，若月逢禄刃，无格可取，须审日主之喜忌，另行别支透出天干者，借以为用。然格局有正有变，正者兼五行之常礼也，曰官印，曰煞印，曰财煞，曰食神制杀，曰食神生财；变者，必从五行之气势也，曰从财，曰从官杀，曰从食伤，曰从强，曰从弱，曰从势，曰一行得气，曰两气成形。其余外格我端，考群书，俱不从五行正理，尽属诡谈；至于《兰台妙选》，所定一切奇格异局，纳音诸法，尤属不经，不待辩而知识荒唐也。

> **【原注·对杂格的批判】**
> 飞天合禄之类，固为影响遥系而非格也。如四季月生人，只当取土为格，不可言杂气财官；丙丁日生于四季月者，当看人元透出天干者取格，不可以概以杂气财官论之；至于建禄月令无能为了刃，亦当看月令中人元透于天干者取格，若不合气象形局，则又无格之也。止取用神，用神又无所取，只得看其大倾，以皮面上断其穷通。不可执格论也。

> **【任氏曰·续论杂格之谬】**
> 影响遥系者，即暗冲暗合之格也。俗书所谓飞天禄马是也。如丙午日支全三午，亥卯日支三亥，逢三则冲，午去暗冲子水为官，亥去暗合卯木为官。又有冲财合财，如壬子日支全三子，暗冲午火为财，乙卯日支三卯，暗合戌土为财。又云，先要四柱不要财官，方可真方可冲合。夫冲者，散也，合者，化也，何能为我用乎？四柱原有财官，不宜冲合，犹有喜与不喜，况四柱无财官乎？至于杂气财官，亦是画蛇添足。辰戌丑未，无非支藏三干，各为杂气；寅申巳亥，亦有五行，各为杂气；岂故不论？夫库中之气，可可以言格，生地之申，岂非杂气？夫库中余气，可以言格，寅申巳亥之杂气，莫非不可言格？大凡库中余气，可生旺于库中，不必冲也；若生地之杂气，往往与四柱相喜与不喜，何故冲之？若丙丁生于四月，月建本为官星，卯戌时为杂气财官，忌冲喜合，格局破矣，格乃破也，余支皆然，不必透出天干取格为是。

### 2.2 原文逐句精解

#### 【第一段：开篇明义】

**原文**：`自形象气局之外，而格为最。`

**精解**：
- 刘氏（刘诚意）开篇即确立格的至高地位。
- "形象气局"是更宏观的五行气势判断，"格"则是在气势之下、具体到月令提纲的**用神定格**系统。
- 关键词：**"格为最"**——格是论命的核心工具，超出形象气局的具体操作层面。

**原文**：`格之真者，月支之神，透于天干也。`

**精解**：
- "格之真者"：真正的格局，需要满足两个条件——**月支本气**（月令本气）**透出天干**（显现在天干层面）。
- 这句话是八格取法的总纲，是核心定理二"月令定气，天干定格"的直接来源。
- **实战要点**：月令本气不透干，则格不真，需要退而求其次看藏干力量或另寻别支透干。

**原文**：`以散乱之天干，而寻其所得所附于提纲，非格也。`

**精解**：
- 批判一种错误做法：撇开月令，单纯从天干散乱的生克关系中寻找格局。
- 任注的逻辑是：**格局有根有源，根在月令**。不从月令出发，天干的吉凶判断就没有依附，是无根之谈。
- 这是对民间"见官就说官格、见财就说财格"粗浅论法的直接批判。

**原文**：`自八格之外，若曲直五格皆为格，而方已局气象定之者，不可言格也。`

**精解**：
- 刘氏明确区分：**八格**与**五格**（曲直、炎上、从革、润下、稼穑）是**两个不同系统**。
- 八格是**常理系统**（正格），五行五格是**气势系统**（变格/从象）。
- "方已局气象定之者"：五行专旺格（从势格）是由全局气势决定的，不能套用八格的月令定格逻辑。
- 学术争议点：后世《子平真诠》将"曲直炎上"等称为"专旺格"，《滴天髓》则将"从一行得气"归入变格，**两者本质一致**，只是命名不同。

#### 【第二段：任注核心——定格三步法】

**原文**：`八格者，命中之真理也。先观月令所得何支，次看天干透出何神，再究司令以定真假，然后取用，以分清浊。`

**精解——定格三步法**：

| 步骤 | 操作 | 回答的问题 |
|------|------|----------|
| 第一步 | 先观月令所得何支 | 月令的五行是什么？（定气） |
| 第二步 | 次看天干透出何神 | 月令本气是否在天干透出？（定格） |
| 第三步 | 再究司令以定真假 | 用神是否得时令之正？（定真） |

**核心口诀**：`月令定气，天干定格，司令定真`

**引申**：三步之后还要"取用分清浊"——用神取定后，还需判断格局的清纯程度。

**原文**：`若月逢禄刃，无格可取，须审日主之喜忌，另行别支透出天干者，借以为用。`

**精解——禄刃月特殊规则**：

- **禄刃月**：日主在月令为临官（禄）或帝旺（刃），即**日主极旺**之月。
  - 寅月甲木（甲的禄位）、卯月乙木（乙的刃位）
  - 午月丙火（丙的刃位）、巳月丁火（丁的禄位）
  - 申月庚金（庚的禄位）、酉月辛金（辛的刃位）
  - 亥月壬水（壬的刃位）、子月癸水（癸的禄位）

- **为何无格可取**：禄刃月日主极旺，月令本气是日主自己（日主的禄或刃），无法取为用神（用神必须是能对日主产生生克作用的十神）。此时格局体系失效，需要**另寻别支透干**作为用神。

**原文**：`然格局有正有变，正者兼五行之常礼也，曰官印，曰煞印，曰财煞，曰食神制杀，曰食神生财；变者，必从五行之气势也。`

**精解——正格vs变格的核心区别**：

| | 正格 | 变格 |
|---|---|---|
| **判别标准** | 兼五行之常礼（依常理生克） | 必从五行之气势（顺势而行） |
| **月令条件** | 月令本气透干 | 全局气势引导 |
| **取用逻辑** | 扶抑日主（身旺则抑，身弱则扶） | 从势而行（日主失势则从） |
| **代表格局** | 官印、煞印、食神制杀、食神生财 | 从财、从官杀、从食伤、从强、从弱、从势 |

#### 【第三段：杂格批判】

**原文**：`至于《兰台妙选》，所定一切奇格异局，纳音诸法，尤属不经，不待辩而知识荒唐也。`

**精解**：
- 任铁樵对《兰台妙选》的评价是**"不待辩而知识荒唐"**——荒唐到不需要辩论，一看就知道。
- "纳音诸法"：以纳音五行定格局命贵贱的做法，在任铁樵看来是伪学。

**原文**：`飞天合禄之类，固为影响遥系而非格也。`

**精解**：
- "飞天禄马"：以地支暗冲暗合（如午暗冲子为官）定格局的体系。
- "影响遥系"：这种做法牵强附会，像影子和回声一样不实在，不是真正的格局。
- 任铁樵的批判逻辑：四柱原局有财官时不靠此法，尚且有理；但四柱无财官时更不能用暗冲暗合来"制造"官星，这种做法**无根无据**。

**原文**：`至于杂气财官，亦是画蛇添足。`

**精解**：
- "杂气财官"：认为辰戌丑未月（四季月）藏有杂气，可以取为财官奇格。
- 任铁樵的批判：辰戌丑未月应以**本气（戌为土、未为土、丑为土、辰为土）**为格，不必另外寻找杂气财官。
- **重要例外**：若杂气月天干透出财官，则以透干为格（这是实战中的灵活处理，不与杂气财官理论混淆）。

#### 【第四段：格局成败的总规律】

**原文**（任注原文最核心一段）：

> 大凡格局真实而纯粹者，百无一二；破败而杂气者，十有八九；无格可取者甚多，无用可引者不少。格正用真，行运不悖，名利自如其人；格破用损，行运又悖，此乃有命无运。

**逐句精解**：

| 句 | 含义 | 实战启示 |
|----|------|---------|
| `格局真实而纯粹者，百无一二` | 真正的上等格局极少 | 不要期望每个八字都有好格局 |
| `破败而杂气者，十有八九` | 格局破败是常态 | 多数命局格局有瑕疵是正常的 |
| `无格可取者甚多，无用可引者不少` | 很多八字根本没有格可取 | 论命不能执格局一端 |
| `格正用真，行运不悖，名利自如其人` | 格局成则命运顺遂 | 格局成功是命运亨通的前提 |
| `格破用损，行运又悖，此乃有命无运` | 有好格局但运不济，仍难得志 | 格局与运程必须配合 |
| `格正用真，五行反悖，一生有志难伸` | 格局虽成但日主五行偏枯，仍有遗憾 | 格局之外还要看日主气势 |

**经典名句背诵**：`格正用真，行运不悖，名利自如其人；格破用损，行运又悖，此乃有命无运`

## 三、核心义理深度解析

### 3.1 八格的根本逻辑：为什么是这八格？

八格（官、印、财、食、伤、煞、偏财、正财）之所以成为子平体系的核心，是因为它们**覆盖了日主与十神之间的全部生克关系**：

```
日主（木）
├── 官星（金）── 克日主（压制）
├── 七杀（金）── 克力更强（威胁）
├── 财星（土）── 日主所克（获取）
├── 印星（水）── 生日主（滋养）
├── 食神（火）── 日主所生（吐秀）
└── 伤官（火）── 日主所生（泄秀但与官不和）
```

这六组关系加上正偏之分（即正官vs七杀、正财vs偏财），构成了八字中**所有主要矛盾**的框架。

### �.2 正格与变格的本质区别

任铁樵在八格篇给出了最清晰的定义：

| 维度 | 正格（兼五行之常礼） | 变格（必从五行之气势） |
|------|---------------------|---------------------|
| **哲学基础** | 日主旺衰是判断基准 | 整个命局的"势"是判断基准 |
| **核心逻辑** | 身旺则抑之，身弱则扶之 | 日主已无能为力，只能顺从全局气势 |
| **格局成立条件** | 月令本气透干 | 整个命局的气势倾向某一五行且日主失势 |
| **典型例子** | 官印格、食神生财格 | 从财格、从官杀格、一行得气格 |
| **《滴天髓》对应篇章** | 八格篇（本章） | 从象篇（第48章）、化象篇（第49章） |

### 3.3 禄刃月"无格可取"的深层原理

禄刃月为何不按八格论？这是初学者最容易出错的地方。

**原理**：月令为日主的**临官（禄）或帝旺（刃）**，意味着日主在月令当旺。此时月令本气**就是日主本身**，无法作为对日主起作用的用神。

以甲木日主为例：
```
甲木日主，生于寅月 → 寅为甲的禄
月令本气：甲（=日主自己）
问：月令透出什么？ → 透出的是甲（比肩）
比肩的作用：助日主同类 → 无助于建立格局体系
结论：无格可取，需另寻他支透干为用
```

### 3.4 "用神"与"格"的关系（常被混淆的概念）

```
格局 ≠ 用神

格（格局）：月令+透干的组合
  ↓
用什么格，就有什么格神
  ↓
格神就是这一格的"主治之药"
  ↓
用神（最关键的那个字）：能够平衡日主与格局的关键字
```

例如正官格中，官星是格神，但用神可能是印制伤官（若伤官破官），也可能是财星生官（若官星无根）。**用神是为格局服务的**，但两者并不总是一回事。

### 4.2 第一造：癸卯 己未 乙未 丙子——正官成格

- 日主乙木有卯木为根，未月土旺克水，但癸水印星有子水为根，贴身相生

  - 印星作用①：印制时干丙火食神（免食神制官）
  - 印星作用②：化泄官星戾气，护持日主

**任注原文理解**：

> 官星独发而清，印星贴身相生，亦通根身库。科甲出身，仕至藩臬。

**断语**：科甲出身，仕至藩臬（省级官员）

**格局成败核心**：印星有力贴身相生 → 官星无伤 → 格正用真

### 5.2 实战三步案例演示

**以第一造 癸卯 己未 乙未 丙子 为例**：

**第一步：定日主强弱**
- 得令：未月土令 → 木失令 → 否
- 得地：年支卯木为乙木根 → 是
- 得势：天干癸水印星生乙木 → 是
- 结论：身弱（有根有印，但月令克身）→ **身弱，用神需扶**

**第二步：定月令格局**
- 月令未月 → 本气己土（正官），天干透出己土
- → **正官格**

**第三步：判断用神真假**
- 得位：己土在月令 → 是
- 通根：己土通根未（坐支本气根）→ 是
- 透干：己土在天干透出 → 是
- 结论：正官格，用神得位通根透干 → **真格**

**总评**：身弱正官格，印星贴身相生护官 → 格局纯粹 → **上等命**

## 六、任铁樵杂格批判清单

> 任铁樵在八格篇对大量流行于世的命理体系进行了系统批判。学习这一篇不仅是学正面知识，更是学**如何识别伪学**。

### 6.1 批判总览表

| 序号 | 被批判的体系/方法 | 任注原文关键词 | 核心批判要点 |
|------|----------------|-------------|------------|
| 1 | 《兰台妙选》 | "尤属不经，不待辩而知识荒唐" | 纳音诸法、奇格异局皆虚妄 |
| 2 | 影响遥系（暗冲暗合） | "固为影响遥系而非格也" | 暗冲暗合非实格，无根无据 |
| 3 | 杂气财官 | "杂气财官不可执" | 杂气月仍取本气为格 |
| 4 | 飞天禄马 | "不可执为格局" | 以暗冲暗合制造官星，不足为凭 |
| 5 | 倒冲刑冲 | "不可执为格局" | 倒冲非正统论法 |
| 6 | 合禄合贵 | "不可执为格局" | 合禄非格局正途 |

### 6.2 重点批判详解

#### 批判一：《兰台妙选》——纳音体系的崩溃

**原文批判**：
> 至于《兰台妙选》，所定一切奇格异局，纳音诸法，尤属不经，不待辩而知识荒唐也。

**任铁樵的批判逻辑**：
- 《兰台妙选》以纳音五行配六十甲子，再以各种组合命名奇格（如壬骑龙背、六阴朝阳等）。
- 任铁樵认为这些格是"**穿凿附会**"——先有了结果（命好），再去找依据（纳音组合），**倒因为果**。
- 学术立场：纳音本身并非全无道理（如在神煞体系中），但**以纳音定格局**违反了子平体系的基本原理（格局以月令为本）。

#### 批判二：飞天禄马——暗冲暗合的虚妄

**原文批判**：
> 飞天禄马之类，固为影响遥系而非格也。如丙午日支全三午，亥卯日支三亥，逢三则冲，午去暗冲子水为官，亥去暗合卯木为官。

**任铁樵的反驳逻辑**：
1. "冲者散也，合者化也，何能为我用乎？"——冲和合本身是**破坏性的**，不是建设性的取用方法。
2. "四柱原有财官，不宜冲合，犹有喜与不喜，况四柱无财官乎？"——原局有财官不需此法；原局无财官，用暗冲暗合来"制造"官星，是**无根无据**的做法。

**实战启示**：断格局必须以原局天干地支的**实际透藏**为依据，不能凭空"制造"。

#### 批判三：杂气财官——画蛇添足

**原文批判**：
> 至于杂气财官，亦是画蛇添足。辰戌丑未，无非支藏三干，各为杂气……岂故不论？

**任铁樵的处理原则（重要！**）：
- **杂气月（辰戌丑未月）以本气为格**，不取杂气。
- **例外**：若杂气月天干透出杂气中的某一干（如辰月天干透出癸水），则以透干为格——这是**灵活处理**，不是承认"杂气财官"的理论。

### 练习二：命造分析

**题目**：分析命造「己丑 丁丑 乙丑 丙子」（第二造）为何破格？用三步法框架。

<details>
<summary>参考答案</summary>

**第一步：定日主强弱**
- 得令：丑月土令 → 乙木失令 → 否
- 得地：乙木无根（三丑土，无木根）→ 否
- 得势：天干无印比生助（癸水印星无根，丙火食神泄身）→ 否
- 结论：**身弱**

**第二步：定月令格局**
- 月令丑月 → 本气己土（正官），天干透出己土
- → **正官格**

**第三步：判断用神真假**
- 得位：己土在月令 → 是
- 通根：己土通根丑（坐支本气根）→ 是
- 透干：己土在天干透出 → 是
- 结论：表面三要素齐备，但**印星无根被晦**（丁火无根，湿土晦火），用神被破坏
- 关键破格点：印星（用神之辅）无根，官星失辅，忌神无制
- → **破格**

**总评**：正官格，表面用神三齐，实则印星无力（丁火无根），无法护官，反被湿土晦火，官星失辅，克泄交加 → 格局破败。
</details>

### 练习三：思考题

**题目**：任铁樵说"格局真纯者，百无一二"。结合现代社会的实际情况，讨论：在现代社会中，格局不纯的人是否就一定没有成就？

<details>
<summary>参考思路</summary>

**任注原文逻辑**：
- 格正用真 → 命运顺遂（但格局纯正是极少数）
- 格破用损 → 需靠运程补救
- 无格可取 → 只能"看其大倾"，以大势判断

**现代社会的反思**：
1. 格局理论是农业社会的产物，其"功名富贵"标准（科举、出仕）已不适用于现代职业体系
2. 现代社会的成就标准多元化（商业、技术、艺术等），格局框架仍有效，但判断维度需扩展
3. 格局破败但运程补救（如第二造，晚年木水运仍可发家）的现象在现代社会同样存在
4. 结论：任铁樵的格局理论作为**技术框架**仍然有效，但其"贵贱"判断标准需结合时代调整
</details>
