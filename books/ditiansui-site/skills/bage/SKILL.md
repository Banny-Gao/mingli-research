# 《滴天髓》·八格篇

> **原著：** 《滴天髓》上篇第十二章·八格
> **注疏：** 任铁樵（徐乐吾评注本）
> **正文：** 财官印绶发偏正，兼论食伤八格定

---

## 一、核心定理（必须背诵）

### 定理一：八格定义

```
八格 = 财、官、印、绶、食、伤、偏（偏官/七杀）、正（正财/正官）
```

### 定理二：八格取用步骤（核心三步）

```
第一步：先观月令所得何支      → 确定月令五行
第二步：次看天干透出何神      → 确定透干十神
第三步：再究司令以定真假      → 确认用神是否得位
```

> 口诀：**月令定气，天干定格，司令定真**

### 定理三：正格 vs 变格判别标准

| 类型     | 判别标准     | 特征                     |
| -------- | ------------ | ------------------------ |
| **正格** | 兼五行之常礼 | 月令本气透干，依常理取用 |
| **变格** | 必从五行之气 | 气势引导，顺势而行       |

### 定理四：禄刃月特殊规则

```
若月逢禄刃，无格可取
  → 禄刃月（日主得令极强），不按八格论，另寻他支透出者为用
```

### 定理五：格局成败比例

```
格局真纯者，百无一二
破败而杂气者，十有八九
  → 实战中破格是常态，纯格罕见
```

## 三、格局分析三步法（实战框架）

### 第一步：定日主强弱

```
日主得令否？  → 月令是否为日主同类五行
日主得地否？  → 地支是否有根（禄、刃、余气）
日主得势否？  → 天干是否有印比生助
↓
综合判断：身旺 / 身弱 / 中和
```

### 第二步：定月令格局

```
月令本气透干否？
  ├─ 是 → 取该透干十神为格（正格）
  └─ 否 → 看月令藏干中哪个力量强，取为格
          若禄刃月 → 无格可取，另寻别支透干
          若杂气月 → 取本气为格（不可执杂气财官为奇格）
```

### 第三步：判断用神真假

```
用神得位否？   → 用神是否在月令或在坐支
用神通根否？   → 用神是否有根（旺衰关键）
用神透干否？   → 用神是否在天干显透
  ↓
三者齐备 → 真格（极罕见）
缺一       → 破格（常态）
缺二以上   → 贱格
```

## 四、杂格批判

任铁樵明确批判以下谬说，论命时**不可执为格局**：

| 被批判的谬说                 | 任注原话                               | 核心观点                 |
| ---------------------------- | -------------------------------------- | ------------------------ |
| **兰台妙选**                 | "尤属不经，不待辩而知识荒唐"           | 奇格异局、纳音诸法皆虚妄 |
| **影响遥系**（暗冲暗合）     | "固为影响遥系而非格也"                 | 暗冲暗合非实格           |
| **杂气财官**                 | "杂气财官不可执"                       | 杂气月仍取本气为格       |
| **飞天禄马**                 | "不可执为格局"                         | 不可执此为奇格           |
| **倒冲刑冲**                 | "不可执为格局"                         | 倒冲非正统论法           |
| **合禄合贵**                 | "不可执为格局"                         | 合禄非格局正途           |
| **日时双头算法**（子时算法） | "何以壬子时独可朝子"                   | 子时算法混乱，不足为凭   |
| **壬骑龙背**                 | "何不取壬子、壬申、壬戌为骑猴马狗之背" | 纯属穿凿附会             |
| **六阴朝阳**                 | "夫阴阳皆阴也，何独辛金可朝阳"         | 以偏概全，逻辑荒谬       |

## 五、命造实例速记（六个命造核心结论）

| 命造       | 四柱                | 格局   | 成败关键                       | 任注断语                     |
| ---------- | ------------------- | ------ | ------------------------------ | ---------------------------- |
| **第一造** | 癸卯 己未 乙未 丙子 | 正官格 | 印星有力贴身相生，格局成       | 科甲出身，仕至藩臬           |
| **第二造** | 己丑 丁丑 乙丑 丙子 | 正官格 | 印星无根，湿土晦火，格局破     | 功名未遂，耗散资财，刑妻克子 |
| **第三造** | 戊寅 甲子 壬子 辛亥 | 印绶格 | 官星无根太弱，才华露而不能纳牝 | 品行超群，才华卓越           |
| **第四造** | 辛酉 丁酉 丁酉 辛亥 | 印绶格 | 壬水药病相济，格局纯粹         | 联登甲第，名利两全           |
| **第五造** | 辛酉 丁酉 丁酉 丙午 | 印绶格 | 无壬水药，丙火助日夺财，格局破 | 机杼空抛，株守待兔           |

**命造对比记忆法**：

- 一、四造成 → 印星有力，用神得位
- 二、五造败 → 印星无力或被夺，格局破败
- 三、六造中 → 格局有瑕疵，中上之命

## 六、格局清浊判断速记

```
格局清 → 纯清（百无一二）→ 上等命
格局浊 → 驳杂（十有八九）→ 中下命

清浊判断四要素：
  ① 用神是否有力（得根/透干/得位）
  ② 忌神是否受制（有无制化）
  ③ 气势是否流通（无阻隔/逆乱）
  ④ 财官印是否纯粹（不混杂）
```

## 七、常见错误警示

| 错误类型         | 正确做法                 |
| ---------------- | ------------------------ |
| 执杂气财官为奇格 | 杂气月仍取本气为格       |
| 以神煞论命为主   | 神煞为辅助，八格为根本   |
| 忽略月令透干条件 | 必须先观月令透干         |
| 禄刃月仍套八格   | 禄刃月无格可取，另寻透干 |
| 以纳音定格局     | 纳音诸法不经，不可执     |
| 格局破败仍认贵   | 格局破则命不贵           |

## 八、任注名句背诵

> **1.** 八格者，命中之真理也
> **2.** 先观月令所得何支，次看天干透出何神，再究司令以定真假
> **3.** 格局有正有变，正者兼五行之常礼，变者必从五行之气
> **4.** 若月逢禄刃，无格可取
> **5.** 格局真纯者，百无一二；破败而杂气者，十有八九
> **6.** 由是数造观之，格局不可执一论也
> **7.** 格正用真，行运不悖，名利自如其人；格破用损，行运又悖，此乃有命无运
