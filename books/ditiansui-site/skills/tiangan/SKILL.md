# 《滴天髓》·天干篇

> **原著：** 《滴天髓》上篇第七章·天干
> **正文：** 五阳皆阳丙为最，五阴皆阴癸为至

---

## 一、核心定理

### 定理一：十干分类

```
五阳干：甲、丙、戊、庚、壬（阳刚健动）
五阴干：乙、丁、己、辛、癸（阴柔顺静）
```

### 定理二：丙癸为最

```
丙火 → 阳中之阳 → 纯阳之精，万物莫不由此而发
癸水 → 阴中之阴 → 纯阴之精，万物莫不由此而生
```

### 定理三：五阳从气，五阴从势

```
五阳从气 → 不畏财杀之势
五阴从势 → 木盛从木，火盛从火，见势衰则忌
```

---

## 二、破世俗谬说

任注批判世俗以拟象论十干（甲为梁栋、乙为花果等），指出：

> 阳木 ≠ 死木，阴木 ≠ 活木，生死应从配合判断，非从天干阴阳属性。

## 三、十干专论速记

### 甲木

```
甲木参天，脱胎要火。春不容金，秋不容土。
火炽乘龙，水宕骑虎。地润天和，植立千古。
要点：春要火，不容金；秋不容土；火炽坐辰，水宕坐寅
```

### 乙木

```
乙木虽柔，刲羊解牛。怀丁抱丙，跨凤乘猴。
虚湿之地，骑马亦忧。藤萝系甲，可春可秋。
要点：刲羊解牛（丑未），藤萝系甲，可春可秋
```

### 丙火

```
丙火猛烈，欺霜侮雪。能煅庚金，逢辛反怯。
土众成慈，水猖显节。虎马犬乡，甲木若来，必当焚灭。
要点：欺霜侮雪，煅庚；逢辛反怯（化水）；虎马犬乡畏甲
```

### 丁火

```
丁火柔中，内性昭融。抱乙而孝，合壬而忠。
旺而不烈，衰而不穷，如有嫡母，可秋可冬。
要点：柔中昭融，抱乙孝，合壬忠；可秋可冬
```

### 戊土

```
戊土固重，既中且正。静翕动辟，万物司命。
水润物生，火燥物病。若在艮坤，怕冲宜静。
要点：万物司命；水润则生，火燥则病；艮坤怕冲
```

### 己土

```
己土卑湿，中正蓄藏。能收元阳之气，滋生万物。
要点：卑湿滋生，收元阳
```

### 庚金

```
庚金带煞，刚健得水而清，得火而锐。
土润则生，土燥则润。
要点：带煞刚健；得水清，得火锐
```

### 辛金

```
辛金软弱，温润而清。能扶社稷救生灵。
刚柔相济，秀气流行。
要点：软弱温清；扶社稷救生灵
```

### 壬水

```
壬水通河，周流不滞。顺则相济，逆则冲激。
要点：通河泄金，周流不滞；从则相济
```

### 癸水

```
癸水至弱，逢龙而化。化火为神，变坎为明。
要点：至弱从化；逢龙化火，化象斯真
```
