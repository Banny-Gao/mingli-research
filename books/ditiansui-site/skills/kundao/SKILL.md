# 《滴天髓》·坤道篇（地道）

> **原著：** 《滴天髓》上篇第二章·坤道
> **正文：** 坤元合德机缄通，五气偏全会凶吉

---

## 一、核心定理

### 定理一：乾坤对应

```
乾元 = 天元 = 主健 = 万物资始
坤元 = 地元 = 主顺 = 万物资生
```

### 定理二：坤道核心

```
坤元合德 → 地支与天干配合
机缄通   → 藏干流通无阻
五气偏全 → 吉凶判定标准
```

### 定理三：偏全定吉凶

```
五气偏全 → 吉（命吉）
五气偏枯 → 凶（命凶）
```

### 定理四：刚柔与地支

```
地有刚柔
  ├─ 刚：寅申巳亥（驿马冲动）
  ├─ 柔：子午卯酉（专位静守）
  └─ 中：辰戌丑未（杂气）
刚柔相济 → 中和之命
```

## 二、开篇正文解读

> 坤元合德机缄通，五气偏全会凶吉

| 关键词   | 命理含义               |
| -------- | ---------------------- |
| 坤元     | 地之元始，地支之根本   |
| 合德     | 与天德相合，天地配合   |
| 机缄通   | 造化机关通畅，藏干流通 |
| 五气偏全 | 五行在地支分布有偏全   |
| 会凶吉   | 偏全则吉，偏枯则凶     |

---

## 三、【原注】核心

> 地有刚柔，故五行生于东南西北中，与天合德，而感其机缄之妙。赋于人者，有偏全之不一，故吉凶定于此。

- 地支有刚柔两面
- 五行生于东南西北中（四维+中央土）
- 偏全决定吉凶

---

## 四、【任氏曰】核心

> 乾主健，坤主顺，顺以承天，德与天合；昭融覆盖，耨育万物。五行之气有偏全，故万物之命有吉凶。

- 乾刚坤柔，天地相合
- 坤顺承天，德与天合
- 五气偏全定吉凶

---

## 五、命局分析三步法

### 第一步：观地支刚柔

```
刚 → 寅、申、巳、亥
柔 → 子、午、卯、酉
中 → 辰戌丑未
```

### 第二步：观五气偏全

```
五行齐全 → 有根基
缺一五行 → 有缺陷
缺二五行 → 偏枯
```

### 第三步：观机缄通塞

```
机缄通 → 藏干相生，无冲刑破害
机缄塞 → 藏干相克，有冲刑破害
```
