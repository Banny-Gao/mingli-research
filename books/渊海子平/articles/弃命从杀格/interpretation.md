# 弃命从杀格 解读

同书相关篇章是《渊海子平》卷二"格局十神"中专门论"弃命从杀格"的一篇短论。**"弃命从杀"是子平法中"从格"系列的一种特殊格局**——与同卷其他从格类格局（从财、从儿、从旺等）同属"从格"大类，但以"从杀"（从七杀）为最严峻、最特殊的一种。本篇以一句经典范例"乙日见巳酉丑金局"立论，点出此格的核心机理与喜忌。

> 【原文】且如乙日，支见巳酉丑，金局大胜又无制杀，身主无气，只得舍身而从之。要行杀旺及财乡，忌日主有根及比肩之地。

**"乙日"**——乙木日主；**"支见巳酉丑"**——地支见巳酉丑三合金局（金局成——金气大盛）；**"金局大胜"**——金局气势大胜（金局成则金气充沛）；**"又无制杀"**——又无制杀（无食神、伤官制杀——金局虽盛但无物制之）。**"身主无气"**——身主（乙木日主）无气（乙木在地支中无根——金局中藏干皆金、克木之物，乙木极弱无依）。**"只得舍身而从之"**——只得舍身而从之（乙木日主极弱、无法自立——只能"弃命从杀"——舍弃日主本体的独立性、从七杀之强势）。**"要行杀旺及财乡"**——要行杀旺（金旺——金局再行金运）及财乡（财能生杀——如土运生金）。**"忌日主有根及比肩之地"**——忌日主有根（乙木在地支中得根——如寅、卯等木旺之地）及比肩之地（比肩为同类之木——乙的比肩是甲木或乙木——比肩之地会助身破格）。

#### 一、术语当场释义

**"弃命从杀"**——子平法中"从格"系列的一种。"弃命"是舍弃日主（命主）本体的独立性；"从杀"是从七杀（克我之七杀、阴克阳或阳克阴之偏官）之强势。**这是从格中最严峻的一种**——因为七杀本是克身之凶神，命主能"从"之则需身极弱、杀极强，且无食神、印星等制化之物介入。

**"金局"**——巳酉丑三合金局（巳酉丑三支相会而成金气）。**三合局**是地支之间按"生—旺—墓"三阶段组合而成的方局（如申子辰合水、寅午戌合火、巳酉丑合金、亥卯未合木）——**金局成则金气最盛**。

**"无制杀"**——无制七杀之物。子平法中"制杀"主要靠食神（如乙以丁为食神、庚金为杀——丁火克庚金、可以制杀）或印星（印化杀生身）。本例中"又无制杀"即指命中无食神制杀、亦无印星化杀——所以七杀无所制、金局无所破。

**"身主无气"**——日主（命主）无气。子平法中"气"指五行当令与否——有气则旺、无气则衰。"身主无气"即日主极衰、几无自立之能。

**"舍身而从之"**——"舍身"是舍弃自身本体的独立性，"从之"是从其大势（七杀之强势）。这是从格成立的机理——日主既不能自立、亦不能与七杀抗争，则干脆"从"了七杀。

**"杀旺之财乡"**——"杀旺"是七杀旺相（金局再行金运则杀更旺）；"财乡"是财星之乡（土——土生金——土运能助金气）。**这两端都是从格所喜之行运**。

**"日主有根"**——日主在地支中有根（乙木以寅、卯、未、亥等为根——地支中见木则木有依托）。**一旦日主有根，则从格即破**——因日主有了自己的根基、不再"无气"，则不再"从"。

**"比肩之地"**——比肩是同类（同五行同阴阳——乙的比肩是乙木）——比肩之地是比肩当旺之所（乙木的比肩之地为卯）。**比肩助身则从格破**——这是从格最忌的行运。

#### 二、原文逐句拆解

本段只有两句，但信息密度极高。下表逐句拆解：

| 原文 | 字面 | 命理含义 |
| --- | --- | --- |
| "且如乙日" | 比如乙日 | 立范例：乙木日主 |
| "支见巳酉丑" | 地支见巳酉丑 | 三合金局成、金气大盛 |
| "金局大胜" | 金局气势大胜 | 金为七杀（乙以庚为杀、辛为正官——金局为杀之根） |
| "又无制杀" | 又无制杀之物 | 无食神（丁）制庚金、无印星化杀 |
| "身主无气" | 身主无气 | 乙木在地支中无根、日主极弱 |
| "只得舍身而从之" | 只得舍弃本体而从之 | 弃命从杀格成立 |
| "要行杀旺及财乡" | 喜行杀旺（金旺）及财乡（土乡） | 杀旺则从势更明、财（土）能生杀（更助金势） |
| "忌日主有根" | 忌日主有根 | 日主有根（地支见木）则从格破 |
| "及比肩之地" | 及比肩之地 | 比肩（同类之木）助身、从格破 |

**这一格是"从格"系列中最精炼的范例之一**——用"乙日"+"巳酉丑"两个关键字概括了从格的所有要件。

#### 三、句式与用字

**句式**：本段由"且如……只得……要……忌……"的句式串联——"且如"是设例的引子（"且如"即"姑且以……为例"），是子平法论格体篇的常见开头；"只得"是无奈的转折（身主无气、别无选择）；"要"与"忌"是对举（喜与忌的两端判语）。**"且如—只得—要—忌"是子平法"格论"的标准四段式**。

**用字**："舍身而从之"五字最值得玩味——"舍身"是**主动**的（"舍"为舍去、自愿放弃）；"从之"是**顺势**的（"从"为顺从、依从）。**主动舍去 + 顺势依从**——这一组合精炼地表达了"从格"的核心理念：**不是被克制、而是主动选择顺从**。这是命理学家对"从格"的一种哲学化解读——"从"不是消极的屈服，而是积极的顺应。

"大胜"两字也值得留意——金局"大胜"而非"小胜"——这是说金局成、气势盛、压倒性优势——若金局不够"大胜"、则乙木仍有可能与之抗争、不成从格。**"大胜"是从格成立的必要条件**。

#### 四、篇章选址

本篇在卷二"格局十神"中与同卷其他从格类格局论并列——是卷二"格局十神"中"从格"系列的最短一篇。卷二另收从格系列的其他格局论与诗诀——本篇是其中最简明扼要的一篇、点出"乙日"这一经典范例。

#### 五、本篇在全书体系中的位置

本篇属于《渊海子平》卷二"格局十神"中**专论"弃命从杀格"**的格论短篇——是子平格局法"从格"系列的最早样本之一。**从格**是子平法中最特殊的一类格局——它突破了"身旺任财官、身弱用印比"的常规判断框架，**承认在日主极弱、克泄耗之势极强的情况下、日主可以"从"其大势而成另一种格局**——这是子平法"灵活变通"思想的集中体现。

**从全书体系看，本篇有几条贯穿性的特色：**

**第一，"从格"成立条件的精炼归总。**本篇用"乙日 + 巳酉丑金局 + 无制 + 身主无气"四要件概括了"弃命从杀格"的所有核心——这是子平法"从格"类格局最早期的精炼表达。

**第二，"行运喜忌"的最早系统化。**本篇明确指出从格"要行杀旺及财乡、忌日主有根及比肩之地"——这是子平法"从格行运"的最早定型，后世同类论说多沿用此两端。

**第三，"从格"哲学意涵的最早表述。**本篇用"舍身而从之"五字精炼表达了从格的哲学理念——**主动舍去 + 顺势依从**——这是命理学家对"从格"的一种哲学化解读。

**第四，"舍身从杀"与"舍身从财"的对偶。**从格分为从杀与从财两支——**从财者从财之旺、从杀者从杀之旺**——两者机理相通但喜忌不同。

**从解读角度看，本篇最值得现代读者留意的，是它对"从格"成立条件的精炼归总**——读者若想深入理解子平法"从格"的核心机理与"行运喜忌"的两端判语，本篇是不可绕过的"入门典范"——篇文虽短、却字字精要。

#### 六、术语小结

- **弃命从杀**——子平法"从格"系列之一，舍弃日主本体之独立性而从七杀（克我之偏官）之强势。
- **金局**——巳酉丑三合金局；金局成则金气最盛、为七杀之根。
- **制杀**——用食神（我生者同阴阳）克七杀、或用印星化杀生身，使七杀不致克身太过。
- **身主无气**——日主极弱、几无自立之能；是从格成立的根本条件。
- **舍身而从**——主动舍弃日主本体的独立性、顺势依从七杀之强势。
- **杀旺**——七杀旺相（金局再行金运则杀更旺）。
- **财乡**——财星之乡（土——土生金——土运能助金气）。本例中财乡（土）能生杀（金）——是"弃命从杀格"所喜之行运。
- **日主有根**——日主在地支中有根（乙木以寅、卯、未、亥等为根）。
- **比肩之地**——比肩当旺之所（乙木的比肩之地为卯）。日主有根或比肩之地则从格破。
