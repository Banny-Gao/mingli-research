# 解读：论金神

## 总论

金神为子平格局中的特殊神煞，本篇系统阐述其定义、格局要件与行运吉凶。金神格的核心要义在于"以刚制暴"——金神本性威猛难驯，须得火局或七杀羊刃制伏，方可化凶为吉。

## 理论解析

### 一、金神定义与三时

> 【原文】夫金神者，只有三时，癸酉、己巳、乙丑。

金神仅存在于三个时辰：癸酉、己巳、乙丑。此三时皆属金之长生、帝旺、死绝之地，构成金神特殊能量场。

【白话】金神是固定在三个时辰上的神煞，不是任意出现的。了解这一点，是判断金神格的前提。

### 二、本性与制伏之道

> 【原文】金神乃破败之神，要制伏，入火乡为胜。如四柱中更带羊刃七杀，真贵人也。大抵威猛者，以强暴为能威，苟不专，人得以侮。故必狠暴如虎动，群兽既慑，威德行矣。然太刚必折，不有以制之，则宽猛不济，何以上履中和之道。故曰有刚者，在驯服调致其和，福禄踵至。

原注点明金神本质：破败之神，具威猛难驯之性。若任其自然，则刚暴失度、祸患随至。制伏之道在于"入火乡"或"带羊刃七杀"。

【白话】金神就像一匹烈马，野性难驯。只有用正确方法（行火运或配合七杀羊刃）驯服它，才能成为贵人。否则就是凶险之神。

### 三、行运喜忌

> 【原文】其人有刚断明敏之才，屈强不可驯服之志。运至火乡，四柱有火局，便为贵命。惧水乡，则非福也。

金神格之人大多具备刚强果断、聪明敏锐的特质。火运或四柱成火局为吉，水运为忌。

【白话】行运走火（木火运）才能发达，行运遇水则破败。金神格的人最怕水运。

## 命造验证

【命造（眉批）】

> 庚日刃在午，喜得乙卯时，正官制伏去了，所以为福也。

此造庚日刃在午，得乙卯正官制伏，与金神格"带羊刃七杀真贵人"之理相通，亦证制伏之要。

【白话】这个命例说明，庚日刃格的命，带正官制服羊刃，就能转凶为吉。金神格也是同样的道理——必须有制伏才能为福。

## 篇章关联

本篇与"金神诗诀"、"论羊刃"构成金神格完整体系：

| 篇章 | 重点 |
| ---- | ---- |
| 论金神 | 格局定义、制伏原理 |
| 金神诗诀 | 行运要诀、歌括记忆 |
| 论羊刃 | 制伏工具之一 |

金神格与羊刃格存在理论交叉——羊刃可作为制伏金神的工具，形成"杀刃相临真贵人"的格局。

【白话】学习金神格要配合羊刃格一起看。羊刃是制服金神的重要手段，两者配合得当，就能成为贵命。
