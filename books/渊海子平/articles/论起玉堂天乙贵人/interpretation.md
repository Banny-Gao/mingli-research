# 论起玉堂天乙贵人 · 解读

> 卷一·基础理论

"天乙贵人"是子平命理中最具声名的吉神之一——八字四柱（年、月、日、时各以一干一支表示，合共八个字）逢之，主人清高贵显、逢凶化吉。本篇专论天乙贵人从十天干起例的口诀及其"辰戌不临"的特殊规则。

## 天乙贵人的起例口诀

> 【原文】甲戊兼牛羊，乙己鼠猴乡，丙丁猪鸡位，壬癸兔蛇藏，庚辛逢马虎，此是贵人方，命中如遇此，定作紫薇郎。

口诀按十天干分列六组，各指其所临地支方位（即天乙贵人所在的十二支）：

| 天干 | 天乙贵人所在支 |
| ---- | -------------- |
| 甲、戊 | 丑（牛）、未（羊） |
| 乙、己 | 子（鼠）、申（猴） |
| 丙、丁 | 亥（猪）、酉（鸡） |
| 壬、癸 | 卯（兔）、巳（蛇） |
| 庚、辛 | 午（马）、寅（虎） |

"紫薇郎"是古代官名，唐宋时指中书省（草拟诏令的最高行政机构）属官，因其地近帝座、清贵显达，故以之喻天乙贵人所主之品级与人格——命局四柱中若见此神来生扶日主（八字以日柱天干代表命主本人）或与日主相合，主其人气质清贵、处世优渥。

## 辰戌二宫为何贵人不临

> 【原文】十干临十支皆贵人所临之方，惟辰戌二宫贵人不临，殊不知辰戌乃魁罡恶曜之地，天乙不临，所以不为贵也。

口诀所列地支共十位：子丑寅卯巳午未申酉亥——唯独缺少**辰、戌**二支。原文随即揭出原因：辰为"水之库"且为"天罡"，戌为"火之库"且为"魁星"，二支合称"魁罡"，在命理中被视为刚烈刑冲之曜（带有强烈刑冲克破意味的星曜），与天乙贵人的"清贵和雅"之气相悖，故贵人不愿降临此二宫。

> 【眉批】言本乃主天之神而曰天乙，临之方有恶煞相浑，或落空亡之地，不为贵也。

眉批进一步阐发：天乙本为天界主神（即执掌天界事务的高位神祇），其降临之处若被其他凶煞混杂，或落於"空亡"（即所临地支恰处空亡位——以日柱所在旬推算而出的两个无气虚位），则吉力被夺，仍不主贵——此即"吉神怕浊"（吉利的星神最怕被凶煞污浊）的传统义理。

## 起例在推命中的运用

天乙贵人的查法是：先看日柱天干属何（因日干代表命主自身，故以日干起贵人最为通行；亦有以年干起例者），再据口诀查其贵人所在支；最后在四柱地支中看其是否出现（出现即为"命带天乙"）。若再逢天乙所喜之运（如贵人支的三合、六合之运），则清贵之力更显。
