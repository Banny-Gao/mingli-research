# 论运化气 · 解读

> 卷二·格局十神

"化气"是子平命理中格局判定的高阶议题——十天干两两相合（天干一阴一阳之合：甲己、乙庚、丙辛、丁壬、戊癸），合后所化之五行若成局势，则入"化气格"，主其人气象迥异常格。本篇系统阐述化气的起例、五合成局、真假分辨与方向之别。

## 五运化气总诀

> 【原文】夫五运化气者：甲己化土乙庚金，丁壬化木尽成林，丙辛化水分清浊，戊癸南方火焰侵。

十天干分阴分阳，依"阳干配阴干"之序天然成五对，每对所化之五行既定不移：

| 相合两干 | 化气 |
| -------- | ---- |
| 甲、己 | 土 |
| 乙、庚 | 金 |
| 丁、壬 | 木 |
| 丙、辛 | 水 |
| 戊、癸 | 火 |

## 化气之理：正月建天干所生之五行

> 【原文】至於合而化者，取其岁首月建天干五行所生之五行。如甲己之年正月为丙寅，天干丙火生土，所以甲己合就化土；乙庚之年正月为戊寅，天干戊土生金，所以乙庚合化金；丙辛之年正月为庚寅，天干庚金生水，所以丙辛合化水。余仿此。

化气之"理"在于回归到每年正月所建之天干：甲己之年（年干为甲或己）正月建丙寅，丙火能生土（按五行相生序：木生火、火生土），故甲己相合就化土。余四组同理——乙庚之年正月建戊寅，戊土生金；丙辛之年正月建庚寅，庚金生水；丁壬之年正月建壬寅，壬水生木；戊癸之年正月建甲寅，甲木生火。

此理的关键在于：**化气之所成，是"岁首月建天干"五行生旺之功的延伸**，而非两干之间的直接反应——这就把天干合化锚定在月令（八字以月柱地支为主气所在，称"月令"）这一时间维度上，与下文"月令生旺方化"相呼应。

## 五合所成之局

> 【原文】甲己化土，中正之合。辰戌丑未全曰稼穑，勾陈得位。乙庚化金，仁义之合，巳酉丑全曰从革。戊癸化火，无情之合，得火局曰炎上。丙辛化水得申子辰水局曰润下。丁壬化木，得亥卯未全曰仁寿。

十干五合各有所主之性情与地支三合局（地支三支一组按特定五行方位相合的局式）：

| 五合 | 性情 | 所需地支局 | 局名 |
| ---- | ---- | ---------- | ---- |
| 甲己化土 | 中正之合 | 辰戌丑未 | 稼穑（勾陈得位） |
| 乙庚化金 | 仁义之合 | 巳酉丑 | 从革 |
| 丙辛化水 | （清浊之合） | 申子辰 | 润下 |
| 丁壬化木 | （繁林之合） | 亥卯未 | 仁寿 |
| 戊癸化火 | 无情之合 | 寅午戌 | 炎上 |

- **稼穑**：本指农耕稼穑之事，土主孕育万物，故以此喻甲己合化土的厚重化育之象。
- **从革**：金主肃杀变革，故以此喻乙庚合化金的刚断明敏之象。
- **润下**：水主润下流通，故以此喻丙辛合化水的流通不滞之象。
- **仁寿**：木主生发仁寿，故以此喻丁壬合化木的繁茂长寿之象。
- **炎上**：火主炎上热烈，故以此喻戊癸合化火的无情炽烈之象。

## 化之真者贵，化之假者孤

> 【原文】化之真者，名公巨卿。化之假者，孤儿异性。逢龙即化，变作飞龙在天，利见大人。月令生旺，养库临官之地方化。阴阳得合，夫妇匹配，中和之气而化，太过不及皆不能化。

化气有真假之分。真化者，主人贵显——"名公巨卿"是古代形容公卿显爵之词（位居三公九卿的朝廷重臣），喻化气成格可致极品。假化者，主其人形单影只——"孤儿异性"或出家为僧道。这一分野的判定标准有四：

1. **逢龙即化**："龙"指辰（辰为龙位），逢辰即化——意指辰方为化气之关键引动位。所谓"飞龙在天，利见大人"——"飞龙在天"与"利见大人"并见，本是《周易·乾卦》九五爻辞（周人占筮书《易经》乾卦第五爻的爻辞，原指君子得位可大展宏图），原典此处借以喻化气成局后命主如飞龙升天、得遇赏识之人。
2. **月令生旺**：化气须得月令（命主出生月份的月支）生旺之助。
3. **养库临官**：化气须在十二长生（天干在十二支中各有"长生、沐浴、冠带、临官、帝旺、衰、病、死、墓、绝、胎、养"十二种状态）中的"养、库、临官"等吉位方能成立。
4. **阴阳得合、中和之气**：必须阴干与阳干匹配（一夫一妻之合），且五行之气中和（既非太盛亦非太弱），太过或不及皆不能化。

## 化的方向：从化之别与转角化

> 【原文】有夫从妻化，妻从夫化，正化、偏化，日下自化。转角化，乃未坤申，丑艮寅。经云：东北丧朋，西南得朋。甲日见己字化土，己见甲亦然，乃化之真谓之正化。

化的方向有四：

- **夫从妻化**：日干（命主本身的天干）为阳、时干（出生时辰的天干）为阴，阳从阴化。
- **妻从夫化**：日干为阴、时干为阳，阴从阳化。
- **正化、偏化、日下自化**：正化为阴阳各得其位（甲己各以自身为主而非彼此从属）之化；偏化为五行偏枯（某一五行过盛而其他五行过弱）时的化；日下自化为日干自身已化（即日干与月支或时支自化其气）。
- **转角化**：地支"未、坤、申"（未即坤位、申即乾位之余气）与"丑、艮、寅"（丑即艮位、寅即艮位之余气）——所谓"东北丧朋，西南得朋"，本是《周易·坤卦》辞（《易经》坤卦辞，原指君子往西南则得同类相助），原典借此喻化气方向之选。

## 化气有方向：戊癸化火之特例

> 【原文】但戊癸化火，南木化午，北不化子。午乃少明君火，所以不化。寅申乃少阳火，乃化。

化气方向有南北之别：戊癸合化火，南方（地支午方）虽是火位但属"少明君火"（阴火，主柔而不化），故不化；反是**寅、申**二位（寅为东北偏东、申为西南偏西，分属"少阳"）之火方能引动化气。这与"午"为太阳火之极的"物极必反"（旺盛到了极点反不能成）的传统理解有关。

> 【原文】经云：化之格局，玄中又玄，妙中又妙，不可俱述。常观天元神趣八法：返、照、鬼、伏、类、属、从、化，仔细推详。

末段归总化气格局之妙，并提示研习者参看"天元神趣八法"——所谓"返、照、鬼、伏、类、属、从、化"是化气判定的八种细化情境（每种情境代表化气在不同干支关系下的具体表现形态）。

> 【眉批】其法亦从甲己起庚寅丁卯戊辰，逢龙便化，余仿此。诗诀见卷五，此处略。

眉批补充化气起例的另一种起法：从甲己年起庚寅（庚金克甲木、起化之端）、丁卯（丁火燃木、引化之势）、戊辰（戊土归位、辰为龙位即化）。
