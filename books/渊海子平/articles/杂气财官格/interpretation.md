# 解读：杂气财官格

## 总论

杂气财官格以辰戌丑未四季月令为平台，论财官印绶藏于杂气之格局。本篇系统阐述杂气财官格的认定、格局要件与命造验证。

## 理论解析

### 一、杂气财官定义

> 【经】财官印绶全备，藏蓄於四季之中，辰戌丑未是也。

杂气指辰戌丑未四个月令所藏天干，非当令之气，故称"杂气"。财官印绶藏于此四季之中。

【白话】辰戌丑未是四季月令，里面藏着财星、官星、印星。杂气就是这个意思。

### 二、四宫所藏

> 【原文】如官露、印露、财露则不妨也。如辰宫则有乙木、癸水、戊土；戌宫则有辛金、丁火、戊土；丑宫则有癸水、辛金、己土；未宫则有乙木、丁火、己土也。

四宫所藏天干：

| 宫位 | 所藏天干 |
| ---- | -------- |
| 辰 | 乙木、癸水、戊土 |
| 戌 | 辛金、丁火、戊土 |
| 丑 | 癸水、辛金、己土 |
| 未 | 乙木、丁火、己土 |

【白话】四个季月各有三个藏干：辰藏乙癸戊，戌藏辛丁戊，丑藏癸辛己，未藏乙丁己。

### 三、格局要件

> 辰、戌、丑、未为四季财官，印绶居杂气，干头透出格为真，用财多为尊贵，杂气自来就不纯，不逢也出始为宗，身强财旺生官禄，运见冲刑装宝珍。

杂气财官格要件：

- 辰戌丑未为杂气月令
- 干头透出财官印绶方为真格
- 以用财多者为尊贵（与正官格喜官星不同）
- 身强财旺则生官禄
- 行运喜冲刑

【白话】杂气财官格要看月令是不是辰戌丑未，还要看天干有没有透出财官印。透出才算真格，财星多用事，身强财旺就能当官。行运喜欢冲刑。

### 四、行运喜忌

> 【眉批】辰戌丑未为四季，印绶财官居杂气，干头透出格为真，只问财多为尊贵。

强调：杂气格以干头透出为真，以财多为尊贵，行运喜冲刑以开库。

【白话】杂气格只要天干透出财官印就算好命，以财多为主。行运喜欢冲刑来开库。

## 命造验证

【王羲之命造】

> 乙卯 癸未 壬子 辛丑(丁财、己官、辛印)

壬日子月，未为杂气。干透辛印、己官、丁财，财官印齐全，为贵命。

【白话】王羲之的命，壬日子月，未是杂气月，天干透出辛己丁，财官印全，是贵命。

【李柳帝命造】

> 戊子 乙丑 乙未 辛亥(癸财)

丑为杂气月令，干透癸财，杂气财官格成。

【白话】李柳帝的命，丑月杂气，天干透癸财，成格。

【黄状元命造】

> 壬子 甲辰 己卯 壬申(壬财)

辰为杂气月令，干透壬财，杂气财官格成。

【白话】黄状元的命，辰月杂气，天干透壬财，成格。

【宣参政命造】

> 己卯 辛未 壬寅 辛亥(己官)

未为杂气月令，干透己官，杂气财官格成。

【白话】宣参政的命，未月杂气，天干透己官，成格。

【张参政命造】

> 丙寅 戊戌 辛酉 戊子(庚官、甲财)

戌为杂气月令，干透庚官、甲财，杂气财官格成。

【白话】张参政的命，戌月杂气，天干透庚官甲财，成格。
