# 子遥巳诗诀 · 解读

> **定位**：卷五 · 诗诀口诀
> **写作原则**：行文与原文体量匹配，核心任务收束"原文在说什么"+"在全书中的位置"，原文本体分析（句式、用字、篇名选址）不可略过

## 原文与释义

> 【原文】甲子重逢甲子时，休言官旺不相宜，月生日主根元壮，运到金乡反得奇。
> 甲临子字日时全，拟作蟾宫折桂仙，丑绊午冲官杀显，反为淹滞祸绵绵。
> 甲子重逢甲子时，名为遥巳最相宜，才临丑午家须破，岁运官逢亦不奇。
> 甲子日逢甲子时，子来遥合巳中支，戊能动丙丙合酉，甲得辛官贵可知。
> 不喜庚辛申酉出，丑来相绊亦非宜，更嫌午字相冲害，运入官乡旺处奇。

**本义还原**：本篇是"子遥巳格"格局的诗诀合集（凡七言四首加五言一首），用韵语重述正文中格局成立之链与喜忌忌神。诗诀与正文互为表里——正文说理、诗诀记诵；正文细辨、诗诀点睛。

**逐首释义**：

- **首章**（月生日主 / 运到金乡）——破"二甲子不见官星则不吉"的疑虑：甲子重逢甲子时，四柱不见官星本似"无凭"，然若月令生扶日主（"月生日主"，即月令地支中所藏之干或月令之气为日主之印星即生日元之物），则日主根元壮实（"根元"指四柱地支中蕴含的日主之气与月令之气），运行至金乡（西方申、酉、戌月令或西方大运）反得奇贵——这是"无官而待官运催发"的特殊取法。
- **次章**（甲临子字 / 丑绊午冲）——前两句描摹"日时两甲子齐备"之贵格（"蟾宫折桂"即月宫折桂，喻科举高中，蟾宫为月宫别称），后两句转写"破格"：丑字绊子（子丑六合，绊则子不能遥合），午字冲子（子午相冲），官杀（正官与七杀）若显而格局被搅，则主人淹滞不前、祸患绵延。
- **三章**（才临丑午 / 岁运官逢）——"才"指财星（甲木以戊土、己土为财星），财星临丑午（丑为金库、午为火旺，皆非甲木财星所喜之位，且丑绊子、午冲子，皆破遥合之势）则家破；岁运再见官星（"官逢"），因原局遥合之链已被绊住，运来官星反成累赘，"亦不奇"——此句最妙，揭示"原局已破，运来吉神反为累"的反直觉现象。
- **四章**（子来遥合 / 甲得辛官）——前四句重述子遥巳的藏干链：子中癸水遥合巳中戊土、戊动丙、丙合酉中辛金，甲得辛金正官，与正文完全对应。
- **末章**（不喜庚辛 / 丑绊午冲）——重述忌神：庚（甲木七杀）、辛（甲木正官，本为喜神但若透干则失"遥"字机制）、申（七杀之禄）、酉（正官之禄）、丑（绊子）、午（冲子）皆忌；末句"运入官乡旺处奇"作结，呼应首章"运到金乡反得奇"，全篇以"运到金乡"作收。

> 【原注】眉批：以甲子为主，以子合巳中戊土，巳中戊丙同宫，丙动来合酉中辛金为甲之官。如东钱丞相命，己巳 乙亥 甲子 甲子，即合此格。如赵太守，丙子 壬辰 甲子 甲子，亦合此格，所以为贵。或有丑字合子字则遥不成反成下格。

**眉批释义**：眉批先点出"以甲子为主"的取格眼目（必须甲子日或甲子时为枢轴），再补两造佐证：东钱丞相造（己巳 乙亥 甲子 甲子）——年支巳与时支子构成"子合巳"的最直接结构（不必依赖"遥"，而是年支巳、时支子已成对宫合势），月令亥（亥中壬水为甲木正印，生扶日主），故"日时全"之格成立而大贵；赵太守造（丙子 壬辰 甲子 甲子）——日支子、时支子皆甲子，月令辰虽藏戊土（甲木偏财）但与子无冲，壬水透干为正印扶身，亦成此格。眉批末尾补出"丑绊则破"的反面案例，与正文"平常人造"之例前后呼应。

## 句式与用字

- **"甲子重逢"** ——"重逢"二字点"二甲子"之"二"意，把"两甲子齐见"从数量表达升格为"再次相遇"的意象。
- **"根元"** ——元即本元、根气，指四柱地支中所藏的日主本气与月令本气之根。"根元壮"三字把"身旺"翻译成"地下有根可立"的空间隐喻。
- **"蟾宫折桂"** ——典出《晋书·郤诜传》"桂林一枝"之喻，后世以"蟾宫折桂"喻科举高中。诗中以"拟作"二字领起，是宋元诗诀常用的"虚拟"笔法——意谓"可视为……"。
- **"淹滞"** ——指命运淹塞、停滞不前，与"折桂"对偶。
- **"休言"二字** ——转折词，等于现代汉语"且莫说""不要以为"，是宋元诗词常见句首。
- **"旺处奇"** ——末句"奇"字押韵（上古支韵，骑奇同部），同时作"出奇之贵"解，语带双关。

## 诗诀与正文的对应关系

本篇作为"子遥巳格"的诗诀口诀，与同卷同格局正文在内容上完全对应，但功能不同：

| 正文 | 诗诀 |
|------|------|
| 立格之链（子→巳→酉） | 重述于"甲子日逢甲子时"一首 |
| 喜行官旺乡运 | 重述于"运到金乡反得奇""运入官乡旺处奇" |
| 忌庚申七杀 | 重述于"不喜庚辛申酉出" |
| 忌辛字官星 | 同上"辛"字 |
| 忌申酉丑绊 | 重述于"丑来相绊亦非宜" |
| 忌午字冲子 | 重述于"更嫌午字相冲害" |

诗诀把正文的逻辑压缩为可记诵的韵语，是《渊海子平》"正文说理、诗诀记诵"体例的典型体现。

## 在全书体系中的位置

本篇是同书诗诀口诀中与子遥巳格正文对应的格局诗诀。同书诗诀群收有各格局诗诀，是同书格局分类的"诗诀镜像"。本篇与同卷其他格局诗诀的体例一致：每篇以四首或五首七言、五言组成，反复吟咏本格局的成立条件、喜忌忌神与命造范例，形成"读者背诵诗诀即可初步判定格局"的应用手册功能。
