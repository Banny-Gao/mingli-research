# 论小儿关杀例 解读


## 一、关杀为时柱之要

> 【原文】小儿之命，当论时辰为主，先看关杀，次看格局。日主强，财官旺，有关无杀；日干弱，财官少，常病可养；日干弱，财官多，有关有杀。又有三会聚杀者难养，不见刑冲者声音响亮，夜啼急性。

小儿命断法之要旨：

1. **时辰为主**：童命最重**时柱**（出生之小时辰所值干支），因时柱为归宿、最晚定；小儿未涉大运，原局以时为定盘。
2. **先关杀、次格局**：先看**关杀**（详见下节），再参论格局。
3. **日主强弱分四态**：
   - **日主强 + 财官旺**：有关无杀 → 吉。
   - **日干弱 + 财官少**：常病可养 → 平常。
   - **日干弱 + 财官多**：有关有杀 → 凶险。
   - **三会聚杀**（三合会成七杀之局，如申子辰水局而戊土日主见之）：难养。

术语当场释义：

- **关杀**：本节特指的术语，「**偏官为关、偏财为杀**」（见下）。关者，**关**锁难脱之意；杀者，**克**伐日主之意。小儿遇之，多主幼年关煞。
- **三会聚杀**：地支三合（申子辰合水局、亥卯未合木局、寅午戌合火局、巳酉丑合金局），若合出之五行恰为日主之七杀，则「聚杀」——杀力集中，难养。

末句「不见刑冲者声音响亮，夜啼急性」——若无刑冲破局，则童声响亮；反之则有**夜啼急性**（夜里啼哭、性情急躁）之征⚠️。

## 二、关杀与生辰之数

> 【原文】八字有财官，生於富贵之家，偏官生於平常之家，伤官劫财生於贫贱之家。偏官、偏印、偏财，主偏生庶出，不然第三、四胎。子平之法，偏官为关，偏财为杀，取生辰之数断之。水一、火二、木三、金四、土五。且如甲日庚杀，乃四、九岁；丙见壬杀，一、六岁；戊日甲杀，三、八岁；庚日丙杀，二、七岁；壬见戊杀，五、十岁见之。至於阴乾亦如此，无不验矣。

生辰之家世与生数之推算：

1. **生辰之家世**：
   - 八字有**财官** → 生於**富贵之家**。
   - **偏官**生於**平常之家**。
   - **伤官劫财**（伤官为我生而克官，劫财为异性比劫夺财）→ 生於**贫贱之家**。
   - **偏官、偏印、偏财**三柱并列 → 主**偏生庶出**⚠️（非正室所生），不然**第三、四胎**。
2. **生辰之数**（五行之数）：
   - 水一、火二、木三、金四、土五。
   - **关**为偏官、**杀**为偏财——本节特以此为定义。
3. **阳干关杀之岁数**（以生辰之数推关杀应期）：
   - 甲日见庚（七）杀 → **四、九岁**（金数四）
   - 丙日见壬杀 → **一、六岁**（水数一）
   - 戊日见甲杀 → **三、八岁**（木数三）
   - 庚日见丙杀 → **二、七岁**（火数二）
   - 壬日见戊杀 → **五、十岁**（土数五）
4. 阴干「亦如此」：乙日见辛杀、辛见乙杀、癸见己杀等皆以同类推算。

## 三、眉批的异议

> 【眉批】关杀之说，术家多不明也，但有一杀为关、一财为杀，於年时见之，总为关杀。在日月者，俱不准。无关无杀、有关有杀俱不准，若有准，验小儿命上中下存一，若依天罡之说，其谬甚矣。

原书自批对正文之说作了显要的修正与保留：

1. **关杀定义**：原书批注认为，**关**应指**一杀**（七杀），**杀**应指**一财**（偏财）——这与正文「偏官为关、偏财为杀」之定名一致。
2. **位置限制**：关杀须在**年支**或**时支**见之方准；在**日支月支**见者，「俱不准」（皆不作关杀论）。
3. **关杀分布之四态**：
   - 无关无杀 → 不准
   - 有关有杀 → 不准
   - 上述二态皆不足以应验
   - 唯有「**上中下存一**」（关杀见於年时之一）方准。
4. **天罡之说**：若依**天罡**（北斗魁罡之星煞体系）之说推关杀，「其谬甚矣」——原书直指天罡派之谬，不留情面。

此段眉批之要，是把**关杀**之推算收束为更严格的限定：位置须在年时，且须关与杀有其一而非并存。这是子平法对**童命关杀**一说的自我修正，与正文前后呼应、形成方法论上的张力。

## 小儿关杀机理与体系位置

本篇为子平法**童命关杀**的专论。它把**时辰为主、关杀为先**作为童命断法的总纲，以**生辰之数**（水一火二木三金四土五）推阳干关杀之应期岁数，并以眉批对关杀之位置与定义作严格修正——关杀唯在年时见之，且须「**上中下存一**」方准。全文涉及**顶相**、**生数**、**关杀**、**天罡**等古代命学专门术语，构成了子平法童命论的完整一环，与「论小儿」前后呼应：前者论童命之总纲（财、印、身旺、身弱），本篇则专论童命之**关杀**（特定煞星之应期推断）。
