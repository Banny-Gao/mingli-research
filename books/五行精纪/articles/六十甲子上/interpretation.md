六十甲子者，干支相配之总名，纳音五行之分派。

## 六十甲子配纳音五行

> 【原文】甲子　乙丑 海中金 / 丙寅　丁卯 炉中火 / 戊辰　已巳 大林木 / 庚午　辛未 路旁土 / 壬申　癸酉 剑锋金 / 甲戌　乙亥 山头火 / 丙子　丁丑 涧下水 / 戊寅　已卯 城头土 / 庚辰　辛巳 白腊金 / 壬午　癸未 杨柳木 / 甲申　乙酉 泉中水 / 丙戌　丁亥 屋上土 / 戊子　已丑 霹雳火 / 庚寅　辛卯 松柏木 / 壬辰　癸巳 长流水 / 甲午　乙未 砂中金 / 丙申　丁酉 山下火 / 戊戌　已亥 平地木 / 庚子　辛丑 壁上土 / 壬寅　癸卯 金萡金 / 甲辰　乙巳 覆灯火 / 丙午　丁未 天河水 / 戊申　已酉 大驿土 / 庚戌　辛亥 钗钏金 / 壬子　癸丑 桑拓木 / 甲寅　乙卯 大溪水 / 丙辰　丁巳 沙中土 / 戊午　已未 天上火 / 庚申　辛酉 石榴木 / 壬戌　癸亥 大海水

此为六十甲子纳音五行之全表。十干与十二支配为六十对，每对共纳一音五行，以喻其气质与材质。所谓「海中金」者，宝藏于渊深之处，未显于外；「炉中火」者，文明之始，锻炼成器；「大林木」者，栋梁之材，森然而立；「路旁土」者，承道之基，万物所践；「剑锋金」者，锐利之器，刚断万物；「山头火」者，丽于高冈，照见四方；「涧下水」者，澄澈流通，曲折有情；「城头土」者，环卫一方，固守不迁；「白腊金」者，柔润光泽，藏于璞中；「杨柳木」者，柔韧随风，生发不已；「泉中水」者，源自地脉，滋养万物；「屋上土」者，依以为安，覆庇生民；「霹雳火」者，威震奋击，击破阴霾；「松柏木」者，凌寒不凋，岁寒后凋；「长流水」者，源远流长，绵延不绝；「砂中金」者，蕴于矿脉，待匠而显；「山下火」者，温暖柔顺，黎明之光；「平地木」者，独立原野，任其自然；「壁上土」者，环卫内外，界隔内外；「金萡金」者，薄而锋利，可裁可断；「覆灯火」者，夜色照明，温暖幽微；「天河水」者，银河之水，清冷高远；「大驿土」者，通途大道，承载万行；「钗钏金」者，妇女之饰，柔丽精巧；「桑拓木」者，蚕食之利，养民之本；「大溪水」者，奔流而去，浩瀚无涯；「沙中土」者，散漫不聚，随风流动；「天上火」者，丽于日月，光辉普照；「石榴木」者，多子繁茂，果实累累；「大海水」者，浩渺无垂，包纳百川。三十种纳音，循环往复，自甲子起至癸亥终，复归于甲子，圆满一轮。

## 六十甲子之起源

> 【原文】夫六十甲子者，起于轩辕，作于大挠。

此言六十甲子之学术源流。「起于轩辕」者，言其本源于黄帝时代；「作于大挠」者，言其具体由大挠创制推演。文中用「起」「作」二字，「起」言其本，「作」言其成，先后次序分明。

> 【原文】大挠者，轩辕之史官也。

此句释大挠之身份——黄帝之史官。史官掌记王事、推演历法、造立术数，故六十甲子之作由其承担，正合其职分。短短一句，既补足前句「作于大挠」之出处，亦点明六十甲子之创制非凭空而起，乃古圣王廷中史官之专业所为。

> 【原文】考天书三式，以十干十二支衍而成之，为六十甲子，三式者，即轩辕感玄女所授之书（烛神经）

此言六十甲子之推演方法与学术渊源。「考天书三式」——研考上古天书之三式；「以十干十二支衍而成之」——以十天干、十二地支相配，推演而得六十甲子；「三式者，即轩辕感玄女所授之书」——所谓三式，乃黄帝感玄女所授之书，括弧内「（烛神经）」为所引之出处。全文末句既有方法论说明（干支配伍），又有学术源流交代（黄帝、玄女、三式），归结一处。

## 源流句式之义理主张

此节三句合观，句式层次井然。先以「夫……者，……也」之判断句式总立命题（六十甲子之起源与创制者），次以「大挠者，……也」之判断句式补释人物身份，后以「考……以……衍而成之」之推演句式说明方法并标注源出。三句相承，源、流、人、法、书五要素齐备，此种结构本身即在示范：论一术数之源流，必先明所自出，次明创制之人，次明推演之法，终明所据之书。此即言术数源流之通则——非止六十甲子为然也。

## 全书定位

此篇为《五行精纪》论六十甲子之开篇章。原文本由两部组成：前部载六十甲子配纳音五行之全表，为后世纳音推命之总纲；后部追溯六十甲子之学术源流，归于黄帝、玄女、三式之传。与后续篇章合观，构成全书论六十甲子之完整体系，为干神、支神、五行等专题之总起。
