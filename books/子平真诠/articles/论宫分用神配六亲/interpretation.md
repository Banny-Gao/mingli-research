```
> 【徐注】六亲之名，由来甚古，义简而赅。汉代京焦说卦，以克我为官鬼，我克为妻财，生我为父母，我生为子孙，同气为兄弟，并本身为六亲(详见《命理寻源》)。命理之配六亲，实脱胎于此，名目虽殊，其理则一也。
```

六亲之名——父母、兄弟、妻财、子孙、官鬼——其渊源甚古，京房、焦延寿说《易》卦，即以「克我为官鬼，我克为妻财，生我为父母，我生为子孙，同气为兄弟」立论。命理之配六亲，实脱胎于京焦之卦象，名目虽殊，其理则一。徐乐吾注以《命理寻源》为此说之渊薮，正坐实六亲配法之古代哲学根源——非命理凭空捏造，乃《易》理之流衍。

## 宫分配六亲：地支之自然位次

```
> 其由宫分配之者，则年月日时，自上而下，祖父妻子，亦自上而下。以地相配，适得其宜，不易之位也。
>
> 宫分者，地支之宫分也。年支为祖基，月支为父母，日支为妻宫，时支为子孙宫，自上而下，以支辰之地位相配也。凡喜用聚于年月支者，祖基必丰，父母之荫庇必厚，幼年享用现成；喜用聚于日支者，妻宫必得力；聚于时支者，子孙必得力，晚运尤佳。年为出身之区，时为归宿之地，出身美则祖基荫庇可知，结局佳则子孙得力可知，亦自然之理也。
```

宫分之法，以八字四柱之地支为六亲之宫位。年支为祖基之位，月支为父母之位，日支为妻宫，时支为子孙之位——自上而下，与人事伦序天然相应：祖、父、妻、子，恰如年、月、日、时之次第。此法之妙，在于「不易之位」四字：宫分既定，喜忌自有所归。

喜用聚于年月，则祖业丰隆、幼年得父母荫庇，此即出身之美；喜用聚于日支，则妻宫得力，助益一生；喜用聚于时支，则晚运亨通，子孙昌盛。年为出身、时为归宿——出身美者祖基可知，结局佳者子孙可知，此皆「自然之理」，非人力所能强为。

宫分配法之实质：六亲之吉凶，首看其宫位所藏之气是否为命局所喜所用。宫位仅为「容器」，容器中盛的是喜是忌，方为定论。

## 用神配六亲：天干之理义推求

```
> 其由用神配之者，则正印也母，身所自出，取其生我也。若偏财受我克制，何反为父？偏财者，母之正夫也，正印为母，则偏财为父矣。正财为妻，受我克制，夫为妻纲，妻则从夫。若官煞则克制乎我，何以反为子女也？官煞者，财所生也，财为妻妾，则官煞为子女矣。至于比肩为兄弟，又理之显然者。
```

用神配法，则从五行生克之理推求：

- **正印为母**——印者，生我者，恰如母之生育于我
- **偏财为父**——印为母，则母之正夫（偏财）即为父
- **正财为妻**——我克者为财，夫为妻纲，妻从夫制
- **官煞为子女**——官煞克我，何以反为子女？盖官煞乃财所生，财既为妻，则官煞即为妻之所出，亦即子女
- **比肩为兄弟**——同气者，比肩是也

此套配法，看似迂曲（父由母推、子女由妻推），实则严守五行生克之逻辑一贯：母（印）生我，父（偏财）乃母之夫；妻（财）我从之，子女（官煞）乃妻所生。每一推步皆有所据，非可任意挪移。

## 阴干从阳：合化之理

```
> 偏财为母之正夫者，譬如甲以癸为正印，戊为偏财，戊癸合也；丙以乙为正印，庚为偏财，乙庚合也。余可类推。五阴干从阳干取，如六乙日干，亦以癸为母，以戊为父也。甲乙日干如有戊无癸，则以壬水为父母。总之言父母则庇我者皆其类，言妻财则奉我者皆其类，言官鬼则制我者皆其类，言子孙则后我者皆其类，言兄弟则同气者皆其类。
```

阴干之配父母，例从阳干——乙日以癸为母、戊为父（仿甲日之例，戊癸合也）；丙日以乙为母、庚为父（乙庚合也）。此因十天干分阴阳、阳主刚、阴主柔，六亲之配虽因干而异，其合化之理则一贯。

若甲乙日干有戊（偏财）无癸（正印），则退而求其次，以壬水为父母——此则从五行之气类着眼，壬水生甲乙木，亦父母之象也。

> **非可刻舟求剑，以为论定**

此句最宜著眼：六亲之配法，言父母则凡庇我者皆其类，言妻财则凡奉我者皆其类，言官鬼则凡制我者皆其类，言子孙则凡后我者皆其类，言兄弟则凡同气者皆其类——此乃**气类原则**，非刻定之死法。一八字中印可为母、为继母、为伯母、为姑母，随其所处之宫位与喜忌而变；不可执一印字便断言生母之穷通。

## 反诘之解：偏财为父、官煞为子女

```
> 至如我克之偏财，何以为父？克我言官煞，何以为子女？乃出于自然之理，凡人受父母之禁约少，受子女之拘束多也。
```

此设问答之辞，意在疏通初学之疑。我克者为财，本以妻论；偏财既为父，岂非违理？克我者为官煞，本以压我论之，何以反为子女？

其解云：「凡人受父母之禁约少，受子女之拘束多」。此语道破配法之深层依据——非徒以生克论，更以人事伦常之实际体验论。父母生于我先，多则管教于幼年；子女我生之，一生牵系最多，反成终身之约束。故以偏财为父（父在先、以奉我之财论其厚我），以官煞为子女（子女在后、以制我之力论其羁縻于我）。此说亦得近代京焦《易》理之印证。

## 时制与命理之显隐

```
> 命不甚显著。譬如前清时代，父母丁忧为仕宦升沉一大关节，命运之中，每显而著，今者礼制废除，父母存亡，无关进退，则命运中亦不甚显著矣。妻宫为一生幸福所系，得力与否，最为明显，学者神而明之，自能自解也。
```

此段原文本与上段相接（疑为传抄漏段标识），实为论六亲之「显隐」原则。同一六亲，于不同时代、不同礼制下，其在命运中之显著程度各异：

- **前清时代**：父母存亡（丁忧去官、夺情起复）直接关系仕途升降，故父母之吉凶显而著；
- **礼制变革后**：丁忧去职之制渐废，父母之存亡不再关系进退，故父母之象在命理中亦不甚显著；
- **妻宫**：无论何时，妻为一生幸福所系，得力与否最为明显——此为亘古不变之通理。

此论之深意：命理非脱离人事之玄谈，六亲之配法与显隐，须与时代礼制、人事实际相参。时移世易，同一八字、同一六亲，其应验之程度可大相径庭。

## 活看与拘执

```
> 其间有无得力，或吉或凶，则以四柱所存或年月或日时财官伤刃，系是何物，然后以六亲配之用神。局中作何喜忌，参而配之，可以了然矣。
>
> 以印为母，以财为妻，局中如无财印，则将如何？用食而逢印夺食，用印而逢财破印，又将如何？是则当参合活看，未可拘执也。大抵从印之喜忌看父母，非必以印为母也；从财之喜忌看妻宫，非必以财为妻也。日主喜印而逢财破，则败祖业；日主忌印而逢财破，则兴家立业矣。身旺喜财而逢比劫分夺，则克妻，反之身弱财重，则以无比劫分夺为克妻矣。伤刃参配喜忌，见下论妻子节。
```

此段最重「活看」二字。

四柱所存六亲之物（财、官、印、伤、刃等），须先辨识其为喜为忌，再参看其落于何宫，然后六亲之吉凶方能定论。

- **从印之喜忌看父母**——日主喜印，印为父母得力之象；日主忌印，印为父母拖累之象。非一见「印」字便断生母之吉凶。
- **从财之喜忌看妻宫**——日主喜财而财被比劫分夺，则妻宫受侵；日主忌财而财被比劫分夺，反为去忌之象。

「日主喜印而逢财破，则败祖业；日主忌印而逢财破，则兴家立业」——同一「印逢财破」之象，吉凶全视日主之喜忌而定。此即「活看」之实义：

> 局中作何喜忌，参而配之，可以了然矣。

## 杂配诸法

```
> 偏印为继母，比肩为兄，劫财为弟者，亦每有验。总之以用神配六亲，更须察其宫分地位，以及喜忌，则大致不谬。
```

六亲之配，又有诸般变例：

- **偏印为继母**——正印为生母，偏印既非正位，譬之继室；
- **比肩为兄、劫财为弟**——同气之中，阳为兄、阴为弟（按五行分阳阴之例）。

「亦每有验」四字极精审：非言其为定法，乃言其验与不验视具体命局而定。

总之，以用神配六亲之法，**更须察其宫分地位与喜忌**——宫分（地支之位）与喜忌（天干之性）两相参合，方为完备。单用宫分则失其理，单用用神则失其位，须二者交验方为稳妥。

## 此论在子平之位

六亲之配，乃命理应用之归宿。前此诸篇论十干、论阴阳、论格局、论用神，皆所以明命局之结构；至本篇则将命局结构落实于人事伦常——父母之庇、兄弟之同、夫妻之合、子女之续——皆依命局而定其得失。

此篇所论，实为子平命理「由理入事」之关键环节。宫分与用神两法并行，示学者不可执一端而废余；活看与拘执之辨，示学者不可泥死法而昧时宜。时制之论（父母丁忧之例）尤具慧识——命理非悬空之玄谈，须与时代人事相印证方显其实用。
