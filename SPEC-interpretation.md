# 滴天髓阐微 · 人类可读解读规范

> **用途：** 规范 `books/**/interpretations/` 目录下所有解读文件的产出格式
> **前置依赖：** 读取 `SPEC-source.md` 确认原文已合规录入，完全理解并遵守 `SKILL-bazi-research-dispute.md` 规则

---

## 一、文件命名规范

```
interpretations/{skill_name}/tutorial.md
interpretations/{skill_name}/advanced.md      ← 可选，深度进阶解读
```

- `skill_name`：与 skills 目录一致的英文标识（如 `tiandao`、`kundao`、`bage`）
- `tutorial.md`：**必须**，面向初学者的由浅入深解读
- `advanced.md`：**可选**，面向进阶学习者的深度专题
- 两个文件内容**不得重复**，advanced 是补充延伸而非缩略版

---

## 二、tutorial.md 结构规范【必须完整】

每个 tutorial.md 必须包含以下七个章节，**顺序不可调整，内容不可缺失**：

```
## 一、入门导读
### 1.1 这一篇在学什么
### 1.2 本篇在整个子平体系中的地位
### 1.3 学习目标

## 二、原文逐段精读
### 2.1 原文全文（带标点整理）
### 2.2 原文逐句精解

## 三、核心义理深度解析
（至少 2 个小节，涵盖本篇核心概念）

## 四、命造实例拆解
（至少 2 个命造，每个包含：原局→格局→用神→喜忌→结论）

## 五、图解与可视化
（图解用于呈现逻辑关系，非装饰性插图；无固定模板，见下方图解规范）

## 六、常见误解与辨析
（至少 2 条，本篇最容易被曲解的论点，附经典原文反驳）

## 七、本篇学习总结
### 7.1 核心必背内容
### 7.2 与其他篇章的关联脉络
### 7.3 进阶学习路径建议
```

### 各章节详细要求

#### 一、入门导读
- **1.1**：用 3 句话说清楚本篇在讲什么，不超过 100 字
- **1.2**：说清楚本篇与前后篇的逻辑关系（可引用 SPEC-interpretation.md 附录的篇目脉络图）
- **1.3**：列出学完本篇后应掌握的 3-5 项具体能力

#### 二、原文逐段精读
- **2.1**：原文骈文/诗文全文，带现代标点，每句一行
- **2.2**：逐句解读，格式统一：
  ```
  【原注第X句】原文
  → 解读：...
  ```
  解读必须**引用原文**关键字，不脱离原文空谈
- **任氏曰**部分同样逐段解读，格式：
  ```
  【任氏曰第X段】原文
  → 解读：...
  ```

#### 三、核心义理深度解析
- 聚焦本篇 2-3 个核心理论点
- 每个论点必须结构：
  - **论点名称**（加粗）
  - **原文依据**（引用【原注】或【任氏曰】原文）
  - **义理解读**（通俗化解释，面向无基础读者）
  - **命造验证**（至少 1 个八字佐证）
- 避免纯理论推导，必须配合命造实例

#### 四、命造实例拆解
- 每个命造统一格式：
  ```
  【命造一】乾造：XXXX年 XX月 XX日 XX时

  格局：用神（取用依据）——喜忌——成破判断

  逐字分析：
  年柱：... → 月令：... → 用神：...

  结论：此造就...格局，...为关键，...为喜，...为忌。
  ```
- 命造来源：优先使用任铁樵原注中给出的命造，其次使用【任氏曰】中的命造
- **禁止自创命造**，所有命造必须有经典出处

#### 五、图解与可视化【规范见第三节】

#### 六、常见误解与辨析
- 每条必须包含：
  1. 错误说法（原文引用或常见网络论调）
  2. 经典原文反驳（必须有【原注】或【任氏曰】原文支撑）
  3. 正确理解
- 至少 2 条，不设上限

#### 七、本篇学习总结
- **7.1**：提取 3-5 句本篇最核心的名句/定理，供背诵
- **7.2**：列出与本篇有理论关联的其他篇章（可引用 `meta/index.md` 的篇目关联数据）
- **7.3**：推荐下一篇学习顺序（附难度对比）

## 三、图解规范【核心规范 · 优先级高】

### 3.1 图解使用原则

**必须使用图解的场景：**
- 逻辑链条清晰的因果关系（如：月令取格→用神定喜忌→格局成败）
- 三元/三方对比关系（如：天元→地元→人元）
- 流程判断类（如：衰旺判断三步法）
- 五行生克循环关系

**禁止使用图解的场景：**
- 文字已清晰说清楚的简单概念
- 装饰性、重复性图示
- 强行将复杂理论简化为图解（以牺牲准确性为代价）

### 3.2 图解类型与选用

| 类型 | 适用场景 | 模板结构 |
|------|---------|---------|
| **三列流程图** | 三元关系、因果三步 | 顶部核心句 + 三列等宽 + 垂直箭头 |
| **树状图** | 格局分类、月令取格分支 | 顶部分支 → 向下一级展开 |
| **对比表** | 清浊判断、成格破格对比 | 双列对比，左正右负 |
| **循环图** | 五行相生相克循环 | 圆形闭合结构 |
| **流程图** | 衰旺判断、用神选取步骤 | 线性步骤 + 条件分支 |

### 3.3 三列流程图规范【当前最常用 · 重点规范】

**标准模板：**
```
【图解】{图解标题}

{核心总结句，一行居中}

地支藏干 → 与天干合德    机缄通畅 → 气脉流通    五行配合 → 吉凶判定
   ↓                   ↓                   ↓
{结论A}             {结论B}              {结论C}
```

**排版要求：**
- 三列**必须等宽等间距**，使用表格或 CSS grid 布局
- 核心句字号大于三列内容
- 箭头使用 `→`（横）和 `↓`（竖），不得混用
- 每列内容不超过 2 行
- 图解区域**整体居中**，四周有内边距

**当前问题修复方向：**
- 箭头加粗一号（当前过细）
- 三列垂直对齐严格居中
- 右列内容不足时用 `　` 占位而非自然空白
- 图解字体大小不低于正文 90%

### 3.4 Markdown 图解代码规范

所有图解必须使用 Markdown 原生语法（不使用外部图片文件），确保：
1. 生成后直接可读
2. 可被 search-index.json 索引
3. 在移动端也能渲染

```markdown
<!-- 三列流程图 -->
【图解】{标题}

{核心总结句}

| 左侧列 | 中间列 | 右侧列 |
|--------|--------|--------|
| 内容A  | 内容B  | 内容C  |
|   ↓    |   ↓    |   ↓    |
| 结论A  | 结论B  | 结论C  |
```

## 五、语言与风格规范

### 5.1 语气与受众

- **称呼读者为"你"**，不用"大家"、"我们"
- 解释命理术语时假设读者**完全无基础**
- 使用类比时选择现代生活场景（工资、职场、健康），不预设读者懂其他命理知识
- 避免说教语气，用探讨式表达："可以理解为..."，"这背后反映的是..."

### 5.2 专业术语处理

所有专业术语（用神、格局、月令、旺相休囚等）**首次出现时必须加注**：

```markdown
**用神**：八字中用来调和五行、平衡命局的关键要素。任铁樵以月令为取用之宗...
```

加注格式：`**术语**：通俗解释。引用原文...

### 5.3 经典引用格式

每次引用原文必须标注：
- 标记来源：`【原注】` 或 `【任氏曰】`
- 上下文：至少引用完整一句话，不截取半句
- 引用后必须有解读，不引不解

```markdown
【原注】"天有阴阳，故春木、夏火、秋金、冬水、季土，随时候显其神功"

→ 解读：这里说的是五行气运随季节轮转的自然规律...
```

## 六、禁止行为

1. 禁止脱离原文解读（解读必须有原文依据）
2. 禁止传授网络伪论、民间野诀
3. 禁止将流派分歧强行判定为"唯一正确"
4. 禁止堆砌原文而不解读（禁止以典代论）
5. 禁止在 tutorial 中引用非公认名家（袁树珊、韦千里、徐乐吾、梁湘润、钟义明之外）的观点作为主要论据
6. 禁止自创命造，所有命造必须有经典出处
7. 禁止使用非本项目 source 目录以外的原文版本

## 七、产出自检清单【输出前必核】

在输出 interpretation 文件前，逐项检查：

- [ ] 读取对应 source 文件，确认【原注】【任氏曰】原文已合规录入
- [ ] tutorial.md 七个章节齐全，无缺章
- [ ] 所有原文引用标注了【原注】或【任氏曰】来源
- [ ] 所有命造有经典出处，无自创命造
- [ ] 图解仅用于逻辑关系清晰的场景，非装饰性
- [ ] 图解三列对齐，无视觉偏移
- [ ] 首次出现的专业术语均已加注
- [ ] 语言面向无基础读者，无行话堆砌
- [ ] advanced.md（如有）与 tutorial.md 内容不重复
- [ ] 无触碰任何禁止行为条款

## 八、篇目难度分级建议

在产出解读时，可参考以下难度分级，标注在入门导读中：

| 级别 | 篇目 | 特点 |
|------|------|------|
| **入门级** | 天道、坤道、人道 | 核心概念建立，篇幅短 |
| **基础级** | 理气、配合、知命、天干、地支 | 子平体系支柱，篇幅中等 |
| **进阶级** | 八格、源流、干支总论、形象、方局 | 综合运用，篇幅较长 |
| **专题级** | 下篇六亲论各篇 | 按需学习，无需顺序 |
