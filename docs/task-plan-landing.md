## 设计原则
1. **布局**: 双滚动上下文（全局页面滚动 + 内容区独立滚动）
2. **移动端**: 100% 宽度居中、overflow-x 隐藏、底部弹窗搜索
3. **视觉**: 金色为主、紫灰为辅，深色背景营造学术氛围
4. **交互**: hover 反馈 200ms ease-out，scale/translateY 微动效
5. **品牌**: 极简命理符号（☯ 太极、十天干、十二地支意象）
